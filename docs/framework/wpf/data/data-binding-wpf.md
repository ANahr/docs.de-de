# Datenbindung (WPF)
Die WPF\-Datenbindung \(Windows Presentation Foundation\) bietet für Anwendungen eine einfache und konsistente Möglichkeit, Daten darzustellen und mit ihnen zu interagieren.  Elemente können aus einer Vielzahl von Datenquellen in Form von CLR\-Objekten \(Common Language Runtime\) an Daten gebunden werden.
