# Formen
[!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] unterstützt das Zeichnen von grundlegenden Formen mithilfe der <xref:System.Windows.Shapes.Shape>\-Klasse.
