# Zeichnungen
<xref:System.Windows.Media.Drawing>\-Objekte werden verwendet, um Formen, Bilder oder Text effizient zu zeichnen.  Zeichnungen werden verwendet, wenn Sie mit einem <xref:System.Windows.Media.DrawingBrush> zeichnen oder mit <xref:System.Windows.Media.Visual>\-Objekten programmieren.
