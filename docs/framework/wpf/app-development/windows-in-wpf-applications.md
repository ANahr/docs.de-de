# Fenster in WPF-Anwendungen
Benutzer interagieren mit Anwendungen über Fenster.  Der wesentliche Zweck eines Fensters ist, Inhalt zu hosten und anzuzeigen.  Welchen Inhaltstyp ein Fenster hostet, hängt vom Typ der Daten ab, die eine Anwendung bearbeitet. Dabei kann es sich um Medien, XAML \(Extensible Application Markup Language\)\-Seiten, Webseiten, Dokumente, Datenbanktabellen und \-datensätze sowie um Systeminformationen handeln.
