# Erstellen und Bereitstellen von WPF-Anwendungen
Mit dem Erstellungs\- und Bereitstellungsmodell können Sie Anwendungen lokal und remote erstellen und bereitstellen, einschließlich:  
  
-   MSBuild: das .NET\-Buildsystem im Microsoft.Build.Tasks.Windows\-Namespace.  
  
-   Ressourcen: Arbeiten mit Ressourcen für Benutzeroberflächen.  
  
-   ClickOnce\-Bereitstellung: das Veröffentlichungs\- und \-Bereitstellungssystem von .NET.
