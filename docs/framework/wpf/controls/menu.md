# Men&#252;
Ein <xref:System.Windows.Controls.Menu> ist ein Steuerelement zur hierarchischen Organisation von Elementen, die Befehlen und Ereignishandlern zugeordnet sind.  Jedes <xref:System.Windows.Controls.Menu> kann mehrere <xref:System.Windows.Controls.MenuItem>\-Steuerelemente enthalten.  Jedes <xref:System.Windows.Controls.MenuItem> kann einen Befehl oder einen `Click`\-Ereignishandler aufrufen.  Ein <xref:System.Windows.Controls.MenuItem> kann auch über mehrere untergeordnete <xref:System.Windows.Controls.MenuItem>\-Elemente verfügen, die ein Untermenü bilden.  
  
 Die folgende Abbildung zeigt die drei unterschiedlichen Zustände eines Menüsteuerelements.  Im Standardzustand befindet sich kein Gerät \(z. B. ein Mauszeiger\) auf dem <xref:System.Windows.Controls.Menu>.  Der Fokuszustand tritt ein, wenn der Mauszeiger auf das <xref:System.Windows.Controls.Menu> zeigt, und der gedrückte Zustand tritt ein, wenn eine Maustaste auf dem <xref:System.Windows.Controls.Menu> geklickt wird.
