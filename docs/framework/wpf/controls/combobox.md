# ComboBox
Das <xref:System.Windows.Controls.ComboBox>\-Steuerelement stellt Benutzern eine Liste mit Optionen bereit.  Die Liste wird angezeigt bzw. ausgeblendet, wenn das Steuerelement erweitert bzw. reduziert wird.  Im Standardzustand ist die Liste reduziert und zeigt nur eine Auswahl an.  Der Benutzer klickt auf eine Schaltfläche, um die vollständige Liste mit Optionen zu sehen.
