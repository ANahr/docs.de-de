# Stile und Vorlagen
Windows Presentation Foundation \(WPF\)\-Stile und \-Vorlagen beziehen sich auf eine Suite von Features \(Stile, Vorlagen, Trigger und Storyboards\), mit denen eine Anwendung, ein Dokument oder ein Entwickler von Benutzeroberflächen visuell überzeugende Anwendungen erstellt und die Standardisierung auf ein bestimmtes Erscheinungsbild des jeweiligen Produkts erreicht.
