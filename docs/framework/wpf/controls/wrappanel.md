# <a name="wrappanel"></a>WrapPanel
Die <xref:System.Windows.Controls.WrapPanel> Element positioniert untergeordnete Elemente sequenziell von links nach rechts und wichtige Inhalt in die nächste Zeile am Rand des enthaltenden Felds.
