# FlowDocumentPageViewer
Mit dem <xref:System.Windows.Controls.FlowDocumentPageViewer>\-Steuerelement werden <xref:System.Windows.Documents.FlowDocument>\-Inhalte Seite für Seite angezeigt.  Im Gegensatz dazu zeigt <xref:System.Windows.Controls.FlowDocumentScrollViewer> <xref:System.Windows.Documents.FlowDocument>\-Inhalte in einem Bildlauf\-Viewer an.
