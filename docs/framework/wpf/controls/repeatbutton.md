# RepeatButton
Ein <xref:System.Windows.Controls.Primitives.RepeatButton> ähnelt einem <xref:System.Windows.Controls.Button>.  Mit den <xref:System.Windows.Controls.Primitives.RepeatButton>\-Elementen können Sie jedoch steuern, wann und wie das <xref:System.Windows.Controls.Primitives.ButtonBase.Click>\-Ereignis ausgelöst wird.  
  
 Die folgende Abbildung zeigt ein Beispiel für die drei Zustände eines RepeatButton\-Steuerelements: Default, PointerFocused und Pressed.  Die erste Schaltfläche zeigt den Standardzustand von <xref:System.Windows.Controls.Primitives.RepeatButton>.  Die zweite Schaltfläche zeigt, wie das Erscheinungsbild der Schaltfläche sich ändert, wenn der Mauszeiger auf sie zeigt, sodass der Fokus darauf liegt.  Die dritte Schaltfläche zeigt, wie <xref:System.Windows.Controls.Primitives.RepeatButton> aussieht, wenn der Benutzer mit der Maustaste auf das Steuerelement klickt.
