# Bezeichnung
<xref:System.Windows.Controls.Label>\-Steuerelemente stellen normalerweise Informationen in der [!INCLUDE[TLA#tla_ui](../../../../includes/tlasharptla-ui-md.md)] bereit.  Traditionell war in einer <xref:System.Windows.Controls.Label> nur Text enthalten, aber da die <xref:System.Windows.Controls.Label>, die im Lieferumfang von [!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] enthalten ist, ein <xref:System.Windows.Controls.ContentControl> ist, kann sie entweder Text oder ein <xref:System.Windows.UIElement> enthalten.  
  
 Eine <xref:System.Windows.Controls.Label> liefert eine funktionale und visuelle Unterstützung für Tastenkombinationen.  Sie wird häufig verwendet, um einen raschen Zugriff über die Tastatur auf Steuerelemente zu ermöglichen, z. B. ein <xref:System.Windows.Controls.TextBox>.  Um eine <xref:System.Windows.Controls.Label> einem <xref:System.Windows.Controls.Control> zuzuweisen, legen Sie die <xref:System.Windows.Controls.Label.Target%2A?displayProperty=fullName>\-Eigenschaft für das Steuerelement fest, das den Fokus bekommen sollte, wenn der Benutzer die Tastenkombination drückt.  
  
 Das folgende Bild zeigt eine <xref:System.Windows.Controls.Label> "Themes", die auf ein <xref:System.Windows.Controls.ComboBox> abzielt.  Wenn der Benutzer die Tastenkombination drückt, erhält das <xref:System.Windows.Controls.ComboBox> den Fokus.  Weitere Informationen finden Sie unter [How to: Set the Target Property of a Label](http://msdn.microsoft.com/de-de/b24c6977-ebcb-4855-a9bb-3fd4435af8f8).
