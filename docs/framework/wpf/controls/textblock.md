# TextBlock
Das <xref:System.Windows.Controls.TextBlock>\-Steuerelement bietet flexible Textunterstützung für [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)]\-Anwendungen.  Das Element zielt in erster Linie auf grundlegende [!INCLUDE[TLA2#tla_ui](../../../../includes/tla2sharptla-ui-md.md)]\-Szenarien ab, bei denen nicht mehr als ein Textabsatz benötigt wird.
