# Frame
Das <xref:System.Windows.Controls.Frame>\-Steuerelement unterstützt die Inhaltsnavigation in Inhalten.  Das <xref:System.Windows.Controls.Frame>\-Element kann von einem Stammelement wie <xref:System.Windows.Window>, <xref:System.Windows.Navigation.NavigationWindow>, <xref:System.Windows.Controls.Page>, <xref:System.Windows.Controls.UserControl>, <xref:System.Windows.Documents.FlowDocument> oder als Insel innerhalb einer Inhaltsstruktur, die zu einem Stammelement gehört, gehostet werden.
