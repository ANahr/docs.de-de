# <a name="windows-presentation-foundation"></a>Windows Presentation Foundation
Windows Presentation Foundation (WPF) in Visual Studio 2015 bietet Entwicklern ein einheitliches Programmiermodell zum Erstellen von modernen Desktopbranchenanwendungen unter Windows.  
  
> [!NOTE]
>  Im Mozilla Firefox-Browser wird die Erweiterung Microsoft Framework Assistant deaktiviert. Informationen, wie die Erweiterung aus Firefox entfernt wird, finden Sie unter [Entfernen von .NET Framework Assistant für Firefox](https://support.microsoft.com/en-us/kb/963707).
