# Ressourcen (WPF)
Eine Ressource ist ein Objekt, das an verschiedenen Stellen in der Anwendung wiederverwendet werden kann.  WPF unterstützt verschiedene Typen von Ressourcen.  Diese Ressourcen sind im Wesentlichen zwei Typen von Ressourcen: XAML\-Betriebsmittel Ressourcendaten und Dateien.  Beispiele dafür sind XAML\-Betriebsmitteln Pinsel und Formate.  Ressourcendaten von Dateien sind nicht ausführbaren Datendateien, die von einer Anwendung benötigt werden.
