# XAML in WPF
[!INCLUDE[TLA#tla_xaml](../../../../includes/tlasharptla-xaml-md.md)] ist eine Markupsprache für die deklarative Anwendungsprogrammierung.  [!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] führt eine XAML\-Prozessorimplementierung aus und stellt XAML\-Sprachunterstützung bereit.  Die [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)]\-Typen werden so implementiert, dass sie die erforderliche Typunterstützung für eine XAML\-Darstellung bereitstellen können. Im Allgemeinen können Sie den Großteil der Benutzeroberfläche der [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)]\-Anwendung mit der XAML\-Markupsprache erstellen.
