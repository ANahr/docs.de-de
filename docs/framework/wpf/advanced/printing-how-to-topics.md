# Gewusst-wie-Themen zum Drucken
Die Themen in diesem Abschnitt erläutern die Verwendung der Features für Druck\- und Drucksystemverwaltung in [!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] sowie den neuen [!INCLUDE[TLA#tla_xps](../../../../includes/tlasharptla-xps-md.md)]\-Druckpfad.  
  
## In diesem Abschnitt  
 [Aufrufen eines Druckdialogfelds](../../../../docs/framework/wpf/advanced/how-to-invoke-a-print-dialog.md)  
 Anweisungen zum Deklarieren eines [!INCLUDE[TLA#tla_win](../../../../includes/tlasharptla-win-md.md)]\-Druckdialogfeldobjekts im XAML\-Markup und zur Verwendung von Code, um das Dialogfeld innerhalb einer [!INCLUDE[TLA#tla_wpf](../../../../includes/tlasharptla-wpf-md.md)]\-Anwendung aufzurufen.  
  
 [Klonen eines Druckers](../../../../docs/framework/wpf/advanced/how-to-clone-a-printer.md)  
 Anweisungen zum Installieren einer zweiten Druckwarteschlange, die über dieselben Eigenschaften verfügt wie eine bereits vorhandene Druckwarteschlange.  
  
 [Diagnose von Problemen mit Druckaufträgen](../../../../docs/framework/wpf/advanced/how-to-diagnose-problematic-print-job.md)  
 Anweisungen zur Verwendung der Eigenschaften von Druckwarteschlangen und Druckaufträgen, um eine Diagnose für einen Druckauftrag durchzuführen, der nicht gedruckt wird.  
  
 [Ermitteln, ob ein Druckauftrag zu dieser Tageszeit gedruckt werden kann](../../../../docs/framework/wpf/advanced/how-to-discover-whether-a-print-job-can-be-printed-at-this-time-of-day.md)  
 Anweisungen zur Verwendung der Eigenschaften von Druckwarteschlangen und Druckaufträgen, um programmgesteuert festzulegen, zu welchen Tageszeiten der Druckauftrag gedruckt werden kann.  
  
 [Auflisten einer Teilmenge von Druckwarteschlangen](../../../../docs/framework/wpf/advanced/how-to-enumerate-a-subset-of-print-queues.md)  
 Anweisungen zum Generieren einer Liste von Druckern, die über bestimmte Eigenschaften verfügen.  
  
 [NOTINBUILD: How to: Extend the Print Schema and Create New Print System Classes](http://msdn.microsoft.com/de-de/a3600218-1ea5-478a-9853-6560464f2885)  
 Ausführliche Erläuterung der Möglichkeiten, wie Programmierer von verwaltetem Code das Druckschema um spezielle Druckgeräte oder Drucker mit neuen oder ungewöhnlichen Features erweitern können.  
  
 [Abrufen von Drucksystemobjekt\-Eigenschaften ohne Reflektion](../../../../docs/framework/wpf/advanced/how-to-get-print-system-object-properties-without-reflection.md)  
 Anweisungen zum Ermitteln der Eigenschaften von Drucksystemobjekten und deren Typen zur Laufzeit.  
  
 [Programmgesteuertes Drucken von XPS\-Dateien](../../../../docs/framework/wpf/advanced/how-to-programmatically-print-xps-files.md)  
 Anweisungen für das schnelle Drucken von [!INCLUDE[TLA#tla_xps](../../../../includes/tlasharptla-xps-md.md)]\-Dateien, ohne dass eine [!INCLUDE[TLA#tla_ui](../../../../includes/tlasharptla-ui-md.md)] erforderlich ist.  
  
 [Remoteüberwachung des Druckerstatus](../../../../docs/framework/wpf/advanced/how-to-remotely-survey-the-status-of-printers.md)  
 Anweisungen zum Erstellen eines Dienstprogramms, das Drucker auf Papierstaus oder andere Probleme überwacht.  
  
 [Überprüfen und Zusammenführen von PrintTickets](../../../../docs/framework/wpf/advanced/how-to-validate-and-merge-printtickets.md)  
 Anweisungen zum Überprüfen, ob ein Druckticket gültig ist und ob es irgendwelche Funktionen anfordert, die nicht vom Drucker unterstützt werden.
