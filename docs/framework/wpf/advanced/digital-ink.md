# Freihandeingaben
In diesem Abschnitt wird die Verwendung von Freihandeingaben im [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] erläutert.  Freihandfunktionen, die traditionell nur im Tablet PC SDK zu finden waren, stehen jetzt auch als Kernfunktionen von Windows Presentation Foundation zur Verfügung.  Dies bedeutet, dass Sie jetzt die Leistungsfähigkeit von Windows Presentation Foundation nutzen können, um vollwertige Tablet PC\-Anwendungen zu entwickeln.
