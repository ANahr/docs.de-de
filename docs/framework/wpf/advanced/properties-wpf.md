# Eigenschaften
[!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] stellt einen Satz von Diensten bereit, mit denen die Funktionalität einer [!INCLUDE[TLA#tla_clr](../../../../includes/tlasharptla-clr-md.md)]\-Eigenschaft erweitert werden kann.  Zusammen werden diese Dienste normalerweise als WPF\-Eigenschaftensystem bezeichnet.  Eine Eigenschaft, die vom [!INCLUDE[TLA#tla_titlewinclient](../../../../includes/tlasharptla-titlewinclient-md.md)]\-Eigenschaftensystem unterstützt wird, wird als [Abhängigkeitseigenschaft](GTMT) bezeichnet.
