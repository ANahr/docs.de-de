# 1450 - WfMessageSent
## Eigenschaften  
  
|||  
|-|-|  
|ID|1450|  
|Schlüsselwörter|WFServices|  
|Ebene|Information|  
|Kanal|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Beschreibung  
 Gibt an, dass von einem Workflow eine Nachricht gesendet wurde.  
  
## Meldung  
 Nachricht von Workflow gesendet

|Datenelementname|Datenelementtyp|Beschreibung|
