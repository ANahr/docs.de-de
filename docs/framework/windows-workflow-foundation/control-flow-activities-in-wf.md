<span data-ttu-id="ac235-103">[!INCLUDE[netfx_current_long](../../../includes/netfx-current-long-md.md)] stellt mehrere Aktivitäten zum Steuern des Ausführungsflusses innerhalb eines Workflows bereit.</span><span class="sxs-lookup"><span data-stu-id="ac235-103">The [!INCLUDE[netfx_current_long](../../../includes/netfx-current-long-md.md)] provides several activities for controlling flow of execution within a workflow.</span></span> <span data-ttu-id="ac235-104">Einige dieser Aktivitäten (z. B. `Switch` und `If`) implementieren Flusssteuerungsstrukturen ähnlich denen in Programmierumgebungen wie [!INCLUDE[csprcs](../../../includes/csprcs-md.md)], während andere (z. B. `Pick`) neue Programmierungsstrukturen modellieren.</span><span class="sxs-lookup"><span data-stu-id="ac235-104">Some of these activities (such as `Switch` and `If`) implement flow control structures similar to those in programming environments such as [!INCLUDE[csprcs](../../../includes/csprcs-md.md)], while others (such as `Pick`) model new programming structures.</span></span>  
  
 <span data-ttu-id="ac235-105">Beachten Sie, dass Aktivitäten wie die Aktivitäten `Parallel` und `ParallelForEach` mehrere untergeordnete Aktivitäten für eine gleichzeitige Ausführung planen, bei
