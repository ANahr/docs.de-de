# <a name="migration-activity-in-wf"></a>Migrationsaktivität in WF
[!INCLUDE[netfx_current_long](../../../includes/netfx-current-long-md.md)]Stellt die <xref:System.Activities.Statements.Interop> Aktivität zum Ausführen von Aktivitäten, die von der Aktivität innerhalb eines Workflows abgeleitet, die basierend werden auf <xref:System.Activities.Activity>. [!INCLUDE[crdefault](../../../includes/crdefault-md.md)]die [Migrationsanleitung](../../../docs/framework/windows-workflow-foundation/migration-guidance.md) Abschnitt.  
  
> [!NOTE]
>  Die <xref:System.Activities.Statements.Interop> Aktivität wird nicht in der Toolbox des Workflow-Designer angezeigt, es sei denn, das Projekt des Workflows dessen **Zielframework** eingestellt **.Net Framework 4** oder höher.
