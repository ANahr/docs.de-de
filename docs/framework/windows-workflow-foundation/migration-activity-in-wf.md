# Migrationsaktivit&#228;t in WF
[!INCLUDE[netfx_current_long](../../../includes/netfx-current-long-md.md)] Stellt die <xref:> System.Activities.Statements.Interop?qualifyHint=False&autoUpgrade=True Aktivität zum Ausführen von Aktivitäten, die von abgeleitet <xref:System.WorkflowModel.Activities.Activity> innerhalb eines Workflows auf der Grundlage <xref:System.Activities.Activity>. [!INCLUDE[crdefault](../../../includes/crdefault-md.md)] die [Hinweise zur Migration](../../../docs/framework/windows-workflow-foundation//migration-guidance.md) Abschnitt.  
  
> [!NOTE]
>  Die <xref:System.Activities.Statements.Interop> Aktivität wird nicht in der Toolbox des Workflow-Designers angezeigt, es sei denn, der Workflow-Projekt die **Zielframework** eingestellt **.Net Framework 4** oder höher.
