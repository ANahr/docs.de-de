# 4207 - MaximumRetriesExceededForSqlCommand
## Eigenschaften  
  
|||  
|-|-|  
|ID|4207|  
|Schlüsselwörter|Kontingent, WFInstanceStore|  
|Ebene|Information|  
|Kanal|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Beschreibung  
 Gibt an, dass der SQL\-Anbieter nicht weiter versucht, einen SQL\-Befehl abzurufen, da die maximale Anzahl an Wiederholungsversuchen ausgeführt wurde.  
  
## Meldung  
 Es wird nicht weiter versucht, einen SQL\-Befehl abzurufen, da die maximale Anzahl an Wiederholungsversuchen ausgeführt wurde.

|Datenelementname|Datenelementtyp|Beschreibung|  
|----------------------|---------------------|------------------|  
|AppDomain|xs:string|Die von AppDomain.CurrentDomain.FriendlyName zurückgegebene Zeichenfolge.|
