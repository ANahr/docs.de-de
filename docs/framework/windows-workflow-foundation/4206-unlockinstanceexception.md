# 4206 - UnlockInstanceException
## Eigenschaften  
  
|||  
|-|-|  
|ID|4206|  
|Schlüsselwörter|WFInstanceStore|  
|Ebene|Fehler|  
|Kanal|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Beschreibung  
 Gibt an, dass bei dem Versuch, eine Instanz zu entsperren, eine Ausnahme aufgetreten ist.  
  
## Meldung  
 Ausnahme %1 beim Versuch, eine Instanz zu entsperren.  
  
## Details  
  
|Datenelementname|Datenelementtyp|Beschreibung|  
|----------------------|---------------------|------------------|  
|ExceptionMessage|xs:string|Die Nachricht aus der SQL\-Ausnahme.|  
|AppDomain|xs:string|Die von AppDomain.CurrentDomain.FriendlyName zurückgegebene Zeichenfolge.|
