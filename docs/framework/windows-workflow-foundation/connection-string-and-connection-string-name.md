# Verbindungszeichenfolge und Name der Verbindungszeichenfolge
Die **Verbindungszeichenfolgen\-Eigenschaft** gibt die Verbindungszeichenfolge an, mit der der SQL\-Workflowinstanzspeicher eine Verbindung zur persistenten Datenbank herstellen soll.Dieser Parameter ist ein optionaler Parameter.Die **Verbindungszeichenfolgennamen\-Eigenschaft** gibt den Namen der benannten Verbindungszeichenfolge an, mit der der SQL\-Workflowinstanzspeicher eine Verbindung mit der persistenten Datenbank herstellen soll.Dieser Parameter ist ein optionaler Parameter.Es empfiehlt sich, einen Wert für die Verbindungszeichenfolgenname\-Eigenschaft oder die Verbindungszeichenfolgen\-Eigenschaft anzugeben, wenn Sie nicht möchten, dass der SQL\-Workflowinstanzspeicher die benannte Standard\-Verbindungszeichenfolge **DefaultSqlWorkflowInstanceStoreConnectionString** verwendet.
