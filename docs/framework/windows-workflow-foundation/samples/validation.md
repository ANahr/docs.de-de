# <a name="validation"></a>Validierung
Dieser Abschnitt enthält Beispiele, in denen die Aktivitätsvalidierung veranschaulicht wird.  
  
## <a name="in-this-section"></a>In diesem Abschnitt  
 [Externe Aktivitätsvalidierung](../../../../docs/framework/windows-workflow-foundation/samples/external-activity-validation.md)  
 In diesem Beispiel wird gezeigt, wie einer integrierten Aktivität Validierungslogik hinzugefügt wird, die Sie nicht erstellt haben.  
  
 [Grundlegende Validierung](../../../../docs/framework/windows-workflow-foundation/samples/basic-validation.md)  
 Veranschaulicht eine Aktivität, `CreateProduct`, die überprüft, ob das `Cost`-Argument kleiner oder gleich dem `Price`-Argument ist.  
  
 [OverloadGroups](../../../../docs/framework/windows-workflow-foundation/samples/overloadgroups.md)  
 Veranschaulicht eine Aktivität, `CreateLocation`, die erforderliche Argumente und optionale Argumente aufweist und es dem Benutzer ermöglicht, einen von zwei verschiedenen Sätzen von Argumenten bereitzustellen.  
  
 [Validierung von Aktivitätsbeziehungen](../../../../docs/framework/windows-workflow-foundation/samples/activity-relationships-validation.md)  
 Veranschaulicht, wie Beziehungen zwischen Aktivitäten überprüft werden.
