# Validierung
Dieser Abschnitt enthält Beispiele, in denen die Aktivitätsvalidierung zur Entwurfszeit in [!INCLUDE[wf](../../../../includes/wf-md.md)] veranschaulicht wird.  
  
## In diesem Abschnitt  
 [Einschränkungstypen](../../../../docs/framework/windows-workflow-foundation/samples/constraint-types.md)  
 Veranschaulicht zwei verschiedene Möglichkeiten zum Anwenden von Einschränkungen auf einen Workflow. Eine Möglichkeit besteht darin, dies innerhalb der Aktivität auszuführen \(Build\), die andere Möglichkeit besteht darin, dies außerhalb der Aktivität auszuführen \(Richtlinie\).
