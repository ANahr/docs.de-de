 <span data-ttu-id="cd7ee-108">Wenn der Beispielworkflow ausgeführt wird, erstellt eine benutzerdefinierte Aktivität mit dem Namen `CreateTextWriter` ein Objekt des Typs <xref:System.IO.TextWriter> und speichert es in einer Workflowvariablen.</span><span class="sxs-lookup"><span data-stu-id="cd7ee-108">When the sample workflow runs, a custom activity called `CreateTextWriter` creates an object of type <xref:System.IO.TextWriter> and saves it into a workflow variable.</span></span> <span data-ttu-id="cd7ee-109"><xref:System.IO.TextWriter> ist ein <xref:System.IDisposable>-Objekt.</span><span class="sxs-lookup"><span data-stu-id="cd7ee-109"><xref:System.IO.TextWriter> is an <xref:System.IDisposable> object.</span></span> <span data-ttu-id="cd7ee-110">Dieser <xref:System.IO.TextWriter>, der zum Schreiben in eine Datei mit dem Namen 'out.txt' in dem Verzeichnis, in dem das Beispiel ausgeführt wird, konfiguriert ist, wird von einer <xref:System.Activities.Statements.WriteLine>-Aktivität verwendet, da sie jeden an der Konsole eingegebenen Text wiederholt.</span><span class="sxs-lookup"><span data-stu-id="cd7ee-110">This <xref:System.IO.TextWriter>, which is configured to write to a file named ‘out.txt’ in the directory in which the sample runs, is used by a <xref:System.Activities.Statements.WriteLine> activity as it echoes any text you type in at the console.</span></span>
