 <span data-ttu-id="16022-108">Der Abbruch kann innerhalb einer Aktivität oder über das übergeordnete Element einer Aktivität ausgelöst werden.</span><span class="sxs-lookup"><span data-ttu-id="16022-110">Die übergeordnete Aktivität kann untergeordnete Aktivitäten aus jedem beliebigen Grund abbrechen.</span><span class="sxs-lookup"><span data-stu-id="16022-110">The parent
