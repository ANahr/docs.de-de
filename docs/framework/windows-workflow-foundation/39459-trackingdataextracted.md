# 39459 - TrackingDataExtracted
## Eigenschaften  
  
|||  
|-|-|  
|ID|39459|  
|Schlüsselwörter|WFRuntime|  
|Ebene|Ausführlich|  
|Kanal|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Beschreibung  
 Gibt an, dass Nachverfolgungsdaten in einer Aktivität extrahiert wurden.  
  
## Meldung  
 Überwachungsdaten %1 wurden in Aktivität %2 extrahiert.  
  
## Details  
  
|Datenelementname|Datenelementtyp|Beschreibung|  
|----------------------|---------------------|------------------|  
|Daten|xs:string|Der Name der extrahierten Daten.|  
|Aktivität|xs:string|Der Name der Aktivität.|  
|AppDomain|xs:string|Die von AppDomain.CurrentDomain.FriendlyName zurückgegebene Zeichenfolge.|
