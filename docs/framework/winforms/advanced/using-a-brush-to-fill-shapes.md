# Verwenden eines Pinsels zum Ausf&#252;llen von Formen
Das <xref:System.Drawing.Brush>\-Objekt in [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] wird verwendet, um das Innere einer geschlossenen Form auszufüllen.  In [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] sind mehrere Füllstile definiert: Volltonfarbe, Schraffur, Bildstruktur und Farbverlauf.  
  
## In diesem Abschnitt

 Beschreibt, wie ein Volltonfarbenpinsel verwendet wird, um Formen auszufüllen.

 [Gewusst wie: Ausfüllen einer Form mit einer Bildstruktur](../../../../docs/framework/winforms/advanced/how-to-fill-a-shape-with-an-image-texture.md)  
 Beschreibt, wie ein Texturpinsel verwendet wird, um Formen auszufüllen.  
  
 [Gewusst wie: Kacheln einer Form mit einem Bild](../../../../docs/framework/winforms/advanced/how-to-tile-a-shape-with-an-image.md)  
 Beschreibt, wie die Tile\-Ansicht für ein Bild in einer Form verwendet wird.
