# Benutzerf&#252;hrung mithilfe von QuickInfos
Die <xref:System.Windows.Forms.ToolTip>\-Komponente kann für bestimmte Steuerelemente in Windows Forms zum Anzeigen einer kurzen, spezifischen Hilfemeldung verwendet werden.  Mit der <xref:System.Windows.Forms.ToolTip>\-Komponente wird eine Eigenschaft verfügbar, die den Text für die einzelnen Steuerelemente des Formulars festlegt.  Weitere Informationen über diese Verwendung der <xref:System.Windows.Forms.ToolTip>\-Komponente finden Sie unter [Gewusst wie: Festlegen von QuickInfos für Steuerelemente auf einem Windows Form zur Entwurfszeit](../../../../docs/framework/winforms/controls/how-to-set-tooltips-for-controls-on-a-windows-form-at-design-time.md).  Zusätzlich können Sie die <xref:System.Windows.Forms.ToolTip>\-Komponente so konfigurieren, dass sie mit Verzögerung angezeigt wird.  Weitere Informationen hierzu finden Sie unter [Gewusst wie: Ändern der Verzögerung der ToolTip\-Komponente in Windows Forms](../../../../docs/framework/winforms/controls/how-to-change-the-delay-of-the-windows-forms-tooltip-component.md).
