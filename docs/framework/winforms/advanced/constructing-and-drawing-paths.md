# Erstellen und Zeichnen von Pfaden
Bei einem Pfad handelt es sich um eine Abfolge von Grundelementen \(Linien, Rechtecken, Kurven, Text usw.\), die als eine zusammenhängende Einheit bearbeitet und gezeichnet werden kann.  Ein Pfad kann in *Figuren* eingeteilt werden, die entweder offen oder geschlossen sind.  Eine Figur kann aus mehreren Grundelementen bestehen.  
  
 Um einen Pfad zu zeichnen, rufen Sie die <xref:System.Drawing.Graphics.DrawPath%2A>\-Methode der <xref:System.Drawing.Graphics>\-Klasse auf. Um den Pfad auszufüllen, rufen Sie die <xref:System.Drawing.Graphics.FillPath%2A>\-Methode der <xref:System.Drawing.Graphics>\-Klasse auf.
