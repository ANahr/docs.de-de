# Verwenden von Bereichen
Die <xref:System.Drawing.Region>\-Klasse in [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] ermöglicht die Definition einer benutzerdefinierten Form.  Diese Form kann aus Linien, Polygonen und Kurven bestehen.  
  
 Zwei häufige Verwendungszwecke für Bereiche sind Trefferüberprüfung und Clipping.  Durch die Trefferüberprüfung wird ermittelt, ob in einem bestimmten Bildschirmbereich mit der Maus geklickt wurde.  Durch das Clipping wird der Zeichenvorgang auf einen bestimmten Bereich beschränkt.  
  
## In diesem Abschnitt  
 [Gewusst wie: Trefferüberprüfung mit einem Bereich](../../../../docs/framework/winforms/advanced/how-to-use-hit-testing-with-a-region.md)  
 Zeigt, wie mit einem <xref:System.Drawing.Region> eine Trefferüberprüfung durchgeführt wird.  
  
 [Gewusst wie: Ausschneiden mit einem Bereich](../../../../docs/framework/winforms/advanced/how-to-use-clipping-with-a-region.md)  
 Erklärt, wie der Clippingbereich für ein <xref:System.Drawing.Graphics>\-Objekt festgelegt wird.
