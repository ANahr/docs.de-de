# Gewusst wie: &#196;ndern des Werts einer vorhandenen Einstellung zur Entwurfszeit
Sie können Visual Studio verwenden, um die Werte vorhandener Einstellungen im Projekt zu bearbeiten.  
  
### So ändern Sie den Wert einer vorhandenen Einstellung zur Entwurfszeit in C\#  
  
1.  Erweitern Sie im **Projektmappen\-Explorer** den Knoten **Eigenschaften** des Projekts.  
  
2.  Doppelklicken Sie auf die SETTINGS\-Datei, der Sie eine neue Einstellung hinzufügen möchten.  Der Standardname dieser Datei ist Settings.settings.  
  
3.  Suchen Sie im Einstellungs\-Designer die Einstellung, für die Sie Wert und Typ des neuen Wertes ändern möchten, in der Spalte Wert.  
  
### So ändern Sie den Wert einer vorhandenen Einstellung zur Entwurfszeit in Visual Basic  
  
1.  Klicken Sie im **Projektmappen\-Explorer** mit der rechten Maustaste auf den Projektknoten, und wählen Sie **Eigenschaften**.  
  
2.  Wählen Sie auf der Seite **Eigenschaften** die Registerkarte **Einstellungen** aus.  
  
3.  Suchen Sie im Einstellungs\-Designer die Einstellung, für die Sie Wert und Typ des neuen Wertes ändern möchten, in der Spalte Wert.
