# Tastatureingaben in einer Windows Forms-Anwendung
Windows Forms enthält Standardtastaturereignisse, mit denen Sie auf bestimmte gedrückte Tasten reagieren können. Außerdem bietet es Möglichkeiten zum Abfangen, Ändern und Konsumieren von gedrückten Tasten in der Anwendung, im Formular oder auf Steuerelementebene.  
  
## In diesem Abschnitt  
 [Funktionsweise von Tastatureingaben](../../../docs/framework/winforms/how-keyboard-input-works.md)  
 Beschreibt die Verarbeitung von Tastaturmeldungen und deren Umwandlung in Tastaturereignisse.  
  
 [Verwenden von Tastaturereignissen](../../../docs/framework/winforms/using-keyboard-events.md)  
 Enthält Informationen zu den Typen von Tastaturereignissen sowie zu den Informationen, die von den Tastaturereignishandlern empfangen werden.  
  
 [Gewusst wie: Ändern von Tastatureingaben in ein Standardsteuerelement](../../../docs/framework/winforms/how-to-modify-keyboard-input-to-a-standard-control.md)  
 Enthält ein Codebeispiel, durch das veranschaulicht wird, wie Tastenwerte geändert werden, bevor sie ein Steuerelement erreichen.  
  
 [Gewusst wie: Ermitteln der gedrückten Modifizierertaste](../../../docs/framework/winforms/how-to-determine-which-modifier-key-was-pressed.md)  
 Veranschaulicht, wie Sie feststellen, ob zusätzlich zu einer anderen Taste UMSCHALT, ALT oder STRG gedrückt wurde.  
  
 [Gewusst wie: Behandeln von Tastatureingaben auf Formularebene](../../../docs/framework/winforms/how-to-handle-keyboard-input-at-the-form-level.md)  
 Enthält ein Codebeispiel, durch das veranschaulicht wird, wie Tastenwerte abgefangen werden, bevor sie ein Steuerelement erreichen.
