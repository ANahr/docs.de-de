# Verwenden des verwalteten HTML-Dokumentobjektmodells
Das verwaltete HTML\-Dokumentobjektmodell \(DOM\) stellt einen auf [!INCLUDE[dnprdnshort](../../../../includes/dnprdnshort-md.md)] basierenden Wrapper für die von Internet Explorer verfügbar gemachten HTML\-Klassen bereit.  Verwenden Sie diese Klassen, um im <xref:System.Windows.Forms.WebBrowser>\-Steuerelement gehostete HTML\-Seiten zu bearbeiten oder neue Seiten zu erstellen.  
  
## In diesem Abschnitt  
 [Gewusst wie: Zugreifen auf das verwaltete HTML\-Dokumentobjektmodell](../../../../docs/framework/winforms/controls/how-to-access-the-managed-html-document-object-model.md)  
 Beschreibt, wie Sie eine gültige Instanz von <xref:System.Windows.Forms.HtmlDocument> entweder über eine Windows Forms\-Anwendung oder über ein in Internet Explorer gehostetes <xref:System.Windows.Forms.UserControl> abrufen.  
  
 [Gewusst wie: Zugreifen auf den HTML\-Quellcode im verwalteten HTML\-Dokumentobjektmodell](../../../../docs/framework/winforms/controls/how-to-access-the-html-source-in-the-managed-html-document-object-model.md)  
 Beschreibt, wie Sie die ursprüngliche, nicht bearbeitete HTML\-Quelle und die aktive Quelle abrufen, die den aktuellen Zustand des DOMs wiedergibt.  
  
 [Gewusst wie: Ändern von Formaten eines Elements im verwalteten HTML\-Dokumentobjektmodell](../../../../docs/framework/winforms/controls/how-to-change-styles-on-an-element-in-the-managed-html-document-object-model.md)  
 Beschreibt, wie Sie Formate bearbeiten, mit denen die visuelle Anzeige von Elementen gesteuert werden kann.  
  
 [Zugreifen auf Frames im verwalteten HTML\-Dokumentobjektmodell](../../../../docs/framework/winforms/controls/accessing-frames-in-the-managed-html-document-object-model.md)  
 Beschreibt, was Frames und Framesets sind und wie auf das DOM eines Frames zugegriffen wird.  
  
 [Zugreifen auf nicht verfügbar gemachte Member des verwalteten HTML\-Dokumentobjektmodells](../../../../docs/framework/winforms/controls/accessing-unexposed-members-on-the-managed-html-document-object-model.md)  
 Beschreibt, wie auf Member des zugrunde liegenden DOMs zugegriffen wird, die über keine verwaltete Entsprechung verfügen.
