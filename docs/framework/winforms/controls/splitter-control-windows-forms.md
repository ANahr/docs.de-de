# Splitter-Steuerelement (Windows&#160;Forms)
`Splitter`\-Steuerelemente werden in Windows Forms dazu verwendet, die Größe angedockter Steuerelemente zur Laufzeit anzupassen.  Das `Splitter`\-Steuerelement wird häufig in Formularen mit Steuerelementen eingesetzt, in denen Daten variabler Länge dargestellt werden müssen. Beispielsweise enthalten die Fensterbereiche in Windows Explorer zu unterschiedlichen Zeiten Informationen unterschiedlicher Länge bzw. Breite.  
  
> [!NOTE]
>  Obwohl <xref:System.Windows.Forms.SplitContainer> das `Splitter`\-Steuerelement vorheriger Versionen ersetzt und funktionell erweitert, wird das `Splitter`\-Steuerelement sowohl aus Gründen der Abwärtskompatibilität als auch, falls gewünscht, für die zukünftige Verwendung beibehalten.
