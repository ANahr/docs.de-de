# &#220;bersicht &#252;ber das TabControl-Steuerelement (Windows&#160;Forms)
Mit <xref:System.Windows.Forms.TabControl> in Windows Forms werden mehrere Registerkarten wie Karteireiter in einem Notizbuch oder Reiter in einem Satz von Ordnern einer Ablageregistratur angezeigt.  Die Registerkarten können Bilder und weitere Steuerelemente enthalten.  Mit dem Registersteuerelement kann ein mehrseitiges Dialogfeld erzeugt werden, wie es an vielen Stellen im Windows\-Betriebssystem eingesetzt wird, z. B. das Anzeigeeigenschaftenfenster in der Systemsteuerung.  Darüber hinaus können mithilfe von <xref:System.Windows.Forms.TabControl> Eigenschaftenseiten erstellt werden, die zum Festlegen einer Gruppe verwandter Eigenschaften verwendet werden.  
  
## Arbeiten mit TabControl  
 Die wichtigste Eigenschaft von <xref:System.Windows.Forms.TabControl> lautet <xref:System.Windows.Forms.TabControl.TabPages%2A>; sie enthält die einzelnen Registerkarten.  Jede einzelne Registerkarte stellt ein <xref:System.Windows.Forms.TabPage>\-Objekt dar.  Wenn auf eine Registerkarte geklickt wird, wird das <xref:System.Windows.Forms.Control.Click>\-Ereignis für dieses <xref:System.Windows.Forms.TabPage>\-Objekt ausgelöst.
