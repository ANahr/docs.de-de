# NotifyIcon-Komponente (Windows&#160;Forms)
Die `NotifyIcon`\-Komponente in Windows Forms zeigt im Statusbenachrichtigungsbereich der Taskleiste Symbole für Prozesse an, die im Hintergrund ausgeführt werden und über keine andere Benutzeroberfläche verfügen.  Ein Beispiel hierfür wäre ein Virenschutzprogramm, das Sie öffnen können, indem Sie auf ein Symbol in dem Bereich der Taskleiste klicken, in dem die Statusbenachrichtigungen angezeigt werden.
