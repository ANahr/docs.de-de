# Panel-Steuerelement (Windows&#160;Forms)
`Panel`\-Steuerelemente in Windows Forms werden dazu verwendet, andere Steuerelemente zu identifizierbaren Gruppen zusammenzufassen.  Normalerweise werden Auswahlbereiche verwendet, um ein Formular nach Funktionsbereichen zu unterteilen.  Das `Panel`\-Steuerelement ist grundsätzlich mit dem <xref:System.Windows.Forms.GroupBox>\-Steuerelement vergleichbar. Allerdings kann nur das `Panel`\-Steuerelement mit Bildlaufleisten und nur das <xref:System.Windows.Forms.GroupBox>\-Steuerelement mit einer Beschriftung versehen werden.
