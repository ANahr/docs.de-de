# ErrorProvider-Komponente (Windows&#160;Forms)
Die `ErrorProvider`\-Komponente in Windows Forms wird verwendet, um den Benutzer diskret auf einen Fehler hinweisen.  Diese wird üblicherweise bei der Validierung der Benutzereingabe in einem Formular oder beim Anzeigen von Fehlern innerhalb eines Datasets verwendet.
