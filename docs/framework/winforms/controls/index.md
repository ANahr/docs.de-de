# Windows Forms-Steuerelemente
Beim Entwerfen und Anpassen der Benutzeroberfläche von Windows Forms\-Anwendungen ist es zeitweise erforderlich, Steuerelemente hinzuzufügen, auszurichten und zu positionieren.  Steuerelemente sind in Formularobjekten enthaltene Objekte.  Jeder Steuerelementtyp verfügt über eigene Eigenschaften\-, Methoden\- und Ereignissets, durch die er für einen bestimmten Verwendungszweck prädestiniert ist.  Sie können Steuerelemente im Designer anpassen und Code schreiben, durch den Steuerelemente dynamisch zur Laufzeit hinzugefügt werden.
