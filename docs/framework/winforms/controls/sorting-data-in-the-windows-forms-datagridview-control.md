# Sortieren von Daten im DataGridView-Steuerelement in Windows Forms
Benutzer können standardmäßig Daten in einem `DataGridView`\-Steuerelement sortieren, indem sie auf den Header einer Textfeldspalte klicken.  Sie können die `SortMode`\-Eigenschaft bestimmter Spalten ändern, um Benutzern das Sortieren nach anderen Spaltentypen zu ermöglichen, sofern dies sinnvoll erscheint.  Sie können die Daten nach einer oder mehreren beliebigen Spalten auch programmgesteuert sortieren.
