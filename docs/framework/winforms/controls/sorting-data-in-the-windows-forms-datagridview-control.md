# <a name="sorting-data-in-the-windows-forms-datagridview-control"></a>Sortieren von Daten in das Windows Forms-DataGridView-Steuerelement

Standardmäßig können Benutzer sortieren Sie die Daten in einem <xref:System.Windows.Forms.DataGridView> Steuerelement, indem Sie auf die Kopfzeile der Spalte ein Text-Feld (oder drücken F3, wenn sich der Fokus einer Zelle der Text im Feld unter .NET Framework 4.7.2 und höheren Versionen ist). Sie können die <xref:System.Windows.Forms.DataGridViewColumn.SortMode> Eigenschaft bestimmter Spalten, damit Benutzer nach anderen Spaltentypen zu sortieren, wenn es sinnvoll. Sie können die Daten auch Sortieren programmgesteuert durch eine Spalte oder mehreren Spalten.
