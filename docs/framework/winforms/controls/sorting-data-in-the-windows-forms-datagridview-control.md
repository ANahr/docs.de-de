# <a name="sorting-data-in-the-windows-forms-datagridview-control"></a>Sortieren von Daten im DataGridView-Steuerelement in Windows Forms
Standardmäßig können Benutzer sortieren Sie die Daten in einem `DataGridView` Steuerelement, indem Sie auf die Kopfzeile der Spalte ein Text-Feld. Sie können die `SortMode` Eigenschaft bestimmter Spalten, damit Benutzer nach anderen Spaltentypen zu sortieren, wenn es sinnvoll. Sie können die Daten auch Sortieren programmgesteuert durch eine Spalte oder mehreren Spalten.
