# RadioButton-Steuerelement (Windows&#160;Forms)
`RadioButton`\-Steuerelemente in Windows Forms bieten dem Benutzer zwei oder mehr Optionen, die sich gegenseitig ausschließen.  Obwohl Optionsfelder und Kontrollkästchen eine relativ ähnliche Funktionsweise besitzen, weisen Sie doch einen wichtigen Unterschied auf: Sobald der Benutzer ein Optionsfeld aktiviert, können in derselben Gruppe keine zusätzlichen Optionsfelder aktiviert werden.
