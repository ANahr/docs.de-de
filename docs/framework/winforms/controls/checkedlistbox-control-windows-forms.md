# <a name="checkedlistbox-control-windows-forms"></a>CheckedListBox-Steuerelement (Windows Forms)
Windows Forms `CheckedListBox` -Steuerelement zeigt eine Liste von Elementen, z. B. die <xref:System.Windows.Forms.ListBox> gesteuert und können auch ein Häkchen neben Elementen angezeigt.  
  
## <a name="in-this-section"></a>In diesem Abschnitt

 Hier werden das Steuerelement und seine wichtigsten Funktionen und Eigenschaften erläutert.  
  
 [Gewusst wie: Bestimmen der aktivierten Elemente im CheckedListBox-Steuerelement von Windows Forms](../../../../docs/framework/winforms/controls/how-to-determine-checked-items-in-the-windows-forms-checkedlistbox-control.md)  
 Beschreibt das Durchlaufen einer Liste, um zu bestimmen, welche Elemente aktiviert sind.
