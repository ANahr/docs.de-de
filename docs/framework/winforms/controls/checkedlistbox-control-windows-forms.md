# CheckedListBox-Steuerelement (Windows&#160;Forms)
Das `CheckedListBox`\-Steuerelement in Windows Forms zeigt, ähnlich wie das <xref:System.Windows.Forms.ListBox>\-Steuerelement, eine Liste von Elementen an und kann zusätzlich Häkchen neben Listeneinträgen einblenden.
