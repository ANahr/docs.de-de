# Verwenden des Designers mit dem DataGridView-Steuerelement in Windows Forms
Visual Studio bietet Designerunterstützung für das `DataGridView`\-Steuerelement, über das Sie viele Einrichtungsaufgaben ohne Schreiben von Code ausführen können.  Diese Aufgaben umfassen das Binden des Steuerelements an eine Datenquelle, das Bearbeiten der Spalten für die Datenanzeige und das Anpassen sowohl von Darstellung als auch grundlegendem Verhalten des Steuerelements.
