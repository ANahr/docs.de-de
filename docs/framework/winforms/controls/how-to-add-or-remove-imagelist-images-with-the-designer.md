# Gewusst wie: Hinzuf&#252;gen oder Entfernen von ImageList-Bildern mithilfe des Designers
Es gibt verschiedene Möglichkeiten, um einer <xref:System.Windows.Forms.ImageList>\-Komponente Bilder hinzuzufügen.  Die schnelle Methode besteht darin, das mit <xref:System.Windows.Forms.ImageList> verknüpfte Smarttag zu verwenden. Falls Sie jedoch mehrere zusätzliche Eigenschaften für <xref:System.Windows.Forms.ImageList> festlegen, erscheint es Ihnen möglicherweise einfacher, Bilder über das Eigenschaftenfenster hinzuzufügen.  Bilder können auch mithilfe von Code hinzugefügt werden.  Weitere Informationen zum Hinzufügen von Bildern mithilfe von Code finden Sie unter [Gewusst wie: Hinzufügen oder Entfernen von Bildern mit der ImageList\-Komponente in Windows Forms](../../../../docs/framework/winforms/controls/how-to-add-or-remove-images-with-the-windows-forms-imagelist-component.md).  Normalerweise füllen Sie eine <xref:System.Windows.Forms.ImageList>\-Komponente mit Bildern, bevor Sie diese mit einem Steuerelement verknüpfen. Dies ist jedoch nicht zwingend erforderlich.  
  
> [!NOTE]
>  Je nach den aktiven Einstellungen oder der Version unterscheiden sich die Dialogfelder und Menübefehle auf Ihrem Bildschirm möglicherweise von den in der Hilfe beschriebenen.  Wählen Sie im Menü **Extras** die Option **Einstellungen importieren und exportieren** aus, um die Einstellungen zu ändern.  Weitere Informationen finden Sie unter [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/de-de/22c4debb-4e31-47a8-8f19-16f328d7dcd3).  
  
### So fügen Sie Bilder mithilfe des Eigenschaftenfensters hinzu oder entfernen sie  
  
1.  Wählen Sie die <xref:System.Windows.Forms.ImageList>\-Komponente aus, oder fügen Sie dem Formular eine hinzu.  
  
2.  Klicken Sie im Eigenschaftenfenster auf die Schaltfläche mit dem Auslassungszeichen \(![VisualStudioEllipsesButton&#45;Bildschirmabbildung](../../../../docs/framework/winforms/media/vbellipsesbutton.png "vbEllipsesButton")\), die sich neben der <xref:System.Windows.Forms.ImageList.Images%2A>\-Eigenschaft befindet.  
  
3.  Klicken Sie im **Bildauflistungs\-Editor** auf **Hinzufügen** oder **Entfernen**, um Bilder in die Liste aufzunehmen oder daraus zu entfernen.  
  
### So fügen Sie Bilder mithilfe des Smarttags hinzu oder entfernen sie  
  
1.  Wählen Sie die <xref:System.Windows.Forms.ImageList>\-Komponente aus, oder fügen Sie dem Formular eine hinzu.  
  
2.  Klicken Sie auf das Smarttagsymbol \(![Smarttag&#45;Glyphe](../../../../docs/framework/winforms/controls/media/vs-winformsmttagglyph.png "VS\_WinFormSmtTagGlyph")\).  
  
3.  Wählen Sie im Dialogfeld **ImageList\-Aufgaben** die Option **Bilder auswählen**.  
  
4.  Klicken Sie im **Bildauflistungs\-Editor** auf **Hinzufügen** oder **Entfernen**, um Bilder in der Liste hinzuzufügen oder zu entfernen.
