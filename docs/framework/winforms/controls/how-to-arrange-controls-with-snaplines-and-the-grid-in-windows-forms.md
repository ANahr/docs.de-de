# Gewusst wie: Anordnen von Steuerelementen mithilfe der Ausrichtungslinien und des Rasters in Windows Forms
Mithilfe der Layoutfeatures von [!INCLUDE[vsprvs](../../../../includes/vsprvs-md.md)] können Sie genau steuern, wo Steuerelemente auf einem Formular platziert werden.  Steuerelemente, die einem Formular hinzugefügt oder darauf verschoben werden, können automatisch an den Zeilen und Spalten des Rasters im Windows Forms\-Designer oder mithilfe von Ausrichtungslinien ausgerichtet werden.  
  
> [!NOTE]
>  Je nach den aktiven Einstellungen oder der Version unterscheiden sich die Dialogfelder und Menübefehle auf Ihrem Bildschirm möglicherweise von den in der Hilfe beschriebenen.  Wählen Sie im Menü **Extras** die Option **Einstellungen importieren und exportieren** aus, um die Einstellungen zu ändern.  Weitere Informationen finden Sie unter [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/de-de/22c4debb-4e31-47a8-8f19-16f328d7dcd3).  
  
### So richten Sie alle Steuerelemente am Raster aus  
  
-   Wählen Sie den Layoutmodus **SnapToGrid** im Dialogfeld **Optionen** des Windows Forms\-Designers aus.  
  
     Weitere Informationen finden Sie unter [General, Windows Forms Designer, Options Dialog Box](http://msdn.microsoft.com/de-de/8dd170af-72f0-4212-b04b-034ceee92834).  Daraufhin werden alle Steuerelemente entlang den Rasterpunkten ausgerichtet.  
  
     Sie können einzelne Steuerelemente am Raster ausrichten, indem Sie sie an ihrer Position sperren.  Gesperrte Steuerelemente können jedoch weder verschoben noch vergrößert oder verkleinert werden.  Weitere Informationen zum Sperren von Steuerelementen finden Sie unter [Gewusst wie: Sperren von Steuerelementen für Windows Forms](../../../../docs/framework/winforms/controls/how-to-lock-controls-to-windows-forms.md).  
  
### So richten Sie Steuerelemente mithilfe von Ausrichtungslinien aus  
  
-   Wählen Sie den Layoutmodus **SnapLines** im Dialogfeld **Optionen** des Windows Forms\-Designers aus.  
  
     Weitere Informationen finden Sie unter [Exemplarische Vorgehensweise: Anordnen von Steuerelementen in Windows Forms mithilfe von Ausrichtungslinien](../../../../docs/framework/winforms/controls/walkthrough-arranging-controls-on-windows-forms-using-snaplines.md).  Sie können nun mithilfe von Ausrichtungslinien Steuerelemente auf dem Formular ausrichten und anordnen.
