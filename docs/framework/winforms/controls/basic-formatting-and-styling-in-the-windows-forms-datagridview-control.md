# Grundlegende Formatierungen und Formate im DataGridView-Steuerelement in Windows Forms
Mithilfe des `DataGridView`\-Steuerelements können Sie schnell und einfach die grundlegende Darstellung von Zellen und das Anzeigeformat von Zellwerten definieren.  Sie können die Darstellung und Formatierung für einzelne Zellen, für Zellen in bestimmten Spalten und Zeilen oder für alle Zellen im Steuerelement definieren, indem Sie die Eigenschaften der `DataGridViewCellStyle`\-Objekte festlegen, auf die über verschiedene Eigenschaften des `DataGridView`\-Steuerelements zugegriffen wird.  Außerdem können Sie diese Formate anhand verschiedener Faktoren, z. B. dem Zellwert, dynamisch ändern, indem Sie das `CellFormatting`\-Ereignis behandeln.
