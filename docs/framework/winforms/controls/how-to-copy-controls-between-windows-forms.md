# Gewusst wie: Kopieren von Steuerelementen zwischen Windows Forms
Ein Steuerelement kann wie folgt kopiert werden: innerhalb desselben Formulars, in ein anderes Formular innerhalb des Projekts oder in die Zwischenablage zur Verwendung in anderen Projektmappen.  
  
### So kopieren Sie ein Steuerelement  
  
1.  Markieren Sie das Steuerelement, und wählen Sie im Menü **Bearbeiten** die Option **Kopieren**.  
  
 Das Steuerelement kann nun in ein Formular eingefügt werden, das für den jeweiligen Steuerelementtyp geeignet ist.  Darüber hinaus wurde das Steuerelement der Zwischenablage hinzugefügt.
