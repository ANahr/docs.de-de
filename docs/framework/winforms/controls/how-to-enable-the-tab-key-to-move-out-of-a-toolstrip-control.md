# <a name="how-to-enable-the-tab-key-to-move-out-of-a-toolstrip-control"></a>Gewusst wie: Aktivieren der TAB-TASTE zum Verlassen eines ToolStrip-Steuerelements
Verwenden Sie das folgende Verfahren zum Aktivieren des Benutzers zum Drücken der TAB-Taste zum Verschieben von einem <xref:System.Windows.Forms.ToolStrip> auf das nächste Steuerelement in der Aktivierreihenfolge.  
  
 Die <xref:System.Windows.Forms.ToolStrip> akzeptiert das erste Drücken der TAB-Taste und der Pfeil Schlüssel wählen Sie Elemente innerhalb der <xref:System.Windows.Forms.ToolStrip>. Wenn der Benutzer ein zweites Mal die TAB-Taste drückt, wird es der Benutzer auf das nächste Steuerelement in der Aktivierreihenfolge.  
  
### <a name="to-enable-the-user-to-press-the-tab-key-to-move-out-of-a-toolstrip-to-the-next-control"></a>So aktivieren Sie den Benutzer zum Drücken der TAB-Taste zum Verlassen eines ToolStrip zum nächsten Steuerelement  
  
-   Festlegen der <xref:System.Windows.Forms.ToolStrip.TabStop%2A> Eigenschaft von der <xref:System.Windows.Forms.ToolStrip> auf `true`.
