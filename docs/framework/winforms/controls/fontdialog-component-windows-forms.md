# <a name="fontdialog-component-windows-forms"></a>FontDialog-Komponente (Windows Forms)
Bei der Windows Forms-Komponente <xref:System.Windows.Forms.FontDialog> handelt es sich um ein vorkonfiguriertes Dialogfeld. Er ist der gleiche **Schriftart** (Dialogfeld), die von Windows-Betriebssystems verfügbar gemacht werden. Die Komponente erbt von der <xref:System.Windows.Forms.CommonDialog> Klasse.  
  
## <a name="in-this-section"></a>In diesem Abschnitt  
 [Übersicht über die FontDialog-Komponente](../../../../docs/framework/winforms/controls/fontdialog-component-overview-windows-forms.md)  
 Stellt die allgemeinen Konzepte von der <xref:System.Windows.Forms.FontDialog> -Komponente, die Sie verwenden, um die Anzeige eines vorkonfigurierten Dialogfelds. Benutzer können das Dialogfeld zum Bearbeiten von Schriftarten und ihre Einstellungen.  
  
 [Gewusst wie: Anzeigen einer Schriftartenliste mit der FontDialog-Komponente](../../../../docs/framework/winforms/controls/how-to-show-a-font-list-with-the-fontdialog-component.md)  
 Erläutert, wie das auswählen eine Schriftart zur Laufzeit über eine Instanz der <xref:System.Windows.Forms.FontDialog> Komponente.
