# FontDialog-Komponente (Windows&#160;Forms)
Die <xref:System.Windows.Forms.FontDialog>\-Komponente in Windows Forms ist ein vorkonfiguriertes Dialogfeld.  Es entspricht dem Dialogfeld **Schriftart** des Betriebssystems Windows.  Die Komponente erbt von der <xref:System.Windows.Forms.CommonDialog>\-Klasse.  
  
## In diesem Abschnitt  
 [Übersicht über die FontDialog\-Komponente](../../../../docs/framework/winforms/controls/fontdialog-component-overview-windows-forms.md)  
 Führt die allgemeinen Konzepte der <xref:System.Windows.Forms.FontDialog>\-Komponente ein, die Sie zum Anzeigen eines vorkonfigurierten Dialogfelds verwenden.  Benutzer können in diesem Dialogfeld Schriftarten und zugehörige Einstellungen ändern.  
  
 [Gewusst wie: Anzeigen einer Schriftartenliste mit der FontDialog\-Komponente](../../../../docs/framework/winforms/controls/how-to-show-a-font-list-with-the-fontdialog-component.md)  
 Erläutert das Auswählen einer Schriftart zur Laufzeit über eine Instanz der <xref:System.Windows.Forms.FontDialog>\-Komponente.
