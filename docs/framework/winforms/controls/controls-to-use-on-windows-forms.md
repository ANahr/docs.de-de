# Steuerelemente f&#252;r Windows Forms
Im folgenden wird eine alphabetische Liste der Steuerelemente und Komponenten in Windows Forms verwendet werden können. Zusätzlich zu den Windows Forms-Steuerelementen in diesem Abschnitt behandelt können Sie Windows Forms ActiveX und benutzerdefinierte Steuerelemente hinzufügen. Wenn Sie nicht finden, dass das Steuerelement, das Sie benötigen hier aufgeführt ist, können Sie auch Ihre eigenen erstellen. Weitere Informationen finden Sie unter [Entwickeln von Windows Forms-Steuerelementen zur Entwurfszeit](../../../../docs/framework/winforms/controls/developing-windows-forms-controls-at-design-time.md). Weitere Informationen zum Auswählen des Steuerelements Sie benötigen, finden Sie unter [Windows Forms-Steuerelemente nach Funktion](../../../../docs/framework/winforms/controls/windows-forms-controls-by-function.md).

>  [!INCLUDE[vbprvb](../../../../includes/vbprvb-md.md)]Steuerelemente basieren auf von bereitgestellten Klassen der [!INCLUDE[dnprdnshort](../../../../includes/dnprdnshort-md.md)].
