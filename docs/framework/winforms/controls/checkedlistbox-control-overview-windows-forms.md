# &#220;bersicht &#252;ber das CheckedListBox-Steuerelement (Windows&#160;Forms)
Das <xref:System.Windows.Forms.CheckedListBox>\-Steuerelement in Windows Forms stellt eine Erweiterung des <xref:System.Windows.Forms.ListBox>\-Steuerelements dar.  Es hat nahezu dieselben Funktionen wie ein Listenfeld, und auch die darin enthaltenen Listenelemente können mit einem Häkchen versehen werden.  Unterschiede zwischen den beiden Steuerelementen bestehen darin, dass aktivierte Listenfelder nur <xref:System.Windows.Forms.DrawMode?displayProperty=fullName> unterstützen und dass jeweils nur ein Element bzw. kein Element ausgewählt sein kann.  Beachten Sie, dass ein ausgewähltes Element im Formular hervorgehoben ist, was jedoch nicht bedeutet, dass das Element aktiviert ist.  
  
 Mithilfe des **Zeichenfolgen\-Editors** können zur Entwurfszeit Elemente zu aktivierten Listenfeldern hinzugefügt werden. Außerdem können die Listenfeldelemente auch aus einer Auflistung dynamisch mithilfe der <xref:System.Windows.Forms.CheckedListBox.Items%2A>\-Eigenschaft hinzugefügt werden.  Weitere Informationen finden Sie unter [Gewusst wie: Hinzufügen und Entfernen von Elementen in bzw. aus ComboBox\-, ListBox\- oder CheckedListBox\-Steuerelementen in Windows Forms](../../../../docs/framework/winforms/controls/add-and-remove-items-from-a-wf-combobox.md).
