# Steuerelemente in Windows&#160;Forms zum Auflisten von Optionen
Windows Forms können vielfältige Steuerelemente hinzugefügt werden, die dem Benutzer die Auswahl aus einer Liste von Optionen ermöglichen.  Je nachdem, wie weit die Eingabemöglichkeiten des Benutzers beschränkt werden sollen, fügen Sie eines der Steuerelemente <xref:System.Windows.Forms.ListBox>, <xref:System.Windows.Forms.ComboBox> oder <xref:System.Windows.Forms.CheckedListBox> hinzu.  Anhand der folgenden Links können Sie prüfen, welches Steuerelement Ihre Anforderungen am besten erfüllt.
