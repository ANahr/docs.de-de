# <a name="how-to-remove-autogenerated-columns-from-a-windows-forms-datagridview-control"></a>Gewusst wie: Entfernen von automatisch generierten Spalten aus einem DataGridView-Steuerelement in Windows Forms
Wenn Ihre <xref:System.Windows.Forms.DataGridView> Steuerelement zur automatischen Generierung Spalten basierend auf Daten aus ihrer Datenquelle festgelegt ist, können Sie bestimmte Spalten selektiv weglassen. Hierzu können Sie durch Aufrufen der <xref:System.Windows.Forms.DataGridViewColumnCollection.Remove%2A> Methode für die <xref:System.Windows.Forms.DataGridView.Columns%2A> Auflistung. Alternativ können Sie Spalten aus der Ansicht ausblenden, indem Sie die Einstellung der <xref:System.Windows.Forms.DataGridViewColumn.Visible%2A> Eigenschaft `false`. Dieses Verfahren eignet sich, wenn in bestimmten Situationen die ausgeblendeten Spalten angezeigt werden soll, oder wenn Sie die Daten in den Spalten zuzugreifen, ohne diese anzuzeigen.  
  
### <a name="to-remove-autogenerated-columns"></a>So entfernen Sie den automatisch generierten Spalten  
  
-   Rufen Sie die <xref:System.Windows.Forms.DataGridViewColumnCollection.Remove%2A> Methode für die <xref:System.Windows.Forms.DataGridView.Columns%2A> Auflistung.
