# FileDialog-Klasse
Die <xref:System.Windows.Forms.FileDialog>\-Klasse von Windows Forms ist die allgemeine Basisklasse für die <xref:System.Windows.Forms.OpenFileDialog>\-Komponente und die <xref:System.Windows.Forms.SaveFileDialog>\-Komponente.  Sie können Änderungen an der <xref:System.Windows.Forms.FileDialog>\-Klasse vornehmen, die sich je nach der Windows\-Version, unter der die Anwendung ausgeführt wird, auf die Darstellung und das Verhalten dieser Dialogfelder auswirken.

 Beschreibt, wie die automatische Formataktualisierung eines Dateidialogfelds abgelehnt wird.  
  
 [Gewusst wie: Hinzufügen eines gemeinsamen Orts zum Dialogfeld "Datei"](../../../../docs/framework/winforms/controls/how-to-add-a-custom-place-to-a-file-dialog-box.md)  
 Beschreibt das Hinzufügen eines benutzerdefinierten Dateispeicherorts zu einem Dateidialogfeld.

 Listet die Ordnernamen mit ihren zugehörigen GUIDs auf.
