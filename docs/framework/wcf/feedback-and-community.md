# Feedback und Community
Wir schätzen Ihre Kommentare und Hinweise zur [!INCLUDE[indigo1](../../../includes/indigo1-md.md)]\- und [!INCLUDE[infocard](../../../includes/infocard-md.md)]\-Dokumentation.  Unten auf jeder Seite des SDK steht der Satz "Kommentare zu diesem Thema an Microsoft senden". Klicken Sie auf den Link "Kommentare", um Ihre Ansichten oder Vorschläge zum jeweiligen Thema zu senden.  
  
 Wenn Sie weitere Informationen suchen oder Vorgehensweisen mit anderen Personen austauschen möchten, die an [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] interessiert sind, können Sie das [MSDN Windows Communication Foundation\-Forum](http://go.microsoft.com/fwlink/?LinkId=186463) nutzen.
