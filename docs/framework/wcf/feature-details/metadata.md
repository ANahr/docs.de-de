# Metadaten
[!INCLUDE[indigo1](../../../../includes/indigo1-md.md)] bietet eine Infrastruktur für den Export, die Veröffentlichung, den Abruf und den Import von Dienstmetadaten.[!INCLUDE[indigo2](../../../../includes/indigo2-md.md)]\-Dienste beschreiben mithilfe von Metadaten, wie mit den Endpunkten des Diensts zusammengearbeitet werden kann, sodass Tools, wie Svcutil.exe, automatisch Clientcode für den Zugriff auf den Dienst generieren können.
