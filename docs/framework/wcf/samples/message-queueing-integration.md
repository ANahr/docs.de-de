# <a name="message-queueing-integration"></a>Message Queueing-Integration
Dieser Abschnitt enthält Beispiele für die Integration zwischen Message Queuing und Windows Communication Foundation (WCF).  
  
## <a name="in-this-section"></a>In diesem Abschnitt  
 [Message Queuing zu Windows Communication Foundation](../../../../docs/framework/wcf/samples/message-queuing-to-wcf.md)  
 Veranschaulicht, wie eine Message Queuing (MSMQ)-Anwendung eine MSMQ-Nachricht an einen WCF-Dienst senden kann.  
  
 [Benutzerdefinierter Demux](../../../../docs/framework/wcf/samples/custom-demux.md)  
 Veranschaulicht, wie MSMQ-Nachrichtenheader unterschiedlichen Dienstvorgängen zugeordnet werden können, damit WCF-Dienste, bei denen <xref:System.ServiceModel.MsmqIntegration.MsmqIntegrationBinding> sind nicht auf die Verwendung eines einzigen Dienstvorgangs beschränkt.  
  
 [Windows Communication Foundation zu Message Queuing](../../../../docs/framework/wcf/samples/wcf-to-message-queuing.md)  
 Veranschaulicht, wie eine WCF-Anwendung eine Nachricht an einen Message Queuing (MSMQ)-Anwendung senden kann.  
  
 [Nachrichtenkorrelation](../../../../docs/framework/wcf/samples/message-correlation.md)  
 Veranschaulicht, wie eine Message Queuing (MSMQ)-Anwendung eine MSMQ-Nachricht an einen WCF-Dienst senden kann und wie Nachrichten zwischen Absender und Empfänger in einer Anforderung/Antwort-Szenario korreliert werden können.
