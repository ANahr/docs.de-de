 [ASP.NET-Zwischenspeicherungsintegration](../../../../docs/framework/wcf/samples/aspnet-caching-integration.md)

 [UriTemplate-Tabellenverteiler](../../../../docs/framework/wcf/samples/uritemplate-table-dispatcher-sample.md)
