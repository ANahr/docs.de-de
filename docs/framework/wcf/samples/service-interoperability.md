# <a name="service-interoperability"></a>Dienst-Interoperabilität
Dieser Abschnitt enthält Beispiele, in denen die Interaktion zwischen [!INCLUDE[indigo1](../../../../includes/indigo1-md.md)] und anderen Diensttechnologien veranschaulicht wird.  
  
## <a name="in-this-section"></a>In diesem Abschnitt  
 [Mithilfe des WCF-Monikers mit COM-Clients](../../../../docs/framework/wcf/samples/using-the-wcf-moniker-with-com-clients.md)  
 Veranschaulicht, wie Webdienste mithilfe des [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)]-Dienstmonikers in COM-Entwicklungsumgebungen wie Microsoft Office Visual Basic for Applications (Office VBA) oder Visual Basic 6.0 integriert werden.  
  
 [ASMX-Client mit einem WCF-Dienst](../../../../docs/framework/wcf/samples/asmx-client-with-a-wcf-service.md)  
 Zeigt, wie ein Dienst erstellt wird, der [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] verwendet, und wie dann aus einem Nicht-[!INCLUDE[indigo2](../../../../includes/indigo2-md.md)]-Client (wie einem ASMX-Client) auf den Dienst zugegriffen wird.
