# <a name="service-interoperability"></a>Dienst-Interoperabilität
Dieser Abschnitt enthält Beispiele, in denen Interaktion zwischen Windows Communication Foundation (WCF) und anderen diensttechnologien veranschaulicht.  
  
## <a name="in-this-section"></a>In diesem Abschnitt  
 [Verwenden des WCF-Monikers mit COM-Clients](../../../../docs/framework/wcf/samples/using-the-wcf-moniker-with-com-clients.md)  
 Veranschaulicht, wie den WCF-Dienstmoniker Webdienste in COM-basierten entwicklungsumgebungen wie Microsoft Office Visual Basic for Applications (Office VBA) oder Visual Basic 6.0 integriert.  
  
 [ASMX-Client mit einem WCF-Dienst](../../../../docs/framework/wcf/samples/asmx-client-with-a-wcf-service.md)  
 Veranschaulicht das Erstellen eines Diensts mithilfe von WCF, und klicken Sie dann auf den Dienst von einem nicht-WCF-Client, wie einem ASMX-Client zugreifen.
