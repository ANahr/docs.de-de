# <a name="data-binding-scenarios"></a>Datenbindungsszenarien
Dieser Abschnitt enthält Beispiele, in denen Windows Communication Foundation (WCF)-Datenbindungsszenarien veranschaulichen.  
  
## <a name="in-this-section"></a>In diesem Abschnitt  
 [Datenbindung in einem Windows Forms-Client](../../../../docs/framework/wcf/samples/data-binding-in-a-windows-forms-client.md)  
 Veranschaulicht das Binden an Daten, die von einem WCF-Dienst in einer Windows Forms-Anwendung zurückgegeben.  
  
 [Datenbindung in einem ASP.NET-Client](../../../../docs/framework/wcf/samples/data-binding-in-an-aspnet-client.md)  
 Veranschaulicht das Binden von Daten, die durch ein normaler WCF-Dienst in einer Web Forms-Anwendung zurückgegeben.  
  
 [Datenbindung in einem Windows Presentation Foundation-Client](../../../../docs/framework/wcf/samples/data-binding-in-a-wpf-client.md)  
 Veranschaulicht die Verwendung der Datenbindung in einem Windows Presentation Foundation (WPF)-Client.
