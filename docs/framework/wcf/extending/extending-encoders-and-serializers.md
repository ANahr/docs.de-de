# <a name="extending-encoders-and-serializers"></a>Erweitern von Encodern und Serialisierungsprogrammen
Die Themen in diesem Abschnitt wird beschrieben, wie das Serialisierungssystem Windows Communication Foundation (WCF) zu erweitern.  
  
## <a name="in-this-section"></a>In diesem Abschnitt  
 [Datenvertrag-Ersatzzeichen](../../../../docs/framework/wcf/extending/data-contract-surrogates.md)  
 Beschreibt, wie Serialisierung, Deserialisierung, Schemaexport und Schemaimport mithilfe einer Datenvertrag-Ersatzzeichenklasse angepasst werden können.  
  
 [Benutzerdefinierte Nachrichtenformatierung](../../../../docs/framework/wcf/extending/custom-message-formatters.md)  
 Beschreibt benutzerdefinierte Meldungsformatierer, die zum Steuern der Serialisierung und Deserialisierung einer [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)]-Meldung in CLR-Typen verwendet werden können.
