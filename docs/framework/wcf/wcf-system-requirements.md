# <a name="wcf-system-requirements"></a>WCF-Systemanforderungen
Windows Communication Foundation (WCF) ist eine Kommunikationsinfrastruktur, die zum Erstellen verteilter Anwendungen verwendet wird. Im folgenden werden Anforderungen ermöglichen es WCF ausgeführt.  
  
## <a name="system-requirements"></a>Systemanforderungen  
 WCF wird standardmäßig installiert, auf [!INCLUDE[wv](../../../includes/wv-md.md)].  
  
 WCF kann auch installiert werden, auf [!INCLUDE[wxpsp2](../../../includes/wxpsp2-md.md)], [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)], oder [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)].  
  
 **Hinweis** der Message Queuing (MSMQ)-Funktionalität von WCF wird nur unterstützt unter [!INCLUDE[wv](../../../includes/wv-md.md)], [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)], [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)], und [!INCLUDE[wxppro](../../../includes/wxppro-md.md)].
