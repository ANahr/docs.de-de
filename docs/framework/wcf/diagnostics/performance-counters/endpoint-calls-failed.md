# <a name="endpoint-calls-failed"></a>Endpunkt: Aufrufe fehlgeschlagen
Indikatorname: Aufrufe fehlgeschlagen  
  
## <a name="description"></a>Beschreibung  
 Anzahl von Aufrufen mit nicht behandelten Ausnahmen an diesem Endpunkt. Der Wert dieses Zählers erhöht sich, sobald an diesem Endpunkt eine nicht behandelte Ausnahme auftritt.
