# <a name="reliable-messaging-sessions-faulted-per-second"></a>Endpunkt: Fehlerhafte zuverlässige Messagingsitzungen pro Sekunde
Indikatorname: Reliable Messaging Sessions Faulted Per Second.  
  
## <a name="description"></a>Beschreibung  
 Fehlerhafte zuverlässige Messagingsitzungen pro Sekunde in diesem Dienst (Anzahl).  
  
 Dieser Indikator wird der Leistungsindikator vom Typ [PERF_COUNTER_COUNTER](https://go.microsoft.com/fwlink/?LinkID=94649), dessen Wert anhand der folgenden Formel berechnet wird.  
  
 (N 1 - N 0)/( (D 1 - D 0)/F)
