# Dienst: Aufrufdauer
Indikatorname: Aufrufdauer  
  
## Beschreibung  
 Die durchschnittliche Dauer der Aufrufe dieses Dienstes.  Die durchschnittliche Dauer wird anhand der folgenden Formel berechnet: \(N1\-N0\)\/\(D1\-D0\).
