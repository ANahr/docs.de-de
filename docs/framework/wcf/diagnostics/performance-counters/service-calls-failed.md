# <a name="service-calls-failed"></a>Dienst: Fehler beim Aufrufen
Indikatorname: Aufrufe fehlgeschlagen  
  
## <a name="description"></a>Beschreibung  
 Anzahl von Aufrufen mit nicht behandelten Ausnahmen in diesem Dienst. Der Wert dieses Indikators erhöht sich jedes Mal, wenn in diesem Dienst eine nicht behandelte Ausnahme auftritt.
