# Aufrufe fehlgeschlagen
Indikatorname: Aufrufe fehlgeschlagen  
  
## Beschreibung  
 Anzahl von Aufrufen mit nicht behandelten Ausnahmen in diesem Vorgang.Der Wert dieses Indikators erhöht sich, sobald in diesem Vorgang eine nicht behandelte Ausnahme auftritt.
