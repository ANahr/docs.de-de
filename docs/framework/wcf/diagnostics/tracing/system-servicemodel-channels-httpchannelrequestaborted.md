# System.ServiceModel.Channels.HttpChannelRequestAborted
Es konnte keine Anforderungsnachricht über HTTP gesendet werden.  
  
## Beschreibung  
 Diese Ablaufverfolgung auf Warnungsebene gibt an, dass eine HTTP\-Kanalanforderung abgebrochen wurde.  Weitere Informationen in dieser Ablaufverfolgung geben Aufschluss über den Grund oder die Nachricht, die der Abort\-Methode übergeben wurde.
