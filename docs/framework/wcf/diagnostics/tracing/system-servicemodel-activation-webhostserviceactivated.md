# System.ServiceModel.Activation.WebHostServiceActivated
System.ServiceModel.Activation.WebHostServiceActivated  
  
## Beschreibung  
 Tritt auf, wenn ein in Internetinformationsdienste \(IIS\) oder in Windows Process Activation Service \(WAS\) gehosteter Dienst vollständig aktiviert ist und die Überwachung von Nachrichten gestartet wird.
