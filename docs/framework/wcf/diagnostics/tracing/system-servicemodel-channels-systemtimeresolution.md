# System.ServiceModel.Channels.SystemTimeResolution
System.ServiceModel.Channels.SystemTimeResolution  
  
## Beschreibung  
 Die Zeitgeberauflösung des Betriebssystems wurde als *n* Zeiteinheiten erkannt. Dies entspricht ungefähr *m* Millisekunden.  *n* und *m* stehen hierbei für numerische Werte.
