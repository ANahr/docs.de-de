# System.ServiceModel.CommunicationObjectCloseFailed
Ein Kommunikationsobjekt konnte nicht geschlossen werden.  
  
## Beschreibung  
 Ein Kommunikationsobjekt konnte nicht geschlossen werden, da während der Ausführung der Close\-Methode eine Ausnahme ausgelöst wurde.Eine andere nahegelegene Ablaufverfolgung enthält möglicherweise nähere Angaben zur Ausnahme.
