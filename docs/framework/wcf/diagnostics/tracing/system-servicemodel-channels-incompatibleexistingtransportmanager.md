# <a name="systemservicemodelchannelsincompatibleexistingtransportmanager"></a>System.ServiceModel.Channels.IncompatibleExistingTransportManager
Es wurde ein bereits vorhandener inkompatibler Transport-Manager für den angegebenen URI gefunden.  
  
## <a name="description"></a>Beschreibung  
 Diese Ablaufverfolgung auf Informationsebene zeigt an, dass für den zu öffnenden Listener zwar eine passende Registrierung gefunden wurde, dass der vorhandene Transport-Manager jedoch inkompatibel ist. Der Listener versucht, einen anderen Transport-Manager zu verwenden und löst eine Ausnahme aus, wenn dies fehlschlägt.
