# System.ServiceModel.DroppedAMessage
System.ServiceModel.DroppedAMessage  
  
## Beschreibung  
 Es wurde eine Antwortnachricht empfangen, aber keine ausstehenden Anforderungen warten auf diese Nachricht.Die Nachricht wurde von der [!INCLUDE[indigo1](../../../../../includes/indigo1-md.md)]\-Laufzeit verworfen.  
  
 Diese Ablaufverfolgung kann zum Debuggen verlorener Nachrichten verwendet werden.
