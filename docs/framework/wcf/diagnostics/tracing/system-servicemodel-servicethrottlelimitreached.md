# <a name="systemservicemodelservicethrottlelimitreached"></a>System.ServiceModel.ServiceThrottleLimitReached
System.ServiceModel.ServiceThrottleLimitReached  
  
## <a name="description"></a>Beschreibung  
 Eine der Drosselungen für MaxConcurrentCalls/Instanzen/Sitzungen wurde erreicht, und eine Nachricht wird in die Warteschlange gestellt.  
  
 Sie können diese Ablaufverfolgung verwenden, um die Drosselungswerte auf Basis der erwarteten und der tatsächlichen Produktionslast anzupassen.
