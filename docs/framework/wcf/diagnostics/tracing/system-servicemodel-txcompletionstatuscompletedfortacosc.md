# <a name="systemservicemodeltxcompletionstatuscompletedfortacosc"></a>System.ServiceModel.TxCompletionStatusCompletedForTACOSC
Die bestimmte Transaktion wurde abgeschlossen, als die Sitzung geschlossen wurde.  
  
## <a name="description"></a>Beschreibung  
 Wurde aufgezeichnet, als die aktuelle aktive Sitzung geschlossen wurde, und `TransactionAutoCompleteOnSessionClose` wird auf `true` festgelegt.
