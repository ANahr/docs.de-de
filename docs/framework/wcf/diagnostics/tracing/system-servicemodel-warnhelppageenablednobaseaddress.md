System.ServiceModel.WarnHelpPageEnabledNoBaseAddress  
  
## <a name="description"></a>Beschreibung  
 Die ServiceDebugBehavior-Hilfeseite wird an einer relativen Adresse aktiviert und kann nicht erstellt werden, da keine Basisadresse vorhanden ist.
