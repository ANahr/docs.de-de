# System.ServiceModel.ServiceHostFaulted
System.ServiceModel.ServiceHostFaulted  
  
## Beschreibung  
 Im ServiceHost ist ein Fehler aufgetreten.  
  
 Diese Ablaufverfolgung gibt an, dass einer der Listener einen Fehler verursacht hat.Diese Ablaufverfolgung erleichtert das Debuggen.
