# Microsoft.Transactions.TransactionBridge.RecoveredParticipantInvalidMetadata
Der in einem Wiederherstellungsprotokolleintrag für einen Teilnehmer gefundene Endpunktverweis enthielt ungültige oder nicht kompatible Metadaten.  
  
## Beschreibung  
 Das System kann keinen Proxy für eine wiederhergestellte Teilnehmereintragung erstellen.  
  
## Problembehandlung  
 Wenden Sie sich an Microsoft, wenn das Problem weiterhin auftritt.
