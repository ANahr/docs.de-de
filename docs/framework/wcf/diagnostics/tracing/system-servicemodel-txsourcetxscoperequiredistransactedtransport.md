# <a name="systemservicemodeltxsourcetxscoperequiredistransactedtransport"></a>System.ServiceModel.TxSourceTxScopeRequiredIsTransactedTransport
Die angegebene Transaktion für den angegebenen Vorgang wurde von einem transaktiven Transport empfangen, z. B. von MSMQ.  
  
## <a name="description"></a>Beschreibung  
 Wird nachverfolgt, wenn die für Benutzercode verfügbar gemachte Transaktion vom Transport erstellt wurde.
