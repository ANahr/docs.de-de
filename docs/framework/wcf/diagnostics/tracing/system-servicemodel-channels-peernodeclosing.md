# <a name="systemservicemodelchannelspeernodeclosing"></a>System.ServiceModel.Channels.PeerNodeClosing
Der PeerNode hat mit dem Schließvorgang begonnen.  
  
## <a name="description"></a>Beschreibung  
 Diese Ablaufverfolgung tritt am Anfang der Ausführung des Schließen-Vorgangs auf dem PeerNode auf.
