# System.ServiceModel.TxSourceTxScopeRequiredIsAttachedTransaction
Die angegebene Transaktion wurde für den angegebenen Vorgang von einer InstanceContext\-Transaktion empfangen.  
  
## Beschreibung  
 Verfolgte nach, ob die dem Benutzercode verfügbar gemachte Transaktion durch eine AutoComplete\=false\-Methode zurückgelassen wurde, die vor diesem Aufruf abgerufen wurde.
