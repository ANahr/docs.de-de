# <a name="microsofttransactionstransactionbridgeenlisttransaction"></a>Microsoft.Transactions.TransactionBridge.EnlistTransaction
Der WS-Atomic-Transaktionsprotokolldienst hat sich erfolgreich in einer Transaktion eingetragen, die den bereitgestellten Koordinationskontext verwendet.  
  
## <a name="description"></a>Beschreibung  
 Eine eingehende Transaktion wurde erfolgreich verarbeitet.
