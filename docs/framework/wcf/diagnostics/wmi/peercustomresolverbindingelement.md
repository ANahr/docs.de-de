# PeerCustomResolverBindingElement
PeerCustomResolverBindingElement  
  
## Syntax  
 `class PeerCustomResolverBindingElement : PeerResolverBindingElement`  
  
 `{`  
  
 `string Address;`  
  
 `string Binding;`  
  
 `};`  
  
## Methoden  
 Die Klasse PeerCustomResolverBindingElement definiert keine Methoden.  
  
## Eigenschaften  
 Die KlassePeerCustomResolverBindingElement weist die folgenden Eigenschaften auf:  
  
### Address  
 Datentyp: Zeichenfolge  
  
 Zugriffstyp: Schreibgeschützt  
  
 Die Adresse des benutzerdefinierten Peerresolvers.  
  
### Bindung  
 Datentyp: string \(Zeichenfolge\)  
  
 Zugriffstyp: Schreibgeschützt  
  
 Der Konfigurationsname der Bindung.
