# GetOperationCounterInstanceName
Ruft den Instanznamen des Vorgangsleistungsindikators ab.  
  
## Syntax  
  
```  
string GetOperationCounterInstanceName(  
  string Operation  
);  
```  
  
## Parameter  
  
### Operation  
 \[in\] string  
  
## Anforderungen  
  
|MOF|Deklariert in Servicemodel.mof.|  
|---------|-------------------------------------|  
|Namespace|Definiert in root\\ServiceModel|
