# <a name="msmqintegrationbindingelement"></a>MsmqIntegrationBindingElement
MsmqIntegrationBindingElement  
  
## <a name="syntax"></a>Syntax  
  
```  
class MsmqIntegrationBindingElement : MsmqBindingElementBase  
{  
  string SerializationFormat;  
};  
```  
  
## <a name="methods"></a>Methoden  
 Von der MsmqIntegrationBindingElement-Klasse werden keine Methoden definiert.  
  
## <a name="properties"></a>Eigenschaften  
 Die MsmqIntegrationBindingElement-Klasse verfügt über die folgende Eigenschaft:  
  
### <a name="serializationformat"></a>SerializationFormat  
 Datentyp: string (Zeichenfolge)  
  
 Zugriffstyp: Schreibgeschützt  
  
 Das Format, das von der Bindung zum Serialisieren von Nachrichten verwendet wird.
