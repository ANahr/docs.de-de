# ProtocolStopFailure
Id: 147  
  
 Schweregrad: Fehler  
  
 Kategorie: TransactionBridge  
  
## Beschreibung  
 Dieses Ereignis gibt an, dass der WS\-Atomic Transaction\-Protokolldienst nicht beendet werden konnte.Das Ereignis listet die Elemente Protokoll\-ID, Protokollname, Ausnahme, Prozessname und Prozess\-ID auf.
