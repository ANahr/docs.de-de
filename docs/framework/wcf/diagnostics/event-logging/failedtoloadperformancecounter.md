# FailedToLoadPerformanceCounter
ID: 118  
  
 Schweregrad: Fehler  
  
 Kategorie: ServiceModel  
  
## Beschreibung  
 Dieses Ereignis gibt an, dass kein Leistungsindikator geladen wurde.  Das Ereignis listet die Elemente Kategoriename, Indikatorname, Instanzname, Ausnahme, Prozessname und Prozess\-ID auf.
