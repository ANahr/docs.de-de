# ParticipantRecoveryLogEntryCorrupt
ID: 138  
  
 Schweregrad: Fehler  
  
 Kategorie: TransactionBridge  
  
## Beschreibung  
 Dieses Ereignis gibt an, dass ein Wiederherstellungsprotokolleintrag für einen Teilnehmer beschädigt war und nicht deserialisiert werden konnte.Aufgrund dieses Fehlers können Daten verloren gehen.Das Ereignis führt die Transaktions\-ID, die Wiederherstellungsdaten \(Base64\-codiert\), die Ausnahme, den Prozessnamen und die Prozess\-ID auf.
