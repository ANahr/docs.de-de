# ServiceStartFailed
ID: 164  
  
 Schweregrad: Fehler  
  
 Kategorie: SMSvcHost  
  
## Beschreibung  
 Dieses Ereignis gibt an, dass eine Anforderung zum Starten des Diensts nicht erfüllt werden konnte.Das Ereignis führt den Fehlercode, den Prozessnamen und die Prozess\-ID auf.
