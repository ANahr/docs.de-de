# ComPlusDllHostInitializerStartingError
ID: 131  
  
 Schweregrad: Fehler  
  
 Kategorie: ServiceModel  
  
## Beschreibung  
 Dieses Ereignis gibt an, dass beim Starten des DllHost\-Initialisieres ein Fehler aufgetreten ist.Das Ereignis listet die Elemente AppID, Ausnahme, Prozessname und Prozess\-ID auf.
