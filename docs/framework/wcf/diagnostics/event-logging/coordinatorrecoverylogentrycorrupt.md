# CoordinatorRecoveryLogEntryCorrupt
ID: 139  
  
 Schweregrad: Fehler  
  
 Kategorie: TransactionBridge  
  
## Beschreibung  
 Dieses Ereignis gibt an, dass ein Koordinatorwiederherstellungsprotokolleintrag fehlerhaft war und nicht deserialisiert werden konnte.Aufgrund dieses Fehlers können Daten verloren gehen.Das Ereignis führt die Transaktions\-ID, die Wiederherstellungsdaten \(Base64\-codiert\), die Ausnahme, den Prozessnamen und die Prozess\-ID auf.
