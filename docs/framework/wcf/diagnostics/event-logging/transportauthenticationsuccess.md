# TransportAuthenticationSuccess
ID: 173  
  
 Schweregrad: Information  
  
 Kategorie: SecurityAudit  
  
## Beschreibung  
 Dieses Ereignis gibt an, dass die Transportauthentifizierung erfolgreich war.Das Ereignis listet den Dienst, die Clientidentität und die Aktivitäts\-ID auf.
