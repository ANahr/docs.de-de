# PiiLoggingOn
Dd: 107  
  
 Schweregrad: Fehler  
  
 Kategorie: Ablaufverfolgung  
  
## Beschreibung  
 Dieses Ereignis gibt an, dass die PII\-Protokollierung aktiviert wurde.Vertrauliche Informationen werden protokolliert.Das Ereignis listet den Prozessnamen und die Prozess\-ID auf.
