# FailFastException
ID: 103  
  
 Schweregrad: Fehler  
  
 Kategorie: Ablaufverfolgung  
  
## Beschreibung  
 Während FailFast wurde eine Ausnahme ausgelöst.  Das Ereignis führt die Ausnahme, den Prozessnamen und die Prozess\-ID auf.
