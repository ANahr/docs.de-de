# <a name="wasconnectiontimedout"></a>WasConnectionTimedout
ID: 163  
  
 Schweregrad: Fehler  
  
 Kategorie: SMSvcHost  
  
## <a name="description"></a>Beschreibung  
 Dieses Ereignis gibt eine Verbindung an, für die der Windows-Prozessaktivierungsdienst eine Zeitüberschreitung erfordert. Dabei wird auch der Prozessname und die -ID aufgeführt.
