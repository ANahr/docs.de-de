# StartErrorPublish
ID: 158  
  
 Schweregrad: Fehler  
  
 Kategorie: SMSvcHost  
  
## Beschreibung  
 Dieses Ereignis gibt an, dass der NT\-Dienst nicht gestartet werden konnte.  Der Endpunkt zum Abhören konnte nicht veröffentlicht werden.  Das Ereignis führt den Prozessnamen und die Prozess\-ID auf.
