# WmiRegistrationFailed
ID: 126  
  
 Schweregrad: Fehler  
  
 Kategorie: ServiceModel  
  
## Beschreibung  
 Dieses Ereignis gibt an, dass die Registrierung des WMI\-Anbieters nicht registriert wurde.Das Ereignis listet das WMI\-Objekt, den Fehler, den Prozessnamen und die Prozess\-ID auf.
