# MessageQueueDuplicatedSocketLeak
ID: 165  
  
 Schweregrad: Fehler  
  
 Kategorie: SMSvcHost  
  
## Beschreibung  
 Dieses Ereignis gibt an, dass beim Senden eines duplizierten Sockets ein Fehler aufgetreten ist.Dieser Handle geht im Verlauf verloren.Das Ereignis listet die Quelle, die Ausnahme, den Prozessnamen und die Prozess\-ID auf.
