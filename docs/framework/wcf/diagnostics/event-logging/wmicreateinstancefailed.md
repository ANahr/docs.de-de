# WmiCreateInstanceFailed
ID: 123  
  
 Schweregrad: Fehler  
  
 Kategorie: ServiceModel  
  
## Beschreibung  
 Dieses Ereignis gibt an, dass die WMI\-CreateInstance\-Anforderung nicht verarbeitet wurde.  Das Ereignis listet den Klassennamen, die Ausnahme, den Prozessnamen und die Prozess\-ID auf.
