# FatalUnexpectedStateMachineEvent
ID: 137  
  
 Schweregrad: Fehler  
  
 Kategorie: TransactionBridge  
  
## Beschreibung  
 Dieses Ereignis gibt an, dass ein Zustandsautomat versuchte, ein unerwartetes Ereignis zu verarbeiten.Das Ereignis wurde als schwerwiegend eingestuft.Das Ereignis listet die Transaktions\-ID, den Zustandsautomaten, den aktuellen Zustand, den Verlauf, den Ereignisnamen, die Ereignisdetails, den Prozessnamen und die Prozess\-ID auf.
