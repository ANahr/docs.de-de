# <a name="fatalunexpectedstatemachineevent"></a>FatalUnexpectedStateMachineEvent
ID: 137  
  
 Schweregrad: Fehler  
  
 Kategorie: TransactionBridge  
  
## <a name="description"></a>Beschreibung  
 Dieses Ereignis gibt an, dass ein Zustandsautomat versucht hat, ein unerwartetes Ereignis zu verarbeiten. Das Ereignis wurde als schwerwiegend eingestuft. Das Ereignis listet die Transaktions-ID, den Zustandsautomaten, den aktuellen Status, den Verlauf, den Ereignisnamen, die Ereignisdetails, den Prozessnamen und die Prozess-ID auf.
