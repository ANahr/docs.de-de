# <a name="serviceauthorizationsuccess"></a>ServiceAuthorizationSuccess
ID: 176  
  
 Schweregrad: Information  
  
 Kategorie: SecurityAudit  
  
## <a name="description"></a>Beschreibung  
 Dieses Ereignis gibt an, dass die Dienstautorisierung erfolgreich war. Das Ereignis führt den Dienst, die Aktion, die Clientidentität, den Autorisierungskontext, die Aktivitäts-ID und den ServiceAuthorizationManager auf.
