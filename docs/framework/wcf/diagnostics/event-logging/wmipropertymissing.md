# WmiPropertyMissing
ID: 129  
  
 Schweregrad: Fehler  
  
 Kategorie: ServiceModel  
  
## Beschreibung  
 Dieses Ereignis gibt an, dass ein Typenkonflikt aufgetreten ist.Die angegebene CIM\-Klasse besitzt keine bestimmte Eigenschaft des angegebenen Typs.Das Ereignis listet den Prozessnamen und die Prozess\-ID auf.
