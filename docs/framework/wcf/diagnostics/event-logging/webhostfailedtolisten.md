# WebHostFailedToListen
ID: 112  
  
 Schweregrad: Fehler  
  
 Kategorie: ServiceModel  
  
## Beschreibung  
 Dieses Ereignis gibt an, dass beim Abhören der angegebenen URL ein Fehler aufgetreten ist.Dieser Arbeitsprozess wird beendet.Das Ereignis führt die Ausnahme, den Prozessnamen und die Prozess\-ID auf.
