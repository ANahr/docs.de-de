# MessageAuthenticationSuccess
ID: 169  
  
 Schweregrad: Information  
  
 Kategorie: SecurityAudit  
  
## Beschreibung  
 Dieses Ereignis gibt an, dass die Nachrichtenauthentifizierung erfolgreich war.Das Ereignis listet den Dienst, die Aktion, die Clientidentität und die Aktivitäts\-ID auf.
