# MessageLoggingOff
Id: 117  
  
 Schweregrad: Fehler  
  
 Kategorie: ServiceModel  
  
## Beschreibung  
 Dieses Ereignis gibt an, dass die Meldungsprotokollierung ausgeschaltet wurde.Das Ereignis listet den Prozessnamen und die Prozess\-ID auf.
