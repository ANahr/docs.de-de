# WmiGetObjectFailed
ID: 120  
  
 Schweregrad: Fehler  
  
 Kategorie: ServiceModel  
  
## Beschreibung  
 Dieses Ereignis gibt an, dass die WMI\-GetObject\-Anforderung nicht verarbeitet wurde.Das Ereignis listet das WMI\-Objekt, die Ausnahme, den Prozessnamen und die Prozess\-ID auf.
