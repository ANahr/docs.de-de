# ProtocolRecoveryBeginningFailure
ID: 144  
  
 Schweregrad: Fehler  
  
 Kategorie: TransactionBridge  
  
## Beschreibung  
 Dieses Ereignis gibt an, dass das MSDTC WS\-Atomic Transaction\-Protokoll beim Start der Wiederherstellung fehlgeschlagen ist.Aus diesem Grund ist die WS\-Atomic Transaction\-Funktionalität deaktiviert.Das Ereignis führt die Protokoll\-ID, den Protokollnamen, die Ausnahme, den Prozessnamen und die Prozess\-ID auf.
