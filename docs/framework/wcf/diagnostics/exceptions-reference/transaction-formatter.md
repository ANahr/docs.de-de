# <a name="transaction-formatter"></a>Transaktionsformatierung
In diesem Thema sind alle von der Transaktionsformatierung generierten Ausnahmen aufgeführt.  
  
## <a name="exception"></a>Ausnahme  
  
|Ressourcencode|Ressourcenzeichenfolge|  
|-------------------|---------------------|  
|TMCommunicationError|Während der Kommunikation mit dem verteilten Transaktions-Manager ist ein Fehler aufgetreten.|  
|WhereaboutsNoHostName|Die MSDTC-Speicherortinformationen enthielten keinen Hostnamen.|  
|WhereaboutsReadFailed|Die MSDTC-Speicherortinformationen konnten nicht deserialisiert werden.|  
|WhereaboutsSignatureMissing|Die Standardspeicherortsignatur fehlte in den MSDTC-Speicherortinformationen.|  
|WsatUriCreationFailed|Eine Registrierungsdienstadresse konnte nicht aus den MSDTC-Speicherortinformationen erstellt werden.|
