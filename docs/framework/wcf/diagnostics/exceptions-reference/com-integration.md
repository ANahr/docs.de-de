# <a name="com-integration"></a>COM+-Integration
In diesem Thema sind alle Ausnahmen aufgeführt, die von der COM+Integration generiert werden.  
  
## <a name="exception-list"></a>Ausnahmeliste  
  
|Ressourcencode|Ressourcenzeichenfolge|  
|-------------------|---------------------|  
|CannotResolveTypeForParamInMessageDescription|Der angegebene Parametertyp im angegebenen Namespace kann nicht aufgelöst werden.|  
|NoEqualSignFound|Auf das angegebene Schlüsselwort folgt kein Gleichheitszeichen. Vergewissern Sie sich, dass auf jedes Schlüsselwort ein Gleichheitszeichen sowie ein Wert folgen.|  
|UdtNotFoundInAssembly|Der angegebene benutzerdefinierte Typ kann gefunden werden. Vergewissern Sie sich, dass der richtige Typ und die richtige Typbibliothek registriert und angegeben wurden.|
