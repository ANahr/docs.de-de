# Peerkanal
In diesem Thema werden alle Ausnahmen aufgelistet, die vom [!INCLUDE[indigo1](../../../../../includes/indigo1-md.md)]\-Peerkanal generiert werden.  
  
## Ausnahmeliste  
  
|Ressourcencode|Ressourcenzeichenfolge|  
|--------------------|----------------------------|  
|InsufficientCredentials|Die angegebenen Anmeldeinformationen sind für den angeforderten Vorgang unzureichend.  Geben Sie einen gültigen Wert für den angegebenen Vorgang an.|  
|InvalidResolverMode|Der angegebene PeerResolverMode\-Wert ist ungültig.  Geben Sie entweder "PeerResolveMode.Auto", "Default" oder "PNRP" an.|  
|PeerNodeAborted|Der PeerNode kann nicht geöffnet werden, weil er beendet wurde.|  
|PeerNullRegistrationInfo|Die Registrierungsinformationen können nicht NULL sein.  Stellen Sie sicher, dass der Registrierungsvorgang mit einem gültigen RegistrationInfo\-Objekt aufgerufen wird.|  
|PeerNullResolveInfo|Die Auflösungsinformationen können nicht NULL `null` sein.  Stellen Sie sicher, dass der Auflösungsvorgang mit einem gültigen ResolveInfo\-Objekt aufgerufen wird.|
