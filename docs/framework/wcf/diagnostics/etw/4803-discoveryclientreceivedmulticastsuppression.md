|||  
|-|-|  
|ID|4803|  
|Stichwörter|Suche|  
|Ebene|Information|  
|Kanal|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Beschreibung  
 Dieses Ereignis wird ausgegeben, wenn vom DiscoveryClient eine Multicastunterdrückungsmeldung von einem DiscoveryProxy empfangen wurde.  
  
## <a name="message"></a>Meldung  
 Vom DiscoveryClient wurde eine Multicastunterdrückungsmeldung von einem DiscoveryProxy empfangen.
