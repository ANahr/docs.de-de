# 3827 - RoutingServiceTransmitFailed
## Eigenschaften  
  
|||  
|-|-|  
|ID|3827|  
|Stichwörter|RoutingServices|  
|Ebene|Warnung|  
|Kanal|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Beschreibung  
 Dieses Ereignis wird ausgegeben, wenn die Übertragung des Routingdiensts fehlgeschlagen ist.  
  
## Meldung  
 Der Routingdienst hat beim Senden an '%1' einen Fehler ausgelöst.
