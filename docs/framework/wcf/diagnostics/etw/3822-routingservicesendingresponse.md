# 3822 - RoutingServiceSendingResponse
## Eigenschaften  
  
|||  
|-|-|  
|ID|3822|  
|Stichwörter|RoutingServices|  
|Ebene|Information|  
|Kanal|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Beschreibung  
 Dieses Ereignis wird ausgegeben, wenn der Routingdienst eine Antwortnachricht zurücksendet.  
  
## Meldung  
 Der Routingdienst sendet eine Antwortnachricht mit Aktion '%1' zurück.
