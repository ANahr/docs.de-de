# 3428 - WebSocketConnectionClosed
## Eigenschaften  
  
|||  
|-|-|  
|ID|3428|  
|Schlüsselwörter|HTTP|  
|Ebene|Ausführlich|  
|Kanal|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Beschreibung  
 Dieses Ereignis wird ausgegeben, wenn die WebSocket\-Verbindung geschlossen wird.  
  
## Meldung  
 Die Verbindung mit WebSocketId: %1 wurde geschlossen.
