# 62326 - HttpHandlerPickedForUrl
## Eigenschaften  
  
|||  
|-|-|  
|ID|62326|  
|Schlüsselwörter|WebHost|  
|Ebene|Information|  
|Kanal|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Beschreibung  
 Dieses Ereignis wird ausgegeben, wenn der HTTP\-Handler ausgewählt wird, um alle Anforderungen an diese URL zu bedienen.  
  
## Meldung  
 Die URL '%1' hostet ein XAML\-Dokument mit Stammelementtyp '%2'.  Der HTTP\-Handlertyp '%3' wurde zum Verarbeiten aller Anforderungen an diese URL ausgewählt.
