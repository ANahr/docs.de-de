# <a name="4815---endpointdiscoverabilityenabled"></a>4815 - EndpointDiscoverabilityEnabled
## <a name="properties"></a>Eigenschaften  
  
|||  
|-|-|  
|ID|4815|  
|Schlüsselwörter|Suche|  
|Ebene|Information|  
|Kanal|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Beschreibung  
 Dieses Ereignis wird ausgegeben, wenn die Erkennbarkeit des Endpunkts aktiviert wurde.  
  
## <a name="message"></a>Meldung  
 Die Erkennbarkeit des Endpunkts mit EndpointAddress='%1' und ListenUri='%2' wurde aktiviert.
