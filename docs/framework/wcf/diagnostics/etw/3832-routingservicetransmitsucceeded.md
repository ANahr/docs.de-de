# 3832 - RoutingServiceTransmitSucceeded
## Eigenschaften  
  
|||  
|-|-|  
|ID|3832|  
|Stichwörter|RoutingServices|  
|Ebene|Information|  
|Kanal|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Beschreibung  
 Dieses Ereignis wird ausgegeben, wenn der Routingdienst erfolgreich eine Nachricht mit einer angegebenen ID gesendet hat.  
  
## Meldung  
 Der Routingdienst hat die Nachricht mit der ID '%1 \[Vorgang %2\] erfolgreich an '%3' übermittelt.
