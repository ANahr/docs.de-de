# <a name="4819---innerchannelopensucceeded"></a>4819 - InnerChannelOpenSucceeded
## <a name="properties"></a>Eigenschaften  
  
|||  
|-|-|  
|ID|4819|  
|Stichwörter|Suche|  
|Ebene|Information|  
|Kanal|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Beschreibung  
 Dieses Ereignis wird ausgegeben, wenn der DiscoveryClientChannel erfolgreich einen Endpunkt ermittelt und den verwendenden Kanal geöffnet hat.  
  
## <a name="message"></a>Meldung  
 Der DiscoveryClientChannel hat erfolgreich einen Endpunkt erkannt und über diesen den Kanal geöffnet. Der Client ist über EndpointAddress='%1' und Via='%2' mit einem Dienst verbunden.
