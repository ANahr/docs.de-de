# <a name="3415---websocketconnectionrequestsendstart"></a>3415 - WebSocketConnectionRequestSendStart
## <a name="properties"></a>Eigenschaften  
  
|||  
|-|-|  
|ID|3415|  
|Stichwörter|HTTP|  
|Ebene|Ausführlich|  
|Kanal|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Beschreibung  
 Dieses Ereignis wird ausgegeben, wenn die WebSocket-Verbindungsanforderung zum Senden gestartet wurde.  
  
## <a name="message"></a>Meldung  
 Das Senden der WebSocket-Verbindungsanforderung an "%1" wurde gestartet.
