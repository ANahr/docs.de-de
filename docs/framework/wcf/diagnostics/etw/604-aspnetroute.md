|-|-|

## <a name="description"></a>Beschreibung  
 Dieses Ereignis wird ausgegeben, wenn eine neue Asp.Net-Route '%1' mit serviceType '%2' und serviceHostFactoryType '%3' hinzugefügt wird.  
  
## <a name="message"></a>Meldung  
 Eine neue Asp.Net-Route '%1' mit serviceType '%2' und serviceHostFactoryType '%3' wird hinzugefügt.
