# 3416 - WebSocketConnectionRequestSendStop
## Eigenschaften  
  
|||  
|-|-|  
|ID|3416|  
|Stichwörter|HTTP|  
|Ebene|Ausführlich|  
|Kanal|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Beschreibung  
 Dieses Ereignis wird ausgegeben, wenn die WebSocket\-Verbindungsanforderung zum Senden beendet wurde.  
  
## Meldung  
 Die Verbindungsanforderung für WebSocketId:%1 wurde gesendet.
