# <a name="603---aspnetroutingservice"></a>603 - AspNetRoutingService
## <a name="properties"></a>Eigenschaften  
  
|||  
|-|-|  
|ID|603|  
|Stichwörter|RoutingServices|  
|Ebene|Ausführlich|  
|Kanal|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Beschreibung  
 Dieses Ereignis wird ausgegeben, wenn die eingehende Anforderung mit einem WCF-Dienst übereinstimmt, der in der Asp.Net-Route mit der Adresse %1 definiert wird.  
  
## <a name="message"></a>Meldung  
 Die eingehende Anforderung stimmt mit einem WCF-Dienst überein, der in der Asp.Net-Route mit der Adresse %1 definiert wird.
