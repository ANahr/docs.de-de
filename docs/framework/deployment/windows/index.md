# <a name="installing-the-net-framework-on-windows"></a>Installieren von .NET Framework unter Windows

In den folgenden Dokumenten wird beschrieben, wie .NET Framework für jede Windows-Version installiert wird.

## <a name="version-differences"></a>Versionsunterschiede

.NET Framework ist in Windows enthalten. Jede Windows-Version enthält eine einzelne .NET Framework-Version, die möglicherweise nicht die ist, die Sie benötigen. Wenn Sie eine andere Version von .NET Framework für eine bestimmte Version von Windows verwenden möchten, müssen Sie diese installieren.

Im Allgemeinen führt eine höhere Version von .NET Framework eine Anwendung aus, die für eine niedrigere Version von .NET Framework erstellt wurde. .NET Framework 4.6.2 führt z.B. eine Anwendung für .NET Framework 4.0 aus. Es gibt allerdings eine Ausnahme zu dieser Regel. Anwendungen für .NET Framework 1.0 über 3.5 können unter .NET Framework 3.5 ausgeführt werden, aber nicht unter .NET Framework 4.0 oder höher.
