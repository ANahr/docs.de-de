# Sicherheit in der Netzwerkprogrammierung
Die Namespaceklassen .NET Framework <xref:System.Net> bieten integrierte Unterstützung für gängige Internet\-Anwendungsauthentifizierungsmechanismen und für .NET Framework\-Codezugriffsberechtigungen.  
  
## In diesem Abschnitt  
 [Verwenden von Secure Sockets Layer](../../../docs/framework/network-programming/using-secure-sockets-layer.md)  
 Beschreibt, wie Verbindungen von SSL \(Secure Sockets Layer\) verwendet.  
  
 [Internetauthentifizierung](../../../docs/framework/network-programming/internet-authentication.md)  
 Beschreibt, wie HTTP\-Authentifizierungsmethoden verwendet, um authentifizierte Verbindungen zu den HTTP\-Servern herzustellen.  
  
 [Web\- und Socketberechtigungen](../../../docs/framework/network-programming/web-and-socket-permissions.md)  
 Beschreibt, wie Codezugriffssicherheit für Anwendungen festlegen, die Internetverbindungen verwenden.

 [Netzwerkprogrammierung in .NET Framework](../../../docs/framework/network-programming/index.md)  
 Stellt die Klassen in <xref:System.Net> und in den <xref:System.Net.Sockets>\-Namespaces vor.
