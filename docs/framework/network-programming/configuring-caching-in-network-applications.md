# Konfigurieren der Zwischenspeicherung in den Netzwerkanwendungen
Um die Zwischenspeicherung zu konfigurieren, müssen Sie eine Cacherichtlinie auf der Anwendung oder der <xref:System.Net.WebRequest> Ebene angeben.  Die folgenden Themen enthalten Codebeispiele, die das Konfigurieren von Anwendungen und von Anforderungen, Zwischenspeichern zu verwenden zeigen.  
  
-   [Gewusst wie: Legen Sie eine ortsbasierte Cache\-Richtlinie für eine Anwendung fest](../../../docs/framework/network-programming/how-to-set-a-location-based-cache-policy-for-an-application.md)  
  
-   [Gewusst wie: Legen Sie die zeitgebundenen Cache\-Richtlinie für eine Anwendung fest](../../../docs/framework/network-programming/how-to-set-the-default-time-based-cache-policy-for-an-application.md)  
  
-   [Gewusst wie: Passen Sie eine zeitbasierte Cache\-Richtlinie an](../../../docs/framework/network-programming/how-to-customize-a-time-based-cache-policy.md)  
  
-   [Gewusst wie: Legen Sie Cache\-Richtlinie für eine Anforderung fest](../../../docs/framework/network-programming/how-to-set-cache-policy-for-a-request.md)  
  
 Sie können Cacherichtlinie mithilfe der Anwendung oder der Computerkonfigurationsdateien auch konfigurieren.  Weitere Informationen finden Sie unter &#124; [\<requestCaching\>\-Element \(Netzwerkeinstellungen\)](../../../docs/framework/configure-apps/file-schema/network/requestcaching-element-network-settings.md).
