# Socketcodebeispiele
Die folgenden Codebeispiele zeigen, wie die <xref:System.Net.Sockets.Socket>\-Klasse als Client verwendet, um an Remote Netzwerkdienste herzustellen und als Server, auf von Links von den Remoteclients zu überwachen.  
  
## In diesem Abschnitt  
 [Synchrone Clientsockets \- Beispiel](../../../docs/framework/network-programming/synchronous-client-socket-example.md)  
 Zeigt, wie ein synchronen <xref:System.Net.Sockets.Socket> Client implementiert, der an einen Server herstellt und die Daten anzeigt, die vom Server zurückgegeben werden.  
  
 [Synchroner Serversocket, Beispiel](../../../docs/framework/network-programming/synchronous-server-socket-example.md)  
 Zeigt, wie ein synchronen <xref:System.Net.Sockets.Socket> Server, der einen akzeptiert, Verbindungen von Clients und von den Echos unterstützen die Daten implementiert, die vom Client empfangen werden.  
  
 [Asynchrone Clientsockets \- Beispiel](../../../docs/framework/network-programming/asynchronous-client-socket-example.md)  
 Zeigt, wie ein asynchroner <xref:System.Net.Sockets.Socket> Client implementiert, der an einen Server herstellt und die Daten anzeigt, die vom Server zurückgegeben werden.  
  
 [Asynchroner Serversocket, Beispiel](../../../docs/framework/network-programming/asynchronous-server-socket-example.md)  
 Zeigt, wie ein asynchroner <xref:System.Net.Sockets.Socket> Server, der einen akzeptiert, Verbindungen von Clients und von den Echos unterstützen die Daten implementiert, die vom Client empfangen werden.
