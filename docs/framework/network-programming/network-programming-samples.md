# <a name="network-programming-samples"></a>Beispiele zur Netzwerkprogrammierung
Dieser Abschnitt enthält Beschreibungen und Links zu herunterladbaren Netzwerkprogrammierungs-Beispielen, die Klassen in <xref:System.Net>, <xref:System.Net.Cache>, <xref:System.Net.Configuration>, <xref:System.Net.Mail>, <xref:System.Net.Mime>, <xref:System.Net.NetworkInformation>, <xref:System.Net.PeerToPeer>, <xref:System.Net.Security>, <xref:System.Net.Sockets> und in den verwandten Namespaces verwenden.  
  
## <a name="in-this-section"></a>In diesem Abschnitt  
 [Beispiel zur Technologie der Downloadstatusanzeige](http://go.microsoft.com/fwlink/?LinkID=179556)  
 Zeigt, wie der Fortschritt eines Dateidownloads angezeigt wird.  
  
 [Beispiel zur FTP-Clienttechnologie](http://go.microsoft.com/fwlink/?LinkID=179557)  
 Zeigt, wie Dateien in und aus einem FTP-Server hoch-und heruntergeladen werden.  
  
 [Beispiel zur HttpListener-Technologie](http://go.microsoft.com/fwlink/?LinkID=179558)  
 Zeigt, wie HTTP-Anforderungen aus einer Anwendung heraus verarbeitet werden.  
  
 [Beispiel zur HttpListener ASPX-Hostanwendung](http://go.microsoft.com/fwlink/?LinkID=179560)  
 Veranschaulicht, wie die Funktionen der <xref:System.Net.HttpListener>-Klasse verwendet werden, um einen HTTP-Server zu erstellen, der Aufrufe an eine gehostete ASP.NET-Anwendung weiterleitet.  
  
 [Beispiel zur Mailer-Technologie](http://go.microsoft.com/fwlink/?LinkID=179561)  
 Zeigt, wie E-Mail-Nachrichten aus einer Clientanwendung gesendet werden.  
  
 [Beispiel zur Technologie des NetStat-Tools](http://go.microsoft.com/fwlink/?LinkID=179562)  
 Veranschaulicht das NCLNetStat-Netzwerk-Informationstool.  
  
 [Beispiel zur Netzwerk-Informationstechnologie](http://go.microsoft.com/fwlink/?LinkID=179564)  
 Zeigt, wie Netzwerkinformationen überwacht und angezeigt werden.  
  
 [Beispiel für die Technologie eines Ping-Clients](http://go.microsoft.com/fwlink/?LinkID=179565)  
 Veranschaulicht eine Clientanwendung, die einen Remotehost Pingen kann.  
  
 [Beispiele für die Web-Clienttechnologie](http://go.microsoft.com/fwlink/?LinkID=179566)  
 Veranschaulicht, wie allgemeine Vorgänge, z.B. das Hoch- oder Herunterladen von Dateien oder Daten ausgeführt werden.  
  
 [Beispiel für sichere Streams](http://go.microsoft.com/fwlink/?LinkID=179567)  
 Zeigt, wie ein sicherer Stream dazu verwendet wird, um die Kommunikation zwischen einem Client und einem Server zu ermöglichen.  
  
 [Beispiel für Socket-IPv6](http://go.microsoft.com/fwlink/?LinkID=179568)  
 Veranschaulicht, wie Sockets verwendet werden, wenn IPv6 aktiviert ist.  
  
 [Beispiel zur FTP-Explorer-Technologie](http://go.microsoft.com/fwlink/?LinkID=179569)  
 Veranschaulicht, wie der Inhalt eines FTP-Servers aufgelistet wird.  
  
 [Beispiel zur Technologie der Socketleistung](http://go.microsoft.com/fwlink/?LinkID=179570)  
 Zeigt, wie Erweiterungen der <xref:System.Net.Sockets.Socket>-Klasse verwendet werden, um eine Serveranwendung zu erstellen, die asynchrone Netzwerk-E/A verwendet, um die höchste Leistung zu erzielen.  
  
 [Beispiele für die PeerToPeer-Technologie](http://go.microsoft.com/fwlink/?LinkID=179571)  
 Zeigt, wie die neuen Klassen im <xref:System.Net.PeerToPeer>-Namespace verwendet werden, um einen Peernamen zu registrieren und zu veröffentlichen und anschließend einen Peernamen aufzulösen.
