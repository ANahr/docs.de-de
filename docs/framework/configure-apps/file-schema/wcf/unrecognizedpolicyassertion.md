# &lt;unrecognizedPolicyAssertion&gt;
Stellt ein Bindungselement dar, das die Richtlinienassertion angibt.  Dieses Element verfügt über kein Attribut und ist als leerer Schalter vorhanden.  
  
## Syntax  
  
```  
  
<unrecognizedPolicyAssertion/>  
```  
  
## Typ  
 `Type`  
  
## Attribute und Elemente  
 In den folgenden Abschnitten werden Attribute sowie untergeordnete und übergeordnete Elemente beschrieben.  
  
### Attribute  
 Keine  
  
### Untergeordnete Elemente  
 Keine  
  
### Übergeordnete Elemente  
  
|Element|Beschreibung|  
|-------------|------------------|  
|[\<Bindung\>](../../../../../docs/framework/misc/binding.md)|Definiert alle Bindungsmöglichkeiten der benutzerdefinierten Bindung.|
