### <a name="to-create-a-new-exception-class-and-map-it-to-an-hresult"></a><span data-ttu-id="12c54-114">Erstellen einer neuen Ausnahmeklasse und Zuweisen zu einem HRESULT</span><span class="sxs-lookup"><span data-stu-id="12c54-114">To create a new exception class and map it to an HRESULT</span></span>  
  
1.  <span data-ttu-id="12c54-115">Verwenden Sie den folgenden Code zum Erstellen einer neuen Ausnahmeklasse namens `NoAccessException`, und ordnen Sie sie dem HRESULT `E_ACCESSDENIED` zu.</span><span class="sxs-lookup"><span data-stu-id="12c54-115">Use the following code to create a new exception class called `NoAccessException` and map it to the HRESULT `E_ACCESSDENIED`.</span></span>  
  
    ```cpp

    {  
        NoAccessException () {  
        HResult = E_ACCESSDENIED;   
    }  
    }  
    CMyClass::MethodThatThrows  
    {  
    throw new NoAccessException();  
    }  
    ```  
  
 <span data-ttu-id="12c54-116">Es wird möglicherweise ein Programm (in einer beliebigen Programmiersprache) auftreten, das verwalteten und nicht verwalteten Code gleichzeitig verwendet.</span><span class="sxs-lookup"><span data-stu-id="12c54-116">You might encounter a program (in any programming language) that uses both managed and unmanaged code at the same time.</span></span> <span data-ttu-id="12c54-117">Beispielsweise verwendet der benutzerdefinierte Marshaller im folgenden Codebeispiel die Methode **Marshal.ThrowExceptionForHR(Int HResult)**, um eine Ausnahme mit einem bestimmten HRESULT-Wert auszulösen.</span><span class="sxs-lookup"><span data-stu-id="12c54-117">For example, the custom marshaler in the following code example uses the **Marshal.ThrowExceptionForHR(int HResult)** method to throw an exception with a specific HRESULT value.</span></span> <span data-ttu-id="12c54-118">Die Methode sucht das HRESULT und generiert den entsprechenden Ausnahmetyp.</span><span class="sxs-lookup"><span data-stu-id="12c54-118">The method looks up the HRESULT and generates the appropriate exception type.</span></span> <span data-ttu-id="12c54-119">Im folgenden Codefragment generiert HRESULT z.B. **ArgumentException**.</span><span class="sxs-lookup"><span data-stu-id="12c54-119">For example, the HRESULT in the following code fragment generates **ArgumentException**.</span></span>
