# ETW Events in the .NET Framework
Die Ereignisablaufverfolgung für Windows \(ETW\) ist ein leistungsstarkes, geringen Aufwand verursachendes und skalierbares Ablaufverfolgungssystem, das in Windows\-Betriebssystemen bereitgestellt wird.  Sie ergänzt die Profilerstellungs\- und Debuggingunterstützung in .NET Framework und kann bei der Problembehandlung in einer Vielzahl von Szenarien verwendet werden.  Allgemeine Informationen zu ETW finden Sie unter [Bessere Debuggen und Leistungsoptimierung mit ETW](http://go.microsoft.com/fwlink/?LinkID=161142) in der MSDN Library.  
  
 In .NET Framework ist die ETW\-Ereignisablaufverfolgung für die Common Language Runtime \(CLR\), die [Task Parallel Library](../../../docs/standard/parallel-programming/task-parallel-library-tpl.md) und [Parallel LINQ \(PLINQ\)](../../../docs/standard/parallel-programming/parallel-linq-plinq.md) verfügbar.

 Beschreibt, wie ein Profil für parallelen Anwendungscode erstellt wird.
