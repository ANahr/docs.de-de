# <a name="icordebugmdagetxml-method"></a>ICorDebugMDA::GetXML-Methode
Ruft den vollständigen XML-Stream der Assistent für verwaltetes Debuggen (MDA) dargestellte zugeordneten [ICorDebugMDA](../../../../docs/framework/unmanaged-api/debugging/icordebugmda-interface.md).  
  
## <a name="syntax"></a>Syntax  
  
```  
HRESULT GetXML (  
    [in] ULONG32   cchName,  
    [out] ULONG32  *pcchName,  
    [out, size_is(cchName), length_is(*pcchName)]  
        WCHAR      szName[]  
);  
```  
  
#### <a name="parameters"></a>Parameter  
 `cchName`  
 [in] Die Größe des `szName`-Arrays.  
  
 `pcchName`  
 [out] Ein Zeiger auf die Länge des XML-Streams.  
  
 `szName`  
 [out] Ein Array, in dem die XML-Stream gespeichert werden soll. Das Array kann leer sein.  
  
## <a name="remarks"></a>Hinweise  
 Die `GetXML` Methode kann abhängig von der Größe des zugeordneten XML-Streams die Leistung beeinträchtigen.
