# <a name="icordebugtypeenum-interface1"></a>ICorDebugTypeEnum Schnittstelle1
Implementiert "ICorDebugEnum"-Methoden und listet "ICorDebugType"-Arrays.  
  
## <a name="methods"></a>Methoden  
  
|Methode|Beschreibung|  
|------------|-----------------|  
|[Next-Methode](../../../../docs/framework/unmanaged-api/debugging/icordebugtypeenum-next-method.md)|Ruft die angegebene Anzahl von `ICorDebugType` Instanzen aus der Enumeration, die an der aktuellen Position ab.|  
  
## <a name="remarks"></a>Hinweise  
  
> [!NOTE]
>  Diese Schnittstelle kann weder computerübergreifend noch prozessübergreifend remote aufgerufen werden.
