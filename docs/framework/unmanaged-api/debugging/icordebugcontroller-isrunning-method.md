# <a name="icordebugcontrollerisrunning-method"></a>ICorDebugController::IsRunning-Methode
Ruft einen Wert, der angibt, ob die Threads im Prozess zurzeit frei ausgeführt werden.  
  
## <a name="syntax"></a>Syntax  
  
```  
HRESULT IsRunning (  
    [out] BOOL *pbRunning  
);  
```  
  
#### <a name="parameters"></a>Parameter  
 `pbRunning`  
 [out] Ein Zeiger auf einen Wert, der `true` , wenn die Threads im Prozess, frei ist, andernfalls ausgeführt werden `false`.
