# <a name="icordebugprocess7-interface"></a>ICorDebugProcess7-Schnittstelle
[Wird nur in .NET Framework 4.5.2 und neueren Versionen unterst√ºtzt]  
  
 Bietet eine Methode, die den Debugger so konfiguriert, dass speicherinterne Metadatenaktualisierungen im Zielprozess behandelt werden.  
  
## <a name="methods"></a>Methoden  
  
|Methode|Beschreibung|  
|------------|-----------------|  
|[SetWriteableMetadataUpdateMode-Methode](../../../../docs/framework/unmanaged-api/debugging/icordebugprocess7-setwriteablemetadataupdatemode-method.md)|Legt einen Wert fest, der bestimmt, wie der Debugger speicherinterne Aktualisierungen von Metadaten innerhalb des Zielprozesses behandelt.|
