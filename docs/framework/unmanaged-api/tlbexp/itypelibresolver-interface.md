# <a name="itypelibresolver-interface"></a>ITypeLibResolver-Schnittstelle
Stellt die [ResolveTypeLib-Methode](../../../../docs/framework/unmanaged-api/tlbexp/resolvetypelib-method.md), der den Dateipfad einer Typbibliothek auflöst.  
  
## <a name="methods"></a>Methoden  
  
|Methode|Beschreibung|  
|------------|-----------------|  
|[ResolveTypeLib-Methode](../../../../docs/framework/unmanaged-api/tlbexp/resolvetypelib-method.md)|Löst den einfachen Namen einer Typbibliothek durch den vollqualifizierten Pfad zurückgeben.|

 **Header:** TlbRef.idl, TlbRef.h  
  
 **Bibliothek:** TlbRef.lib  
  
 **.NET Framework-Version:** 4, 3.5, 3.0 und 2.0
