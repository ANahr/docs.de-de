# <a name="corprfgcreason-enumeration"></a>COR_PRF_GC_REASON-Enumeration
Zeigt den Grund, weshalb die Garbage Collection stattfindet.  
  
## <a name="syntax"></a>Syntax  
  
```  
typedef enum {  
    COR_PRF_GC_INDUCED = 1,  
    COR_PRF_GC_OTHER = 0  
} COR_PRF_GC_REASON;  
```
