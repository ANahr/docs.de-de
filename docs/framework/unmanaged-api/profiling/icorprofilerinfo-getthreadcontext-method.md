# <a name="icorprofilerinfogetthreadcontext-method"></a>ICorProfilerInfo::GetThreadContext-Methode
Ruft die Kontextidentität, die derzeit zugewiesen ist, mit dem angegebenen Thread ab.  
  
## <a name="syntax"></a>Syntax  
  
```  
HRESULT GetThreadContext(  
    [in]  ThreadID  threadId,  
    [out] ContextID *pContextId);  
```  
  
#### <a name="parameters"></a>Parameter  
 `threadId`  
 [in] Die ID des Threads.  
  
 `pContextId`  
 [out] Ein Zeiger auf die Kontext-ID, die derzeit mit dem angegebenen Thread zugewiesen ist. Wenn der Thread kein Kontext derzeit zugeordnet verfügt, wird diese Funktion CORPROF_E_DATAINCOMPLETE zurück.
