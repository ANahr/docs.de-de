# <a name="isymunmanageddocumentwriter-interface"></a>ISymUnmanagedDocumentWriter-Schnittstelle
Stellt Methoden zum Schreiben in ein Dokument bereit, auf das ein Symbolspeicher verweist.  
  
## <a name="methods"></a>Methoden  
  
|Methode|Beschreibung|  
|------------|-----------------|  
|[SetCheckSum-Methode](../../../../docs/framework/unmanaged-api/diagnostics/isymunmanageddocumentwriter-setchecksum-method.md)|Legt Prüfsummeninformationen fest.|  
|[SetSource-Methode](../../../../docs/framework/unmanaged-api/diagnostics/isymunmanageddocumentwriter-setsource-method.md)|Legt die eingebettete Quelle für ein Dokument, das geschrieben wird.|  
  
## <a name="requirements"></a>Anforderungen  
 **Header:** CorSym.idl, CorSym.h
