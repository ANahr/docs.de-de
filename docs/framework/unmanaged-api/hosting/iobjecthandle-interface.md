|Methode|Beschreibung|  
|------------|-----------------|  
|[Unwrap-Methode](../../../../docs/framework/unmanaged-api/hosting/iobjecthandle-unwrap-method.md)|Eine Marshal-by-Value-Objekt aus der Dereferenzierung entpackt.|
