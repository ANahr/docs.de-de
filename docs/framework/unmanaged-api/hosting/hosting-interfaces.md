# <a name="hosting-interfaces"></a>Hostingschnittstellen
Dieser Abschnitt beschreibt die Schnittstellen, die nicht verwaltete Hosts können Sie die common Language Runtime (CLR) in ihre Anwendungen integrieren.  
  
 Die .NET Framework Version 2.0 Hostschnittstellen der .NET Framework Version 1.0 und 1.1-Schnittstellen. Es gibt bedeutende Unterschiede zwischen den beiden Sätzen von Schnittstellen. Mischen sie kann unerwartetes Verhalten verursachen und wird nicht empfohlen.  
  
 .NET Framework-Versionen 3.0 und 3.5 verwenden die .NET Framework Version 2.0-Hostingschnittstellen und Hostingfunktionen führen.  
  
 Die [!INCLUDE[net_v40_long](../../../../includes/net-v40-long-md.md)] Hostschnittstellen das .NET Framework 2.0-Schnittstellen.
