# <a name="iclrcontrol-interface"></a>ICLRControl-Schnittstelle
Stellt Methoden, die zum Zulassen von eines Hosts zum Abrufen der Verweise auf und Konfigurieren der Aspekte, die common Language Runtime (CLR) bereit.  
  
## <a name="methods"></a>Methoden  
  
|Methode|Beschreibung|  
|------------|-----------------|  
|[GetCLRManager-Methode](../../../../docs/framework/unmanaged-api/hosting/iclrcontrol-getclrmanager-method.md)|Ruft einen Schnittstellenzeiger zu einer Instanz von den Manager-Typen, die der Host so konfigurieren Sie die CLR verwenden kann.|  
|[SetAppDomainManagerType-Methode](../../../../docs/framework/unmanaged-api/hosting/iclrcontrol-setappdomainmanagertype-method.md)|Legt einen von abgeleiteten Typ <xref:System.AppDomainManager> als Typ für Anwendungsdomänen-Manager.|
