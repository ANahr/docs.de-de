# <a name="iassemblynamefinalize-method"></a>IAssemblyName::Finalize-Methode
Dies ermöglicht [IAssemblyName](../../../../docs/framework/unmanaged-api/fusion/iassemblyname-interface.md) -Objekt, Ressourcen freizugeben und andere Bereinigungen durchzuführen, bevor der Destruktor aufgerufen wird.  
  
## <a name="syntax"></a>Syntax  
  
```  
HRESULT Finalize ();  
```
