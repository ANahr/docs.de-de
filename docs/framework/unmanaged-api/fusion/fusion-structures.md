# <a name="fusion-structures"></a>Fusionsstrukturen
In diesem Abschnitt wird beschrieben, die nicht verwalteten Strukturen, die Fusion-API verwendet, wird.  
  
## <a name="in-this-section"></a>In diesem Abschnitt  
 [ASSEMBLY_INFO-Struktur](../../../../docs/framework/unmanaged-api/fusion/assembly-info-structure.md)  
 Enthält Informationen zu einer Assembly, die im globalen Assemblycache registriert ist.  
  
 [FUSION_INSTALL_REFERENCE-Struktur](../../../../docs/framework/unmanaged-api/fusion/fusion-install-reference-structure.md)  
 Stellt einen Verweis, den eine Anwendung eine Assembly ändert, der die Anwendung im globalen Assemblycache installiert wurde.  
  
 [IDENTITY_ATTRIBUTE-Struktur](../../../../docs/framework/unmanaged-api/fusion/identity-attribute-structure.md)  
 Enthält Informationen über Bildattribute Metadaten über eine [IDefinitionIdentity](../../../../docs/framework/unmanaged-api/fusion/idefinitionidentity-interface.md) Instanz.  
  
 [IDENTITY_ATTRIBUTE_BLOB-Struktur](../../../../docs/framework/unmanaged-api/fusion/identity-attribute-blob-structure.md)  
 Enthält die Offsetinformationen für eine verknüpfte [IDENTITY_ATTRIBUTE](../../../../docs/framework/unmanaged-api/fusion/identity-attribute-structure.md) Struktur.
