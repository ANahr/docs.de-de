# <a name="iassemblycacheitem-interface"></a>IAssemblyCacheItem-Schnittstelle
Stellt eine einzelne Assembly im globalen Assemblycache.  
  
## <a name="methods"></a>Methoden  
  
|Methode|Beschreibung|  
|------------|-----------------|  
|[AbortItem-Methode](../../../../docs/framework/unmanaged-api/fusion/iassemblycacheitem-abortitem-method.md)|Ermöglicht es, die Assembly im globalen Assemblycache Bereinigungsvorgänge durchzuführen, bevor er freigegeben wird.|  
|[Commit-Methode](../../../../docs/framework/unmanaged-api/fusion/iassemblycacheitem-commit-method.md)|Führt einen Commit für den Assemblyverweis zwischengespeicherte Arbeitsspeicher.|  
|[CreateStream-Methode](../../../../docs/framework/unmanaged-api/fusion/iassemblycacheitem-createstream-method.md)|Erstellt einen Stream mit dem angegebenen Namen und das Format an.|
