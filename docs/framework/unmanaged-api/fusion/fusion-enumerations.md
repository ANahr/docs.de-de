# <a name="fusion-enumerations"></a>Fusionsenumerationen
In diesem Abschnitt wird beschrieben, die nicht verwalteten Enumerationen, die die Fusion-API verwendet wird.  
  
## <a name="in-this-section"></a>In diesem Abschnitt  
 [ASM_CACHE_FLAGS-Enumeration](../../../../docs/framework/unmanaged-api/fusion/asm-cache-flags-enumeration.md)  
 Gibt die Quelle einer Assembly dargestellte [IAssemblyCacheItem](../../../../docs/framework/unmanaged-api/fusion/iassemblycacheitem-interface.md) im globalen Assemblycache.  
  
 [ASM_CMP_FLAGS-Enumeration](../../../../docs/framework/unmanaged-api/fusion/asm-cmp-flags-enumeration.md)  
 Gibt die Version, Build, Kultur, Signatur und So weiter, der zwei Assemblys, die von verglichen werden soll die [IAssemblyName:: IsEqual](../../../../docs/framework/unmanaged-api/fusion/iassemblyname-isequal-method.md) Methode.  
  
 [ASM_DISPLAY_FLAGS-Enumeration](../../../../docs/framework/unmanaged-api/fusion/asm-display-flags-enumeration.md)  
 Gibt an, die Version, Build, Kultur, Signatur und usw., der die Assembly, deren Anzeigename abgerufen werden soll, indem, die [IAssemblyName:: GetDisplayName](../../../../docs/framework/unmanaged-api/fusion/iassemblyname-getdisplayname-method.md) Methode.  
  
 [ASM_NAME-Enumeration](../../../../docs/framework/unmanaged-api/fusion/asm-name-enumeration.md)  
 Gibt die Version, Build, Kultur, Signatur und So weiter, der die Assembly, deren Eigenschaften abgerufen oder festgelegt werden [IAssemblyName](../../../../docs/framework/unmanaged-api/fusion/iassemblyname-interface.md) Methoden.  
  
 [AssemblyComparisonResult-Enumeration](../../../../docs/framework/unmanaged-api/fusion/assemblycomparisonresult-enumeration.md)  
 Gibt die Gleichwertigkeit von zwei Assemblyidentit√§ten an, durch die [CompareAssemblyIdentity](../../../../docs/framework/unmanaged-api/fusion/compareassemblyidentity-function.md) Funktion.  
  
 [CREATE_ASM_NAME_OBJ_FLAGS-Enumeration](../../../../docs/framework/unmanaged-api/fusion/create-asm-name-obj-flags-enumeration.md)  
 Gibt die Attribute eines ein `IAssemblyName` -Objekt nach der Erstellung durch die [CreateAssemblyNameObject](../../../../docs/framework/unmanaged-api/fusion/createassemblynameobject-function.md) Funktion.
