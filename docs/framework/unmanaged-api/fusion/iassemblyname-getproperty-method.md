# <a name="iassemblynamegetproperty-method"></a>IAssemblyName::GetProperty-Methode
Ruft einen Zeiger auf die Eigenschaft, die vom Bezeichner angegebene Eigenschaft verwiesen wird.  
  
## <a name="syntax"></a>Syntax  
  
```  
HRESULT GetProperty (  
    [in]      DWORD    PropertyId,  
    [out]     LPVOID   pvProperty,  
    [in, out] LPDWORD  pcbProperty  
);  
```  
  
#### <a name="parameters"></a>Parameter  
 `PropertyId`  
 [in] Der eindeutige Bezeichner für die angeforderte Eigenschaft.  
  
 `pvProperty`  
 [out] Die zurückgegebenen Daten.  
  
 `pcbProperty`  
 [in, out] Die Größe in Bytes, der `pvProperty`.
