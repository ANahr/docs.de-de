# UI Automation Specification and Community Promise
Die Benutzeroberflächenautomatisierung stellt programmgesteuerten Zugriff auf die meisten Benutzeroberflächenelemente auf dem Desktop zur Verfügung, sodass Hilfstechnologieprodukte, z. B. die Bildschirmsprachausgabe, Informationen über die Benutzeroberfläche für die Endbenutzer bereitstellen und die Benutzeroberfläche mit anderen Verfahren als Standardeingaben ändern können.
