# .NET Framework 4.6 und 4.5
> [!NOTE]
>  Dieser .NET Framework\-Inhaltssatz enthält Informationen für die .NET Framework\-Versionen 4.5, 4.5.1, 4.5.2, 4.6 und 4.6.1. Weitere Informationen zum Herunterladen von .NET Framework finden Sie unter [Installieren von .NET Framework](../../docs/framework/install/guide-for-developers.md). Eine Liste der neuen Funktionen und Änderungen in .NET Framework 4.5, [!INCLUDE[net_v46](../../includes/net-v46-md.md)] und deren Nebenversionen finden Sie unter [Neuerungen](../../docs/framework/whats-new/index.md). Eine Liste der unterstützten Plattformen finden Sie unter [Systemanforderungen für .NET Framework](../../docs/framework/get-started/system-requirements.md).  
  
 .NET Framework ist eine Entwicklungsplattform zum Erstellen von Apps für Windows, Windows Phone, Windows Server und Microsoft Azure. Es besteht aus der CLR \(Common Language Runtime\) und der .NET Framework\-Klassenbibliothek mit Klassen, Schnittstellen und Werttypen, die eine Vielzahl von Technologien unterstützen. .NET Framework stellt neben einer verwalteten Ausführungsumgebung eine vereinfachte Entwicklung und Bereitstellung sowie die Integration einer Vielfalt von Programmiersprachen, einschließlich Visual Basic und Visual C\#, zur Verfügung.  
  
> [!NOTE]
>  Weitere Informationen zum Herunterladen von .NET Framework finden Sie unter [Installieren von .NET Framework](../../docs/framework/install/guide-for-developers.md).  
  
 Eine allgemeine Einführung in .NET Framework für Benutzer und Entwickler erhalten Sie unter [Erste Schritte](../../docs/framework/get-started/index.md). Eine Einführung in die Architektur und die Hauptfunktionen von .NET Framework finden Sie unter [Übersicht](../../docs/framework/get-started/overview.md).  
  
 Die Dokumentation für .NET Framework enthält eine umfassende Klassenbibliothekreferenz, konzeptionelle Übersichten, schrittweise Anweisungen und Informationen über Beispiele, Compiler und Befehlszeilentools. Die für Sie relevanten Informationen können Sie in der folgenden Liste der Hauptthemenbereiche finden.
