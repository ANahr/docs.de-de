# Erste Schritte mit WCF Data Services
Die Themen in diesem Abschnitt helfen Ihnen beim schnellen Einstieg in [!INCLUDE[ssODataFull](../../../../includes/ssodatafull-md.md)] und erläutern die zugrunde liegenden Technologien, um [!INCLUDE[ssODataShort](../../../../includes/ssodatashort-md.md)]\-Feeds mithilfe von [!INCLUDE[ssAstoria](../../../../includes/ssastoria-md.md)] zu verwenden und verfügbar zu machen.  Dieser Abschnitt enthält sowohl konzeptionelle Inhalte als auch das Lernprogramm [Schnellstart](../../../../docs/framework/data/wcf/quickstart-wcf-data-services.md).  
  
## In diesem Abschnitt  
 In den folgenden Themen wird beschrieben, wie Sie Datendienste mit [!INCLUDE[ssAstoria](../../../../includes/ssastoria-md.md)] erstellen.  
  
 [Verfügbarmachen der Daten als Dienst](../../../../docs/framework/data/wcf/exposing-your-data-as-a-service-wcf-data-services.md)  
 Beschreibt die erforderlichen Schritte zum Erstellen eines Datendiensts mit [!INCLUDE[ssAstoria](../../../../includes/ssastoria-md.md)].  
  
 [Zugreifen auf Datendienstressourcen](../../../../docs/framework/data/wcf/accessing-data-service-resources-wcf-data-services.md)  
 Beschreibt das Arbeiten mit [!INCLUDE[ssODataShort](../../../../includes/ssodatashort-md.md)]\-Feeds.  
  
 [Verwenden eines Datendiensts in einer Clientanwendung](../../../../docs/framework/data/wcf/using-a-data-service-in-a-client-application-wcf-data-services.md)  
 Beschreibt das Arbeiten mit einem [!INCLUDE[ssODataShort](../../../../includes/ssodatashort-md.md)]\-Feed in einer .NET Framework\-Clientanwendung.  
  
 [Schnellstart](../../../../docs/framework/data/wcf/quickstart-wcf-data-services.md)  
 Erläutert, wie ein einfacher [!INCLUDE[ssODataShort](../../../../includes/ssodatashort-md.md)]\-basierter Dienst erstellt wird, der einen auf der Northwind\-Beispieldatenbank basierenden Feed zugänglich macht, und wie darauf zugegriffen wird.  
  
 [Anwendungszenarien](../../../../docs/framework/data/wcf/application-scenarios-wcf-data-services.md)  
 Erläutert die wichtigsten [!INCLUDE[ssODataShort](../../../../includes/ssodatashort-md.md)]\-Szenarien, die von [!INCLUDE[ssAstoria](../../../../includes/ssastoria-md.md)] unterstützt werden.  
  
 [Ressourcen](../../../../docs/framework/data/wcf/wcf-data-services-resources.md)  
 Stellt Links zu [!INCLUDE[ssAstoria](../../../../includes/ssastoria-md.md)]\- und [!INCLUDE[ssODataShort](../../../../includes/ssodatashort-md.md)]\-Ressourcen bereit.
