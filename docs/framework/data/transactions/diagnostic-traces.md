## <a name="enabling-tracing"></a><span data-ttu-id="bdad1-116">Aktivieren der Ablaufverfolgung</span><span class="sxs-lookup"><span data-stu-id="bdad1-116">Enabling Tracing</span></span>  
 <span data-ttu-id="bdad1-117">Um die Ablaufverfolgung während der Transaktionsverarbeitung zu aktivieren, müssen Sie die Konfigurationsdatei der Anwendung bearbeiten.</span><span class="sxs-lookup"><span data-stu-id="bdad1-117">To enable traces during transaction processing, you should edit your application’s configuration file.</span></span> <span data-ttu-id="bdad1-118">Nachfolgend finden Sie ein Beispiel:</span><span class="sxs-lookup"><span data-stu-id="bdad1-118">The following is an example.</span></span>  
  
```xml  
<configuration>  
<system.diagnostics>  
     <sources>  
          <source name="System.Transactions" switchValue="Warning">  
               <listeners>  
                    <add name="tx"   
                     type="System.Diagnostics.XmlWriterTraceListener"   
                     initializeData= "tx.log" />  
               </listeners>  
          </source>  
     </sources>  
</system.diagnostics>  
</configuration>  
```
