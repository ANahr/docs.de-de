# Arbeiten mit Objekten
In [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] können Sie Daten abfragen, einfügen, aktualisieren und löschen, die als typisierte Objekte der Common Language Runtime \(CLR\) ausgedrückt werden, die Instanzen von Entitätstypen sind. Die Entitätstypen stellen die Entitäten dar, die im konzeptionellen Modell definiert sind.  In [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] werden Entitäten und Beziehungen zugeordnet, die in einem konzeptionellen Modell für eine Datenquelle definiert werden.  [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] stellt Funktionen bereit, mit denen die folgenden Schritte durchgeführt werden können: Materialisieren der von der Datenquelle als Objekte zurückgegebenen Daten, Nachverfolgen von Änderungen an den Objekten, Behandeln der Parallelität, Zurückgeben von Objektänderungen an die Datenquelle und Binden von Objekte an Steuerelemente.  
  
 Weitere Informationen zum Arbeiten mit Objekten in der neuesten Entity Framework\-Version finden Sie unter [Arbeiten mit Objekten](http://go.microsoft.com/fwlink/?LinkId=235289).
