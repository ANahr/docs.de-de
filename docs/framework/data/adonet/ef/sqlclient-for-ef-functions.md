# SqlClient f&#252;r Entity Framework-Funktionen
Der .NET Framework\-Datenanbieter für SQL Server \(SqlClient\) für das Entity Framework stellt einen Satz an Funktionen zur Durchführung von mathematischen und Aggregationsberechnungen sowie Funktionen zur Durchführung von `System.DateTime`\- und `string`\-Vorgängen bereit.  Diese Funktionen befinden sich im `SQLServer`\-Namespace.  
  
 Eine Liste von Funktionen, die mit jedem Anbieter funktionieren sollten, finden Sie unter [Kanonische Funktionen](../../../../../docs/framework/data/adonet/ef/language-reference/canonical-functions.md).  
  
 Weitere Informationen zur Zuordnung von kanonischen Funktionen zu SQL Serverfunktionen finden Sie unter [Zuordnen von kanonischen Funktionen \(konzeptionelles Modell\) zu SQL Server\-Funktionen](../../../../../docs/framework/data/adonet/ef/conceptual-model-canonical-to-sql-server-functions-mapping.md).
