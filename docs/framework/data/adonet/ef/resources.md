# <a name="entity-framework-resources"></a>Entity Framework-Ressourcen
Die folgenden externen Ressourcen bieten Informationen und Unterstützung zum Erstellen von Entity Framework-Anwendungen.  
  
 [Data Developer Center](http://go.microsoft.com/fwlink/?LinkId=213876)  
 Zentraler Ort für Datenentwicklung mit Microsoft-Technologien.  
  
 [ADO.NET-Teamblog](http://go.microsoft.com/fwlink/?LinkId=91905)  
 Blog, das Updates für ADO.NET-Funktionen und Diskussionen über dessen Funktionen und Funktionalitäten enthält.  
  
 [Entwurfsblog für Entity Framework](http://go.microsoft.com/fwlink/?LinkId=186888)  
 Entwurfsdiskussionen und Vorschau auf Funktionen für zukünftige Entity Framework-Versionen.  
  
 [Datenplattform wie kann ich? Videos: Entity Framework-Reihe](http://go.microsoft.com/fwlink/?LinkId=124600)  
 Enthält eine Reihe von Videos, die die Erstellung von Anwendungen unter Verwendung verschiedener Entity Framework-Komponenten veranschaulichen.  
  
 [Entity Framework häufig gestellte Fragen](http://go.microsoft.com/fwlink/?LinkID=213877)  
 Blog, das häufig gestellte Fragen über das Entity Framework enthält.
