# XML-Daten in SQL Server
SQL Server macht die Funktionen von SQLXML innerhalb von .NET Framework verfügbar.  Entwickler können Anwendungen schreiben, die aus einer Instanz von SQL Server auf XML\-Daten zugreifen, die Daten in die .NET Framework\-Umgebung importieren, sie verarbeiten und die Updates zurück zu SQL Server senden.  In SQL Server gibt es verschiedene Verwendungsmöglichkeiten für XML\-Daten, z. B. beim Speichern von Daten oder als Parameterwerte beim Abrufen von Daten.  Die **SqlXml**\-Klasse in .NET Framework bietet clientseitige Unterstützung für die Arbeit mit Daten, die in SQL Server in einer XML\-Spalte gespeichert sind.  Weitere Informationen finden Sie unter "SQLXML Managed Classes" in der SQL Server\-Onlinedokumentation.
