# Bearbeiten von Daten in einer DataTable
Nach dem Erstellen einer <xref:System.Data.DataTable> in einem <xref:System.Data.DataSet> können Sie die gleichen Aktionen ausführen, die Sie auch bei der Verwendung einer Tabelle in einer Datenbank ausführen würden.  Sie können Daten in der Tabelle hinzufügen, anzeigen, bearbeiten und löschen, Fehler und Ereignisse überwachen und die Abfragen über die Daten in der Tabelle ausführen.  Beim Bearbeiten von Daten in einer **DataTable** können Sie auch überprüfen, ob die Änderungen stimmen und festlegen, ob die Änderungen programmgesteuert übernommen oder zurückgewiesen werden sollen.
