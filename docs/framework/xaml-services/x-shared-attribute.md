# <a name="xshared-attribute"></a><span data-ttu-id="1aee5-102">x:Shared-Attribut</span><span class="sxs-lookup"><span data-stu-id="1aee5-102">x:Shared Attribute</span></span>
<span data-ttu-id="1aee5-103">Bei Festlegung auf `false`, WPF Abrufen von Ressourcen Verhalten ändert, so, dass Anforderungen für die attributierte Ressource einer neuen Instanz für jede Anforderung erstellen und für alle Anforderungen dieselbe Instanz gemeinsam.</span><span class="sxs-lookup"><span data-stu-id="1aee5-103">When set to `false`, modifies WPF resource-retrieval behavior so that requests for the attributed resource create a new instance for each request instead of sharing the same instance for all requests.</span></span>

```xaml  
</ResourceDictionary>  
```
