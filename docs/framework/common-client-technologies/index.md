# Allgemeine Clienttechnologien in .NET&#160;Framework
In diesem Abschnitt werden die verschiedenen Technologien beschrieben, die Sie in Ihren Clientanwendungen verwenden können.  
  
## In diesem Abschnitt  
 [Manipulations and Inertia](../../../docs/framework/common-client-technologies/manipulations-and-inertia.md)  
 Beschreibt das Verwenden der Prozessorklassen Manipulationen und Trägheit in verschiedenen Benutzeroberflächen\-Frameworks, wie z. B. Microsoft Windows Presentation Foundation \(WPF\) oder Microsoft XNA.  
  
 [Clientanwendungsdienste](../../../docs/framework/common-client-technologies/client-application-services.md)  
 Beschreibt, wie die [!INCLUDE[ajax_current_short](../../../includes/ajax-current-short-md.md)]\-Anmelde\-, \-Rollen\- und \-Profilanwendungsdienste verwendet werden, die in den Microsoft ASP.NET 2.0 AJAX\-Erweiterungen Ihrer Windows\-basierten Anwendungen enthalten sind.
