# <a name="install-the-net-framework-on-windows-10-and-windows-server-2016"></a>Installieren Sie .NET Framework auf Windows 10 und WindowsServer 2016

.NET Framework ist erforderlich, auf viele Anwendungen unter Windows ausgeführt wird. Die Anweisungen in diesem Artikel sollte helfen Sie bei der Installation von .NET Framework-Versionen, die Sie benötigen. Die [.NET Framework 4.7.1](https://www.microsoft.com/download/details.aspx?id=56115&desc=dotnet47) ist die neueste verfügbare Version.

Sie können auf dieser Seite angekommen sind, nach dem Versuch, eine Anwendung ausgeführt werden und ein Dialogfeld wird angezeigt, auf dem Computer, die ähnlich dem folgenden:

![Diese Anwendung konnte nicht gestartet werden](./media/this-application-could-not-be-started.png)

## <a name="net-framework-471"></a>.NET Framework 4.7.1

.NET Framework 4.7.1 ist im Lieferumfang:

* [Update für Windows 10 fallen Ersteller (Version 1709)](https://www.microsoft.com/software-download/windows10)
* [Windows Server 2016 (Version 1709)](https://docs.microsoft.com/windows-server/get-started/get-started-with-1709)

> [!div class="button"]
[.NET Framework 4.7.1 herunterladen](https://www.microsoft.com/net/download/thank-you/net471?utm_source=ms-docs&utm_medium=referral)

Die [.NET Framework 4.7.1](https://www.microsoft.com/download/details.aspx?id=56115&desc=dotnet47) zum Ausführen von Anwendungen für .NET Framework 4.0 über 4.7.1 erstellten verwendet werden können.

Sie können die [.NET Framework 4.7.1](https://www.microsoft.com/en-us/download/details.aspx?id=56115&desc=dotnet47) auf:

* Ersteller-Update für Windows 10 (Version 1703)
* Anniversary-Update für Windows 10 (Version 1607)
* Windows Server 2016

.NET Framework 4.7.1 wird auf nicht unterstützt:

* Windows 10 1507
* Windows 10-1511

Wenn Sie Windows 10 1507 oder 1511 verwenden, und Sie .NET Framework 4.7.1 installieren möchten, müssen Sie zunächst auf eine höhere Version von Windows 10 zu aktualisieren.

## <a name="net-framework-461"></a>.NET Framework 4.6.1

Die [.NET Framework 4.6.1](https://www.microsoft.com/download/details.aspx?id=49981) neueste unterstützte Version von .NET Framework auf Windows 10 1507 und 1511 ist.

.NET Framework 4.6.1 unterstützt apps für .NET Framework 4.0 über 4.6.1 erstellt wurden.

## <a name="net-framework-35"></a>.NET Framework 3,5

Folgen Sie den Anweisungen zum Installieren von [.NET Framework 3.5 unter Windows 10](dotnet-35-windows-10.md).

.NET Framework 3.5 unterstützt apps für .NET Framework 1.0 bis 3.5 erstellt wurden.

## <a name="additional-information"></a>Zusätzliche Informationen

.NET Framework 4.x-Versionen sind direkte Updates früherer Versionen. Dies bedeutet Folgendes:

- Sie können nur eine Version von .NET Framework 4.x auf Ihrem Computer installiert.

- Sie können nicht auf dem Computer eine frühere Version von .NET Framework installieren, wenn bereits eine neuere Version installiert ist.

- 4.x-Versionen von .NET Framework können verwendet werden, zum Ausführen von Anwendungen für .NET Framework 4.0 über diese Version erstellt wurden. .NET Framework 4.7 kann beispielsweise verwendet werden, zum Ausführen von Anwendungen für .NET Framework 4.0 über 4.7 erstellt wurden. Die neueste Version ((.NET Framework 4.7.1) kann verwendet werden, zum Ausführen von Anwendungen, die erstellt werden alle Versionen von .NET Framework 4.0 ab.

Eine Liste aller Versionen von .NET Framework heruntergeladen, finden Sie unter der [.NET Downloads](https://www.microsoft.com/net/download?utm_source=ms-docs&utm_medium=referral) Seite.
