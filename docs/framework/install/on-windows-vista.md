# <a name="install-the-net-framework-on-windows-vista"></a>Installieren von .NET Framework unter Windows Vista

Möglicherweise wird .NET Framework zum Ausführen einer App auf Windows Vista benötigt. Die folgenden Anweisungen helfen Ihnen. Die Anweisungen sind von der benötigten .NET Framework-Version abhängig.

## <a name="net-framework-461-and-later"></a>.NET Framework 4.6.1 und höher

Sie können .NET Framework 4.6.1 oder höher nicht auf Windows Vista installieren. Stattdessen müssen Sie auf eine neuere Windows-Version aktualisieren, um .NET Framework 4.6.1 oder höher verwenden zu können.

.NET Framework 4.6.2 ist im Lieferumfang von [Windows 10 Anniversary Update](https://www.microsoft.com/software-download/windows10) enthalten.

## <a name="net-framework-46"></a>.NET Framework 4.6

Sie können [.NET Framework 4.6](http://go.microsoft.com/fwlink/?LinkID=213834&dotnetdocs) auf Windows Vista installieren.

## <a name="net-framework-35"></a>.NET Framework 3,5

Sie können [.NET Framework 3.5](http://go.microsoft.com/fwlink/?LinkID=213834&dotnetdocs) auf Windows Vista installieren.

.NET Framework 3.5 unterstützt Apps, die für .NET Framework 1.0 bis 3.5 entwickelt wurden.

## <a name="help"></a>Hilfe

Wenn Sie die richtige Version von .NET Framework nicht installieren können, [wenden Sie sich an Microsoft, um Hilfe zu erhalten](mailto:dotnet-install-help@microsoft.com?subject=Help installing .NET Framework).
