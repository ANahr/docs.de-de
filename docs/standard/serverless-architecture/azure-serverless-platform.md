# <a name="azure-serverless-platform"></a>Serverlose Azure-Plattform

Die serverlose Azure-Plattform umfasst Azure Functions, Logic Apps und Event Grid. Diese Dienste zusammenarbeiten und Herstellen einer Verbindung mit unzähligen anderen Ressourcen. Die serverlose Plattform funktioniert alles, was von Datenbanken und Speicher, Analyse und Machine Learning/künstliche Intelligenz.

Sie können auch über die Application Insights, einer serverlosen Plattform für die Erfassung von ablaufverfolgungen zur Diagnose und Telemetrie. Application Insights für Anwendungen aller Art verfügbar sind (Desktop-, Mobile oder Web) sowie serverlose Implementierungen. Die Plattform wird in der folgenden Abbildung veranschaulicht:
