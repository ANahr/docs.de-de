*   <span data-ttu-id="516ef-127">Strukturen</span><span class="sxs-lookup"><span data-stu-id="516ef-127">Structures</span></span>

*   <span data-ttu-id="516ef-129">Schnittstellen</span><span class="sxs-lookup"><span data-stu-id="516ef-129">Interfaces</span></span>

## <a name="common-language-specification"></a><span data-ttu-id="516ef-133">Common Language Specification</span><span class="sxs-lookup"><span data-stu-id="516ef-133">Common Language Specification</span></span>

<span data-ttu-id="516ef-134">Um vollständige Interoperabilitätsszenarien zu aktivieren, müssen alle Objekte, die im Code erstellt werden, sich auf eine gewisse Gemeinsamkeit in den Sprachen verlassen, von denen sie verwendet werden (die ihre _Aufrufer_ sind).</span><span class="sxs-lookup"><span data-stu-id="516ef-134">To enable full interoperability scenarios, all objects that are created in code must rely on some commonality in the languages that are consuming them (are their _callers_).</span></span> <span data-ttu-id="516ef-135">Da es zahlreiche verschiedene Sprachen gibt, legt .NET diese Gemeinsamkeiten in der sogenannten **Common Language Specification** (CLS) fest.</span><span class="sxs-lookup"><span data-stu-id="516ef-135">Since there are numerous different languages, .NET has specified those commonalities in something called the **Common Language Specification** (CLS).</span></span> <span data-ttu-id="516ef-136">Die CLS definiert einen Satz von Funktionen, die viele gängige Anwendungen benötigen.</span><span class="sxs-lookup"><span data-stu-id="516ef-136">CLS defines a set of features that are needed by many common applications.</span></span> <span data-ttu-id="516ef-137">Darüber hinaus bietet sie für jede Sprache, die in .NET implementiert wird, Anweisungen, was diese unterstützen muss.</span><span class="sxs-lookup"><span data-stu-id="516ef-137">It also provides a sort of recipe for any language that is implemented on top of .NET on what it needs to support.</span></span>

<span data-ttu-id="516ef-138">Die CLS ist eine Teilmenge des CTS.</span><span class="sxs-lookup"><span data-stu-id="516ef-138">CLS is a subset of the CTS.</span></span> <span data-ttu-id="516ef-139">Dies bedeutet, dass alle Regeln im CTS auch für die CLS gelten, es sei denn, die CLS-Regeln sind strikter.</span><span class="sxs-lookup"><span data-stu-id="516ef-139">This means that all of the rules in the CTS also apply to the CLS, unless the CLS rules are more strict.</span></span> <span data-ttu-id="516ef-140">Wenn eine Komponente nur gemäß der Regeln in der CLS erstellt wird, d.h., nur die CLS-Funktionen werden in der API verfügbar gemacht, gilt sie als **CLS-kompatibel**.</span><span class="sxs-lookup"><span data-stu-id="516ef-140">If a component is built using only the rules in the CLS, that is, it exposes only the CLS features in its API, it is said to be **CLS-compliant**.</span></span> <span data-ttu-id="516ef-141">Die `<framework-librares>` sind z.B. genau deshalb CLS-kompatibel, weil sie in allen von .NET unterstützten Sprachen übergreifend funktionieren müssen.</span><span class="sxs-lookup"><span data-stu-id="516ef-141">For instance, the `<framework-librares>` are CLS-compliant precisely because they need to work across all of the languages that are supported on .NET.</span></span>
