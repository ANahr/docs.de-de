-   <span data-ttu-id="45e51-121">Hyper-V-Container erweitern die durch Windows Server-Container bereitgestellte Isolierung, indem jeder Container in einem hochgradig optimierten virtuellen Computer ausgeführt wird.</span><span class="sxs-lookup"><span data-stu-id="45e51-121">Hyper-V Containers expand on the isolation provided by Windows Server Containers by running each container in a highly optimized virtual machine.</span></span> <span data-ttu-id="45e51-122">In dieser Konfiguration wird der Kernel des Containerhosts nicht für den Hyper-V-Container freigegeben, was die Isolierung verbessert.</span><span class="sxs-lookup"><span data-stu-id="45e51-122">In this configuration, the kernel of the container host is not shared with the Hyper-V Containers, providing better isolation.</span></span> <span data-ttu-id="45e51-123">Hyper-V-Container lassen als nicht vertrauenswürdig eingestufte und *schädliche Anwendungen mit mehreren Mandanten* auf demselben Host zu.</span><span class="sxs-lookup"><span data-stu-id="45e51-123">Hyper-V Containers allow untrusted and *hostile multi-tenant* applications to run on the same host.</span></span> <span data-ttu-id="45e51-124">Hyper-V-Container sind, was die Startzeit und die Dichte betrifft, etwas weniger effizient als Windows Server-Container.</span><span class="sxs-lookup"><span data-stu-id="45e51-124">Hyper-V Containers have a bit less efficiency in startup times and density than Windows Server Containers.</span></span>

<span data-ttu-id="45e51-125">Die Images für diese Container werden genauso erstellt und funktionieren auch so.</span><span class="sxs-lookup"><span data-stu-id="45e51-125">The images for these containers are created and function the same way.</span></span> <span data-ttu-id="45e51-126">Der Unterschied besteht darin, wie der Container erstellt wird.</span><span class="sxs-lookup"><span data-stu-id="45e51-126">They differ in how the container is created.</span></span> <span data-ttu-id="45e51-127">Weitere Informationen finden Sie unter [Windows-Container](https://msdn.microsoft.com/en-us/virtualization/windowscontainers/about/about_overview).</span><span class="sxs-lookup"><span data-stu-id="45e51-127">For details, see [Hyper-V Containers](https://msdn.microsoft.com/en-us/virtualization/windowscontainers/about/about_overview).</span></span>
