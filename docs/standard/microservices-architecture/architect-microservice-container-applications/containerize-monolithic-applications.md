## <a name="deploying-a-monolithic-application-as-a-container"></a><span data-ttu-id="94b32-142">Bereitstellen einer monolithischen-Anwendung als container</span><span class="sxs-lookup"><span data-stu-id="94b32-142">Deploying a monolithic application as a container</span></span>
