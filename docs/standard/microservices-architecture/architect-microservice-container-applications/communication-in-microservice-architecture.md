## <a name="communication-types"></a><span data-ttu-id="0af9e-124">Kommunikationstypen</span><span class="sxs-lookup"><span data-stu-id="0af9e-124">Communication types</span></span>

<span data-ttu-id="0af9e-125">Clients und Dienste können über viele verschiedene Kommunikationstypen miteinander kommunizieren, von denen alle unterschiedliche Szenarios und Ziele zum Gegenstand haben.</span><span class="sxs-lookup"><span data-stu-id="0af9e-125">Client and services can communicate through many different types of communication, each one targeting a different scenario and goals.</span></span> <span data-ttu-id="0af9e-126">Diese Kommunikationstypen können zunächst in zwei Achsen klassifiziert werden.</span><span class="sxs-lookup"><span data-stu-id="0af9e-126">Initially, those types of communications can be classified in two axes.</span></span>
