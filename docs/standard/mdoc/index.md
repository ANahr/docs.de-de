# <a name="building-documentation-with-mdoc"></a>Erstellen von Dokumentation mit mdoc

mdoc ist ein Dokumentationserstellungssystem für verwaltete Referenzdokumentation. Durch Reflexion Ihrer Assemblys erstellt mdoc das Basisformat Ihrer verwalteten Referenzdokumentation und ergänzt diese um Autoreninhalte.
