# Parallel Diagnostic Tools
[!INCLUDE[vs_dev10_ext](../../../includes/vs-dev10-ext-md.md)] bietet umfangreiche Unterstützung für das Debuggen und die Profilerstellung in Multithreadanwendungen.  
  
## Debuggen  
 Im Visual Studio Debugger stehen neue Fenster zum Debuggen von parallelen Anwendungen zur Verfügung.  Weitere Informationen finden Sie unter den folgenden Themen:  
  
-   [Verwenden des Fensters "Parallele Stapel"](../Topic/Using%20the%20Parallel%20Stacks%20Window.md)  
  
-   [Verwenden des Fensters "Aufgaben"](../Topic/Using%20the%20Tasks%20Window.md)  
  
-   [Exemplarische Vorgehensweise: Debuggen einer parallelen Anwendung](../Topic/Walkthrough:%20Debugging%20a%20Parallel%20Application.md).  
  
## Profilerstellung  
 Mithilfe der Berichtansichten der Parallelitätsschnellansicht können Sie die Interaktion der Threads in einem parallelen Programm miteinander und mit Threads von anderen Prozessen im System visualisieren.  Weitere Informationen finden Sie unter [Parallelitätsschnellansicht](../Topic/Concurrency%20Visualizer.md).
