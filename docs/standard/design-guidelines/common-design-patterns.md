# Allgemeine Entwurfsmuster
Es sind zahlreiche Bücher Softwaremuster, Mustersprachen und Antimuster, die sehr weit gefasste Gegenstand Muster zu behandeln. Dieses Kapitel stellt daher Richtlinien und Diskussionen im Zusammenhang mit der eine sehr begrenzte Anzahl von Mustern, die häufig in der Entwicklung von .NET Framework\-APIs verwendet werden.
