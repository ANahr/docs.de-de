 <span data-ttu-id="e2c18-104">Die <xref:System.Xml.XmlDocument>-Klasse implementiert die W3C-Empfehlungen "Document Object Model Level 1" und "Document Object Model (DOM) Level 2 Core Specification".</span><span class="sxs-lookup"><span data-stu-id="e2c18-104">The <xref:System.Xml.XmlDocument> class implements the W3C document object model (DOM) level 1 core and the core DOM level 2 recommendations.</span></span> <span data-ttu-id="e2c18-105">DOM ist eine Strukturdarstellung eines XML-Dokuments im Arbeitsspeicher (Cache).</span><span class="sxs-lookup"><span data-stu-id="e2c18-105">The DOM is an in-memory (cache) tree representation of an XML document.</span></span> <span data-ttu-id="e2c18-106">Mit dem <xref:System.Xml.XmlDocument> und seinen verwandten Klassen können Sie XML-Dokumente erstellen, Daten laden und auf diese zugreifen, Daten ändern und Änderungen speichern.</span><span class="sxs-lookup"><span data-stu-id="e2c18-106">With the <xref:System.Xml.XmlDocument> and its related classes, you can construct XML documents, load and access data, modify data, and save changes.</span></span>  
  
 <span data-ttu-id="e2c18-107">Die <xref:System.Xml.XPath.XPathDocument>-Klasse ist ein schreibgeschützter Datenspeicher im Arbeitsspeicher, der auf dem XPath-Datenmodell basiert.</span><span class="sxs-lookup"><span data-stu-id="e2c18-107">The <xref:System.Xml.XPath.XPathDocument> class is a read-only, in-memory data store that is based on the XPath data model.</span></span> <span data-ttu-id="e2c18-108">Die <xref:System.Xml.XPath.XPathNavigator>-Klasse bietet Optionen zur Bearbeitung und zum Navigieren mit einem Cursormodell in XML-Dokumenten, die in der schreibgeschützten <xref:System.Xml.XPath.XPathDocument>-Klasse oder in der <xref:System.Xml.XmlDocument>-Klasse enthalten sind.</span><span class="sxs-lookup"><span data-stu-id="e2c18-108">The <xref:System.Xml.XPath.XPathNavigator> class offers several editing options and navigation capabilities using a cursor model over XML documents contained in the read-only <xref:System.Xml.XPath.XPathDocument> class as well as the <xref:System.Xml.XmlDocument> class.</span></span>  
  
 <span data-ttu-id="e2c18-109">[LINQ to XML](https://msdn.microsoft.com/library/f0fe21e9-ee43-4a55-b91a-0800e5782c13) ist das neue Modell in .NET Framework 3.5 zum Verarbeiten von XML-Daten.</span><span class="sxs-lookup"><span data-stu-id="e2c18-109">[LINQ to XML](https://msdn.microsoft.com/library/f0fe21e9-ee43-4a55-b91a-0800e5782c13) is the new model in the .NET Framework version 3.5 for processing XML data.</span></span> <span data-ttu-id="e2c18-110">Hierbei handelt es sich um ein In-Memory-Modell, das auf [LINQ (Language Integrated Query)](https://msdn.microsoft.com/library/a73c4aec-5d15-4e98-b962-1274021ea93d) basiert.</span><span class="sxs-lookup"><span data-stu-id="e2c18-110">It is an in-memory model that leverages [LINQ (Language-Integrated Query)](https://msdn.microsoft.com/library/a73c4aec-5d15-4e98-b962-1274021ea93d).</span></span> <span data-ttu-id="e2c18-111">LINQ erweitert die Sprachsyntax von C# und Visual Basic um neue Abfragefunktionen.</span><span class="sxs-lookup"><span data-stu-id="e2c18-111">LINQ extends the language syntax of C# and Visual Basic to provide new query capabilities.</span></span>
