# <a name="inserting-nodes-into-an-xml-document"></a>Einfügen von Knoten in ein XML-Dokument
Sie können das Dokument durch Einfügen von neuen Knoten ändern. Die Knoten können neu erstellt werden, im gleichen Dokument vorhanden sein oder aus einem anderen Dokument importiert werden. Weitere Informationen zum Erstellen neuer Knoten finden Sie unter [Erstellen von neuen Knoten im DOKUMENTOBJEKTMODELL](../../../../docs/standard/data/xml/create-new-nodes-in-the-dom.md). Weitere Informationen zum Kopieren von vorhandener Knoten finden Sie unter [Kopieren bestehender Knoten](../../../../docs/standard/data/xml/copy-existing-nodes.md), [Kopieren bestehender Knoten aus einem Dokument in eine andere](../../../../docs/standard/data/xml/copying-existing-nodes-from-one-document-to-another.md), oder [Kopieren von Dokumentfragmenten](../../../../docs/standard/data/xml/copying-document-fragments.md).
