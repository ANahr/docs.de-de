# <a name="copying-document-fragments"></a>Kopieren von Dokumentfragmenten
Sie erstellen eine **XmlDocumentFragment** Knoten, und klicken Sie dann den Knoten hinzufügen. Wenn die **XmlDocumentFragment** eingefügt wird die **InsertNode** -Methode, die **XmlDocumentFragment** -Knoten nicht kopiert, aber seine untergeordneten Knoten in der XML-Code eingefügt werden Dokumentobjektmodell (DOM).
