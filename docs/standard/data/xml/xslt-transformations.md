# XSLT-Transformationen
Mit XSLT \(Extensible Stylesheet Language Transformation\) können Sie den Inhalt eines XML\-Quelldokuments in ein anderes Dokument mit anderem Format oder einer anderen Struktur transformieren.  Mit XSLT können Sie beispielsweise XML in HTML für eine Website transformieren oder in ein Dokument, das nur die für eine Anwendung erforderlichen Felder enthält.  Dieser Transformationsprozess wird durch die [Empfehlung der W3C XSL Transformations \(XSLT\) Version 1.0](http://go.microsoft.com/fwlink/?LinkID=49919) angegeben.  
  
 Die <xref:System.Xml.Xsl.XslCompiledTransform>\-Klasse ist der XSLT\-Prozessor in .NET Framework.  Die <xref:System.Xml.Xsl.XslCompiledTransform>\-Klasse unterstützt die W3C\-Empfehlung zu XSLT 1.0.  
  
> [!NOTE]
>  Die <xref:System.Xml.Xsl.XslTransform>\-Klasse ist in .NET Framework 2.0 veraltet.  Die <xref:System.Xml.Xsl.XslCompiledTransform>\-Klasse ist eine neue Implementierung des XSLT\-Moduls.  Sie enthält Leistungsverbesserungen und neue Sicherheitsfunktionen.  Es wird empfohlen, XSLT\-Anwendungen mit der <xref:System.Xml.Xsl.XslCompiledTransform>\-Klasse zu erstellen.
