Visual Basic Power Packs-Steuerelemente sind zusätzliche Windows Forms-Steuerelemente. Sie sind nicht in Visual Studio enthalten.  Können Sie [herunterladen](http://go.microsoft.com/fwlink/?LinkId=321343), aber wie sie sind angeboten – ist, ohne Unterstützung.
