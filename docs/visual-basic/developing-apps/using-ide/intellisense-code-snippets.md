# <a name="visual-basic-intellisense-code-snippets"></a>Visual Basic IntelliSense-Codeausschnitte
[!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]enthält eine Codebibliothek mit mehreren hundert Codesegmenten von Code, dem IntelliSense-Codeausschnitte, die in der Anwendung eingefügt werden können. Jeder Ausschnitt führt eine vollständige Programmieraufgabe z. B. Erstellen einer benutzerdefinierten Ausnahme, das Senden einer e-Mail-Nachricht oder das Zeichnen eines Kreises. Sie können einen Ausschnitt in den Quellcode mit wenigen Mausklicks einfügen. Sie können auch eigene Ausschnitte erstellen, die entsprechend dem Bedarf Ihrer Business, sie der Bibliothek hinzufügen und anschließend bei Bedarf verwenden. Weitere Informationen finden Sie unter [Codeausschnitte](https://docs.microsoft.com/visualstudio/ide/code-snippets).  
  
## <a name="a-productivity-tool-for-visual-basic-users"></a>Ein Produktivitätstool für Visual Basic-Benutzer  
 Diese Codeausschnitte erhöhen die Produktivität, da Sie weniger Zeit für Suche nach Codebeispielen benötigen, reduziert den Zeitaufwand für eine unbekannte Funktion verwenden und durch die Unterstützung der Wiederverwendung von Code.  
  
 Die Codebibliothek unterstützt die folgenden Funktionen:  
  
-   Legen Sie die Codeausschnitte in die **Code-Editor**.  
  
-   Erstellen Sie neue Aufgaben, die Sie in Ihren Projekten wiederverwenden können.  
  
-   Erstellen Sie neue Vorgänge für Ihre Arbeitsgruppe und Ihren Kollegen freigeben.  
  
-   Bearbeiten Sie die Aufgaben.  
  
-   Downloaden Sie weitere Aufgaben von Drittanbietern.  
  
 Diese Codeblöcke stehen überall in [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs_md.md)]:  
  
-   Sie können sie über das Kontextmenü des Hinzufügen der **Code-Editor**.  
  
-   Sie können die XML-Codedateien aus ziehen **Datei-Explorer** auf die Quellcodedatei.
