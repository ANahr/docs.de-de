# <a name="using-the-visual-basic-development-environment"></a>Verwenden der Visual Basic-Entwicklungsumgebung
Zusätzlich zu den wichtigsten Features der Benutzeroberfläche, die Visual Studio bereitstellt, fügt der Visual Basic-IDE (IDE), Produktivität und einfache Bedienung helfen.
