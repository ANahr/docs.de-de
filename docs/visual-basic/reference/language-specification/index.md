# <a name="visual-basic-language-specification"></a>Visual Basic-Sprachspezifikation
Die [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)]-Programmiersprachenspezifikation enthält verbindliche Antworten auf alle Fragen zu Grammatik und Syntax von [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)]. Sie enthält ausführliche Informationen zu allen Aspekten der Programmiersprache, darunter auch viele Punkte, die in der Referenzdokumentation zu [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] nicht behandelt werden.  
  
 Die Spezifikation ist im [Microsoft Download Center](http://go.microsoft.com/fwlink/?LinkId=188623) verfügbar.  
  
Diese Site enthält die [VB 11-Spezifikation](../../../../_vblang/spec/introduction.md). Diese wird aus den Markdowndateien erstellt, die im [dotnet/vblang-GitHub-Repository](https://github.com/dotnet/vblang/blob/master/spec/README.md) enthalten sind.

Probleme mit der Entwurfsspezifikation sollten im Repository [dotnet/vblang](https://github.com/dotnet/vblang/issues) erstellt werden. Wenn Sie daran interessiert sind, die Fehler zu beheben, die Sie finden, können Sie einen [Pull Request](https://github.com/dotnet/vblang/pulls) an dasselbe Repository übermitteln.
