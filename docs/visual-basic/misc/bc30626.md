# <a name="39namespace39-statement-must-end-with-a-matching-39end-namespace39"></a>&#39;Namespace&#39; Anweisung enden mit einem übereinstimmenden &#39;End Namespace&#39;
`Namespace` -Blöcke müssen mit einer `End Namespace` -Anweisung enden.  
  
 **Fehler-ID:** BC30626  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Fügen Sie eine `End Namespace` -Anweisung ans Ende des `Namespace` -Blocks hinzu.
