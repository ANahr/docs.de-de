# <a name="members-in-a-structure-cannot-be-declared-39ltspecifiergt39"></a>Member in einer Struktur nicht deklariert werden &#39; &lt;Spezifizierer&gt;&#39;
Sie haben einen Spezifizierer verwendet, der für einen Member innerhalb einer `Structure` -Anweisung ungültig ist.  
  
 **Fehler-ID:** BC30435  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie den Spezifizierer.
