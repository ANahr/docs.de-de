# <a name="39ltnamegt39-cannot-expose-type-lttype1gt-in-ltspecifiergt-lttype2gt-39lttypenamegt39-through-ltnamegt"></a>&#39;&lt;Namen&gt; &#39; kann nicht verfügbar machen &lt;Typ1&gt; in &lt;Spezifizierer&gt; &lt;Typ2&gt; &#39; &lt;Typename&gt; &#39; über &lt;Name&gt;
Sie haben versucht, einen privaten Typ außerhalb der öffentlichen Klasse `MyClass`verfügbar zu machen.  
  
 **Fehler-ID:** BC30508  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Deklarieren Sie den Typ `Public`.
