# <a name="ltnamegt-cannot-expose-type-lttype1gt-in-ltspecifiergt-lttype2gt-lttypenamegt-through-ltnamegt"></a>"&lt;Namen&gt;' kann nicht verfügbar machen &lt;type1&gt; in &lt;Spezifizierer&gt; &lt;Typ2&gt; "&lt;Typename&gt;' über &lt;Name&gt;
Sie haben versucht, einen privaten Typ außerhalb der öffentlichen Klasse `MyClass`verfügbar zu machen.  
  
 **Fehler-ID:** BC30508  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Deklarieren Sie den Typ `Public`.
