# Initialisierungsfehler
Fehler beim Initialisieren des Programms. Möglicherweise wurde eine Installation während ihrer Ausführung abgebrochen.  
  
 **Fehler\-ID:** BC2000  
  
### So beheben Sie diesen Fehler  
  
-   Wiederholen Sie die Installation.
