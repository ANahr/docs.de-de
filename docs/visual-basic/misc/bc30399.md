# <a name="39mybase39-cannot-be-used-with-method-39ltmethodnamegt39-because-it-is-declared-39mustoverride39"></a>&#39; MyBase &#39; kann nicht mit der-Methode &#39;verwendet werden. &lt;Methodname&gt;&#39; da sie deklariert wurde &#39; MustOverride &#39;
Sie haben versucht, `MyBase` mit einer Methode zu verwenden, die als `MustOverride`deklariert wurde.  
  
 **Fehler-ID:** BC30399  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie die `MustOverride` -Deklaration.
