# <a name="39catch39-cannot-catch-type-39lttypenamegt39-because-it-is-not-39systemexception39-or-a-class-that-inherits-from-39systemexception39"></a>&#39; Catch &#39; Geben Sie &#39;nicht abfangen werden; &lt;Typename&gt;&#39; da er nicht ist &#39; System.Exception &#39; oder eine Klasse, die von erben &#39; System.Exception &#39;
`Catch` kann nur Ausnahmen abfangen, und Sie haben versucht, etwas abzufangen, das nicht von einer Ausnahme abgeleitet ist.  
  
 **Fehler-ID:** BC30392  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Entfernen Sie die `Catch` -Anweisung, oder ändern Sie das Ziel von `Catch` in eine tatsächliche Ausnahme.
