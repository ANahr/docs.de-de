# <a name="the-file-39ltfilenamegt39-is-not-a-text-file"></a>Die Datei &#39; &lt;Filename&gt;&#39; ist keine Textdatei.
Die angegebene Datei kann nicht geschrieben werden, weil sie eine Binärdatei ist.  
  
 **Fehler-ID:** BC2015  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Überprüfen Sie, ob Sie den richtigen Dateinamen und -pfad angegeben haben.
