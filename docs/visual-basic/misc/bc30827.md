## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Verwenden Sie den Operator `^` oder den Buchstaben `E+` , um anzuzeigen, dass ein Exponent vorhanden ist. Zum Beispiel:
