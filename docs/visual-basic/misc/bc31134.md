# <a name="39addhandler39-and-39removehandler39-method-parameters-cannot-be-declared-39byref39"></a>&#39;AddHandler&#39; und &#39;RemoveHandler&#39; Parameter der Methode nicht deklariert werden &#39;ByRef&#39;
Die Parameter der `AddHandler` - und der `RemoveHandler` -Methode können nicht mit dem `ByRef` -Modifizierer deklariert werden.  
  
 **Fehler-ID:** BC31134  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie das `ByRef` -Schlüsselwort aus der Anweisung.
