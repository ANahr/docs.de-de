# <a name="39addressof39-expressions-are-not-valid-in-debug-windows"></a>"AddressOf"-Ausdrücke sind in Debugfenstern nicht gültig
`AddressOf`Anweisungen können nur im Quellcode verwendet werden.  
  
 **Fehler-ID:** BC30731  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Entfernen Sie `AddressOf` -Anweisungen aus dem Debugcode.
