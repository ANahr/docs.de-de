# <a name="39end-removehandler39-must-be-the-first-statement-on-a-line"></a>&#39; End RemoveHandler &#39; die erste Anweisung in einer Zeile muss sein
Einem Doppelpunkt (:) als Trennzeichen folgt eine `End RemoveHandler` -Anweisung. `End RemoveHandler` muss die einzige Anweisung auf der jeweiligen Quellzeile sein.  
  
 **Fehler-ID:** BC31119  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Platzieren Sie jede Anweisung in einer eigenen Zeile.
