# <a name="39lttypegt39-cannot-be-inherited-more-than-once"></a>&#39;&lt;Typ&gt; &#39; kann nicht mehrmals geerbt werden kann
Sie haben eine `Inherits` -Anweisung dupliziert.  
  
 **Fehler-ID:** BC30584  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie die redundante `Inherits` -Anweisung.
