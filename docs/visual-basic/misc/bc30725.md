# <a name="39throw39-statements-are-not-valid-in-the-immediate-window"></a>Die throw-Anweisungen sind im Direktfenster ungültig
`Throw`Anweisungen können nur im Quellcode verwendet werden.  
  
 **Fehler-ID:** BC30725  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie `Throw` -Anweisungen aus Debugcode.
