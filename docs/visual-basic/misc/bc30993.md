## <a name="example"></a><span data-ttu-id="9b247-110">Beispiel</span><span class="sxs-lookup"><span data-stu-id="9b247-110">Example</span></span>  
 <span data-ttu-id="9b247-111">Die folgende Klasse enthält drei Eigenschaftendefinitionen: eine für `TotalItems` und zwei für `Item`. Letztere ist überladen.</span><span class="sxs-lookup"><span data-stu-id="9b247-111">The following class contains three property definitions: one for `TotalItems` and two for `Item`, which is overloaded.</span></span>  
  
```  
Class CollectionOfItems  
    Property TotalItems() As Integer  
        Get  
        End Get  
        Set(ByVal value As Integer)  
        End Set  
    End Property  
    Property Item(ByVal Key As String) As Object  
        Get  
        End Get  
        Set(ByVal value As Object)  
        End Set  
    End Property  
    Property Item(ByVal Index As Integer) As Object  
        Get  
        End Get  
        Set(ByVal value As Object)  
        End Set  
    End Property  
End Class  
```  
  
 <span data-ttu-id="9b247-112">Die `TotalItems` -Eigenschaft erfordert keine Argumente und kann in einer Objektinitialisiererliste initialisiert werden, wie in der folgenden Deklaration dargestellt.</span><span class="sxs-lookup"><span data-stu-id="9b247-112">The `TotalItems` property requires no arguments and can be initialized in an object initialization list, as shown in the following declaration.</span></span>  
  
```

```  
  
 <span data-ttu-id="9b247-113">Die `Item` -Eigenschaft ist überladen, und jede Überladung erfordert ein Argument.</span><span class="sxs-lookup"><span data-stu-id="9b247-113">The `Item` property is overloaded, and each overload requires an argument.</span></span> <span data-ttu-id="9b247-114">Aus diesem Grund darf `Item` nicht in einer Objektinitialisiererliste enthalten sein.</span><span class="sxs-lookup"><span data-stu-id="9b247-114">Therefore, `Item` cannot appear in an object initializer list.</span></span>  
  
```  
' The following declaration is not valid.  
' Dim coinCollection As New CollectionOfItems With { .TotalItems = 0, _  
'    .Item = aCoinObject }  
```  
  
 <span data-ttu-id="9b247-115">Um diesen Fehler zu vermeiden, initialisieren Sie die `Item` -Eigenschaft außerhalb des Objektinitialisierers.</span><span class="sxs-lookup"><span data-stu-id="9b247-115">To avoid this error, initialize the `Item` property outside the object initializer.</span></span>

```  
coinCollection.Item(1) = aCoinObject  
```
