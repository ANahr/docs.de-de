# <a name="39custom-event39-must-end-with-a-matching-39end-event39"></a>&#39;Benutzerdefiniertes Ereignis&#39; enden mit einem übereinstimmenden &#39;End-Ereignis&#39;
Eine `Custom Event` -Deklaration muss mit einer `End Event` -Anweisung enden.  
  
 **Fehler-ID:** BC31114  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Überprüfen Sie, ob die `Custom Event` -Deklaration mit einer `End Event` -Anweisung endet.
