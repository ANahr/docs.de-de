# Arraydimensionen k&#246;nnen keine negative Gr&#246;&#223;e haben
Mindestens eine der festgelegten Arraygrenzen ist eine negative Zahl. Ein negativer Index kann nur angegeben werden, wenn eine Obergrenze von \-1 verwendet wird, um ein leeres Array zu deklarieren. Ein solches Array enth√§lt null Elemente, aber es ist nicht `Nothing`.  
  
 **Fehler\-ID:** BC30611  
  
### So beheben Sie diesen Fehler  
  
-   Ersetzen Sie die negative Arraygrenze durch eine positive Zahl.
