# <a name="method-arguments-must-be-enclosed-in-parentheses"></a>Methodenargumente müssen in Klammern stehen.
Die Regeln, die Prozeduraufrufe steuern, sind in neueren Versionen von [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)]einfacher. Alle Argumente müssen in Klammern stehen.  
  
 **Fehler-ID:** BC30800  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Schließen Sie die Argumentliste in Klammern ein. Beispiel:  
  
    ```  
    MySub(ArrayIndex, NewValue)  
    ```
