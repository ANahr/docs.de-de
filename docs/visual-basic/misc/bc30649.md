# <a name="39lttypenamegt39-is-an-unsupported-type"></a>&#39; &lt;Typename&gt;&#39; ein nicht unterstützter Typ
Der angegebene Typ kann nicht von [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)]dargestellt werden.  
  
 **Fehler-ID:** BC30649  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Verwenden Sie einen unterstützten Typ.  
  
-   Entwickeln Sie einen neuen Typ.
