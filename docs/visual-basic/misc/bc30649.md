# <a name="39lttypenamegt39-is-an-unsupported-type"></a>'&lt;Typename&gt;' ist ein nicht unterstützter Typ
Der angegebene Typ kann nicht von dargestellt werden [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)].  
  
 **Fehler-ID:** BC30649  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Verwenden Sie einen unterstützten Typ.  
  
-   Entwickeln Sie einen neuen Typ.
