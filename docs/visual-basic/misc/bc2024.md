# <a name="the-default-assembly-reference-39ltreferencegt39-has-already-been-automatically-added-ignored"></a>Der Standardassemblyverweis &#39; &lt;Verweis&gt; &#39; wurde bereits wurde automatisch hinzugefügt; ignoriert
Es wurde versucht, den Standardassemblyverweis hinzuzufügen, aber die Assembly wurde bereits hinzugefügt, weshalb der Versuch ignoriert wird.  
  
 **Fehler-ID:** BC2024  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie den überflüssigen Code.
