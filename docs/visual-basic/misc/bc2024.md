# <a name="the-default-assembly-reference-ltreferencegt-has-already-been-automatically-added-ignored"></a>Der Standardassemblyverweis "&lt;Verweis&gt;' ist bereits wurde automatisch hinzugefügt; wird ignoriert
Es wurde versucht, den Standardassemblyverweis hinzuzufügen, aber die Assembly wurde bereits hinzugefügt, weshalb der Versuch ignoriert wird.  
  
 **Fehler-ID:** BC2024  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie den überflüssigen Code.
