# <a name="39systemruntimeinteropservicesdllimportattribute39-cannot-be-applied-to-a-39get39-or-39set39"></a>&#39; System.Runtime.InteropServices.DllImportAttribute &#39; kann nicht angewendet werden, um eine &#39; Get &#39; oder &#39; Set &#39;
Das `DllImportAttribute` -Attribut wurde auf eine `Get` - oder `Set` -Eigenschaftenprozedur angewendet.  
  
 **Fehler-ID:** BC31524  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Entfernen Sie `DllImportAttribute` aus `Get` - und `Set` -Eigenschaftenprozeduren.
