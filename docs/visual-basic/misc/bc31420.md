# <a name="39ltmembergt39-conflicts-with-the-reserved-member-by-this-name-that-is-implicitly-declared-in-all-enums"></a>&#39;&lt;Member&gt; &#39; steht in Konflikt mit dem reservierten Member mit diesem Namen, der implizit in allen Enumerationen explizit deklariert ist
Der Name eines Typmembers steht in Konflikt mit dem Namen eines implizit in allen Enumerationen deklarierten Members.  
  
 Visual Basic-Compiler erstellt implizite Member, bestimmte deklarierten Programmierelementen entsprechen. Enumerationen deklarieren den `value__member`-Member implizit.  
  
 **Fehler-ID:** BC31420  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Ändern Sie den Namen des Members.
