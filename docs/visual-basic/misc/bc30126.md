 **Fehler-ID:** BC30126  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Zwischen der `Get` -Anweisung und der `Property` -Anweisung muss eine `End Property` -Prozedur eingef√ºgt sein.  
  
2.  Stellen Sie sicher, dass andere Prozeduren innerhalb der `Property` -Deklaration korrekt abgeschlossen sind.
