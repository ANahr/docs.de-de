# <a name="39ltmembername1gt39-cannot-override-39ltmembername2gt39-because-they-differ-by-the-types-of-optional-parameters"></a>&#39; &lt;membername1&gt;&#39; kann nicht überschreiben &#39;&lt; membername2&gt;&#39; da sie von den Typen von optionalen Parametern unterscheiden
Überladene Member müssen für mindestens einen erforderlichen Parameter unterschiedliche Datentypen aufweisen.  
  
 **Fehler-ID:** BC30697  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Definieren Sie mindestens einen erforderlichen Parameter als anderen Datentyp.
