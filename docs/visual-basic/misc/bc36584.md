# <a name="parameter-39ltparameternamegt39-in-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39-already-has-a-matching-argument"></a>Parameter &#39; &lt;Parametername&gt;&#39; in der Erweiterungsmethode &#39;&lt; Methodenname&gt;&#39; im definierten &#39;&lt; TypeName&gt;&#39; hat bereits ein entsprechendes Argument
Ein Argument wird innerhalb des gleichen Prozeduraufrufs mehrfach anhand des Namens übergeben.  
  
 **Fehler-ID:** BC36584  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Überprüfen Sie, ob einer der Argumentnamen falsch geschrieben ist.  
  
-   Entfernen Sie das doppelt vorkommende Argument aus der Argumentliste.
