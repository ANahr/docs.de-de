# <a name="39ltmodifiergt39-is-not-valid-on-a-local-variable-declaration"></a>&#39; &lt;Modifizierer&gt;&#39; ist in einer lokalen Variablendeklaration ungültig.
Eine `Dim` -Anweisung innerhalb einer Prozedur enthält ein ungültiges Schlüsselwort, z. B. `Overloads`.  
  
 **Fehler-ID:** BC30247  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie das ungültige Schlüsselwort aus der `Dim` -Anweisung.
