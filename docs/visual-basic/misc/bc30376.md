# <a name="39new39-cannot-be-used-on-class-39ltclassnamegt39-because-it-contains-a-39mustoverride39-member-that-has-not-been-overridden"></a>&#39; Neue &#39; kann nicht auf Klasse &#39;verwendet werden. &lt;Classname&gt;&#39; da er enthält eine &#39; MustOverride &#39; Member, der nicht überschrieben wurde
Sie haben versucht, `New` für eine Klasse zu verwenden, die einen `MustOverride` -Member enthält, der nicht überschrieben wurde.  
  
 **Fehler-ID:** BC30376  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie die `New` -Anweisung.
