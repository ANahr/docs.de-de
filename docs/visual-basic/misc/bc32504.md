Eine Klasse mit einem `COMClassAttribute` -Attributblock ist innerhalb einer Klasse deklariert, die nicht als `Public`deklariert ist. Wenn eine Klasse als COM-Objekt verf√ºgbar gemacht werden soll, muss die gesamte Kapselungshierarchie mit `Public` -Zugriff deklariert werden.
