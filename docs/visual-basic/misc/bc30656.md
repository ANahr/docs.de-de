# <a name="field-39ltfieldnamegt39-is-of-an-unsupported-type"></a>Feld '&lt;Fieldname&gt;' wird nicht unterstützt
Ein Feld wird deklariert als Typ, der von dargestellt werden kann [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)].  
  
 **Fehler-ID:** BC30656  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Definieren Sie das Feld als unterstützten Typ.  
  
2.  Entwickeln Sie einen neuen Typ, der die Daten im Feld beschreibt.
