# <a name="field-39ltfieldnamegt39-is-of-an-unsupported-type"></a>Feld &#39; &lt;Fieldname&gt; &#39; weist einen nicht unterstützten Typ
Ein Feld wird als Typ deklariert, der von Visual Studio nicht dargestellt werden kann.  
  
 **Fehler-ID:** BC30656  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Definieren Sie das Feld als unterstützten Typ.  
  
2.  Entwickeln Sie einen neuen Typ, der die Daten im Feld beschreibt.
