 Die Klasse ist in einem Modul definiert, auf das Ihr Projekt nicht direkt verweist. Visual Basic-Compiler erfordert einen Verweis auf die Mehrdeutigkeit zu vermeiden, falls die Klasse in mehr als einem Modul definiert ist.
