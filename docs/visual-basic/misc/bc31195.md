# <a name="operator-39operator39-is-not-defined-for-types-39type139-and-39type239"></a>Operator &#39; Operator &#39; für Typen &#39; Typ1 &#39;ist nicht definiert werden. und &#39; Typ2 &#39;
Operator '<Operatorname>' ist für die Typen 'Typ1' und 'Typ2' nicht definiert. Sie können die Eigenschaft "Value" den Zeichenfolgenwert des ersten Elements der abzurufenden "\<ParentElement >'.  
  
 Es wurde ein XML-Literal in einer Binärdatei oder einem `Boolean` -Vergleich verwendet.  
  
 **Fehler-ID:** BC31195  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Verwenden Sie die `Value` -Eigenschaft des XML-Literals für die Binärdatei oder den `Boolean` -Vergleich.
