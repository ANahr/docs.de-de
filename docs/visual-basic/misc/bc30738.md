# <a name="sub-main-is-declared-more-than-once-in-ltnamespacegt-ltmessagegt"></a>"Sub Main" werden in mehr als einmal deklariert "&lt;Namespace&gt;': &lt;Nachricht&gt;
`Sub Main` kann in einem Namespace nur einmal deklariert werden.  
  
 **Fehler-ID:** BC30738  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Stellen Sie sicher, dass nur eine einzige `Sub Main` -Prozedur in Ihrem Projekt auftritt.
