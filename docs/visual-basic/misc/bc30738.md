# <a name="39sub-main39-is-declared-more-than-once-in-39ltnamespacegt39-ltmessagegt"></a>&#39;Sub Main&#39; ist in mehr als einmal deklariert &#39; &lt;Namespace&gt;&#39;: &lt;Nachricht&gt;
`Sub Main` kann in einem Namespace nur einmal deklariert werden.  
  
 **Fehler-ID:** BC30738  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Stellen Sie sicher, dass nur eine einzige `Sub Main` -Prozedur in Ihrem Projekt auftritt.
