# <a name="events-declared-with-an-39as39-clause-must-have-a-delegate-type"></a>Deklarierte Ereignisse mit einem &#39;als&#39; -Klausel muss einen Delegattyp aufweisen
Es wurde ein Typ angegeben, der kein Delegat ist.  
  
 **Fehler-ID:** BC31044  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Entfernen Sie die `As` -Klausel.  
  
2.  Geben Sie mit der `As` -Klausel einen Delegaten an.
