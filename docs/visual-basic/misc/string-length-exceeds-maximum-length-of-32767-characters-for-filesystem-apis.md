# <a name="string-length-exceeds-maximum-length-of-32767-characters-for-39filesystem39-apis"></a>Die Zeichenfolgenlänge überschreitet die maximale Länge von 32767 Zeichen für &#39; FileSystem &#39; APIs
Eine Zeichenfolge überschreitet die maximale Länge von 32767 Zeichen.  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Kürzen Sie die Zeichenfolge.
