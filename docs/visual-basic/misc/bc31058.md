# <a name="conflicts-with-39ltname1gt39-which-is-implicitly-created-for-39ltname2gt39-in-lttypegt-39ltname3gt39"></a>Steht in Konflikt mit &#39; &lt;name1&gt;&#39;, die implizit für &#39; &lt;name2&gt; &#39; in &lt;Typ&gt; &#39; &lt;name3&gt;&#39;
Der Name eines Typmembers steht in Konflikt mit einem anderen implizit erstellten Member.  
  
 **Fehler-ID:** BC31058  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Benennen Sie zur Behebung des Namenskonflikts den explizit deklarierten Member um.
