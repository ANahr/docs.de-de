# <a name="ltpropertygt-parameters-cannot-be-declared-39byref39"></a>&lt;Eigenschaft&gt; Parameter können nicht deklariert werden &#39; ByRef &#39;
Parameter für Eigenschaften müssen mit dem `ByVal` -Modifizierer deklariert werden.  
  
 **Fehler-ID:** BC30651  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Definieren Sie Eigenschaftenparameter als `ByVal`.
