# <a name="reference-required-to-assembly-39ltassemblynamegt39-containing-the-type-39ltmembernamegt39"></a>Verweis auf die Assembly erforderlich '&lt;Assemblyname&gt;'mit dem Typ'&lt;Membername&gt;'
Verweis auf die Assembly erforderlich '\<Assemblyname > "mit dem Typ '\<Membername >'. Fügen Sie dem Projekt einen Verweis hinzu.  
  
 Das Projekt enthält einen Typ, der von einem anderen Typ aus einer Assembly abhängt, die nicht in den Projektverweisen enthalten ist.  
  
 **Fehler-ID:** BC30652  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Fügen Sie einen Projektverweis für die abhängigen Typen im Projekt hinzu.
