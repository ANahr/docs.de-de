# With-Kontexte und -Anweisungen sind in Debugfenstern ung&#252;ltig.
`With`\-Anweisungen können nur im Quellcode verwendet werden.  
  
 **Fehler\-ID:** BC30726  
  
### So beheben Sie diesen Fehler  
  
-   Entfernen Sie `With`\-Anweisungen aus Debugcode.
