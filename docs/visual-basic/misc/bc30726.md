# <a name="39with39-contexts-and-statements-are-not-valid-in-debug-windows"></a>&#39; Mit &#39; Kontexte und-Anweisungen sind in Debugfenstern nicht gültig
`With` -Anweisungen können nur im Quellcode verwendet werden.  
  
 **Fehler-ID:** BC30726  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie `With` -Anweisungen aus Debugcode.
