# <a name="39case39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Groß-/Kleinschreibung &#39; Anweisungen sind im Direktfenster nicht gültig
`Case` -Anweisungen können nur im Quellcode verwendet werden.  
  
 **Fehler-ID:** BC30719  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie `Case` -Anweisungen aus dem **Direktfenster** .
