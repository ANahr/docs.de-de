# <a name="ltproperty1gt-and-ltproperty2gt-cannot-overload-each-other-because-only-one-is-declared-default"></a>"&lt;" Property1 "&gt;'und'&lt;Eigenschaft2&gt;' kann nicht gegenseitig überladen, da nur eine"Default"deklariert ist.
Wenn eine Eigenschaft `Default`angibt, müssen alle für diesen Namen überladenen Eigenschaften ebenfalls `Default`angeben.  
  
 **Fehler-ID:** BC30361  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Stellen Sie sicher, dass alle überladenen Eigenschaften als `Default`deklariert werden.
