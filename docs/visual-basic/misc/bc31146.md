# XML-Name erwartet
Ein XML\-Literal oder eine XML\-Achseneigenschaft wurde ohne Name angegeben.  
  
 **Fehler\-ID:** BC31146  
  
### So beheben Sie diesen Fehler  
  
-   Fügen Sie einen Namen für das XML\-Literal oder die XML\-Achseneigenschaft hinzu.
