# <a name="only-one-of-39public39-39private39-39protected39-39friend39-or-39protected-friend39-can-be-specified"></a>Nur eine der &#39; Öffentliche &#39; &#39; Private &#39; &#39; Geschützte &#39; &#39; "Friend" &#39; oder &#39; Protected Friend &#39; kann angegeben werden
In einer Elementdeklaration tritt inkonsistenter Zugriff auf. Es kann nur eine Form von Zugriff angegeben werden.  
  
 **Fehler-ID:** BC30176  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Wählen Sie einen Zugriff aus, und entfernen Sie die anderen Schlüsselwörter aus der Deklaration.
