# <a name="character-sequence-39--39-is-not-allowed-in-an-xml-comment"></a>Zeichen Sequence &#39;--&#39; ist in ein XML-Kommentar nicht zulässig
Ein XML-Kommentarliteral enthält die Zeichenfolge „--“, die nicht unterstützt wird.  
  
 **Fehler-ID:** BC31171  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie die Zeichenfolge „--“ aus dem XML-Kommentarliteral.
