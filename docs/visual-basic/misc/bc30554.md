# <a name="39ltnamespacenamegt39-is-ambiguous"></a>'&lt;Namespacename&gt;' ist nicht eindeutig
Sie haben einen Namen angegeben, der nicht eindeutig ist und daher einen Konflikt mit einem anderen Namen verursacht. Die [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] Compiler keine Auflösung des Konflikts zwischen Regeln; Sie müssen Namen sich unterscheiden.  
  
 **Fehler-ID:** BC30554  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Geben Sie einen vollqualifizierten Namen an.
