# <a name="local-variable-39ltvariablenamegt39-cannot-be-referred-to-before-it-is-declared"></a>Lokale Variable &#39; &lt;Variablename&gt; &#39; kann nicht verwiesen werden, bevor es deklariert ist
Eine Variable innerhalb einer Prozedur wird vor ihrer Deklaration verwendet.  
  
 **Fehler-ID:** BC32000  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Verschieben Sie die Deklaration vor den ersten Verweis auf die Variable.
