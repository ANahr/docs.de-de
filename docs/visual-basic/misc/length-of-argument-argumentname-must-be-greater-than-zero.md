# <a name="length-of-argument-39ltargumentnamegt39-must-be-greater-than-zero"></a>Die Länge von Argument &#39; &lt;Argumentname&gt;&#39; muss größer als 0 (null) sein
Ein Wert eines Arguments ist gleich oder kleiner als 0 (null).  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Geben Sie ein Argument an, das größer als 0 (null) ist.
