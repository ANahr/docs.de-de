# <a name="39catch39-block-never-reached-because-39ltname1gt39-inherits-from-39ltname2gt39"></a>&#39;Abfangen&#39; -Block wurde nie erreicht, da &#39; &lt;name1&gt; &#39; erbt &#39; &lt;name2&gt;&#39;
Ein `Catch` -Block im Code wird niemals erreicht.  
  
 Standardmäßig ist diese Meldung eine Warnung. Weitere Informationen zum Ausblenden von Warnungen aus, oder zum Behandeln von Warnungen als Fehler finden Sie unter [Configuring Warnings in Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic).  
  
 **Fehler-ID:** BC42029  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie den überflüssigen `Catch` -Block.
