# <a name="resource-name-39ltresourcenamegt39-cannot-be-used-more-than-once"></a>Ressourcenname &#39; &lt;Resourcename&gt; &#39; kann nicht mehr als einmal verwendet werden
Der Name für eine angegebene Ressource in einer DLL oder ausführbaren Datei wurde mehrfach verwendet.  
  
 **Fehler-ID:** BC31502  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Geben Sie Ressourcen eindeutige Namen.
