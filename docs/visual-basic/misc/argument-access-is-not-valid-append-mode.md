# <a name="argument-39access39-is-not-valid-valid-values-for-append-mode-are-39openaccesswrite39-and-39openaccessdefault39"></a>Argument &#39; Zugriff &#39; ist ungültig. Gültige Werte für den Append-Modus sind &#39;OpenAccess.Write &#39; und &#39;OpenAccess.Default &#39;
Ein Zugriffswert ist im Anfügemodus ungültig.  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Ändern Sie den Wert entweder in `OpenAccess.Write` oder `OpenAccess.Default`.
