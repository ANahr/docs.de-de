# <a name="39ltmembergt39-is-already-declared-in-this-structure"></a>'&lt;Element&gt;' ist bereits in dieser Struktur deklariert
Ein Enumerationsmember wurde in dieser Struktur mehrmals deklariert.  
  
 **Fehler-ID:** BC31421  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie die überflüssige Deklaration.
