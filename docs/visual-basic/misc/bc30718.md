# <a name="39select39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Wählen &#39; Anweisungen sind im Direktfenster nicht gültig
`Select` -Anweisungen können nur im Quellcode verwendet werden.  
  
 **Fehler-ID:** BC30718  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie `Select` -Anweisungen aus dem Debugcode.
