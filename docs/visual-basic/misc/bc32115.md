<span data-ttu-id="7b331-103">Ein generischer Typ wird aufgerufen, indem ein Typargument vom Typ <xref:System.Nullable%601> an einen Typparameter mit einer `Structure` -Einschränkung übergeben wird.</span><span class="sxs-lookup"><span data-stu-id="7b331-103">A generic type is invoked passing a type argument of <xref:System.Nullable%601> to a type parameter with a `Structure` constraint.</span></span>
