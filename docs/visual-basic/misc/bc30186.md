# <a name="39using39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Mit&#39; Anweisungen sind im Direktfenster nicht gültig
`Using` -Anweisungen können nur im Quellcode verwendet werden.  
  
 **Fehler-ID:** BC30186  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie die `Using` -Anweisung aus dem **Direktfenster** .
