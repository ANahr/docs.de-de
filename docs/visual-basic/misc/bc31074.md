# <a name="delegate-constructor-for-type-39lttypenamegt39-necessary-for-the-implementation-of-delegates-cannot-be-found"></a>Delegatkonstruktor für Typ &#39; &lt;Typename&gt;&#39; erforderlich für die Implementierung von Delegaten, wurde nicht gefunden
Die Assembly, die den Typ enthält, ist möglicherweise beschädigt oder ungültig.  
  
 **Fehler-ID:** BC31074  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Stellen Sie sicher, dass die Assembly mit diesem Typ gültig ist.  
  
2.  Kompilieren Sie das Programm erneut, um festzustellen, ob der Fehler erneut auftritt.  
  
3.  Wenn der Fehler weiterhin besteht, speichern Sie Ihre Arbeit, und starten Sie [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)]erneut.
