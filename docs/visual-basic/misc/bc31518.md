# <a name="assembly-ltassemblynamegt-cannot-be-created-because-its-path-is-longer-than-259-characters"></a>Assembly '&lt;Assemblyname&gt;' kann nicht erstellt werden, da ihr Pfad länger als 259 Zeichen ist.
Der Pfad für diese Assembly überschreitet die maximale Länge von 259 Zeichen.  
  
 **Fehler-ID:** BC31518  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Erstellen Sie einen kürzeren Pfad, indem Sie kürzere Namen verwenden oder eine flachere Dateihierarchie auswählen.
