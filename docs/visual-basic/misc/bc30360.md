# <a name="property-39ltpropertynamegt39-must-be-declared-39default39-because-it-overrides-a-default-property-on-the-base-39ltnamegt39"></a>Eigenschaft &#39; &lt;Propertyname&gt; &#39; muss deklariert werden &#39;Standard&#39; , da es sich um eine Standardeigenschaft für die Basis überschreibt &#39; &lt;Name&gt;&#39;
Eine `Property` -Anweisung im Code überschreibt eine `Default` -Eigenschaft in der Basisklasse, -struktur oder -schnittstelle und muss daher als `Default`deklariert werden.  
  
 **Fehler-ID:** BC30360  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Deklarieren Sie die Eigenschaft als `Default`.
