# <a name="ltmembername1gt-and-ltmembername2gt-cannot-overload-each-other-because-they-differ-only-by-the-types-of-optional-parameters"></a>"&lt;membername1&gt;'und'&lt;membername2&gt;' kann nicht gegenseitig überladen, da sie nur von den Typen der optionalen Parameter unterscheiden
Überladene Member müssen für mindestens einen erforderlichen Parameter unterschiedliche Datentypen aufweisen.  
  
 **Fehler-ID:** BC30696  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Definieren Sie mindestens einen erforderlichen Parameter als anderen Datentyp.
