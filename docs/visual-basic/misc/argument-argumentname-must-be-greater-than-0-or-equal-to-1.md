# <a name="argument-39ltargumentnamegt39-must-be-greater-than-0-or-equal-to--1"></a>Argument &#39; &lt;Argumentname&gt;&#39; muss größer als 0 oder gleich-1 sein
Der Wert eines Arguments ist kleiner als 0, aber ungleich -1.  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Stellen Sie sicher, dass der Argumentwert innerhalb des gültigen Bereichs liegt.  
  
2.  Wenn das Argument durch eine Berechnung bereitgestellt wird, überprüfen Sie die bei der Berechnung verarbeiteten Werte.
