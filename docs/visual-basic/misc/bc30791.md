# <a name="declare-statements-in-a-structure-cannot-be-declared-ltspecifiergt"></a>"Declare"-Anweisungen in einer Struktur können nicht deklariert werden "&lt;Spezifizierer&gt;"
Eine `Declare` -Deklaration enthält einen Spezifizierer, der in einer `Structure` -Deklaration ungültig ist.  
  
 **Fehler-ID:** BC30791  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie den Spezifizierer.
