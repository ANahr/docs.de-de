# Auf die Definition der &lt;Methode&gt;-Methode kann in diesem Kontext nicht zugegriffen werden.
Es wurde eine nicht vorhandene Aggregatfunktion mit dem `Into`\-Operator einbezogen.  
  
 **Fehler\-ID:** BC36594  
  
### So beheben Sie diesen Fehler  
  
-   Überprüfen Sie die Schreibweise des Namens der Aggregatfunktion, und nehmen Sie notwendige Korrekturen vor.  
  
-   Fügen Sie die Methode als benutzerdefinierte Aggregatfunktion \(Erweiterungsmethode\) hinzu. Weitere Informationen zum Erstellen einer benutzerdefinierten Aggregatfunktion finden Sie unter [Aggregate Clause](../../visual-basic/language-reference/queries/aggregate-clause.md).
