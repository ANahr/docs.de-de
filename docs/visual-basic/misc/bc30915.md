# <a name="39ltnamegt39-cannot-expose-the-underlying-delegate-type-39ltdelegatetypegt39-of-the-event-it-is-implementing-outside-the-project-through-ltspecifiergt-39lttypegt39"></a>&#39; &lt;Namen&gt;&#39; kann nicht verfügbar zu machen, die zugrunde liegenden Delegattyp &#39;&lt; Delegattyp&gt;&#39; des Ereignisses, außerhalb des Projekts durch implementiert wird &lt;Spezifizierer&gt; &#39;&lt; Typ&gt;&#39;
Der Code macht einen privaten Typ außerhalb der öffentlichen Klasse verfügbar.  
  
 **Fehler-ID:** BC30915  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Deklarieren Sie den Typ `Public`.  
  
     - oder -   
  
-   Verwenden Sie einen anderen Typ.
