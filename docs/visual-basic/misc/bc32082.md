<span data-ttu-id="68987-103">Ein generischer Typ wird mit einer `MustInherit` -Klasse als Typargument aufgerufen, während der entsprechende Typparameter mit der `New` -Einschränkung deklariert ist.</span><span class="sxs-lookup"><span data-stu-id="68987-103">A generic type is invoked with a `MustInherit` class as a type argument, while the corresponding type parameter is declared with the `New` constraint.</span></span>  
  
 <span data-ttu-id="68987-104">Für die `New` -Einschränkung muss der im entsprechenden Typargument übergebene Typ die Erstellung von Objekten unterstützen.</span><span class="sxs-lookup"><span data-stu-id="68987-104">The `New` constraint requires that the type passed in the corresponding type argument must support the creation of objects.</span></span> <span data-ttu-id="68987-105">Eine *abstrakte* Klasse, also eine Klasse, die als `MustInherit`deklariert ist, macht jedoch keine Konstruktoren verfügbar, da keine Objekte aus ihr erstellt werden können.</span><span class="sxs-lookup"><span data-stu-id="68987-105">However, an *abstract* class, that is, a class declared as `MustInherit`, does not expose any constructors because you cannot create any objects from it.</span></span>  
  
 <span data-ttu-id="68987-106">**Fehler-ID:** BC32082</span><span class="sxs-lookup"><span data-stu-id="68987-106">**Error ID:** BC32082</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="68987-107">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="68987-107">To correct this error</span></span>

2.  <span data-ttu-id="68987-109">Wenn die im Typargument verwendete Klasse abstrakt sein muss, aber nicht für die Konstruktion des generischen Typs verwendet werden muss, übergeben Sie im Typargument eine andere Klasse.</span><span class="sxs-lookup"><span data-stu-id="68987-109">If the class used in the type argument needs to be abstract but does not need to be used to construct the generic type, then pass a different class in the type argument.</span></span>  
  
3.  <span data-ttu-id="68987-110">Wenn der entsprechende Typparameter keine Objekte des ihm übergebenen Typs erstellen muss, entfernen Sie die Einschränkung `New` aus seiner Deklaration.</span><span class="sxs-lookup"><span data-stu-id="68987-110">If the corresponding type parameter does not need to create any objects from the type passed to it, then remove the `New` constraint from its declaration.</span></span>
