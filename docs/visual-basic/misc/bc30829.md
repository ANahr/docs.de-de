# <a name="39get39-statements-are-no-longer-supported"></a>"Get"-Anweisungen werden nicht mehr unterstützt.
`Get`Anweisungen werden nicht mehr unterstützt. Datei-E/A-Funktionalität ist im `Microsoft.VisualBasic` -Namespace verfügbar.  
  
 `Get`wird für Dateioperationen nicht unterstützt und kann nur in der Syntax von Eigenschaftenprozeduren verwendet werden.  
  
 **Fehler-ID:** BC30829  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Führen Sie Dateioperationen mit den Membern von `System.IO`, `FileSystemObject`, und [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] -Laufzeitfunktionen.
