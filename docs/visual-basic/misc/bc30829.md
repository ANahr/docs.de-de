# <a name="39get39-statements-are-no-longer-supported"></a>&#39; Get &#39; Anweisungen werden nicht mehr unterstützt.
`Get` -Anweisungen werden nicht mehr unterstützt. Datei-E/A-Funktionalität ist im `Microsoft.VisualBasic` -Namespace verfügbar.  
  
 `Get` wird für Dateivorgänge nicht unterstützt und kann nur in der Syntax von Eigenschaftsprozeduren verwendet werden.  
  
 **Fehler-ID:** BC30829  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Führen Sie Dateivorgänge mit den Membern der Laufzeitfunktionen `System.IO`, `FileSystemObject`und [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] aus.
