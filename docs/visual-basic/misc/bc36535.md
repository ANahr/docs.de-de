# <a name="instance-members-and-39me39-cannot-be-used-in-a-query-expression"></a><span data-ttu-id="c8f58-102">Instanzmember und &#39;mich&#39; kann nicht in einem Abfrageausdruck verwendet werden</span><span class="sxs-lookup"><span data-stu-id="c8f58-102">Instance members and &#39;Me&#39; cannot be used in a query expression</span></span>
<span data-ttu-id="c8f58-103">Eine LINQ-Abfrage in einer `Structure` enthält einen Verweis auf `Me` oder auf ein Instanzmember der Struktur.</span><span class="sxs-lookup"><span data-stu-id="c8f58-103">A LINQ query in a `Structure` includes a reference to `Me` or to an instance member of the structure.</span></span> <span data-ttu-id="c8f58-104">Verweise auf `Me` oder Instanzmember sind in Abfrageausdrücken innerhalb einer `Structure`nicht zulässig.</span><span class="sxs-lookup"><span data-stu-id="c8f58-104">References to `Me` or instance members are not allowed in query expressions within a `Structure`.</span></span>  
  
 <span data-ttu-id="c8f58-105">**Fehler-ID:** BC36535</span><span class="sxs-lookup"><span data-stu-id="c8f58-105">**Error ID:** BC36535</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="c8f58-106">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="c8f58-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="c8f58-107">Erstellen Sie eine Kopie des Instanzmembers oder des vom Verweis auf `Me` zurückgegebenen Werts, und verwenden Sie die Kopie im Abfrageausdruck, wie im folgenden Beispiel gezeigt.</span><span class="sxs-lookup"><span data-stu-id="c8f58-107">Create a copy of the instance member or the value returned by the reference to `Me` and use the copy in the query expression, as shown in the following example.</span></span>  
  
```vb  
Structure SampleStructure  
    Public SearchValue As Integer  
  
    Public Sub SetSearchValue(ByVal number As Integer)  
      SearchValue = number  
    End Sub  
  
    Public Sub GetData()  
        Dim sv = SearchValue  
        Dim SampleData = New Integer() {1, 2, 3, 4}  
        Dim query = From number In SampleData _  
                    Where number < sv  
    End Sub  
End Structure  
```
