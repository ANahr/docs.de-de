<span data-ttu-id="8c06c-103">Ein für ein Attributargument verwendeter Ausdruck wird als Datentyp ausgewertet, der sich von dem des entsprechenden Attributparameters unterscheidet, und [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] lässt die erforderliche Typkonvertierung für Attributargumente nicht zu.</span><span class="sxs-lookup"><span data-stu-id="8c06c-103">An expression used for an attribute argument evaluates to a data type different from that of the corresponding attribute parameter, and [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] does not allow the required type conversion for attribute arguments.</span></span>  
  
 <span data-ttu-id="8c06c-104">Ein Attribut stellt Metadaten für das Element bereit, auf das es angewendet wird, und der Compiler muss zur Kompilierzeit in der Lage sein, alle Metadaten zu erstellen.</span><span class="sxs-lookup"><span data-stu-id="8c06c-104">An attribute provides metadata for the element it is applied to, and the compiler must be able to construct all the metadata at compile time.</span></span> <span data-ttu-id="8c06c-105">Daher muss jedes Attribut Werte verwenden, die zur Kompilierzeit konstant sind, und jedes Attributargument muss als ein zur Kompilierzeit konstanter Wert ausgewertet werden.</span><span class="sxs-lookup"><span data-stu-id="8c06c-105">For this reason, every attribute must use values that are constant at compile time, and therefore every attribute argument must evaluate to a compile-time constant value.</span></span>
