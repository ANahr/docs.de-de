# <a name="raiseevent-is-already-declared"></a>"RaiseEvent" wurde bereits deklariert
In einer benutzerdefinierten Ereignisdeklaration ist mehr als eine `RaiseEvent`-Deklaration vorhanden. In einer `RaiseEvent` -Deklaration wird eine Prozedur deklariert, mit der das Ereignis ausgel√∂st wird.  
  
 **Fehler-ID:** BC31129  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie die redundante `RaiseEvent` -Anweisung.
