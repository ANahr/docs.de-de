# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-type-39ltmembernamegt39"></a>Referenz zum Modul "&lt;Modulename&gt;'mit dem Typ'&lt;Membername&gt;'
Referenz zum Modul "\<Modulename >" mit dem Typ '\<Membername >'. Fügen Sie dem Projekt einen Verweis hinzu.  
  
 Das Projekt enthält einen Typ, der von einem anderen Typ aus einer Assembly abhängt, die nicht in den Projektverweisen enthalten ist.  
  
 **Fehler-ID:** BC30653  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Fügen Sie einen Projektverweis für die abhängigen Typen im Projekt hinzu.
