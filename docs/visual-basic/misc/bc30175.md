# <a name="lttype1gt-lttypenamegt-and-lttype2gt-lttypenamegt-declared-in-ltnamegt-conflict-in-ltnamespacegt-ltnamespacenamegt"></a>&lt;type1&gt; "&lt;Typename&gt;" und &lt;Typ2&gt; "&lt;Typename&gt;", deklariert in"&lt;Namen&gt;', verursachen einen Konflikt in &lt;Namespace &gt; "&lt;Namespacename&gt;"
Zwei deklarierte Elemente, z. B. eine Klasse und ein Modul, haben den gleichen Namen im selben Namespace.  
  
 **Fehler-ID:** BC30175  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Verschieben Sie die Deklaration eines der Elemente in einen anderen Namespace.
