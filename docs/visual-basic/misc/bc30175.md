# <a name="lttype1gt-39lttypenamegt39-and-lttype2gt-39lttypenamegt39-declared-in-39ltnamegt39-conflict-in-ltnamespacegt-39ltnamespacenamegt39"></a>&lt;Typ1&gt; &#39;&lt; TypeName&gt;&#39; und &lt;Typ2&gt; &#39;&lt; TypeName&gt;&#39; die im deklariert &#39;&lt; Namen&gt;&#39; Konflikt in &lt;Namespace&gt; &#39;&lt; Namespacename&gt;&#39;
Zwei deklarierte Elemente, z. B. eine Klasse und ein Modul, haben den gleichen Namen im selben Namespace.  
  
 **Fehler-ID:** BC30175  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Verschieben Sie die Deklaration eines der Elemente in einen anderen Namespace.  
  
-   Ändern Sie den Namen eines der beiden Elemente.
