# <a name="39end-operator39-must-be-the-first-statement-on-a-line"></a>&#39; End-Operator &#39; die erste Anweisung in einer Zeile muss sein
Einem Doppelpunkt ( `End Operator` ) als Trennzeichen folgt eine`:`-Anweisung. `End Operator` muss die einzige Anweisung der jeweiligen Quellzeile sein.  
  
 **Fehler-ID:** BC33006  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Platzieren Sie jede Anweisung in einer eigenen Zeile.
