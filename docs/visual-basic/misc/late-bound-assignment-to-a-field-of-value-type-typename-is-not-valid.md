# <a name="late-bound-assignment-to-a-field-of-value-type-39lttypenamegt39-is-not-valid-when-39ltnamegt39-is-the-result-of-a-late-bound-expression"></a>Spät gebundene Zuweisung an ein Feld des Werttyps &#39; &lt;Typename&gt;&#39; ist nicht gültig, wenn &#39;&lt; Namen&gt;&#39; ist das Ergebnis eines spät gebundenen Ausdrucks
Sie haben versucht, eine spät gebundene Zuweisung zu verwenden, die nicht gültig ist.

-   Ändern Sie die Zuweisung in eine früh gebundene Zuweisung.
