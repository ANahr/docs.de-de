# <a name="file-ltfilenamegt-cannot-be-deleted-because-it-is-open"></a>Datei "&lt;Filename&gt;' kann nicht gelöscht werden, da sie geöffnet ist
Es wird versucht, eine geöffnete Datei zu löschen.  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Schließen Sie die Datei, bevor Sie sie löschen.
