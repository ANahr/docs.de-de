# <a name="file-39ltfilenamegt39-cannot-be-deleted-because-it-is-open"></a>Datei &#39; &lt;Filename&gt; &#39; kann nicht gelöscht werden, da er geöffnet ist.
Es wird versucht, eine geöffnete Datei zu löschen.  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Schließen Sie die Datei, bevor Sie sie löschen.
