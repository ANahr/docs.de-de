# <a name="39implements39-statement-must-follow-any-39inherits39-statement-and-precede-all-declarations-in-a-class"></a>&#39;Implementiert&#39; Anweisung folgen alle &#39;Inherits&#39; Anweisung und vor allen Deklarationen in einer Klasse
Eine `Implements` -Anweisung wurde an einer ung√ºltigen Stelle gefunden.  
  
 **Fehler-ID:** BC31053  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Ordnen Sie `Implements` -Anweisungen unmittelbar nach `Inherits` -Anweisungen, aber noch vor Deklarationen an. Zum Beispiel:  
  
    ```  
    Class Derived  
       Inherits Base  
       Implements One  
       Sub SubOne() Implements One.Method1  
          ' Add code to implement the method.  
       End Sub  
    End Class  
    ```
