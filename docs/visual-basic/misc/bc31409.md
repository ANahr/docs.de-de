# <a name="ltmethodgt-39ltmethodname1gt39-must-be-declared-39overloads39-because-another-39ltmethodname2gt39-is-declared-39overloads39"></a>&lt;Methode&gt; &#39;&lt; Methodenname1&gt;&#39; muss deklariert werden &#39; die Überladungen &#39; da eine andere &#39;&lt; methodenname2&gt;&#39; deklariert &#39; die Überladungen &#39;
Eine andere Eigenschaft oder Methode mit demselben Namen wurde mit dem `Overloads` -Modifizierer deklariert.  
  
 **Fehler-ID:** BC31409  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Fügen Sie den `Overloads` -Modifizierer zur Eigenschaft oder Methode hinzu.  
  
2.  Entfernen Sie den `Overloads` -Modifizierer aus allen Eigenschaften oder Methoden mit demselben Namen.
