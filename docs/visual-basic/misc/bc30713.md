# <a name="39exit39-statements-are-not-valid-in-the-immediate-window"></a>'Beenden'-Anweisungen sind im Direktfenster ungültig
`Exit`Anweisungen können nur im Quellcode verwendet werden.  
  
 **Fehler-ID:** BC30713  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie `Exit` -Anweisungen aus dem Debugcode.
