# <a name="39let39-and-39set39-assignment-statements-are-no-longer-supported"></a>&#39;Lassen Sie&#39; und &#39;festgelegt&#39; zuweisungsanweisungen werden nicht mehr unterstützt.
`Let` und `Set` können nicht in Zuweisungen verwendet werden.  
  
 **Fehler-ID:** BC30807  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Verzichten Sie in Zuweisungsanweisungen auf `Let` und `Set` . Beispiel:  
  
     `X = 2`
