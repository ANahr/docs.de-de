# <a name="warning-number-39ltnumbergt39-for-the-option-39ltoptionnamegt39-is-either-not-configurable-or-not-valid"></a>Nummer der codeanalysewarnung &#39; &lt;Anzahl&gt; &#39; für die Option &#39; &lt;Optionsname&gt; &#39; ist ungültig oder kann nicht konfiguriert
Die ID einer Warnung ist ungültig. IDs müssen ganzzahlige Werte größer als 0 sein.  
  
 **Fehler-ID:** BC2026  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Überprüfen Sie die angegebene Warnungs-ID, und stellen Sie sicher, dass diese gültig ist.
