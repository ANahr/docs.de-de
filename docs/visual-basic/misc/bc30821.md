# <a name="39ltrsetstmtgt39-is-not-declared"></a>&#39;&lt;Rsetstmt&gt; &#39; ist nicht deklariert
"\<Rsetstmt >" ist nicht deklariert. RSet-Anweisungen werden nicht mehr unterstützt. Verwenden Sie stattdessen Microsoft.VisualBasic.RSet.  
  
 Einige systeminterne Funktionen in Visual Basic in früheren Versionen wurden an verschoben wurden die <xref:Microsoft.VisualBasic?displayProperty=nameWithType> Namespace. Dadurch wird ihre Funktionalität allgemeiner verfügbar für alle Programmiersprachen.  
  
 **Fehler-ID:** BC30821  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Verwenden Sie stattdessen die `RSet` -Funktion im `Microsoft.VisualBasic` -Namespace.
