# <a name="39end-using39-must-be-preceded-by-a-matching-39using39"></a>&#39; Verwenden End &#39; muss ein entsprechender &#39;vorangestellt werden; Mit &#39;
Eine `End Using` -Anweisung wird ohne die entsprechende vorangehende `Using` -Deklaration angezeigt.  
  
 **Fehler-ID:** BC36007  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie die `End Using` -Anweisung, falls sie redundant ist.  
  
-   Geben Sie die fehlende [Using-Anweisung](../../visual-basic/language-reference/statements/using-statement.md) Falls diese fehlt.  
  
-   Verschieben Sie die `End Using` -Anweisung im Code an die entsprechende Position.
