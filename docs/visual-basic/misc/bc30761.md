# <a name="project-ltprojectnamegt-cannot-reference-project-ltprojectnamegt-because-ltprojectnamegt-directly-or-indirectly-references-ltprojectnamegt"></a>Projekt '&lt;Projectname&gt;"kann nicht auf Projekt verweisen"&lt;Projectname&gt;"da"&lt;Projectname&gt;"direkt oder indirekt verweist"&lt;Projektname &gt;'
Ein Projekt enthält einen Verweis auf ein zweites Projekt. Das zweite Projekt enthält wiederum einen Verweis auf das Projekt, in dem darauf verwiesen wird.  
  
 **Fehler-ID:** BC30761  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie einen der Verweise.
