# <a name="39if39-operands-cannot-be-named-arguments"></a><span data-ttu-id="5f050-102">&#39;Wenn&#39; Operanden können keine benannten Argumente</span><span class="sxs-lookup"><span data-stu-id="5f050-102">&#39;If&#39; operands cannot be named arguments</span></span>
