# Komma erwartet.
In einer Argumentliste fehlt ein Komma \(,\) zwischen zwei Argumenten.  
  
 **Fehler\-ID:** BC30196  
  
### So beheben Sie diesen Fehler  
  
-   Stellen Sie sicher, dass alle Argumente durch Kommas voneinander getrennt sind.
