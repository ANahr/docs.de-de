# <a name="partial-method-39ltmethodnamegt39-cannot-use-the-39implements39-keyword"></a>Partielle Methode &#39; &lt;Methodname&gt; &#39; können keine der &#39;implementiert&#39; Schlüsselwort
Die Definition einer partiellen Methode kann keine Schnittstelle und Schnittstellenmember implementieren.  
  
 **Fehler-ID:** BC31439  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Entfernen Sie die `Implements` -Anweisung aus der Definition der partiellen Methode.
