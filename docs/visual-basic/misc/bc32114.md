# <a name="type-parameter-with-a-39structure39-constraint-cannot-be-used-as-a-constraint"></a><span data-ttu-id="5bbe4-102">Geben Sie Parameter mit einem &#39;Struktur&#39; Einschränkung kann nicht als Einschränkung verwendet werden</span><span class="sxs-lookup"><span data-stu-id="5bbe4-102">Type parameter with a &#39;Structure&#39; constraint cannot be used as a constraint</span></span>
<span data-ttu-id="5bbe4-103">Ein Typparameter mit einer `Structure` -Einschränkung wird als Einschränkung für einen anderen Typparameter verwendet.</span><span class="sxs-lookup"><span data-stu-id="5bbe4-103">A type parameter with a `Structure` constraint is used as the constraint for another type parameter.</span></span>  
  
 <span data-ttu-id="5bbe4-104">Die `Structure` -Einschränkung setzt voraus, dass das an den Typparameter übergebene Typargument ein Werttyp ist.</span><span class="sxs-lookup"><span data-stu-id="5bbe4-104">The `Structure` constraint requires that the type argument passed to its type parameter be a value type.</span></span> <span data-ttu-id="5bbe4-105">Allerdings kann ein Werttyp nicht implementiert oder geerbt werden. Daher es nicht sinnvoll, ihn als Einschränkung zu verwenden, da der andere Typparameter ihn implementieren oder erben müsste.</span><span class="sxs-lookup"><span data-stu-id="5bbe4-105">However, a value type cannot be implemented or inherited, so it is not meaningful to use it as a constraint, which would require the other type parameter to implement it or inherit from it.</span></span>  
  
 <span data-ttu-id="5bbe4-106">Diese Situation ergibt nur dann einen Sinn, wenn beide Typargumente genau denselben Typ aufweisen müssen.</span><span class="sxs-lookup"><span data-stu-id="5bbe4-106">The only meaningful interpretation of this situation is that both type arguments must be of the exact same type.</span></span> <span data-ttu-id="5bbe4-107">In diesem Fall braucht der generische Typ nur ein Typparameter zu sein.</span><span class="sxs-lookup"><span data-stu-id="5bbe4-107">If that is the case, your generic type needs only one type parameter.</span></span>  
  
 <span data-ttu-id="5bbe4-108">Dieser Fehler kann durch die folgende Anweisung generiert werden.</span><span class="sxs-lookup"><span data-stu-id="5bbe4-108">The following statement can generate this error.</span></span>  
  
 `Class c1(Of t1 As Structure, t2 As t1)`  
  
 <span data-ttu-id="5bbe4-109">Der an `t2` übergebene Typ kann den an `t1`übergebenen Typ nicht implementieren oder erben, da der an `t1` übergebene Typ ein Werttyp sein muss.</span><span class="sxs-lookup"><span data-stu-id="5bbe4-109">The type passed to `t2` cannot implement or inherit the type passed to `t1`, because the type passed to `t1` must be a value type.</span></span>  
  
 <span data-ttu-id="5bbe4-110">**Fehler-ID:** BC32114</span><span class="sxs-lookup"><span data-stu-id="5bbe4-110">**Error ID:** BC32114</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="5bbe4-111">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="5bbe4-111">To correct this error</span></span>  
  
-   <span data-ttu-id="5bbe4-112">Entfernen Sie den mit `Structure` eingeschränkten Typparameter aus der Einschränkungsliste des anderen Typparameters.</span><span class="sxs-lookup"><span data-stu-id="5bbe4-112">Remove the type parameter constrained to `Structure` from the constraint list on the other type parameter.</span></span>  
  
-   <span data-ttu-id="5bbe4-113">Wenn beide Typparameter denselben Werttyp erfordern, definieren Sie den generischen Typ mit nur einem Typparameter.</span><span class="sxs-lookup"><span data-stu-id="5bbe4-113">If both type parameters require the same value type, define the generic type with only one type parameter.</span></span>
