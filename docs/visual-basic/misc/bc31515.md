# <a name="project-39ltprojectnamegt39-makes-an-indirect-reference-to-assembly-39ltassemblynamegt39-which-contains-39ltclassnamegt39"></a>Projekt '&lt;Projektname&gt;'macht einen indirekten Verweis auf Assembly'&lt;Assemblyname&gt;'enthält'&lt;Classname&gt;'
Projekt '\<Projektname > "macht einen indirekten Verweis auf Assembly '\<Assemblyname >' enthält '\<Classname >'. Fügen Sie einen Verweis auf "\<Assemblyname >" zu Ihrem Projekt.  
  
 Dieses Projekt enthält Member, die von einer Assembly abhängen, die nicht in den Projektverweisen enthalten ist.  
  
 **Fehler-ID:** BC31515  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Fügen Sie der Assembly einen Verweis hinzu, indem Sie im **Projektmappen-Explorer** mit der rechten Maustaste auf **Verweise** klicken. Klicken Sie dann auf **Verweis hinzufügen**.
