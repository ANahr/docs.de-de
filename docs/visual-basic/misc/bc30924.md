# <a name="39ltnamegt39-cannot-expose-type-39lttype1gt39-used-in-the-underlying-delegate-type-39ltdelegatetypegt39-of-the-event-it-is-implementing-outside-the-project-through-ltspecifiergt-39lttype2gt39"></a>&#39;&lt;Namen&gt; &#39; kann nicht verfügbar machen &#39; &lt;Typ1&gt; &#39; in den zugrunde liegenden Delegattyp verwendet &#39; &lt;Delegattyp&gt; &#39; des Ereignisses implementiert wird außerhalb des Projekts durch &lt;Spezifizierer&gt; &#39; &lt;Typ2&gt;&#39;
Der Code macht einen privaten Typ außerhalb der öffentlichen Klasse verfügbar.  
  
 **Fehler-ID:** BC30924  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Deklarieren Sie den Typ `Public` , oder verwenden Sie einen anderen Typ.
