# <a name="array-initializer-cannot-be-specified-for-a-non-constant-dimension-use-the-empty-initializer-"></a><span data-ttu-id="3b543-102">Der Arrayinitialisierer kann nicht für eine nicht Konstante Dimension angegeben werden; Verwenden Sie den leeren Initialisierer '{}"</span><span class="sxs-lookup"><span data-stu-id="3b543-102">Array initializer cannot be specified for a non constant dimension; use the empty initializer '{}'</span></span>
<span data-ttu-id="3b543-103">Ein Array initialisiert eine Dimension, die zum Zeitpunkt der Kompilierung nicht bekannt ist.</span><span class="sxs-lookup"><span data-stu-id="3b543-103">An array initializes a dimension that is not known at compile time.</span></span>  
  
 <span data-ttu-id="3b543-104">Der folgende Code generiert diese Warnung.</span><span class="sxs-lookup"><span data-stu-id="3b543-104">The following code generates this error.</span></span>  
  
```  
Dim j As Integer  
Dim intArray As Integer = New Integer(1, j) {{0, 100}, {1,101}}  
```  
  
 <span data-ttu-id="3b543-105">Mit folgendem Code wird der Fehler vermieden.</span><span class="sxs-lookup"><span data-stu-id="3b543-105">The following code avoids the error.</span></span>  
  
```  
Dim intArray As Integer = New Integer(1, j) {}  
For i As Integer = 0 To j  
    intArray(0, i) = i  
    intArray(1, i) = 100 + i  
Next i  
```  
  
 <span data-ttu-id="3b543-106">**Fehler-ID:** BC30949</span><span class="sxs-lookup"><span data-stu-id="3b543-106">**Error ID:** BC30949</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="3b543-107">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="3b543-107">To correct this error</span></span>  
  
-   <span data-ttu-id="3b543-108">Geben Sie nach Möglichkeit eine konstante Dimension in der Arraydeklaration an.</span><span class="sxs-lookup"><span data-stu-id="3b543-108">If possible, specify a constant dimension in the array declaration.</span></span>  
  
-   <span data-ttu-id="3b543-109">Wenn Sie keine konstante Dimension angeben können, müssen Sie das Array mit einer Schleife initialisieren, wenn die nicht konstante Dimension bekannt ist.</span><span class="sxs-lookup"><span data-stu-id="3b543-109">If you cannot specify a constant dimension, then you must initialize the array using a loop when the nonconstant dimension becomes known.</span></span>
