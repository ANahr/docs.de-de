# <a name="39ltspecifiergt39-is-not-valid-on-a-structure-declaration"></a>&#39; &lt;Spezifizierer&gt;&#39; ist ungültig für Structure-Deklarationen
Sie haben einen Spezifizierer wie etwa `Dim` oder `Override`verwendet, der in einer `Structure` -Deklaration nicht gültig ist.  
  
 **Fehler-ID:** BC30395  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie den Spezifizierer.
