# <a name="class-39ltclassnamegt39-cannot-be-indexed-because-it-has-no-default-property"></a>Klasse &#39; &lt;Classname&gt; &#39; kann nicht indiziert werden, da sie keine Standardeigenschaft vorhanden ist.
Sie haben keine Standardeigenschaft für eine Klasse angeben.  
  
 **Fehler-ID:** BC30367  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Geben Sie eine Eigenschaft als Standardeigenschaft an, indem Sie ihre Deklaration mit dem `Default` -Schlüsselwort beginnen.
