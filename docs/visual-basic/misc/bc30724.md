# <a name="39error39-statements-are-not-valid-in-the-immediate-window"></a>'Fehler'-Anweisungen sind im Direktfenster ungültig
`Error`Anweisungen können nur im Quellcode verwendet werden.  
  
 **Fehler-ID:** BC30724  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie `Error` -Anweisungen aus Debugcode.
