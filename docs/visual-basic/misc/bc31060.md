# <a name="39ltpropertynamegt39-implicitly-defines-39ltmembernamegt39-which-conflicts-with-a-member-of-the-same-name-in-lttypegt-39lttypenamegt39"></a>&#39;&lt;PropertyName&gt; &#39; definiert implizit &#39; &lt;Membername&gt;&#39;, was zu einem Konflikt mit einem Member mit demselben Namen in &lt;Typ&gt; &#39; &lt; Typname&gt;&#39;
Der Name eines Typmembers steht in Konflikt mit dem Namen eines implizit für eine Eigenschaft erstellten Members. Eigenschaften erstellen implizit mehrere implizite Variablen, die mit definierten Membern in Konflikt stehen können.  
  
 **Fehler-ID:** BC31060  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Benennen Sie zur Behebung des Namenskonflikts den explizit deklarierten Member um.
