# <a name="39try39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Wiederholen Sie den &#39; Anweisungen sind im Direktfenster nicht gültig
`Try` -Anweisungen können nur im Quellcode verwendet werden.  
  
 **Fehler-ID:** BC30717  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie `Try` -Anweisungen aus Debugcode.
