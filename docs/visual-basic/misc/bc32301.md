# <a name="project-has-been-closed"></a>Das Projekt wurde geschlossen.

 **Fehler-ID:** BC32301  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler

2.  Wenn der Fehler erneut auftritt, installieren Sie den [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] -Compiler neu.
