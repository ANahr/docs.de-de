# <a name="unable-to-open-key-file-39ltfilenamegt39-lterrorgt"></a>Kann nicht geöffnet werden Schlüsseldatei &#39; &lt;Filename&gt;&#39;: &lt;Fehler&gt;
Für dieses Projekt angegebene Schlüsseldatei konnte nicht geöffnet werden.  
  
 **Fehler-ID:** BC31025  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler
