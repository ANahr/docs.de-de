# <a name="39new39-cannot-be-used-on-a-type-parameter-that-does-not-have-a-39new39-constraint"></a><span data-ttu-id="c1317-102">&#39; Neue &#39; kann nicht verwendet werden, für einen Typparameter, der keinem &#39; Neue &#39; Einschränkung</span><span class="sxs-lookup"><span data-stu-id="c1317-102">&#39;New&#39; cannot be used on a type parameter that does not have a &#39;New&#39; constraint</span></span>
<span data-ttu-id="c1317-103">Eine deklarationsanweisung verwendet eine [New-Operator](../../visual-basic/language-reference/operators/new-operator.md) -Klausel einen Typparameter als Typ erstellt werden soll, und die Typparameter deklariert, ohne eine `New` Einschränkung.</span><span class="sxs-lookup"><span data-stu-id="c1317-103">A declaration statement uses a [New Operator](../../visual-basic/language-reference/operators/new-operator.md) clause specifying a type parameter as the type to be created, and the type parameter is declared without a `New` constraint.</span></span>  
  
 <span data-ttu-id="c1317-104">Eine *Einschränkung* für einen Typparameter ist eine Anforderung an jedes Typargument, das beim Erstellen des generischen Typs an diesen Typparameter übergeben wird.</span><span class="sxs-lookup"><span data-stu-id="c1317-104">A *constraint* on a type parameter imposes a requirement on any type argument passed to that type parameter when the generic type is created.</span></span> <span data-ttu-id="c1317-105">Die `New` -Einschränkung gibt an, dass das Typargument einen parameterlosen Konstruktor verfügbar machen muss, auf den der erstellende Code zugreifen kann.</span><span class="sxs-lookup"><span data-stu-id="c1317-105">The `New` constraint specifies that the type argument must expose a parameterless constructor that the creating code can access.</span></span> <span data-ttu-id="c1317-106">Dadurch wird eine `New` -Klausel in einer Deklarationsanweisung und das Erstellen einer Instanz dieses Typs ermöglicht.</span><span class="sxs-lookup"><span data-stu-id="c1317-106">This is what allows a `New` clause in a declaration statement to create an instance of that type.</span></span>  
  
 <span data-ttu-id="c1317-107">**Fehler-ID:** BC32046</span><span class="sxs-lookup"><span data-stu-id="c1317-107">**Error ID:** BC32046</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="c1317-108">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="c1317-108">To correct this error</span></span>  
  
-   <span data-ttu-id="c1317-109">Wenn Sie für das Typargument das Verfügbarmachen eines parameterlosen Konstruktors, auf den der Code zugreifen kann, verbindlich festlegen können, fügen Sie der Deklaration des Typparameters die `New` -Einschränkung hinzu.</span><span class="sxs-lookup"><span data-stu-id="c1317-109">If you can require the type argument to expose an accessible parameterless constructor, add the `New` constraint to the declaration of the type parameter.</span></span>
