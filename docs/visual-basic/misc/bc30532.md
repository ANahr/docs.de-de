# <a name="conversion-from-39date39-to-39double39-requires-calling-the-39datetooadate39-method"></a>Konvertierung von &#39; Datum &#39; zu &#39; doppelte &#39; erfordert das Aufrufen der &#39; Datum. ToOADate &#39; -Methode
Sie haben versucht, eine `Date` -Wert an ein `Double` -Wert, der ohne erfolgen kann die <xref:System.DateTime.ToOADate%2A?displayProperty=nameWithType> Methode.  
  
 **Fehler-ID:** BC30532  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Verwenden der <xref:System.DateTime.ToOADate%2A?displayProperty=nameWithType> Methode, um den Wert zu konvertieren.
