# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-implemented-interface-39ltinterfacenamegt39"></a>Verweis auf das Modul erforderlich &#39; &lt;"ModuleName"&gt; &#39; , enthält die implementierte Schnittstelle &#39; &lt;Schnittstellenname&gt;&#39;
Verweis auf das Modul erforderlich "\<" ModuleName ">" mit der ebenfalls implementierten Schnittstelle '\<Schnittstellenname >'. Fügen Sie dem Projekt einen Verweis hinzu.  
  
 Die Schnittstelle ist in einem Modul definiert, auf das Ihr Projekt nicht direkt verweist. Visual Basic-Compiler erfordert einen Verweis auf die Mehrdeutigkeit zu vermeiden, falls die Schnittstelle in mehr als einem Modul definiert ist.  
  
 **Fehler-ID:** BC30010  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Nehmen Sie den Namen des nicht referenzierten Moduls in Ihre Projektverweise auf.
