# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-implemented-interface-39ltinterfacenamegt39"></a>Referenz zum Modul "&lt;Modulename&gt;"enthält die implementierte Schnittstelle"&lt;Interfacename&gt;'
Referenz zum Modul '\<Modulename >', enthält die implementierte Schnittstelle "\<Interfacename >'. Fügen Sie dem Projekt einen Verweis hinzu.  
  
 Die Schnittstelle ist in einem Modul definiert, auf das Ihr Projekt nicht direkt verweist. Die [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] Compiler benötigt einen Verweis, um Mehrdeutigkeit zu vermeiden, falls die Schnittstelle in mehr als einem Modul definiert ist.  
  
 **Fehler-ID:** BC30010  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Nehmen Sie den Namen des nicht referenzierten Moduls in Ihre Projektverweise auf.
