# <a name="instance-of-restricted-type-39lttypenamegt39-cannot-be-used-in-a-query-expression"></a>Instanz des eingeschränkten Typs &#39; &lt;Typename&gt; &#39; kann nicht in einem Abfrageausdruck verwendet werden
Instanzen von eingeschränkten Typen, wie z. B. <xref:System.ArgIterator>, <xref:System.RuntimeArgumentHandle>und <xref:System.TypedReference>dürfen nicht in einem Lambda- oder Abfrageausdruck verwendet werden.  
  
 **Fehler-ID:** BC36598
