# <a name="value-of-type-39lttype1gt39-cannot-be-converted-to-39lttype2gt39-because-39lttype3gt39-is-not-a-reference-type"></a>Ein Wert vom Typ &#39; &lt;Typ1&gt;&#39; kann nicht konvertiert werden, um &#39;&lt; Typ2&gt;&#39; da &#39;&lt; Typ3&gt;&#39; ist kein Verweistyp
Eine Anweisung versucht, einen Arraytyp in einen anderen Arraytyp zu konvertieren, bei dem der Datentyp der Elemente des Zielarrays kein Verweistyp ist.  
  
 **Fehler-ID:** BC30333  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Überprüfen Sie die Datentypen der Elemente beider Arrays, um die Quelle des Fehlers zu bestimmen.
