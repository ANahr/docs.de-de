# <a name="variable-declaration-without-an-39as39-clause-type-of-object-assumed"></a>Variablendeklaration ohne eine As-Klausel; Typ des Objekts, die davon ausgegangen, dass
In einer Variablendeklaration ist keine `As` -Klausel angegeben.  
  
 Eine `As` -Klausel bezeichnet einen Datentyp, der einem Programmierelement zugeordnet werden soll. In einem [Dim-Anweisung](../../visual-basic/language-reference/statements/dim-statement.md), gibt den Datentyp der Variablen. Wenn Sie keine `As` -Klausel in die `Dim` -Anweisung einfügen, ist der Datentyp der Variablen standardmäßig `Object`.  
  
 Standardmäßig ist diese Meldung eine Warnung. Informationen zum Ausblenden von Warnungen oder zum Behandeln von Warnungen als Fehler finden Sie unter [Configuring Warnings in Visual Basic](https://docs.microsoft.com/visualstudio/ide/configuring-warnings-in-visual-basic).  
  
 **Fehler-ID:** BC42020  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Fügen Sie eine `As` -Klausel in die `Dim` -Anweisung ein, um den Datentyp der Variablen anzugeben.
