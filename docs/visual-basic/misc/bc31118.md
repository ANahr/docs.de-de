# <a name="end-addhandler-must-be-the-first-statement-on-a-line"></a>"End AddHandler" muss die erste Anweisung in einer Zeile sein
Eine `End AddHandler`-Anweisung folgt auf einen als Anweisungstrennzeichen verwendeten Doppelpunkt (:). `End AddHandler` muss die einzige Anweisung der jeweiligen Quellzeile sein.  
  
 **Fehler-ID:** BC31118  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Platzieren Sie jede Anweisung in einer eigenen Zeile.
