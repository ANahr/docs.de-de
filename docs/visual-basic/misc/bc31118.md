# <a name="39end-addhandler39-must-be-the-first-statement-on-a-line"></a>&#39; End AddHandler &#39; die erste Anweisung in einer Zeile muss sein
Eine `End AddHandler` -Anweisung folgt auf einen als Anweisungstrennzeichen verwendeten Doppelpunkt (:). In der entsprechenden Quellzeile darf nur`End AddHandler` als Anweisung vorkommen.  
  
 **Fehler-ID:** BC31118  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Platzieren Sie jede Anweisung in einer eigenen Zeile.
