# <a name="39ltqualifiedmethodgt39-is-not-accessible-in-this-context-because-it-is-39ltmodifiergt39"></a>&#39;&lt;qualifizierte_methode&gt; &#39; kann nicht in diesem Kontext zugegriffen werden, da es sich handelt &#39; &lt;Modifizierer&gt;&#39;
Sie haben versucht, auf eine Methode zuzugreifen, auf die in diesem Kontext aufgrund eines Modifizierers, der auf sie angewendet wurde, nicht zugegriffen werden kann.  
  
 **Fehler-ID:** BC30390  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie den Modifizierer.
