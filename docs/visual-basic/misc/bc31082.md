# <a name="39ltvariablenamegt39-is-not-a-local-variable-or-parameter-and-so-cannot-be-used-as-a-39catch39-variable"></a>&#39; &lt;Variablename&gt;&#39; ist keine lokale Variable oder Parameter, und daher nicht verwendet werden, als ein &#39; Catch &#39; Variable
Variablen in `Try...Catch...Finally` -Anweisungen müssen lokal deklariert werden oder Parameter für die aktuelle Prozedur darstellen.  
  
 **Fehler-ID:** BC31082  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Deklarieren Sie lokale Variablen oder Parameter für `Try...Catch...Finally` -Anweisungen.
