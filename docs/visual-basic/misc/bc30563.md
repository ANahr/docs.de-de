# <a name="39ltnamegt39-is-ambiguous-in-the-application-objects-39ltlistgt39"></a>&#39; &lt;Namen&gt;&#39; ist nicht eindeutig, in die Anwendungsobjekte &#39;&lt; Liste&gt;&#39;
Sie haben einen Namen angegeben, der nicht eindeutig ist und daher einen Konflikt mit einem anderen Namen verursacht. Der [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] -Compiler enthält keine Konfliktlösungsregeln. Sie müssen die Mehrdeutigkeit der Namen selbst aufheben. Die in der Fehlermeldung angegebene Liste enthält die Anwendungsobjekte, deren Name mehrdeutig ist.  
  
 **Fehler-ID:** BC30563  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Geben Sie einen vollqualifizierten Namen an.
