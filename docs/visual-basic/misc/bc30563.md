# <a name="39ltnamegt39-is-ambiguous-in-the-application-objects-39ltlistgt39"></a>'&lt;Namen&gt;'ist mehrdeutig Anwendungsobjekte'&lt;Liste&gt;'
Sie haben einen Namen angegeben, der nicht eindeutig ist und daher einen Konflikt mit einem anderen Namen verursacht. Die [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] Compiler muss keine Regeln; Sie müssen Namen sich unterscheiden. Die in der Fehlermeldung angegebene Liste enthält die Anwendungsobjekte, deren Name mehrdeutig ist.  
  
 **Fehler-ID:** BC30563  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Geben Sie einen vollqualifizierten Namen an.
