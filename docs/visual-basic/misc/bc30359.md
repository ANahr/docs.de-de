# <a name="39default39-can-be-applied-to-only-one-property-name-in-a-lttypegt"></a>&#39; Standardeinstellung &#39; kann angewendet werden, um nur einen Eigenschaftsnamen in einem &lt;Typ&gt;
Sie haben versucht, `Default` auf mehrere Eigenschaftsnamen innerhalb einer Klasse, Schnittstelle oder Struktur anzuwenden.  
  
 **Fehler-ID:** BC30359  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie alle `Default` -Deklarationen bis auf eine.
