# <a name="structure-39ltstructurenamegt39-cannot-be-indexed-because-it-has-no-default-property"></a>Struktur &#39; &lt;Structurename&gt; &#39; kann nicht indiziert werden, da es keine Standardeigenschaft vorhanden ist.
Es wurde versucht, die Syntax der Standardeigenschaft mit einer Struktur zu verwenden, die keine Standardeigenschaft hat.  
  
 **Fehler-ID:** BC30690  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Verwenden Sie die Syntax der Standardeigenschaft, um auf die Eigenschaften der Struktur zuzugreifen.
