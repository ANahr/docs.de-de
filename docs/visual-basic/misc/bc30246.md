# <a name="39ltmodifiergt39-is-not-valid-on-a-local-constant-declaration"></a>&#39; &lt;Modifizierer&gt;&#39; ist in der Deklaration von lokalen Konstanten ungültig.
Eine `Const` -Anweisung innerhalb einer Prozedur enthält ein ungültiges Schlüsselwort, z. B. `Overloads`.  
  
 **Fehler-ID:** BC30246  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Entfernen Sie das ungültige Schlüsselwort aus der `Const` -Anweisung.
