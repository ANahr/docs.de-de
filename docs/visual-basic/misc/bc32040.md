# <a name="the-39ltkeywordgt39-keyword-is-used-to-overload-inherited-members-do-not-use-the-39ltkeywordgt39-keyword-when-overloading-39sub-new39"></a>Die &#39; &lt;Schlüsselwort&gt; &#39; -Schlüsselwort wird zum Überladen von geerbten Membern verwendet; verwenden Sie nicht die &#39; &lt;Schlüsselwort&gt; &#39; Schlüsselwort beim Überladen &#39;Sub New&#39;
Ein Konstruktor wird mit dem `Overloads` -Schlüsselwort deklariert.  
  
 Visual Basic unterstützt nicht das Vererben oder Überladen von Konstruktoren.  
  
 **Fehler-ID:** BC32040  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie das `Overloads` -Schlüsselwort aus allen Konstruktordeklarationen.
