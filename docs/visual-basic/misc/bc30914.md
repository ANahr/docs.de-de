# <a name="39ltnamegt39-cannot-expose-the-underlying-delegate-type-39ltdelegatetypegt39-of-the-event-it-is-implementing-to-ltspecifier1gt-39lttype1gt39-through-ltspecifier2gt-39lttype2gt39"></a>&#39;&lt;Namen&gt; &#39; können keine verfügbar machen den zugrunde liegenden Delegattyp &#39; &lt;Delegattyp&gt; &#39; des implementierten Ereignisses &lt;spezifizierer1&gt; &#39; &lt; Typ1&gt; &#39; über &lt;spezifizierer2&gt; &#39; &lt;Typ2&gt;&#39;
