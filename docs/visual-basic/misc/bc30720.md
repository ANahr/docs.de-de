# <a name="39on-error39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Bei einem Fehler&#39; Anweisungen sind im Direktfenster nicht gültig
`On Error` -Anweisungen können nur im Quellcode verwendet werden.  
  
 **Fehler-ID:** BC30720  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie `On Error` -Anweisungen aus dem Debugcode.
