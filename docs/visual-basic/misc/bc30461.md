# <a name="classes-cannot-be-declared-39ltspecifiergt39"></a>Klassen können nicht deklariert werden &#39; &lt;Spezifizierer&gt;&#39;
Sie haben versucht, eine `Class` oder ein `Module` mit einem ungültigen Spezifizierer zu deklarieren.  
  
 **Fehler-ID:** BC30461  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie den Spezifizierer.
