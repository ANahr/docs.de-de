# <a name="39endif39-statements-are-no-longer-supported-use-39end-if39-instead"></a>&#39; EndIf &#39; Anweisungen werden nicht mehr unterstützt. Verwenden Sie &#39; Beenden, wenn &#39; Stattdessen
Die `EndIf` -Anweisung wird nicht mehr unterstützt.  
  
 **Fehler-ID:** BC30826  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Verwenden Sie `End If` , um `If` -Anweisungen zu beenden.
