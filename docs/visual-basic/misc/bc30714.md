# <a name="39resume39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Resume&#39; Anweisungen sind im Direktfenster nicht gültig
Resume-Anweisungen können nur im Quellcode verwendet werden.  
  
 **Fehler-ID:** BC30714  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie `Resume` -Anweisungen aus dem Debugcode.
