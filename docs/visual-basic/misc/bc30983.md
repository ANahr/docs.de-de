# <a name="type-of-39ltvariablenamegt39-is-ambiguous-because-the-loop-bounds-and-the-step-variable-do-not-widen-to-the-same-type"></a><span data-ttu-id="3bf92-102">Geben Sie der &#39; &lt;Variablename&gt; &#39; ist mehrdeutig, weil die schleifenbegrenzungen und die Step-Variable nicht auf den gleichen Typ erweitert werden</span><span class="sxs-lookup"><span data-stu-id="3bf92-102">Type of &#39;&lt;variablename&gt;&#39; is ambiguous because the loop bounds and the step variable do not widen to the same type</span></span>
<span data-ttu-id="3bf92-103">Der Code enthält eine `For...Next` -Schleife, in der der Compiler aufgrund der folgenden Bedingungen keinen Datentyp für die Schleifensteuerungsvariable ableiten kann:</span><span class="sxs-lookup"><span data-stu-id="3bf92-103">Your code contains a `For...Next` loop in which the compiler cannot infer a data type for the loop control variable because the following conditions is true:</span></span>  
  
-   <span data-ttu-id="3bf92-104">Der Datentyp der Schleifensteuerungsvariablen wird nicht mit einer `As` -Klausel angegeben.</span><span class="sxs-lookup"><span data-stu-id="3bf92-104">The data type of the loop control variable is not specified with an `As` clause.</span></span>  
  
-   <span data-ttu-id="3bf92-105">Die Schleifenbegrenzungen und die step-Variable enthalten mindestens zwei Datentypen.</span><span class="sxs-lookup"><span data-stu-id="3bf92-105">The loop bounds and step variable contain at least two data types.</span></span>  
  
-   <span data-ttu-id="3bf92-106">Zwischen den Datentypen sind mehrere Konvertierungen möglich.</span><span class="sxs-lookup"><span data-stu-id="3bf92-106">More than one possible conversion exists between the data types.</span></span>  
  
-   <span data-ttu-id="3bf92-107">Es gibt keinen optimalen Typ unter den Kandidaten, sodass die Auswahl des Typs für die Schleifensteuerungsvariable mehrdeutig ist.</span><span class="sxs-lookup"><span data-stu-id="3bf92-107">There is no best type among the candidates, so that the choice of type for the loop control variable is ambiguous.</span></span>  
  
 <span data-ttu-id="3bf92-108">Die folgende Schleife weist beispielsweise eine Schleifenbegrenzung vom Typ `Integer` und eine Schleifenbegrenzung vom Typ `UInteger`auf:</span><span class="sxs-lookup"><span data-stu-id="3bf92-108">For example, the following loop has one loop bound of type `Integer` and one loop bound of type `UInteger`:</span></span>  
  
```vb  
Dim m As Integer = 1  
Dim n As UInteger = 10  
' Not valid.  
' For i = m To n  
    ' Loop processing.  
' Next  
```  
  
 <span data-ttu-id="3bf92-109">Es wird eine mögliche Konvertierung zwischen `Integer` und `UInteger`sowie eine mögliche Konvertierung zwischen `UInteger` und `Integer`unterstützt, bei beiden handelt es sich jedoch um einschränkende Konvertierungen, sodass keine der beiden eine optimale Wahl darstellt.</span><span class="sxs-lookup"><span data-stu-id="3bf92-109">There is a possible conversion between `Integer` and `UInteger`, and a possible conversion between `UInteger` and `Integer`, but both are narrowing conversions so neither is the best choice.</span></span>  
  
 <span data-ttu-id="3bf92-110">Im nächsten Beispiel wird eine dritte Variable des Typs `Double` hinzugefügt.</span><span class="sxs-lookup"><span data-stu-id="3bf92-110">In the next example, a third variable of type `Double` is added.</span></span> <span data-ttu-id="3bf92-111">Sowohl `Integer` als auch `UInteger` verfügen über standardmäßige Erweiterungskonvertierungen für `Double`, wodurch die Konvertierung in `Double` die beste Wahl darstellt.</span><span class="sxs-lookup"><span data-stu-id="3bf92-111">Both `Integer` and `UInteger` have standard widening conversions to `Double`, which makes conversion to `Double` the best choice.</span></span> <span data-ttu-id="3bf92-112">Durch den Typrückschluss wird dem Datentyp `i` die Schleifensteuerungsvariable `Double`zugewiesen.</span><span class="sxs-lookup"><span data-stu-id="3bf92-112">Type inference assigns to loop control variable `i` the data type `Double`.</span></span>  
  
```vb  
Dim stepVar As Double = 1  
' Valid.  
For i = m To n Step stepVar  
    ' Loop processing.  
Next  
```  
  
 <span data-ttu-id="3bf92-113">**Fehler-ID:** BC30983</span><span class="sxs-lookup"><span data-stu-id="3bf92-113">**Error ID:** BC30983</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="3bf92-114">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="3bf92-114">To correct this error</span></span>  
  
-   <span data-ttu-id="3bf92-115">Konvertieren Sie eine der Variablen explizit in einen Typ, für den die anderen über eine Erweiterungskonvertierung verfügen. Beispiel:</span><span class="sxs-lookup"><span data-stu-id="3bf92-115">Explicitly convert one of the variables to a type that the others have a widening conversion to, for example:</span></span>  
  
    ```  
    For i = m To CLng(n)  
    ```  
  
-   <span data-ttu-id="3bf92-116">Verwenden Sie eine `As` -Klausel, um den Typ der Steuerelementvariablen anzugeben:</span><span class="sxs-lookup"><span data-stu-id="3bf92-116">Use an `As` clause to specify the type of the control variable:</span></span>  
  
    ```  
    For i As Double = m To n   
    ```
