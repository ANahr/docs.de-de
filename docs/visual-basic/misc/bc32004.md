# <a name="39ltnamegt39-is-declared-in-project-39ltprojectname1gt39-which-is-not-referenced-by-project-39ltprojectname2gt39"></a>&#39;&lt;Namen&gt; &#39; im Projekt deklarierten &#39; &lt;projektname1&gt;&#39;, die keinen Verweis durch Projekt &#39; &lt;projektname2&gt;&#39;
Ein Programmierelement ist in einem Projekt definiert, auf das im aktuellen Projekt kein Verweis vorhanden ist.  
  
 **Fehler-ID:** BC32004  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Fügen Sie im aktuellen Projekt einen Verweis auf das Projekt hinzu, das das gewünschte Element enthält.
