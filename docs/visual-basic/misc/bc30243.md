# <a name="39ltmodifiergt39-is-not-valid-on-an-event-declaration"></a>&#39;&lt;Modifizierer&gt; &#39; ist in einer Ereignisdeklaration ungültig.
Eine `Event` -Anweisung enthält ein ungültiges Schlüsselwort, z. B. `ReadOnly`.  
  
 **Fehler-ID:** BC30243  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Entfernen Sie das ungültige Schlüsselwort aus der `Event` -Anweisung.
