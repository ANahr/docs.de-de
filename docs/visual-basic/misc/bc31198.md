<span data-ttu-id="ed4fb-103">Eine Deklaration der XML-Literale wird in einem Ausdruck an einem Ort verwendet, die für die Visual Basic-Compiler mehrdeutig ist.</span><span class="sxs-lookup"><span data-stu-id="ed4fb-103">An XML literal declaration is used in an expression in a location that is ambiguous to the Visual Basic compiler.</span></span> <span data-ttu-id="ed4fb-104">Das heißt, die Visual Basic-Compiler kann nicht bestimmen, ob das kleiner-als Zeichen (<) als Vergleichsoperator oder Startzeichen am Anfang eines XML-Literals vorgesehen ist.</span><span class="sxs-lookup"><span data-stu-id="ed4fb-104">That is, the Visual Basic compiler cannot determine whether the less-than character (<) is intended as a comparative operator or the start of an XML literal.</span></span> <span data-ttu-id="ed4fb-105">Der folgende Code veranschaulicht dies:</span><span class="sxs-lookup"><span data-stu-id="ed4fb-105">The following code provides an example:</span></span>  

```vb  
' Generates an error.  
Dim queryResult = From element In elements _

                  Select element  
```

## <a name="to-correct-this-error"></a><span data-ttu-id="ed4fb-107">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="ed4fb-107">To correct this error</span></span>
