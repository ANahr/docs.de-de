# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-definition-for-event-39lteventnamegt39"></a>Ein Verweis auf das Modul &#39;erforderlich; &lt;"ModuleName"&gt;&#39; mit der Definition für das Ereignis &#39;&lt; Ereignisname&gt;&#39;
Verweis auf das Modul erforderlich "<`modulename`>", enthält die Definition für das Ereignis ' <`eventname`> ". Fügen Sie dem Projekt einen Verweis hinzu.  
  
 Das Ereignis ist in einem Modul definiert, auf das Ihr Projekt nicht direkt verweist. Der [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] -Compiler benötigt einen Verweis, um Mehrdeutigkeiten zu vermeiden, falls das Ereignis in mehr als einem Modul definiert ist.  
  
 **Fehler-ID:** BC30006  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Nehmen Sie den Namen des nicht referenzierten Moduls in Ihre Projektverweise auf.
