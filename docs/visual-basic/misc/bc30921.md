# <a name="39ltderivedtypenamegt39-cannot-inherit-from-lttypegt-39ltconstructedbasetypenamegt39-because-it-expands-the-access-of-type-39ltinternaltypenamegt39-to-ltregiongt-39ltregionnamegt39"></a><span data-ttu-id="fe14d-102">&#39; &lt;Derivedtypename&gt;&#39; kann nicht Vererben &lt;Typ&gt; &#39;&lt; konstruierter_basistypname&gt;&#39; da sie den Zugriff vom Typ &#39; erweitert&lt; Internaltypename&gt;&#39; in &lt;Region&gt; &#39;&lt; Regionsname&gt;&#39;</span><span class="sxs-lookup"><span data-stu-id="fe14d-102">&#39;&lt;derivedtypename&gt;&#39; cannot inherit from &lt;type&gt; &#39;&lt;constructedbasetypename&gt;&#39; because it expands the access of type &#39;&lt;internaltypename&gt;&#39; to &lt;region&gt; &#39;&lt;regionname&gt;&#39;</span></span>
<span data-ttu-id="fe14d-103">Eine abgeleitete Klasse oder Schnittstelle versucht, die Zugriffsebene eines internen Typs zu erweitern, indem sie diesen als Typargument für eine Basisklasse oder Schnittstelle verwendet.</span><span class="sxs-lookup"><span data-stu-id="fe14d-103">A derived class or interface attempts to expand the access level of an internal type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="fe14d-104">Dieser Fehler kann durch folgenden Code generiert werden.</span><span class="sxs-lookup"><span data-stu-id="fe14d-104">The following code can generate this error.</span></span>  
  
```  
Public Class containingClass  
    Public Class baseClass(Of t)  
    End Class  
    Friend Class derivedClass  
        Inherits baseClass(Of internalStructure)  
    End Class  
    Private Structure internalStructure  
        Dim firstMember As Integer  
    End Structure  
End Class  
```  
  
 <span data-ttu-id="fe14d-105">Code außerhalb der `containingClass` darf nicht auf die `internalStructure`zugreifen.</span><span class="sxs-lookup"><span data-stu-id="fe14d-105">Code outside `containingClass` is not allowed to access `internalStructure`.</span></span> <span data-ttu-id="fe14d-106">Allerdings kann jeder Code in derselben Assembly auf die `derivedClass` zugreifen.</span><span class="sxs-lookup"><span data-stu-id="fe14d-106">However, `derivedClass` can be accessed from any code in the same assembly.</span></span> <span data-ttu-id="fe14d-107">Daher kann die `derivedClass` die `baseClass` nicht erben, wenn sie die `internalStructure` als Typargument verwendet, da auf diese Weise die `internalStructure` in der gesamten definierenden Coderegion verfügbar gemacht würde.</span><span class="sxs-lookup"><span data-stu-id="fe14d-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `internalStructure` as a type argument, because that could expose `internalStructure` throughout the defining code region.</span></span>  
  
 <span data-ttu-id="fe14d-108">**Fehler-ID:** BC30921</span><span class="sxs-lookup"><span data-stu-id="fe14d-108">**Error ID:** BC30921</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="fe14d-109">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="fe14d-109">To correct this error</span></span>  
  
-   <span data-ttu-id="fe14d-110">Passen Sie die Zugriffsebenen der Klassen oder Schnittstellen so an, dass der abgeleitete Typ die Zugriffsebene des internen Typs nicht erweitert.</span><span class="sxs-lookup"><span data-stu-id="fe14d-110">Adjust the access levels of the classes or interfaces so that the derived type does not expand the access level of the internal type.</span></span>  
  
     <span data-ttu-id="fe14d-111">- oder -</span><span class="sxs-lookup"><span data-stu-id="fe14d-111">-or-</span></span>
