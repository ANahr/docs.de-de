# <a name="class-39ltclassnamegt39-could-not-be-created-lterrorgt"></a>Klasse&lt;Classname&gt;' konnte nicht erstellt werden: &lt;Fehler&gt;
Bei der Initialisierung einer Klasse ist ein Fehler aufgetreten. Eine mögliche Ursache ist eine beschädigte [!INCLUDE[dnprdnshort](../../csharp/getting-started/includes/dnprdnshort_md.md)] Installation.  
  
 **Fehler-ID:** BC32400  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Überprüfen Sie die angegebene Fehlermeldung, und ergreifen Sie entsprechende Maßnahmen.  
  
2.  Installieren der [!INCLUDE[dnprdnshort](../../csharp/getting-started/includes/dnprdnshort_md.md)] und kompilieren Sie das Programm neu.  
  
3.  Wenn der Fehler erneut auftritt, installieren Sie den [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]-Compiler neu.  
  
4.  Wenn der Fehler nach der Neuinstallation weiterhin besteht, tragen Sie Informationen zu den Umständen zusammen, und benachrichtigen Sie den Produktsupport von Microsoft.
