# <a name="visual-basic-compiler-is-unable-to-recover-from-the-following-error-lterrorgt"></a>Visual Basic-Compiler wird mit folgendem Fehler beendet: &lt;Fehler&gt;
Die [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] Compiler wird mit folgendem Fehler beendet: `error`>.  
  
 Einige IDE-Features wurden deaktiviert.  
  
 Ein interner Fehler der [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] Compiler.  
  
 **Fehler-ID:** BC31021  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Speichern Sie Ihre Arbeit und starten Sie [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)].  
  
2.  Wenn der Fehler erneut auftritt, installieren Sie [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)] neu.  
  
3.  Wenn der Fehler auch nach der erneuten Installation auftritt, informieren Sie den Produktsupport von Microsoft.
