# <a name="visual-basic-compiler-is-unable-to-recover-from-the-following-error-lterrorgt"></a>Visual Basic-Compiler wird der folgende Fehler wiederhergestellt werden konnte: &lt;Fehler&gt;
Visual Basic-Compiler wird der folgende Fehler wiederhergestellt werden konnte: <`error`>.  
  
 Einige IDE-Features wurden deaktiviert.  
  
 In Visual Basic-Compiler ist ein interner Fehler aufgetreten.  
  
 **Fehler-ID:** BC31021  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Speichern Sie Ihre Arbeit, und starten Sie Visual Studio neu.  
  
2.  Wenn der Fehler erneut auftritt, installieren Sie Visual Studio.  
  
3.  Wenn der Fehler auch nach der erneuten Installation auftritt, informieren Sie den Produktsupport von Microsoft.
