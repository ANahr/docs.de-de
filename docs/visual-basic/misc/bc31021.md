# <a name="visual-basic-compiler-is-unable-to-recover-from-the-following-error-lterrorgt"></a>Visual Basic-Compiler wird der folgende Fehler wiederhergestellt werden konnte: &lt;Fehler&gt;
Die [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] -Compiler wird die Wiederherstellung des folgenden Fehlers: <`error`>.  
  
 Einige IDE-Features wurden deaktiviert.  
  
 Ein interner Fehler ist im [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] -Compiler aufgetreten.  
  
 **Fehler-ID:** BC31021  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Speichern Sie Ihre Arbeit, und starten Sie [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)]erneut.  
  
2.  Wenn der Fehler erneut auftritt, installieren Sie [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)]neu.  
  
3.  Wenn der Fehler auch nach der erneuten Installation auftritt, informieren Sie den Produktsupport von Microsoft.
