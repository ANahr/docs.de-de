# <a name="39of39-required-when-specifying-type-arguments-for-a-generic-type-or-method"></a>&#39;Der&#39; erforderlich, wenn die Angabe von Typargumenten für generische Typen oder -Methode
Eine Anweisung versucht, ohne Verwendung einer [Of](../../visual-basic/language-reference/statements/of-clause.md) -Klausel einen Typ aus einem generischen Typ zu erstellen bzw. eine generische Methode aufzurufen.  
  
 Visual Basic-Syntax für generische Typen erfordert, Typparameter und Typargumente eingeführt wird, indem Sie die `Of` Schlüsselwort. Zusätzlich müssen Typparameterliste und Typargumentliste in Klammern eingeschlossen sein.  
  
 **Fehler-ID:** BC32093  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Fügen Sie das `Of` -Schlüsselwort am Anfang der Typargumentliste hinzu, und schließen Sie die gesamte Liste in Klammern ein.
