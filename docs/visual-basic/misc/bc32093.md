# <a name="39of39-required-when-specifying-type-arguments-for-a-generic-type-or-method"></a>"Of" erforderlich, bei der Angabe von Typargumenten für generischen Typs oder einer Methode
Es wurde versucht, einen Typ eines generischen Typs zu erstellen oder eine generische Methode aufzurufen, ohne eine [von](../../visual-basic/language-reference/statements/of-clause.md) Klausel.  
  
 Die [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] Syntax für generische Typen erfordert, dass der Typparameter und Typargumente mit eingeführt werden die `Of` Schlüsselwort. Zusätzlich müssen Typparameterliste und Typargumentliste in Klammern eingeschlossen sein.  
  
 **Fehler-ID:** BC32093  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Fügen Sie das `Of` -Schlüsselwort am Anfang der Typargumentliste hinzu, und schließen Sie die gesamte Liste in Klammern ein.
