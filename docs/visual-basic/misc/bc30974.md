# <a name="comment-statements-cannot-be-evaluated-not-in-toc"></a>Comment-Anweisungen können nicht ausgewertet werden, (nicht im Inhaltsverzeichnis)
In das Direktfenster wurde während einer Debugsitzung eine Comment-Anweisung eingegeben. Comment-Anweisungen werden im Direktfenster nicht ausgewertet.  
  
 **Fehler-ID:** BC30974
