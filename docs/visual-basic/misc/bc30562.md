# <a name="39ltnamegt39-is-ambiguous-between-declarations-in-modules-39ltmodulename1gt39-and-39ltmodulename2gt39"></a>&#39; &lt;Namen&gt;&#39; ist zwischen den Deklarationen in den Modulen &#39;&lt; modulname1&gt;&#39; und &#39;&lt; modulename2&gt;&#39;
Sie haben einen Namen angegeben, der nicht eindeutig ist und daher einen Konflikt mit einem anderen Namen verursacht. Der [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] -Compiler verfügt nicht über Regeln zur Konfliktlösung; Sie müssen selbst für unterscheidbare Namen sorgen.  
  
 **Fehler-ID:** BC30562  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Geben Sie einen vollqualifizierten Namen an.
