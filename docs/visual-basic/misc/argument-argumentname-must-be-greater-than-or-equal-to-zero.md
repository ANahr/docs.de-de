# <a name="argument-39ltargumentnamegt39-must-be-greater-than-or-equal-to-zero"></a>Argument &#39; &lt;Argumentname&gt;&#39; muss größer als oder gleich null sein
Ein Argument ist ungültig, da es kleiner als 0 (null) ist.  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Überprüfen Sie die Schreibweise der Argumente. Ein falsch geschriebener Variablenname kann implizit eine numerische Variable erzeugen, die mit 0 (null) initialisiert wird.  
  
2.  Überprüfen Sie vorherige Vorgänge für Variablen im Ausdruck, insbesondere solche, die als Argumente aus anderen Prozeduren an die Prozedur übergeben werden.
