# Ung&#252;ltiges Bild.
Sie haben versucht, der `Picture`\-Eigenschaft eines Formulars oder Steuerelements ein anderes Grafikformat als ein Bitmap\-, Symbol\- oder Windows Metafile\-Format zuzuweisen.  
  
### So beheben Sie diesen Fehler  
  
1.  Stellen Sie sicher, dass die Datei, die Sie in die `Picture`\-Eigenschaft laden möchten, eine gültige, von [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] unterstützte Grafikdatei ist.
