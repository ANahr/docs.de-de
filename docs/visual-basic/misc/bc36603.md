# <a name="multi-dimensional-array-cannot-be-converted-to-an-expression-tree"></a><span data-ttu-id="7da49-102">Ein mehrdimensionales Array kann nicht in eine Ausdrucksbaumstruktur konvertiert werden.</span><span class="sxs-lookup"><span data-stu-id="7da49-102">Multi-dimensional array cannot be converted to an expression tree</span></span>

```vb  
Module Module1  
    Sub Main()  
        '' A multi-dimensional array cannot be converted.  
        'Dim expTree As Expressions.Expression(Of Func(Of Object)) = _  
        '    Function() New Integer(1, 1) {{1, 2}, {2, 3}}  
  
        ' A one-dimensional array can be converted.  
        Dim expTree2 As Expressions.Expression(Of Func(Of Object)) = _

    End Sub  
End Module  
```
