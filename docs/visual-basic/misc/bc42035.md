# Ung&#252;ltiges GUID-Format.
Eine GUID wurde in einem ungültigen Format angegeben.  
  
 **Fehler\-ID:** BC42035  
  
### So beheben Sie diesen Fehler  
  
-   Überprüfen Sie das GUID\-Format.
