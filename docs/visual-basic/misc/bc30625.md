# <a name="39module39-statement-must-end-with-a-matching-39end-module39"></a>&#39;Modul&#39; Anweisung enden mit einem übereinstimmenden &#39;End Module&#39;
Modulblöcke müssen mit einer `End``Module` -Anweisung enden.  
  
 **Fehler-ID:** BC30625  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Fügen Sie eine `End Module` -Anweisung ans Ende des `Module` -Blocks hinzu.
