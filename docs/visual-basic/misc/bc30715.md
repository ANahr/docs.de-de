# <a name="39catch39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Abfangen&#39; Anweisungen sind im Direktfenster nicht gültig
`Catch` -Anweisungen können nur im Quellcode verwendet werden.  
  
 **Fehler-ID:** BC30715  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie `Catch` -Anweisungen aus dem Debugcode.
