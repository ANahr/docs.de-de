# <a name="39ltprocedurenamegt39-has-a-return-type-that-is-not-supported-or-parameter-types-that-are-not-supported"></a>&#39;&lt;Prozedurname&gt; &#39; verfügt über einen Rückgabetyp aufweisen, die nicht unterstützt wird oder Parametertypen, die nicht unterstützt werden
Eine Prozedur wird deklariert, Parameter oder einen Rückgabetyp aufweisen, der von Visual Studio nicht dargestellt werden kann.  
  
 **Fehler-ID:** BC30657  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Definieren Sie die Rückgabetypen oder Parameter als unterstützten Typ.  
  
-   Entwickeln Sie einen neuen Typ, der die von dieser Prozedur verwendeten Daten beschreibt.
