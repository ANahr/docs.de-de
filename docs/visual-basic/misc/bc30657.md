# <a name="39ltprocedurenamegt39-has-a-return-type-that-is-not-supported-or-parameter-types-that-are-not-supported"></a>'&lt;Prozedurname&gt;"hat einen Rückgabetyp aufweisen, die nicht unterstützt wird oder Parametertypen, die nicht unterstützt werden
Deklariert eine Prozedur mit Parametern oder einen Rückgabetyp aufweisen, die von dargestellt werden kann [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)].  
  
 **Fehler-ID:** BC30657  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Definieren Sie die Rückgabetypen oder Parameter als unterstützten Typ.  
  
-   Entwickeln Sie einen neuen Typ, der die von dieser Prozedur verwendeten Daten beschreibt.
