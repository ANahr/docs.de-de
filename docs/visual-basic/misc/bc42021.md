# <a name="function-without-an-39as39-clause-return-type-of-object-assumed"></a>Funktion ohne eine As-Klausel. der Rückgabetyp des Objekt wird angenommen.
Eine `Function` -Prozedur gibt keine `As` -Klausel an.  
  
 Eine `As` -Klausel bezeichnet einen Datentyp, der einem Programmierelement zugeordnet werden soll. In einem [Function-Anweisung](../../visual-basic/language-reference/statements/function-statement.md), gibt Sie den Datentyp des Werts der `Function` Prozedur an den aufrufenden Code zurückgegeben. Wenn Sie keine `As` -Klausel in die `Function` -Anweisung einfügen, lautet der Rückgabedatentyp standardmäßig `Object`.  
  
 Standardmäßig ist diese Meldung eine Warnung. Informationen zum Ausblenden von Warnungen oder zum Behandeln von Warnungen als Fehler finden Sie unter [Configuring Warnings in Visual Basic](https://docs.microsoft.com/visualstudio/ide/configuring-warnings-in-visual-basic).  
  
 **Fehler-ID:** BC42021  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Fügen Sie eine `As` -Klausel in die `Function` -Anweisung ein, um den Rückgabedatentyp anzugeben.
