# <a name="39ltargumentnamegt39-is-not-a-parameter-of-39ltmethoddeclarationgt39"></a>&#39; &lt;Argumentname&gt;&#39; ist kein Parameter von &#39;&lt; MethodDeclaration&gt;&#39;
Ein Prozeduraufruf liefert ein zusätzliches Argument, das nicht von der Prozedur definiert wird.  
  
 **Fehler-ID:** BC30272  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie das überflüssige Argument aus der Argumentliste.
