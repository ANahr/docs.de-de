# <a name="type-parameter-39lttypeparametername1gt39-must-have-either-a-39new39-constraint-or-a-39structure39-constraint-to-satisfy-the-39new39-constraint-for-type-parameter-39lttypeparametername2gt39"></a><span data-ttu-id="78197-102">Typparameter &#39; &lt;typparametername1&gt;&#39; muss eine &#39; Neue &#39; Einschränkung oder einem &#39; Struktur &#39; Einschränkung erfüllen die &#39; Neue &#39; Einschränkung für den Typparameter &#39; &lt;typparametername2&gt;&#39;</span><span class="sxs-lookup"><span data-stu-id="78197-102">Type parameter &#39;&lt;typeparametername1&gt;&#39; must have either a &#39;New&#39; constraint or a &#39;Structure&#39; constraint to satisfy the &#39;New&#39; constraint for type parameter &#39;&lt;typeparametername2&gt;&#39;</span></span>
<span data-ttu-id="78197-103">Eine Anweisung erstellt einen generischen Typ, der einen Typparameter übergibt, der nicht zur Erfüllung der `New` -Einschränkung beschränkt ist.</span><span class="sxs-lookup"><span data-stu-id="78197-103">A statement constructs a generic type passing a type parameter that is not constrained to satisfy a `New` constraint.</span></span>  
  
 <span data-ttu-id="78197-104">Die `New` -Einschränkung gibt vor, dass das für den Typparameter angegebene Typargument einen parameterlosen Konstruktor für den Code verfügbar machen muss, der Objekte daraus erstellt.</span><span class="sxs-lookup"><span data-stu-id="78197-104">The `New` constraint means that the type argument supplied to that type parameter must expose a parameterless constructor accessible to the code that creates objects from it.</span></span> <span data-ttu-id="78197-105">Alle Werttypen verfügen über parameterlose Konstruktoren, jedoch nicht alle Verweistypen.</span><span class="sxs-lookup"><span data-stu-id="78197-105">All value types have parameterless constructors, but not all reference types do.</span></span> <span data-ttu-id="78197-106">Daher erfüllt die `Structure` -Einschränkung die `New` -Einschränkung, aber die `Class` -Einschränkung bzw. eine Klasse oder ein Schnittstellenname nicht.</span><span class="sxs-lookup"><span data-stu-id="78197-106">Therefore the `Structure` constraint satisfies the `New` constraint, but the `Class` constraint, or a class or interface name, does not.</span></span>  
  
 <span data-ttu-id="78197-107">Dieser Fehler kann durch die folgenden Anweisungen generiert werden.</span><span class="sxs-lookup"><span data-stu-id="78197-107">The following statements can generate this error.</span></span>

```  
End Class

    Public testObject As New c1(Of u)  
End Class  
```  
  
 <span data-ttu-id="78197-108">Wenn die Klasse `c2` ein Objekt aus `c1`erstellt, erfüllt der Typparameter `u` nicht die `New` -Einschränkung für den Typparameter `t`.</span><span class="sxs-lookup"><span data-stu-id="78197-108">When class `c2` creates an object from `c1`, type parameter `u` does not satisfy the `New` constraint on type parameter `t`.</span></span>
