# <a name="39lttypenamegt39-cannot-be-used-as-an-attribute-because-it-is-not-a-class"></a>&#39; &lt;Typename&gt;&#39; kann nicht als Attribut verwendet werden, da es sich nicht um eine Klasse handelt.
Es wurde versucht, einen Typ, der keine Klasse ist, als Attribut zu verwenden.  
  
 **Fehler-ID:** BC31503  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Definieren Sie benutzerdefinierte Attribute als von `System.Attribute`abgeleitete Klassen.
