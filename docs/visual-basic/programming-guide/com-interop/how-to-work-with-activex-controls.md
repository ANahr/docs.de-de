# <a name="how-to-work-with-activex-controls-visual-basic"></a>Gewusst wie: Verwenden von ActiveX-Steuerelementen (Visual Basic)
ActiveX-Steuerelemente sind COM-Komponenten oder Objekte, die Sie in einer Webseite oder einer anderen Anwendung auf eine andere Person programmiert hat wiederverwenden einfügen können. Können ActiveX-Steuerelemente für Visual Basic 6.0 und früheren Versionen entwickelten Features hinzufügen die **Toolbox** von [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs_md.md)].  
  
### <a name="to-add-activex-controls-to-the-toolbox"></a>ActiveX-Steuerelemente der Toolbox hinzu  
  
1.  Auf der **Tools** Menü klicken Sie auf **Toolboxelemente**.  
  
     Die **Toolboxelemente** das Dialogfeld wird angezeigt.  
  
2.  Klicken Sie auf die **COM-Komponenten** Registerkarte.  
  
3.  Aktivieren Sie das Kontrollkästchen neben dem ActiveX-Steuerelement zu verwenden, und klicken Sie dann auf **OK**.  
  
     Das neue Steuerelement angezeigt wird, mit den anderen Tools in der **Toolbox**.  
  
    > [!NOTE]
    >  Das Dienstprogramm Aximp können Sie eine Interop-Assembly für ActiveX-Steuerelemente manuell erstellen. Weitere Informationen finden Sie unter [Aximp.exe (Windows Forms ActiveX Control Importer)](http://msdn.microsoft.com/library/482c0d83-7144-4497-b626-87d2351b78d0).
