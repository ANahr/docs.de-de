# <a name="linq-to-xml-visual-basic"></a>LINQ to XML (Visual Basic)
LINQ to XML stellt eine speicherinterne XML-Programmierschnittstelle, die das .NET Language-Integrated Query (LINQ)-Framework nutzt. LINQ to XML verwendet die neuesten Funktionen der .NET Framework-Sprache und ist vergleichbar mit einer aktualisierten und umgestalteten (DOM = Document Object Model)-XML-Programmierschnittstelle.  
  
 Die LINQ-Technologiefamilie bietet ein konsistentes Abfrageerlebnis für Objekte (LINQ to Objects), relationale Datenbanken (LINQ to SQL) und XML (LINQ to XML).
