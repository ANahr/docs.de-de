# <a name="programming-guide-linq-to-xml-visual-basic"></a>Programmierhandbuch (LINQ to XML) (Visual Basic)
Dieser Abschnitt enthält grundlegende Informationen und Anleitungen zum Programmieren mit [!INCLUDE[sqltecxlinq](../../../../csharp/programming-guide/concepts/linq/includes/sqltecxlinq_md.md)].  
  
## <a name="who-should-read-this-documentation"></a>Zielgruppe  
 Diese Dokumentation wendet sich an Entwickler, die bereits Visual Basic und einige grundlegende Aspekte der kennen die [!INCLUDE[dnprdnshort](../../../../csharp/getting-started/includes/dnprdnshort_md.md)].  
  
 Das Ziel dieser Dokumentation ist es, [!INCLUDE[sqltecxlinq](../../../../csharp/programming-guide/concepts/linq/includes/sqltecxlinq_md.md)] für alle Arten von Entwicklern einfach zu verwenden. [!INCLUDE[sqltecxlinq](../../../../csharp/programming-guide/concepts/linq/includes/sqltecxlinq_md.md)]vereinfacht die XML-Programmierung. und kann auch von Entwicklern verwendet werden, die keine ausgesprochenen Experten auf diesem Gebiet sind.  
  
 [!INCLUDE[sqltecxlinq](../../../../csharp/programming-guide/concepts/linq/includes/sqltecxlinq_md.md)]basiert in weiten Teilen auf generischen Klassen. Folglich ist sehr wichtig, dass Sie die Verwendung von generischen Typen und Klassen verstehen. Weitere Informationen finden Sie unter [generische Typen in Visual Basic](../../../../visual-basic/programming-guide/language-features/data-types/generic-types.md).
