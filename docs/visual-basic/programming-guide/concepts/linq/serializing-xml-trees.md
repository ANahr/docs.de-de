# <a name="serializing-xml-trees-visual-basic"></a>Serialisieren von XML-Strukturen (Visual Basic)
Beim Serialisieren einer XML-Struktur wird aus der XML-Struktur XML generiert. Serialisierung kann in eine Datei, in eine konkrete Implementierung der <xref:System.IO.TextWriter>-Klasse oder in eine konkrete Implementierung einer <xref:System.Xml.XmlWriter>.</xref:System.Xml.XmlWriter> </xref:System.IO.TextWriter>  
  
 Sie können verschiedene Aspekte der Serialisierung steuern. So können Sie z. B. steuern, ob das serialisierte XML mit Einzügen versehen und ob eine XML-Deklaration geschrieben werden soll.
