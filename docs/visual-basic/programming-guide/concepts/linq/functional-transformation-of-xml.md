Zur Form gehört Folgendes:</span><span class="sxs-lookup"><span data-stu-id="a7c46-106">The shape of an XML document is the structure of the document, which includes the following:</span></span>

-   <span data-ttu-id="a7c46-109">die Datentypen der Elemente und Attribute</span><span class="sxs-lookup"><span data-stu-id="a7c46-109">The data types of the elements and attributes.</span></span>
