# <a name="linq-and-reflection-visual-basic"></a>LINQ und Reflektion (Visual Basic)

Da diese Sammlungen unterstützt, die die generische `IEnumerable` -Schnittstelle, sie können mithilfe von LINQ abgefragt werden.
