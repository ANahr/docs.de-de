# <a name="tutorial-deferred-execution-visual-basic"></a>Lernprogramm: Verzögerte Ausführung (Visual Basic)

 Bei Abfragen, die Office Open XML-Dokumente verarbeiten, kommt dieses Verfahren sehr häufig vor.
