<span data-ttu-id="fe4af-103">Stößt der Visual Basic-Compiler einen Aufruf an eine Prozedur, die in mehreren überladenen Versionen definiert ist, muss der Compiler die Überladung aufrufen, entscheiden.</span><span class="sxs-lookup"><span data-stu-id="fe4af-103">When the Visual Basic compiler encounters a call to a procedure that is defined in several overloaded versions, the compiler must decide which of the overloads to call.</span></span> <span data-ttu-id="fe4af-104">Dies geschieht durch die folgenden Schritte ausführen:</span><span class="sxs-lookup"><span data-stu-id="fe4af-104">It does this by performing the following steps:</span></span>  
  
1.  <span data-ttu-id="fe4af-105">**Barrierefreiheit.**</span><span class="sxs-lookup"><span data-stu-id="fe4af-105">**Accessibility.**</span></span> <span data-ttu-id="fe4af-106">Überladung mit der Zugriffsebene, die verhindert, dass den aufrufenden Code Aufrufen dieser beseitigt.</span><span class="sxs-lookup"><span data-stu-id="fe4af-106">It eliminates any overload with an access level that prevents the calling code from calling it.</span></span>  
  
2.  <span data-ttu-id="fe4af-107">**Die Anzahl von Parametern.**</span><span class="sxs-lookup"><span data-stu-id="fe4af-107">**Number of Parameters.**</span></span> <span data-ttu-id="fe4af-108">Überladung, die eine andere Anzahl von Parametern definiert als angegeben wird, werden im Aufruf beseitigt.</span><span class="sxs-lookup"><span data-stu-id="fe4af-108">It eliminates any overload that defines a different number of parameters than are supplied in the call.</span></span>

     <span data-ttu-id="fe4af-113">In diesem Schritt aber es wird Überladung für die die Datentypen der aufrufenden Argumente in die Parametertypen, die in der Überladung definiert konvertiert werden können.</span><span class="sxs-lookup"><span data-stu-id="fe4af-113">In this step, it eliminates any overload for which the data types of the calling arguments cannot be converted to the parameter types defined in the overload.</span></span>

5.  <span data-ttu-id="fe4af-117">**Geringste Erweiterung.**</span><span class="sxs-lookup"><span data-stu-id="fe4af-117">**Least Widening.**</span></span> <span data-ttu-id="fe4af-118">Der Compiler betrachtet die verbleibenden Überladungen paarweise zugeordnet.</span><span class="sxs-lookup"><span data-stu-id="fe4af-118">The compiler considers the remaining overloads in pairs.</span></span> <span data-ttu-id="fe4af-119">Für jedes Paar vergleicht er die Datentypen der definierten Parameter.</span><span class="sxs-lookup"><span data-stu-id="fe4af-119">For each pair, it compares the data types of the defined parameters.</span></span> <span data-ttu-id="fe4af-120">Wenn die Typen in eine der Überladungen, die alle auf die entsprechenden Typen in den anderen erweitert werden, entfällt der Compiler letztere.</span><span class="sxs-lookup"><span data-stu-id="fe4af-120">If the types in one of the overloads all widen to the corresponding types in the other, the compiler eliminates the latter.</span></span> <span data-ttu-id="fe4af-121">D. h., behält es die Überladung, die am wenigsten erweiternde erforderlich sind.</span><span class="sxs-lookup"><span data-stu-id="fe4af-121">That is, it retains the overload that requires the least amount of widening.</span></span>  
  
6.  <span data-ttu-id="fe4af-122">**Einzelne geeignet.**</span><span class="sxs-lookup"><span data-stu-id="fe4af-122">**Single Candidate.**</span></span> <span data-ttu-id="fe4af-123">Sollten Überladungen paarweise zugeordnet, bis die einzige bleibt Überladung und des Aufrufs an diese Überladung löst wird fortgesetzt.</span><span class="sxs-lookup"><span data-stu-id="fe4af-123">It continues considering overloads in pairs until only one overload remains, and it resolves the call to that overload.</span></span> <span data-ttu-id="fe4af-124">Wenn der Compiler nicht die Überladungen, die einem einzelnen Kandidaten reduziert werden kann, wird einen Fehler generiert.</span><span class="sxs-lookup"><span data-stu-id="fe4af-124">If the compiler cannot reduce the overloads to a single candidate, it generates an error.</span></span>
