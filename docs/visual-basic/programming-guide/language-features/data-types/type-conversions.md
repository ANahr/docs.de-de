# <a name="type-conversions-in-visual-basic"></a>Typkonvertierung in Visual Basic
Einen Wert von einem Datentyp in einen anderen Typ umgewandelt wird bezeichnet *Konvertierung*. Konvertierungen sind entweder *erweiternde* oder *einschränkende*, je nach Datenkapazität der betreffenden Typen. Sie sind auch *implizite* oder *explizite*, abhängig von der Syntax im Quellcode.
