# <a name="mycomputerports-object"></a>My.Computer.Ports-Objekt
Stellt eine Eigenschaft und eine Methode für den Zugriff auf die seriellen Anschlüsse des Computers bereit.  
  
## <a name="remarks"></a>Hinweise  
 Weitere Informationen zu den Methoden und Eigenschaften der `My.Computer.Ports` Objekt, finden Sie unter <xref:Microsoft.VisualBasic.Devices.Ports>.</xref:Microsoft.VisualBasic.Devices.Ports>  
  
 Weitere Informationen finden Sie unter [Zugriff auf die Anschlüsse des Computers](../../../visual-basic/developing-apps/programming/computer-resources/accessing-the-computer-s-ports.md).  
  
> [!NOTE]
>  Sie können auch Eigenschaften und Methoden des der <xref:System.IO.Ports.SerialPort?displayProperty=fullName>Klasse, um die seriellen Anschlüsse des Computers zugreifen.</xref:System.IO.Ports.SerialPort?displayProperty=fullName>
