# <a name="special-characters-visual-basic-reference"></a>Sonderzeichen (Visual Basic-Referenz)

Sonderzeichen sind vordefinierte, kontextabhängige Zeichen, die das Programmelement ändert, zu dem sie vorangestellt sind. Visual Basic unterstützt die folgenden Sonderzeichen enthalten: 

- [$](interpolated.md), das Zeichen für interpolierte Zeichenfolgen.
