# <a name="unable-to-write-temporary-file-because-temporary-path-is-not-available"></a>Die temporäre Datei konnte nicht geschrieben werden, da kein temporärer Pfad verfügbar ist.
[!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] konnte den Pfad nicht bestimmen, in dem temporäre Dateien gespeichert sind.  
  
 **Fehler-ID:** BC30698  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Starten Sie [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)] neu.  
  
2.  Wenn das Problem weiterhin auftritt, installieren Sie [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)] neu.
