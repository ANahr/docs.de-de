# <a name="type-mismatch-visual-basic"></a>Typenkonflikt (Visual Basic)
Sie haben versucht, einen Wert in einen anderen Typ auf eine Weise zu konvertieren, der ungültig ist.  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Überprüfen Sie die Zuweisung, stellen Sie sicher, dass er gültig ist.  
  
2.  Stellen Sie sicher, dass ein Objekt nicht an eine Prozedur übergeben, die eine einzelne Eigenschaft oder ein Wert erwartet.  
  
3.  Stellen Sie sicher, dass Sie ein Modul oder ein Projekt nicht verwendet haben, wo ein Ausdruck erwartet wurde.
