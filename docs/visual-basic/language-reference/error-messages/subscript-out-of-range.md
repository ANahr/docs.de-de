# <a name="subscript-out-of-range-visual-basic"></a>Index außerhalb des definierten Bereichs (Visual Basic)
Ein Arrayfeldindex ist ungültig, weil es außerhalb des zulässigen Bereichs liegt. Für eine Dimension der niedrigste Indexwert ist immer 0, und der höchste Indexwert wird zurückgegeben, durch die `GetUpperBound` Methode für diese Dimension.  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Ändern Sie den Index aus, damit sie innerhalb des gültigen Bereichs liegt.
