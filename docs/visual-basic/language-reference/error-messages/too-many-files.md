# <a name="too-many-files"></a>Zu viele Dateien
Wurden weitere Dateien in das Stammverzeichnis erstellt als im Betriebssystem zulässig oder mehr Dateien als die angegebene Zahl geöffnet wurden die **Dateien =** in der Datei CONFIG festlegen. SYS-Datei.  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Wenn das Programm öffnen, schließen oder Speichern von Dateien in das Stammverzeichnis ist, ändern Sie das Programm so, dass es ein Unterverzeichnis verwendet.  
  
2.  Erhöhen Sie die Anzahl der angegebenen Dateien in Ihre **Dateien =** in der Datei CONFIG festlegen. SYS-Datei und den Computer neu starten.
