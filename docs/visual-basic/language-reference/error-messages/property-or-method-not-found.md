# <a name="property-or-method-not-found"></a>Die Eigenschaft oder Methode wurde nicht gefunden
Die referenzierte Objektmethode oder Objekteigenschaft ist nicht definiert.  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Sie können den Namen des Objekts richtig geschrieben haben. Um festzustellen, welche Eigenschaften und Methoden für ein Objekt definiert sind, zeigen Sie den Objektkatalog. Wählen Sie die entsprechenden Objektbibliothek zum Anzeigen einer Liste der verfügbaren Eigenschaften und Methoden.
