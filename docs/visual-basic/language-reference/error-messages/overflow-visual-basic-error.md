# <a name="overflow-visual-basic-error"></a>Überlauf (Visual Basic-Fehler)
Ein Literal stellt einen Wert außerhalb der Grenzen des Datentyps, denen es zugewiesen wird.  
  
 **Fehler-ID:** BC30036  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Wenden Sie sich an den Wertebereich für das Ziel-Datentyp und Schreiben Sie das Literal, um diesem Bereich entsprechen.
