# <a name="unable-to-find-required-file-39ltfilenamegt39"></a>Kann nicht zum Suchen der Datei erforderliche Datei &#39; &lt;Filename&gt;&#39;
Eine Datei, die für Visual Studio erforderlich ist, fehlt oder ist beschädigt.  
  
 **Fehler-ID:** BC30655  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Installieren Sie Visual Studio erneut.
