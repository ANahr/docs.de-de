# <a name="39ltfunctionnamegt39-is-not-declared-smart-devicevisual-basic-compiler-error"></a>&#39; &lt;Funktionsname&gt;&#39; ist nicht deklariert (Compilerfehler für intelligentes Gerät/Visual Basic)
<`functionname`> ist nicht deklariert. Die Datei-E/A-Funktionalität ist gewöhnlich im `Microsoft.VisualBasic` -Namespace verfügbar, allerdings wird sie von der Zielversion von .NET Compact Framework nicht unterstützt.  
  
 **Fehler-ID:** BC30766  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Führen Sie Dateivorgänge mit Funktionen in der `System.IO` Namespace.
