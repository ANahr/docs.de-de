# <a name="permission-denied-visual-basic"></a>Berechtigung verweigert (Visual Basic)
Es wurde in einem schreibgeschützten Datenträger zu schreiben oder auf eine gesperrte Datei zuzugreifen versucht.  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Um eine Datei schreibgeschützt öffnen, ändern Sie das Schreibschutzattribut der Datei.  
  
2.  Stellen Sie sicher, dass die Datei nicht von einem anderen Prozess gesperrt hat, und warten Sie zum Öffnen der Datei, bis die andere Prozess, diesen freigibt.  
  
3.  Für den Zugriff auf die Registrierung, überprüfen Sie, dass Ihre Benutzerberechtigungen dieser Typ von Registrierungszugriff umfassen.
