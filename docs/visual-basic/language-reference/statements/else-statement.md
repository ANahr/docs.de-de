# <a name="else-statement-visual-basic"></a>Else-Anweisung (Visual Basic)
Stellt eine Gruppe von Anweisungen ausgeführt oder kompiliert, wenn keine bedingten Gruppe von Anweisungen ausgeführt oder kompiliert wurde.  
  
## <a name="remarks"></a>Hinweise  
 Das `Else`-Schlüsselwort kann in den folgenden Kontexten verwendet werden:  
  
 [If...Then...Else-Anweisung](../../../visual-basic/language-reference/statements/if-then-else-statement.md)  
  
 [Select...Case-Anweisung](../../../visual-basic/language-reference/statements/select-case-statement.md)  
  
 [#If... ... #Else-Direktive](../../../visual-basic/language-reference/directives/if-then-else-directives.md)
