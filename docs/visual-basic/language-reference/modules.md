# <a name="modules-visual-basic"></a>Module (Visual Basic)
[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]enthält mehrere Module, die ermöglichen es Ihnen, vereinfachen Sie häufige Aufgaben in Ihrem Code, darunter Bearbeiten von Zeichenfolgen, die mathematische Berechnungen, Abrufen von Systeminformationen, Ausführen von Vorgängen für Datei- und Verzeichnisnamen, ausführen und so weiter. Die folgende Tabelle listet die Module von [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)].  
  
|||  
|---|---|  
|<xref:Microsoft.VisualBasic.Constants></xref:Microsoft.VisualBasic.Constants>|Enthält verschiedene Konstanten. Diese Konstanten können überall im Code verwendet werden.|  
|<xref:Microsoft.VisualBasic.ControlChars></xref:Microsoft.VisualBasic.ControlChars>|Enthält Konstanten Steuerzeichen zum Drucken und Anzeigen von Text.|  
|<xref:Microsoft.VisualBasic.Conversion></xref:Microsoft.VisualBasic.Conversion>|Enthält Member, die Dezimalzahlen in andere Basiszahlen konvertiert Zahlen in Zeichenfolgen, Zeichenfolgen in Zahlen und einen Datentyp in einen anderen geben.|  
|<xref:Microsoft.VisualBasic.DateAndTime></xref:Microsoft.VisualBasic.DateAndTime>|Enthält Elemente, die das aktuelle Datum oder Uhrzeit abrufen, Durchführen von datumsberechnungen, ein Datums- oder Zeitwert zurück, Festlegen des Datums oder der Uhrzeit oder Zeit die Dauer eines Prozesses an.|  
|<xref:Microsoft.VisualBasic.ErrObject></xref:Microsoft.VisualBasic.ErrObject>|Enthält Informationen zu Laufzeitfehlern und Methoden auslösen oder einen Fehler zu löschen.|  
|<xref:Microsoft.VisualBasic.FileSystem></xref:Microsoft.VisualBasic.FileSystem>|Enthält Member, die Datei-, Verzeichnis- oder Ordner und System-Vorgänge ausführen.|  
|<xref:Microsoft.VisualBasic.Financial></xref:Microsoft.VisualBasic.Financial>|Enthält Verfahren, mit denen finanzberechnungen ausgeführt.|  
|<xref:Microsoft.VisualBasic.Globals></xref:Microsoft.VisualBasic.Globals>|Enthält Informationen über die aktuelle Version in der scripting-Moduls.|  
|<xref:Microsoft.VisualBasic.Information></xref:Microsoft.VisualBasic.Information>|Enthält die Elemente, die zurückgeben, testen oder Überprüfen Sie die Informationen wie Arraygröße, Typnamen und So weiter.|  
|<xref:Microsoft.VisualBasic.Interaction></xref:Microsoft.VisualBasic.Interaction>|Enthält Elemente mit Objekten, Anwendungen und Systeme zu interagieren.|  
|<xref:Microsoft.VisualBasic.Strings></xref:Microsoft.VisualBasic.Strings>|Enthält Member, die Zeichenfolgenoperationen wie das Formatieren von Zeichenfolgen, Suchen nach Zeichenfolgen, die Länge einer Zeichenfolge abrufen und so weiter.|  
|<xref:Microsoft.VisualBasic.VBMath></xref:Microsoft.VisualBasic.VBMath>|Enthält Elemente führen mathematische Operationen.|
