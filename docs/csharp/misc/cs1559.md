# <a name="compiler-error-cs1559"></a>Compilerfehler CS1559
"Objekt" wurde importiert und kann deshalb nicht für die Main-Methode verwendet werden.  
  
 Wurde eine ungültige Klasse angegeben, um die [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) (Compileroption); die Klasse kann nicht verwendet werden, als Speicherort für die [Main](../../csharp/programming-guide/main-and-command-args/index.md) Methode.
