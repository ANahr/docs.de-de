 <span data-ttu-id="69304-108">Die folgende Sequenz von Kompilierungen führt zu CS0012:</span><span class="sxs-lookup"><span data-stu-id="69304-108">The following sequence of compilations will result in CS0012:</span></span>  
  
```

```  
  
 <span data-ttu-id="69304-109">Dann:</span><span class="sxs-lookup"><span data-stu-id="69304-109">Then:</span></span>  
  
```

{  
   public static A f()  
   {  
      return new A();  
   }  
}  
```  
  
 <span data-ttu-id="69304-110">Dann:</span><span class="sxs-lookup"><span data-stu-id="69304-110">Then:</span></span>  
  
```

   public static void Main()  
   {  
      object o = B.f();   // CS0012  
   }  
}  
```
