<span data-ttu-id="c3479-103">Der Typ "Typ" ist in einer nicht referenzierten Assembly definiert.</span><span class="sxs-lookup"><span data-stu-id="c3479-103">The type 'type' is defined in an assembly that is not referenced.</span></span> <span data-ttu-id="c3479-104">Fügen Sie einen Verweis auf die Assembly "Assembly" hinzu.</span><span class="sxs-lookup"><span data-stu-id="c3479-104">You must add a reference to assembly 'assembly'.</span></span>  
  
 <span data-ttu-id="c3479-105">Die Definition für einen referenzierten Typ wurde nicht gefunden.</span><span class="sxs-lookup"><span data-stu-id="c3479-105">The definition for a referenced type was not found.</span></span> <span data-ttu-id="c3479-106">Dies kann passieren, wenn eine erforderliche DLL-Datei nicht in der Kompilierung enthalten ist.</span><span class="sxs-lookup"><span data-stu-id="c3479-106">This could occur if a required .DLL file is not included in the compilation.</span></span> <span data-ttu-id="c3479-107">Weitere Informationen finden Sie unter [hinzufügen Dialogfelds "Verweis"](http://msdn.microsoft.com/library/2feb0fe2-0805-4cc9-8cba-b0315849dfb7) und [/Reference (C#-Compileroptionen)](../../csharp/language-reference/compiler-options/reference-compiler-option.md).</span><span class="sxs-lookup"><span data-stu-id="c3479-107">For more information, see [Add Reference Dialog Box](http://msdn.microsoft.com/library/2feb0fe2-0805-4cc9-8cba-b0315849dfb7) and [/reference (C# Compiler Options)](../../csharp/language-reference/compiler-options/reference-compiler-option.md).</span></span>  
  
 <span data-ttu-id="c3479-108">Die folgende Sequenz von Kompilierungen führt zu CS0012:</span><span class="sxs-lookup"><span data-stu-id="c3479-108">The following sequence of compilations will result in CS0012:</span></span>  
  
```csharp  
// cs0012a.cs  
// compile with: /target:library  
public class A {}  
```  
  
 <span data-ttu-id="c3479-109">Dann:</span><span class="sxs-lookup"><span data-stu-id="c3479-109">Then:</span></span>  
  
```csharp  
// cs0012b.cs  
// compile with: /target:library /reference:cs0012a.dll  
public class B  
{  
   public static A f()  
   {  
      return new A();  
   }  
}  
```  
  
 <span data-ttu-id="c3479-110">Dann:</span><span class="sxs-lookup"><span data-stu-id="c3479-110">Then:</span></span>  
  
```csharp  
// cs0012c.cs  
// compile with: /reference:cs0012b.dll  
class C  
{  
   public static void Main()  
   {  
      object o = B.f();   // CS0012  
   }  
}  
```  
  
 <span data-ttu-id="c3479-111">Sie können diesen CS0012-Fehler beheben, indem Sie beim Kompilieren mit `/reference:cs0012b.dll;cs0012a.dll`, oder in Visual Studio mithilfe der [hinzufügen Dialogfelds "Verweis"](http://msdn.microsoft.com/library/2feb0fe2-0805-4cc9-8cba-b0315849dfb7) einen Verweis auf "cs0012a.dll" zusätzlich zu "cs0012b.dll" hinzufügen.</span><span class="sxs-lookup"><span data-stu-id="c3479-111">You could resolve this CS0012 by compiling with `/reference:cs0012b.dll;cs0012a.dll`, or in Visual Studio by using the [Add Reference Dialog Box](http://msdn.microsoft.com/library/2feb0fe2-0805-4cc9-8cba-b0315849dfb7) to add a reference to cs0012a.dll in addition to cs0012b.dll.</span></span>
