# <a name="compiler-error-cs0210"></a><span data-ttu-id="7302c-102">Compilerfehler CS0210</span><span class="sxs-lookup"><span data-stu-id="7302c-102">Compiler Error CS0210</span></span>
<span data-ttu-id="7302c-103">Sie müssen in einer fixed- oder using-Anweisungsdeklaration einen Initialisierer bereitstellen.</span><span class="sxs-lookup"><span data-stu-id="7302c-103">You must provide an initializer in a fixed or using statement declaration</span></span>  
  
 <span data-ttu-id="7302c-104">Müssen Sie die Variable in einer [fixed-Anweisung](../../csharp/language-reference/keywords/fixed-statement.md)deklarieren und initialisieren.</span><span class="sxs-lookup"><span data-stu-id="7302c-104">You must declare and initialize the variable in a [fixed statement](../../csharp/language-reference/keywords/fixed-statement.md).</span></span> <span data-ttu-id="7302c-105">Weitere Informationen finden Sie unter [Unsicherer Code und Zeiger](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="7302c-105">For more information, see [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="7302c-106">Im folgenden Beispiel wird CS0210 generiert:</span><span class="sxs-lookup"><span data-stu-id="7302c-106">The following sample generates CS0210:</span></span>  
  
```  
// CS0210a.cs  
// compile with: /unsafe  
  
class Point  
{  
   public int x, y;  
}  
  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
  
      fixed (int i)    // CS0210  
      {  
      }  
      // try the following lines instead  
      /*  
      fixed (int* p = &pt.x)  
      {  
      }  
      fixed (int* q = &pt.y)  
      {  
      }  
      */  
   }  
}  
```  
  
 <span data-ttu-id="7302c-107">Im folgenden Beispiel wird auch CS0210 generiert, da die [using-Anweisung](../../csharp/language-reference/keywords/using-statement.md) keinen Initialisierer aufweist.</span><span class="sxs-lookup"><span data-stu-id="7302c-107">The following sample also generates CS0210 because the [using statement](../../csharp/language-reference/keywords/using-statement.md) has no initializer.</span></span>  
  
```  
// CS0210b.cs  
  
using System.IO;  
class Test   
{  
   static void Main()   
   {  
      using (StreamWriter w) // CS0210  
      // Try this line instead:  
      // using (StreamWriter w = new StreamWriter("TestFile.txt"))   
      {  
         w.WriteLine("Hello there");  
      }  
   }  
}  
```
