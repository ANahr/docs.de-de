# <a name="compiler-error-cs1577"></a>Compilerfehler CS1577
Fehler beim Generieren der Assembly – Grund  
  
 Nach der erfolgreichen Kompilierung des Quellcodes durch den C#-Compiler ist beim Generieren der Assembly ein Fehler aufgetreten. Weitere Informationen finden Sie in der Fehlerdokumentation für das ALink-Dienstprogramm ([Fehler und Warnungen des Al.exe-Tools](http://msdn.microsoft.com/en-us/7f125d49-0a03-47a6-9ba9-d61a679a7d4b)).
