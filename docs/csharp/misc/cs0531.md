# <a name="compiler-error-cs0531"></a><span data-ttu-id="b9aa5-102">Compilerfehler CS0531</span><span class="sxs-lookup"><span data-stu-id="b9aa5-102">Compiler Error CS0531</span></span>
<span data-ttu-id="b9aa5-103">"Member": Schnittstellenmember können keine Definition haben.</span><span class="sxs-lookup"><span data-stu-id="b9aa5-103">'member' : interface members cannot have a definition</span></span>  
  
 <span data-ttu-id="b9aa5-104">Methoden, die in einer [Schnittstelle](../../csharp/language-reference/keywords/interface.md) deklariert sind, müssen in einer Klasse implementiert werden, die von der Schnittstelle erbt, sie dürfen nicht in der Schnittstelle selbst implementiert werden.</span><span class="sxs-lookup"><span data-stu-id="b9aa5-104">Methods that are declared in an [interface](../../csharp/language-reference/keywords/interface.md) must be implemented in a class that inherits from it and not in the interface itself.</span></span>  
  
 <span data-ttu-id="b9aa5-105">Im folgenden Beispiel wird CS0531 generiert:</span><span class="sxs-lookup"><span data-stu-id="b9aa5-105">The following sample generates CS0531:</span></span>  
  
```  
// CS0531.cs  
namespace x  
{  
   public interface clx  
   {  
      int xclx()   // CS0531, cannot define xclx  
      // Try the following declaration instead:  
      // int xclx();  
      {  
         return 0;  
      }  
   }  
  
   public class cly  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
