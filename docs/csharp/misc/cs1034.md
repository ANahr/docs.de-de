# Compilerfehler CS1034
Compilerlimit überschritten: Die Zeile darf nicht mehr als "Anzahl" Zeichen umfassen.  
  
 Der Grenzwert für die in einer Zeile zulässige Anzahl von Zeichen ist 16.777.214.
