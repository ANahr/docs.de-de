<span data-ttu-id="d9ea7-103">Der "Zahl"-Parameter sollte nicht mit dem "Schlüsselwort"-Schlüsselwort deklariert werden.</span><span class="sxs-lookup"><span data-stu-id="d9ea7-103">Parameter 'number' should not be declared with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="d9ea7-104">Dieser Fehler tritt auf, wenn der Parametertypmodifizierer in einer anonymen Methode nicht mit dem übereinstimmt, der in der Deklaration des Delegaten verwendet wird, in den Sie die Methode umwandeln.</span><span class="sxs-lookup"><span data-stu-id="d9ea7-104">This error occurs when the parameter type modifier in an anonymous method does not match that used in the declaration of the delegate, to which you are casting the method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="d9ea7-105">Beispiel</span><span class="sxs-lookup"><span data-stu-id="d9ea7-105">Example</span></span>  
 <span data-ttu-id="d9ea7-106">Im folgenden Beispiel wird CS1677 generiert:</span><span class="sxs-lookup"><span data-stu-id="d9ea7-106">The following sample generates CS1677:</span></span>  
  
```csharp  
// CS1677.cs  
delegate void D(int i);  
class Errors  
{  
    static void Main()   
    {  
        D d = delegate(out int i) { };   // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int i) { };  
  
        D d = delegate(ref int j){}; // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int j){};  
    }  
}  
```
