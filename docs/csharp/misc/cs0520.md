# <a name="compiler-error-cs0520"></a>Compilerfehler CS0520
Der vordefinierte Typ „name“ wurde falsch deklariert.  
  
 Eine oder mehrere vom Compiler benötigte Dateien wurden nicht gefunden. Installieren Sie die Common Language Runtime von .NET Framework neu.
