# <a name="compiler-error-cs0432"></a><span data-ttu-id="1dab4-102">Compilerfehler CS0432</span><span class="sxs-lookup"><span data-stu-id="1dab4-102">Compiler Error CS0432</span></span>
<span data-ttu-id="1dab4-103">Der Alias "identifier" wurde nicht gefunden</span><span class="sxs-lookup"><span data-stu-id="1dab4-103">Alias 'identifier' not found</span></span>  
  
 <span data-ttu-id="1dab4-104">Dieser Fehler tritt auf, wenn Sie "::" auf der rechten Seite eines Bezeichners verwenden, der kein Alias ist.</span><span class="sxs-lookup"><span data-stu-id="1dab4-104">This error occurs when you use "::" to the right of an identifier that is not an alias.</span></span> <span data-ttu-id="1dab4-105">Verwenden Sie ".", um den Fehler zu beheben.</span><span class="sxs-lookup"><span data-stu-id="1dab4-105">To resolve the error, use "." instead.</span></span>  
  
 <span data-ttu-id="1dab4-106">Im folgenden Beispiel wird CS0432 generiert:</span><span class="sxs-lookup"><span data-stu-id="1dab4-106">The following example generates CS0432:</span></span>  
  
```csharp  
// CS0432.cs  
namespace A {  
    public class B {  
        public static void Meth() { }  
    }  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        A::B.Meth();   // CS0432  
       // To resolve, use the following line instead:  
       // A.B.Meth();  
    }  
}  
```
