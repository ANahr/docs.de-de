 <span data-ttu-id="51024-108">Im folgenden Beispiel wird CS0456 generiert:</span><span class="sxs-lookup"><span data-stu-id="51024-108">The following sample generates CS0456.</span></span>  
  
```

{  
   public class G5<T> where T : struct  
   {

   }  
}  
```
