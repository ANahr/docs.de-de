# <a name="compiler-error-cs1621"></a><span data-ttu-id="0be23-102">Compilerfehler CS1621</span><span class="sxs-lookup"><span data-stu-id="0be23-102">Compiler Error CS1621</span></span>
<span data-ttu-id="0be23-103">Die yield-Anweisung kann nicht in einer anonymen Methode oder einem Lambda-Ausdruck verwendet werden.</span><span class="sxs-lookup"><span data-stu-id="0be23-103">The yield statement cannot be used inside an anonymous method or lambda expression</span></span>  
  
 <span data-ttu-id="0be23-104">Die yield-Anweisung kann nicht in einem anonymen Methodenblock in einem Iterator sein.</span><span class="sxs-lookup"><span data-stu-id="0be23-104">The yield statement cannot be in an anonymous method block in an iterator.</span></span>  
  
## <a name="example"></a><span data-ttu-id="0be23-105">Beispiel</span><span class="sxs-lookup"><span data-stu-id="0be23-105">Example</span></span>  
 <span data-ttu-id="0be23-106">Im folgenden Beispiel wird CS1621 generiert:</span><span class="sxs-lookup"><span data-stu-id="0be23-106">The following sample generates CS1621:</span></span>  
  
```csharp  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
