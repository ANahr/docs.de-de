## <a name="example"></a><span data-ttu-id="b4e05-108">Beispiel</span><span class="sxs-lookup"><span data-stu-id="b4e05-108">Example</span></span>  
 <span data-ttu-id="b4e05-109">Im folgenden Beispiel wird CS0403 generiert.</span><span class="sxs-lookup"><span data-stu-id="b4e05-109">The following sample generates CS0403.</span></span>  
  
```  
// CS0403.cs  
// compile with: /target:library  
class C<T>  
{  
   public void f()  
   {  
      T t = null;  // CS0403  
      T t2 = default(T);   // OK  
    }  
}  
  
class D<T> where T : class   
{  
   public void f()  
   {  
      T t = null;  // OK  
    }  
}  
```
