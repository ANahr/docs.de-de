# <a name="compiler-error-cs1613"></a><span data-ttu-id="6988b-102">Compilerfehler CS1613</span><span class="sxs-lookup"><span data-stu-id="6988b-102">Compiler Error CS1613</span></span>
<span data-ttu-id="6988b-103">Die verwaltete Co-Klassen-Wrapperklasse "class" für die "interface"-Schnittstelle kann nicht gefunden werden. (Möglicherweise fehlt ein Assemblyverweis.)</span><span class="sxs-lookup"><span data-stu-id="6988b-103">The managed coclass wrapper class 'class' for interface 'interface' cannot be found (are you missing an assembly reference?)</span></span>  
  
 <span data-ttu-id="6988b-104">Es wurde versucht, ein COM-Objekt über eine Schnittstelle zu instanziieren.</span><span class="sxs-lookup"><span data-stu-id="6988b-104">An attempt was made to instantiate a COM object from an interface.</span></span> <span data-ttu-id="6988b-105">Die Schnittstelle verfügt über das **ComImport** - und das `CoClass` -Attribut, aber der Compiler kann nicht den Typ für das `CoClass` -Attribut finden.</span><span class="sxs-lookup"><span data-stu-id="6988b-105">The interface has the **ComImport** and `CoClass` attributes, but the compiler cannot find the type given for the `CoClass` attribute.</span></span>  
  
 <span data-ttu-id="6988b-106">Um diesen Fehler zu beheben, können Sie eine der folgenden Lösungen versuchen:</span><span class="sxs-lookup"><span data-stu-id="6988b-106">To resolve this error, you can try one of the following:</span></span>  
  
-   <span data-ttu-id="6988b-107">Fügen Sie einen Verweis auf die Assembly hinzu, die die Co-Klasse aufweist (in den meisten Fällen sollten sich die Schnittstelle und die Co-Klasse in der gleichen Assembly befinden).</span><span class="sxs-lookup"><span data-stu-id="6988b-107">Add a reference to the assembly that has the coclass (most of the time the interface and coclass should be in the same assembly).</span></span> <span data-ttu-id="6988b-108">Finden Sie unter [/reference](../../csharp/language-reference/compiler-options/reference-compiler-option.md) oder [hinzufügen Dialogfelds "Verweis"](http://msdn.microsoft.com/library/2feb0fe2-0805-4cc9-8cba-b0315849dfb7) Informationen.</span><span class="sxs-lookup"><span data-stu-id="6988b-108">See [/reference](../../csharp/language-reference/compiler-options/reference-compiler-option.md) or [Add Reference Dialog Box](http://msdn.microsoft.com/library/2feb0fe2-0805-4cc9-8cba-b0315849dfb7) for information.</span></span>  
  
-   <span data-ttu-id="6988b-109">Korrigieren Sie das `CoClass` -Attribut in der Schnittstelle.</span><span class="sxs-lookup"><span data-stu-id="6988b-109">Fix the `CoClass` attribute on the interface.</span></span>  
  
 <span data-ttu-id="6988b-110">Das folgende Beispiel veranschaulicht die richtige Verwendung von **CoClassAttribute**:</span><span class="sxs-lookup"><span data-stu-id="6988b-110">The following sample demonstrates correct usage of **CoClassAttribute**:</span></span>  
  
```csharp  
// CS1613.cs  
using System;  
using System.Runtime.InteropServices;  
  
[Guid("1FFD7840-E82D-4268-875C-80A160C23296")]  
[ComImport()]  
[CoClass(typeof(A))]  
public interface IA{}  
public class A : IA {}  
  
public class AA  
{  
   public static void Main()  
   {  
      IA i;  
      i = new IA(); // This is equivalent to new A().  
                    // because of the CoClass attribute on IA  
   }  
}  
```
