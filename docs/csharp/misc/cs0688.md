## <a name="example"></a><span data-ttu-id="cae87-108">Beispiel</span><span class="sxs-lookup"><span data-stu-id="cae87-108">Example</span></span>  
 <span data-ttu-id="cae87-109">Im folgenden Beispiel wird CS0688 generiert.</span><span class="sxs-lookup"><span data-stu-id="cae87-109">The following sample generates CS0688.</span></span> <span data-ttu-id="cae87-110">Um die Warnung aufzulösen, ohne die Basisklasse zu ändern, entfernen Sie das Sicherheitsattribut aus der überschreibenden Methode.</span><span class="sxs-lookup"><span data-stu-id="cae87-110">To resolve the warning without modifying the base class, remove the security attribute from the overriding method.</span></span> <span data-ttu-id="cae87-111">Das Sicherheitsproblem wird dadurch nicht behoben.</span><span class="sxs-lookup"><span data-stu-id="cae87-111">This will not solve the security problem.</span></span>  
  
```  
// CS0688.cs  
// compile with: /W:1  
using System;  
using System.Security.Permissions;  
  
class Base   
{  
    //Uncomment the following line to close the security hole  
    //[FileIOPermission(SecurityAction.LinkDemand, All=@"C:\\")]  
    public virtual void DoScaryFileStuff()  
    {  
    }  
}  
  
class Derived: Base  
{  
    [FileIOPermission(SecurityAction.LinkDemand, All=@"C:\\")] // CS0688  
    public override void DoScaryFileStuff()  
    {  
    }  
    static void Main()  
    {  
    }  
}  
```
