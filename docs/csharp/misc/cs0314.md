## <a name="example"></a><span data-ttu-id="58317-108">Beispiel</span><span class="sxs-lookup"><span data-stu-id="58317-108">Example</span></span>  
 <span data-ttu-id="58317-109">Mit dem folgenden Code wird CS0314 generiert:</span><span class="sxs-lookup"><span data-stu-id="58317-109">The following code generates CS0314:</span></span>  
  
```  
// cs0314.cs  
// Compile with: /target:library  
public class ClassConstraint { }  
  
public class A<T> where T : ClassConstraint  
{ }  
  
public class B<T> : A<T> //CS0314  
{ }  
  
// Try using this instead.  
public class C<T> : A<T> where T : ClassConstraint  
{ }  
```
