# <a name="compiler-error-cs0586"></a>Compilerfehler CS0586
Interner Compilerfehler: Stufe "Stufe"  
  
 Versuchen Sie festzustellen, ob der Compiler einen Fehler verursacht, weil er unerwartete Syntax nicht analysieren kann. Wenn dies nicht der Fall ist, versuchen Sie [sprechen Sie mit uns](https://docs.microsoft.com/visualstudio/ide/talk-to-us).
