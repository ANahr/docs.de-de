# <a name="compiler-error-cs0586"></a>Compilerfehler CS0586
Interner Compilerfehler: Stufe "Stufe"  
  
 Versuchen Sie festzustellen, ob der Compiler einen Fehler verursacht, weil er unerwartete Syntax nicht analysieren kann. Wenn das nicht der Fall ist, versuchen Sie es [sprechen Sie mit uns](/visualstudio/ide/talk-to-us).
