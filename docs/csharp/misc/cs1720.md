 <span data-ttu-id="026c2-104">Der Fehler tritt auf, wenn Sie einen Ausdruck schreiben, der den Standardwert einer Variablen des generischen Typs enthält, die ein Verweistyp (z. {  
    public static void GenericClass<T>(T t1) where T : class
