# <a name="compiler-warning-level-1-cs1720"></a><span data-ttu-id="a7112-102">Compilerwarnung (Stufe 1) CS1720</span><span class="sxs-lookup"><span data-stu-id="a7112-102">Compiler Warning (level 1) CS1720</span></span>
<span data-ttu-id="a7112-103">Der Ausdruck führt immer zu System.NullReferenceException, da der Standardwert von "generischer Typ" NULL ist.</span><span class="sxs-lookup"><span data-stu-id="a7112-103">Expression will always cause a System.NullReferenceException because the default value of 'generic type' is null</span></span>  
  
 <span data-ttu-id="a7112-104">Der Fehler tritt auf, wenn Sie einen Ausdruck schreiben, der den Standardwert einer Variablen des generischen Typs enthält, die ein Verweistyp (z. B. eine Klasse) ist.</span><span class="sxs-lookup"><span data-stu-id="a7112-104">If you write an expression involving the default of a generic type variable that is a reference type (for example, a class), this error will occur.</span></span> <span data-ttu-id="a7112-105">Betrachten Sie hierzu den folgenden Ausdruck:</span><span class="sxs-lookup"><span data-stu-id="a7112-105">Consider the following expression:</span></span>  
  
```  
default(T).ToString()  
```  
  
 <span data-ttu-id="a7112-106">Da `T` ein Verweistyp ist, lautet der Standardwert NULL. ## <a name="example"></a><span data-ttu-id="a7112-107">Beispiel</span><span class="sxs-lookup"><span data-stu-id="a7112-107">Example</span></span>

```  
// CS1720.cs  
using System;  
public class Tester   
{  
    public static void GenericClass<T>(T t1) where T : class   
    {  
        Console.WriteLine(default(T).ToString());  // CS1720  
    }  
    public static void Main() {}  
}  
```
