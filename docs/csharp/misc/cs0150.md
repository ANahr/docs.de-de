# <a name="compiler-error-cs0150"></a><span data-ttu-id="b19fa-102">Compilerfehler CS0150</span><span class="sxs-lookup"><span data-stu-id="b19fa-102">Compiler Error CS0150</span></span>
<span data-ttu-id="b19fa-103">Konstantenwert erwartet.</span><span class="sxs-lookup"><span data-stu-id="b19fa-103">A constant value is expected</span></span>  
  
 <span data-ttu-id="b19fa-104">Eine Variable wurde gefunden, obwohl eine Konstante erwartet wurde.</span><span class="sxs-lookup"><span data-stu-id="b19fa-104">A variable was found where a constant was expected.</span></span> <span data-ttu-id="b19fa-105">Weitere Informationen finden Sie unter [switch](../../csharp/language-reference/keywords/switch.md).</span><span class="sxs-lookup"><span data-stu-id="b19fa-105">For more information, see [switch](../../csharp/language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="b19fa-106">Im folgenden Beispiel wird CS0150 generiert:</span><span class="sxs-lookup"><span data-stu-id="b19fa-106">The following sample generates CS0150:</span></span>  
  
```  
// CS0150.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         int j = 0;  
  
         switch(i)  
         {  
            case j:   // CS0150, j is a variable int, not a constant int  
            // try the following line instead  
            // case 1:  
         }  
      }  
   }  
}  
```  
  
 <span data-ttu-id="b19fa-107">Dieser Fehler wird auch ausgelöst, wenn eine Arraygröße durch ein Variablenwert angegeben und mit einem Arrayinitialisierer initialisiert wird.</span><span class="sxs-lookup"><span data-stu-id="b19fa-107">This error is also produced when an array size is specified with a variable value and initialized with an array initializer.</span></span> <span data-ttu-id="b19fa-108">Um den Fehler zu beheben, initialisieren Sie das Array in einer oder mehreren separaten Anweisung(en).</span><span class="sxs-lookup"><span data-stu-id="b19fa-108">To remove the error, initialize the array in a separate statement or statements.</span></span>  
  
```  
// CS0150.cs  
    namespace MyNamespace  
    {  
        public class MyClass  
        {  
            public static void Main()  
            {  
                int size = 2;  
                double[] nums = new double[size] { 46.9, 89.4 }; //CS0150  
                // Try the following lines instead  
                // double[] nums = new double[size];  
                // nums[0] = 46.9;   
                // nums[1] = 89.4;  
            }  
        }  
  
    }  
```
