# <a name="compiler-warning-level-1-cs3004"></a>Compilerwarnung (Stufe 1) CS3004
Gemischte und aufgelöste Unicode-Zeichen sind nicht CLS-kompatibel.  
  
 Nur zusammengesetzte Unicode-Zeichen dürfen [öffentlichen](../../csharp/language-reference/keywords/public.md), [geschützt](../../csharp/language-reference/keywords/protected.md), oder `protected internal` Bezeichner, um mit der Common Language Specification (CLS) kompatibel sein. Weitere Informationen zu den CLS-Konformitätsregeln finden Sie unter [Writing CLS-Compliant Code (Schreiben von CLS-kompatiblem Code)](http://msdn.microsoft.com/en-us/4c705105-69a2-4e5e-b24e-0633bc32c7f3)und [Language Independence and Language-Independent Components (Sprachunabhängigkeit und sprachunabhängige Komponenten)](https://msdn.microsoft.com/library/12a7a7h3).
