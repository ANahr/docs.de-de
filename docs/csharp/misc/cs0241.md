 <span data-ttu-id="edd0c-108">Im folgenden Beispiel wird CS0241 generiert:</span><span class="sxs-lookup"><span data-stu-id="edd0c-108">The following sample generates CS0241.</span></span> <span data-ttu-id="edd0c-109">Darüber hinaus veranschaulicht das Beispiel mithilfe der Überladung das Simulieren einer Methode mit Standardargumenten.</span><span class="sxs-lookup"><span data-stu-id="edd0c-109">In addition, the sample shows how to simulate, with overloading, a method with default arguments.</span></span>  
  
```  
// CS0241.cs  
public class A  
{  
   public void Test(int i = 9) {}   // CS0241  
}  
  
public class B  
{  
   public void Test() { Test(9); }  
   public void Test(int i)  {}  
}  
  
public class C  
{  
   public static void Main()  
   {   
      B x = new B();  
      x.Test();  
   }  
}  
```
