# <a name="compiler-warning-level-3-cs0169"></a>Compilerwarnung (Stufe 3) CS0169
Das private Feld "Klassenmember" wird nie verwendet.  
  
 Eine private Variable wurde deklariert, es wurde jedoch nie darauf verwiesen. Diese Warnung wird h√§ufig generiert, wenn Sie einen privaten Member einer Klasse deklarieren und nicht verwenden.  
  
 Im folgenden Beispiel wird CS0169 generiert:  
  
```csharp  
// compile with: /W:3  
using System;  
public class ClassX  
{  
   int i;   // CS0169, i is not used anywhere  
  
   public static void Main()  
   {  
   }  
}  
```
