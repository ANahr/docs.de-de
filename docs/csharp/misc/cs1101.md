# <a name="compiler-error-cs1101"></a><span data-ttu-id="02afb-102">Compilerfehler CS1101</span><span class="sxs-lookup"><span data-stu-id="02afb-102">Compiler Error CS1101</span></span>
<span data-ttu-id="02afb-103">Der Parametermodifizierer „ref“ kann nicht mit „this“ verwendet werden.</span><span class="sxs-lookup"><span data-stu-id="02afb-103">The parameter modifier 'ref' cannot be used with 'this'.</span></span>  
  
 <span data-ttu-id="02afb-104">Wird der erste Parameter einer statischen Methode durch das Schlüsselwort `this` geändert, wird dem Compiler signalisiert, dass die Methode eine Erweiterungsmethode ist.</span><span class="sxs-lookup"><span data-stu-id="02afb-104">When the `this` keyword modifies the first parameter of a static method, it signals to the compiler that the method is an extension method.</span></span> <span data-ttu-id="02afb-105">Für den ersten Parameter einer Erweiterungsmethode sind keine weiteren Modifizierer erforderlich oder zulässig.</span><span class="sxs-lookup"><span data-stu-id="02afb-105">No other modifiers are needed or allowed on the first parameter of an extension method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="02afb-106">Beispiel</span><span class="sxs-lookup"><span data-stu-id="02afb-106">Example</span></span>  
 <span data-ttu-id="02afb-107">Im folgenden Beispiel wird der Fehler CS1101 generiert:</span><span class="sxs-lookup"><span data-stu-id="02afb-107">The following example generates CS1101:</span></span>  
  
```csharp  
// cs1101.cs  
// Compile with: /target:library  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(ref this int i) {} // CS1101  
  
    // Single type parameter.  
        public static void Test<T>(ref this T t) {}// CS1101  
  
    // Multiple type parameters.  
        public static void Test<T,U,V>(ref this U u) {}// CS1101  
}  
```
