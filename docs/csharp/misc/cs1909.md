# <a name="compiler-error-cs1909"></a><span data-ttu-id="d5130-102">Compilerfehler CS1909</span><span class="sxs-lookup"><span data-stu-id="d5130-102">Compiler Error CS1909</span></span>

## <a name="example"></a><span data-ttu-id="d5130-105">Beispiel</span><span class="sxs-lookup"><span data-stu-id="d5130-105">Example</span></span>  
 <span data-ttu-id="d5130-106">Im folgenden Beispiel wird CS1909 generiert:</span><span class="sxs-lookup"><span data-stu-id="d5130-106">The following sample generates CS1909.</span></span>  
  
```csharp  
// CS1909.cs  
// compile with: /target:library  
using System.Runtime.InteropServices;  
  
public interface ISomeInterface  
{  
   void Test1([DefaultParameterValue(new int[] {1, 2})] int[] arr1);   // CS1909  
  
   void Test2([DefaultParameterValue("Test String")] string s);   // OK  
}  
```
