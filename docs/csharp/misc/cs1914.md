## <a name="example"></a><span data-ttu-id="a6776-109">Beispiel</span><span class="sxs-lookup"><span data-stu-id="a6776-109">Example</span></span>  
 <span data-ttu-id="a6776-110">Der folgende Code generiert CS1914, da der Initialisierer versucht, das `TestClass.Number` -Feld zu initialisieren, das `static`ist:</span><span class="sxs-lookup"><span data-stu-id="a6776-110">The following code generates CS1914 because the initializer tries to initialize the `TestClass.Number` field, which is `static`:</span></span>  
  
```csharp  
// cs1914.cs  
using System.Linq;  
public class TestClass  
{  
    public string Message { get; set; }  
    public static int Number { get; set; }      
}  
class Test  
{  
    static void Main()  
    {  
        TestClass b = new TestClass() { Message = "Hello", Number = "555-1212" }; // CS1914  
  
    }  
}  
```
