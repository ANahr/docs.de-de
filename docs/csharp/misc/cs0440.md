## <a name="example"></a><span data-ttu-id="0da48-105">Beispiel</span><span class="sxs-lookup"><span data-stu-id="0da48-105">Example</span></span>  
 <span data-ttu-id="0da48-106">Im folgenden Beispiel wird CS0440 generiert:</span><span class="sxs-lookup"><span data-stu-id="0da48-106">The following example generates CS0440:</span></span>  
  
```csharp  
// CS0440.cs  
// Compile with: /W:1  
  
using global = MyClass;   // CS0440  
class MyClass  
{  
    static void Main()  
    {  
        // Note how global refers to the global namespace  
        // even though it is redefined above.  
        global::System.Console.WriteLine();  
    }  
}  
```
