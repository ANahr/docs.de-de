## <a name="example"></a><span data-ttu-id="4332c-106">Beispiel</span><span class="sxs-lookup"><span data-stu-id="4332c-106">Example</span></span>  
 <span data-ttu-id="4332c-107">Im folgenden Beispiel wird CS0551 generiert:</span><span class="sxs-lookup"><span data-stu-id="4332c-107">The following sample generates CS0551.</span></span>  
  
```  
// CS0551.cs  
// compile with: /target:library  
interface ii  
{  
   int i  
   {  
      get;  
      set;  
   }  
}  
  
public class a : ii  
{  
   int ii.i { set {} }   // CS0551  
  
   // OK  
   int ii.i      
   {  
      set {}  
      get { return 0; }  
   }  
}  
```
