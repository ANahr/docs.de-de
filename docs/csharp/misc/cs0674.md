 <span data-ttu-id="88ce5-105">Der C#-Compiler lässt keine für die Verwendung von <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; verwenden Sie [Params](../../csharp/language-reference/keywords/params.md) stattdessen.</span><span class="sxs-lookup"><span data-stu-id="88ce5-105">The C# compiler does not allow for the use of <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; use [params](../../csharp/language-reference/keywords/params.md) instead.</span></span>

```csharp

{

      Console.WriteLine();  
   }

      UseParams(1, 2, 3);  
   }  
}  
```
