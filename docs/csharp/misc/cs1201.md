# Compilerwarnung (Stufe 1) CS1201
Das Feature "Ungültiges Feature" ist veraltet. Verwenden Sie stattdessen "Gültiges Feature".  
  
 Das Feature, das Sie verwenden möchten, ist veraltet. Aktualisieren Sie den Code, um stattdessen das gültige Feature zu verwenden.
