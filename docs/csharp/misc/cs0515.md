# Compilerfehler CS0515

 Ein statischer Konstruktor kann keinen [Zugriffsmodifizierer](../../csharp/language-reference/keywords/modifiers.md) enthalten.  
  
## Beispiel  
 Im folgenden Beispiel wird CS0515 generiert:  
  
