 <span data-ttu-id="5a9a1-104">Es wurde versucht, auf einen Wert über einen Indexer für einen Datentyp zuzugreifen, der [Indexers](../../csharp/programming-guide/indexers/index.md)nicht unterstützt.</span><span class="sxs-lookup"><span data-stu-id="5a9a1-104">An attempt was made to access a value through an indexer on a data type that does not support [Indexers](../../csharp/programming-guide/indexers/index.md).</span></span>

## <a name="example"></a><span data-ttu-id="5a9a1-108">Beispiel</span><span class="sxs-lookup"><span data-stu-id="5a9a1-108">Example</span></span>  
 <span data-ttu-id="5a9a1-109">Diese Datei wird in eine DLL-Datei kompiliert. Dabei wurde das `DefaultMember` -Attribut auskommentiert, um den Fehler zu generieren.</span><span class="sxs-lookup"><span data-stu-id="5a9a1-109">This file compiles to a .dll file—with the `DefaultMember` attribute commented out—in order to generate the error.</span></span>  
  
```  
// CPP0021.cpp  
// compile with: /clr /LD  
using namespace System::Reflection;  
// Uncomment the following line to resolve  
//[DefaultMember("myItem")]  
public ref class MyClassMC  
{  
        public:  
        property int myItem[int]  
        {  
            int get(int i){  return 5; }  
            void set(int i, int value) {}  
        }  
};  
```
