## <a name="example"></a><span data-ttu-id="8ad7d-108">Beispiel</span><span class="sxs-lookup"><span data-stu-id="8ad7d-108">Example</span></span>  
 <span data-ttu-id="8ad7d-109">Im folgenden Beispiel wird der Fehler CS1102 generiert:</span><span class="sxs-lookup"><span data-stu-id="8ad7d-109">The following example generates CS1102:</span></span>  
  
```  
// cs1102.cs  
// Compile with: /target:library.  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(this out int i) {} // CS1102  
  
    //Single type parameter  
        public static void Test<T>(this out T t) {}// CS1102  
  
    //Multiple type parameters  
        public static void Test<T,U,V>(this out U u) {}// CS1102  
}  
```
