# <a name="compiler-error-cs0669"></a>Compilerfehler CS0669

Eine Klasse mit dem ComImport-Attribut kann nicht auf einen benutzerdefinierten Konstruktor haben.

Die COM-Interop-Ebene in der Common Language Runtime gibt den Konstruktor f√ºr [ComImport](xref:System.Runtime.InteropServices.ComImportAttribute) -Klassen an. Folglich kann ein COM-Objekt als ein verwaltetes Objekt in der Laufzeit verwendet werden.

Im folgenden Beispiel wird CS0669 generiert:

```csharp
// CS0669.cs
using System.Runtime.InteropServices;
[ComImport, Guid("00000000-0000-0000-0000-000000000001")]
class TestClass
{
   TestClass()   // CS0669, delete constructor to resolve
   {
   }

   public static void Main()
   {
   }
}
```
