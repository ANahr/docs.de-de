# <a name="compiler-warning-level-1-cs3001"></a>Compilerwarnung (Stufe 1) CS3001
Der Argumenttyp "Typ" ist nicht CLS-kompatibel  
  
 Ein [öffentlichen](../../csharp/language-reference/keywords/public.md), [geschützt](../../csharp/language-reference/keywords/protected.md), oder `protected internal` Methode muss akzeptieren einen Parameter, dessen Typ mit der Common Language Specification (CLS) kompatibel ist. Weitere Informationen zu den CLS-Konformitätsregeln finden Sie unter [Writing CLS-Compliant Code (Schreiben von CLS-kompatiblem Code)](http://msdn.microsoft.com/en-us/4c705105-69a2-4e5e-b24e-0633bc32c7f3)und [Language Independence and Language-Independent Components (Sprachunabhängigkeit und sprachunabhängige Komponenten)](../../../docs/standard/language-independence-and-language-independent-components.md).  
  
## <a name="example"></a>Beispiel  
 Im folgenden Beispiel wird CS3001 generiert:  
  
```  
// CS3001.cs  
  
[assembly:System.CLSCompliant(true)]  
public class a  
{  
    public void bad(ushort i)   // CS3001  
    {  
    }  
  
    private void OK(ushort i)   // OK, private method  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
