# <a name="compiler-error-cs0030"></a><span data-ttu-id="3a5db-102">Compilerfehler CS0030</span><span class="sxs-lookup"><span data-stu-id="3a5db-102">Compiler Error CS0030</span></span>
<span data-ttu-id="3a5db-103">Eine Konvertierung vom Typ "type" in "type" ist nicht möglich</span><span class="sxs-lookup"><span data-stu-id="3a5db-103">Cannot convert type 'type' to 'type'</span></span>  
  
 <span data-ttu-id="3a5db-104">Sie müssen zur Unterstützung bestimmter Operatorüberladungen Konvertierungsroutinen angeben.</span><span class="sxs-lookup"><span data-stu-id="3a5db-104">You must provide conversion routines to support certain operator overloads.</span></span> <span data-ttu-id="3a5db-105">Weitere Informationen finden Sie unter [Konvertierungsoperatoren](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span><span class="sxs-lookup"><span data-stu-id="3a5db-105">For more information, see [Conversion Operators](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span></span>  
  
 <span data-ttu-id="3a5db-106">Im folgenden Beispiel wird CS0030 generiert:</span><span class="sxs-lookup"><span data-stu-id="3a5db-106">The following sample generates CS0030:</span></span>  
  
```csharp  
// CS0030.cs  
namespace x  
{  
   public class iii  
   {  
      /*  
      public static implicit operator iii(int aa)  
      {  
         return null;  
      }  
  
      public static implicit operator int(iii aa)  
      {  
         return 0;  
      }  
      */  
  
      public static iii operator ++(iii aa)  
      {  
         return (iii)0;   // CS0030  
         // uncomment the conversion routines to resolve CS0030  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
