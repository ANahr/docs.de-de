 <span data-ttu-id="a090c-104">Dieser Fehler tritt auf, wenn keine statischen [Main](../../csharp/programming-guide/main-and-command-args/index.md) Methode mit einer richtigen Signatur befindet sich in den Code, der eine ausführbare Datei erstellt.</span><span class="sxs-lookup"><span data-stu-id="a090c-104">This error occurs when no static [Main](../../csharp/programming-guide/main-and-command-args/index.md) method with a correct signature is found in the code that produces an executable file.</span></span> <span data-ttu-id="a090c-105">Dieser Fehler tritt ebenfalls auf, wenn die Eintrittspunktfunktion `Main`, mit falscher Groß-/Kleinschreibung definiert ist, etwa als klein geschriebenes `main`.</span><span class="sxs-lookup"><span data-stu-id="a090c-105">This error also occurs if the entry point function, `Main`, is defined with the wrong case, such as lower-case `main`.</span></span>
