 <span data-ttu-id="9efe3-104">Die Vererbungsliste für eine [Schnittstelle](../../csharp/language-reference/keywords/interface.md) enthält einen direkten oder indirekten Verweis auf sich selbst.</span><span class="sxs-lookup"><span data-stu-id="9efe3-104">The inheritance list for an [interface](../../csharp/language-reference/keywords/interface.md) includes a direct or indirect reference to itself.</span></span> <span data-ttu-id="9efe3-105">Eine Schnittstelle kann nicht von sich selbst erben.</span><span class="sxs-lookup"><span data-stu-id="9efe3-105">An interface cannot inherit from itself.</span></span>

   {  
   }
