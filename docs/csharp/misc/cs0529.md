<span data-ttu-id="6c9e2-103">Die geerbte Schnittstelle 'Schnittstelle1' verursacht eine Schleife in der Schnittstellenhierarchie von 'Schnittstelle2'.</span><span class="sxs-lookup"><span data-stu-id="6c9e2-103">Inherited interface 'interface1' causes a cycle in the interface hierarchy of 'interface2'</span></span>  
  
 <span data-ttu-id="6c9e2-104">Die Vererbungsliste für eine [Schnittstelle](../../csharp/language-reference/keywords/interface.md) enthält einen direkten oder indirekten Verweis auf sich selbst.</span><span class="sxs-lookup"><span data-stu-id="6c9e2-104">The inheritance list for an [interface](../../csharp/language-reference/keywords/interface.md) includes a direct or indirect reference to itself.</span></span> <span data-ttu-id="6c9e2-105">Eine Schnittstelle kann nicht von sich selbst erben.</span><span class="sxs-lookup"><span data-stu-id="6c9e2-105">An interface cannot inherit from itself.</span></span>  
  
 <span data-ttu-id="6c9e2-106">Im folgenden Beispiel wird CS0529 generiert:</span><span class="sxs-lookup"><span data-stu-id="6c9e2-106">The following sample generates CS0529:</span></span>  
  
```csharp  
// CS0529.cs

{  
   public interface a  
   {  
   }  
  
   public interface b : a, c  
   {  
   }  
  
   public interface c : b   // CS0529, b inherits from c  
   {  
   }  
}  
```
