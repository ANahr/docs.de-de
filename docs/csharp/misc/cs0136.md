## <a name="example"></a><span data-ttu-id="c9a9e-106">Beispiel</span><span class="sxs-lookup"><span data-stu-id="c9a9e-106">Example</span></span>  
 <span data-ttu-id="c9a9e-107">Im folgenden Beispiel wird CS0136 generiert:</span><span class="sxs-lookup"><span data-stu-id="c9a9e-107">The following sample generates CS0136:</span></span>  
  
```csharp
// CS0136.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         {  
            char i = 'a';   // CS0136, hides int i  
         }  
         i++;  
      }  
   }  
}  
```
