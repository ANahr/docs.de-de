# <a name="compiler-error-cs1558"></a><span data-ttu-id="f864b-102">Compilerfehler CS1558</span><span class="sxs-lookup"><span data-stu-id="f864b-102">Compiler Error CS1558</span></span>
<span data-ttu-id="f864b-103">'Klasse' hat keine passende statische Main-Methode.</span><span class="sxs-lookup"><span data-stu-id="f864b-103">'class' does not have a suitable static Main method</span></span>  
  
 <span data-ttu-id="f864b-104">Die [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) -Compileroption angegeben, eine Klasse, suchen Sie nach einer **Main** Methode.</span><span class="sxs-lookup"><span data-stu-id="f864b-104">The [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) compiler option specified a class in which to look for a **Main** method.</span></span> <span data-ttu-id="f864b-105">Allerdings die [Main](../../csharp/programming-guide/main-and-command-args/index.md) Methode wurde nicht ordnungsgemäß definiert.</span><span class="sxs-lookup"><span data-stu-id="f864b-105">However, the [Main](../../csharp/programming-guide/main-and-command-args/index.md) method was not defined correctly.</span></span>  
  
 <span data-ttu-id="f864b-106">Im folgende Beispiel wird CS1558 aufgrund eines ungültigen Rückgabetyps generiert.</span><span class="sxs-lookup"><span data-stu-id="f864b-106">The following example generates CS1558 because of invalid return type.</span></span>  
  
```  
// CS1558.cs  
// compile with: /main:MyNamespace.MyClass  
  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static float Main()   
      {  
         return 0.0; // CS1558 because the return type is a float.  
      }  
   }  
}  
```
