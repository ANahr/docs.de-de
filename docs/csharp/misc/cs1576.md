# <a name="compiler-error-cs1576"></a>Compilerfehler CS1576
Die Zeilennummer, die für die #line-Direktive angegeben wurde, fehlt oder ist ungültig.  
  
 Der Compiler hat einen Fehler bei dem an die [#line](../../csharp/language-reference/preprocessor-directives/preprocessor-line.md) -Direktive übergebenen Wert erkannt.  
  
 Im folgenden Beispiel wird CS1576 generiert:  
  
```  
// CS1576.cs  
public class MyClass  
{  
   static void Main()  
   {  
      #line "abc.sc"         // CS1576  
      // try the following line instead  
      //#line  101 "abc.sc"  
      intt i;  // error will be reported on line 101  
   }  
}  
```
