<span data-ttu-id="ca787-103">Der aufgefangene oder ausgelöste Typ muss von System.Exception abgeleitet werden.</span><span class="sxs-lookup"><span data-stu-id="ca787-103">The type caught or thrown must be derived from System.Exception</span></span>  
  
 <span data-ttu-id="ca787-104">Es wurde versucht, einen Datentyp zu übergeben, der nicht von abgeleitet ist **System.Exception** in einem [catch](../../csharp/language-reference/keywords/try-catch.md) Block.</span><span class="sxs-lookup"><span data-stu-id="ca787-104">An attempt was made to pass a data type that does not derive from **System.Exception** into a [catch](../../csharp/language-reference/keywords/try-catch.md) block.</span></span> <span data-ttu-id="ca787-105">An einen **catch** -Block können nur Datentypen übergeben werden, die von **System.Exception** abgeleitet wurden.</span><span class="sxs-lookup"><span data-stu-id="ca787-105">Only data types that derive from **System.Exception** can be passed into a **catch** block.</span></span> <span data-ttu-id="ca787-106">Weitere Informationen finden Sie unter [Exception Handling Statements](../../csharp/language-reference/keywords/exception-handling-statements.md) und [Ausnahmen und Ausnahmebehandlung](../../csharp/programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="ca787-106">For more information, see [Exception Handling Statements](../../csharp/language-reference/keywords/exception-handling-statements.md) and [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="ca787-107">Im folgenden Beispiel wird CS0155 generiert:</span><span class="sxs-lookup"><span data-stu-id="ca787-107">The following sample generates CS0155:</span></span>  
  
```  
// CS0155.cs  
using System;  
  
namespace MyNamespace  
{  
    public class MyClass2  
    // try the following line instead  
    // public class MyClass2 : Exception  
    {  
    }  
    public class MyClass  
    {  
        public static void Main()  
        {  
            try  
            {  
            }  
            catch (MyClass2)   // CS0155, resolves if you derive MyClass2 from Exception  
            {  
            }  
        }  
    }  
}  
```
