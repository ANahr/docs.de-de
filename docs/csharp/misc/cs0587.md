# <a name="compiler-error-cs0587"></a>Compilerfehler CS0587
Interner Compilerfehler: Stufe 'STAGE'  
  
 Versuchen Sie festzustellen, ob der Compiler einen Fehler verursacht, weil er unerwartete Syntax nicht analysieren kann. Wenn dies nicht der Fall ist, versuchen Sie es [sprechen Sie mit uns](/visualstudio/ide/talk-to-us).
