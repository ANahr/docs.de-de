## <a name="example"></a><span data-ttu-id="75e3c-110">Beispiel</span><span class="sxs-lookup"><span data-stu-id="75e3c-110">Example</span></span>  
 <span data-ttu-id="75e3c-111">Im folgenden Code wird die Warnung CS0728 generiert.</span><span class="sxs-lookup"><span data-stu-id="75e3c-111">The following code will generate warning CS0728.</span></span>  
  
```csharp

using System;  
public class ValidBase : IDisposable  
{  
    public void Dispose() {  }  
}  
  
public class Logger  
{  
    public static void dummy()  
    {  
        ValidBase vb = null;  
        using (vb)   
        {  
            vb = null;  // CS0728  
        }  
        vb = null;  
    }  
    public static void Main() { }  
}  
```
