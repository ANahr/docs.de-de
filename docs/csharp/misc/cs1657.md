 <span data-ttu-id="a6871-104">Dieser Fehler tritt auf, wenn eine Variable als [ref](../../csharp/language-reference/keywords/ref.md) - oder [out](../../csharp/language-reference/keywords/out.md) -Argument in einem Kontext verwendet wird, in dem diese Variable schreibgeschützt ist.</span><span class="sxs-lookup"><span data-stu-id="a6871-104">This error occurs when a variable is passed as a [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out.md) argument in a context in which that variable is readonly.</span></span> <span data-ttu-id="a6871-105">Schreibgeschützte Kontexte schließen [foreach](../../csharp/language-reference/keywords/foreach-in.md) -Iterationsvariablen, [using](../../csharp/language-reference/keywords/using-statement.md) -Variablen und `fixed` -Variablen ein.</span><span class="sxs-lookup"><span data-stu-id="a6871-105">Readonly contexts include [foreach](../../csharp/language-reference/keywords/foreach-in.md) iteration variables, [using](../../csharp/language-reference/keywords/using-statement.md) variables, and `fixed` variables.</span></span> <span data-ttu-id="a6871-106">Um diesen Fehler zu beheben, rufen Sie keine Funktionen auf, die `foreach`-, `using` - oder `fixed` -Variablen als `ref` - oder `out` -Parameter in `using` -Blöcken, `foreach` -Anweisungen und `fixed` -Anweisungen verwenden.</span><span class="sxs-lookup"><span data-stu-id="a6871-106">To resolve this error, do not call functions that take the `foreach`, `using` or `fixed` variable as a `ref` or `out` parameter in `using` blocks, `foreach` statements, and `fixed` statements.</span></span>  
  
## <a name="example"></a><span data-ttu-id="a6871-107">Beispiel</span><span class="sxs-lookup"><span data-stu-id="a6871-107">Example</span></span>  
 <span data-ttu-id="a6871-108">Im folgenden Beispiel wird CS1657 generiert:</span><span class="sxs-lookup"><span data-stu-id="a6871-108">The following sample generates CS1657:</span></span>  
  
```  
// CS1657.cs  
using System;  
class C : IDisposable  
{  
    public int i;  
    public void Dispose() {}  
}  
  
class CMain  
{  
    static void f(ref C c)  
    {  
    }  
    static void Main()  
    {  
        using (C c = new C())  
        {  
            f(ref c);  // CS1657  
        }  
    }  
}  
```
