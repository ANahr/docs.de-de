## <a name="example"></a><span data-ttu-id="82b64-105">Beispiel</span><span class="sxs-lookup"><span data-stu-id="82b64-105">Example</span></span>  
 <span data-ttu-id="82b64-106">Im folgenden Beispiel wird CS0135 generiert:</span><span class="sxs-lookup"><span data-stu-id="82b64-106">The following sample generates CS0135:</span></span>  
  
```csharp
// CS0135.cs  
public class MyClass2  
{  
   public static int i = 0;  
  
   public static void Main()  
   {  
      {  
         int i = 4;  
         i++;  
      }  
      i = 0;   // CS0135  
   }  
}  
```
