## <a name="example"></a><span data-ttu-id="71a2e-105">Beispiel</span><span class="sxs-lookup"><span data-stu-id="71a2e-105">Example</span></span>  
 <span data-ttu-id="71a2e-106">Im folgenden Beispiel wird CS0472 generiert:</span><span class="sxs-lookup"><span data-stu-id="71a2e-106">The following sample generates CS0472.</span></span>  
  
```csharp  
public class Test  
{  
    public static int Main()  
    {  
        int i = 5;  
        int counter = 0;  
  
        // Comparison:  
        if (i == null)  // CS0472  
        // To resolve, use a valid value for i.  
            counter++;   
        return counter;  
    }  
}  
```
