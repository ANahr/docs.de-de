 <span data-ttu-id="426e1-108">Durch den folgenden Code wird der Fehler CS1931 ausgelöst, da der Bezeichner `x` sowohl als lokale Variable in `Main` als auch als Bereichsvariable im Abfrageausdruck verwendet wird:</span><span class="sxs-lookup"><span data-stu-id="426e1-108">The following code generates CS1931 because the identifier `x` is used both as a local variable in `Main` and as the range variable in the query expression:</span></span>
