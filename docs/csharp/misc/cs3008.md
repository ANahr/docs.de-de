# <a name="compiler-warning-level-1-cs3008"></a>Compilerwarnung (Stufe 1) CS3008
Der Bezeichner 'identifier', der sich nur hinsichtlich der Groß- und Kleinschreibung unterscheidet, ist nicht CLS-kompatibel.  
  
 Ein [öffentlichen](../../csharp/language-reference/keywords/public.md), [geschützt](../../csharp/language-reference/keywords/protected.md), oder `protected internal` Bezeichner unterbricht die Kompatibilität mit der Common Language Specification (CLS), wenn er mit einem Unterstrich (_) beginnt. Weitere Informationen zur CLS-Kompatibilität finden Sie unter [Schreiben von CLS-kompatiblem Code](http://msdn.microsoft.com/en-us/4c705105-69a2-4e5e-b24e-0633bc32c7f3) und [Sprachenunabhängigkeit und sprachunabhängige Komponenten](https://msdn.microsoft.com/library/12a7a7h3).  
  
## <a name="example"></a>Beispiel  
 Im folgenden Beispiel wird CS3008 generiert:  
  
```  
// CS3008.cs  
  
using System;  
  
[assembly:CLSCompliant(true)]  
public class a  
{  
    public static int _a = 0;  // CS3008  
    // OK, private  
    // private static int _a1 = 0;  
  
    public static void Main()  
    {  
    }  
}  
```
