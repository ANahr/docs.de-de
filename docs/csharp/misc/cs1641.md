<span data-ttu-id="2e1e7-103">Bei einem Pufferfeld fester Größe muss sich der Arraygrößenspezifizierer hinter dem Feldnamen befinden.</span><span class="sxs-lookup"><span data-stu-id="2e1e7-103">A fixed size buffer field must have the array size specifier after the field name</span></span>

 <span data-ttu-id="2e1e7-106">Im folgenden Beispiel wird CS1641 generiert:</span><span class="sxs-lookup"><span data-stu-id="2e1e7-106">The following sample generates CS1641:</span></span>  
  
```csharp  
// CS1641.cs  
// compile with: /unsafe /target:library  
unsafe struct S {  
   fixed int [] a;  // CS1641  
  
   // OK  
   fixed int b [10];  
   const int c = 10;  
   fixed int d [c];  
}  
```
