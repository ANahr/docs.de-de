 <span data-ttu-id="cc1a8-106">Im folgenden Beispiel wird CS1641 generiert:</span><span class="sxs-lookup"><span data-stu-id="cc1a8-106">The following sample generates CS1641:</span></span>  
  
```  
// CS1641.cs  
// compile with: /unsafe /target:library  
unsafe struct S {  
   fixed int [] a;  // CS1641  
  
   // OK  
   fixed int b [10];  
   const int c = 10;  
   fixed int d [c];  
}  
```
