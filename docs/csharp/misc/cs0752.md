 <span data-ttu-id="34c46-104">Eine partielle Methode darf keinen [out](../../csharp/language-reference/keywords/out.md) -Parameter enthalten.</span><span class="sxs-lookup"><span data-stu-id="34c46-104">A partial method cannot have an [out](../../csharp/language-reference/keywords/out.md) parameter.</span></span> <span data-ttu-id="34c46-105">Out-Parameter sind nicht zulässig, da es beim Entfernen der partiellen Methode durch den Compiler keine Garantie gibt, dass der out-Parameter jemals zugewiesen wird.</span><span class="sxs-lookup"><span data-stu-id="34c46-105">Out parameters are not allowed because if the partial method is removed by the compiler then there is no guarantee that the out parameter is ever assigned.</span></span>
