## <a name="example"></a><span data-ttu-id="f94b2-110">Beispiel</span><span class="sxs-lookup"><span data-stu-id="f94b2-110">Example</span></span>  
 <span data-ttu-id="f94b2-111">Im folgende Beispiel wird CS1059 generiert, da `i` eine Konstante und keine Variable und `E` ein `Enum` -Typ ist, dessen Elemente auch Konstantenwerte sind.</span><span class="sxs-lookup"><span data-stu-id="f94b2-111">The following example generates CS1059 because `i` is a constant, not a variable, and `E` is an `Enum` type, whose elements are also constant values.</span></span>  
  
```csharp  
// CS1059.cs  
    class Program  
    {  
        public enum E : sbyte  
        {  
            a = 1,  
            b = 2  
        }  
  
        static void Main(string[] args)  
        {  
            const int i = 0;  
            i++;            // CS1059  
            E test = E.a++; // CS1059  
        }  
    }  
```
