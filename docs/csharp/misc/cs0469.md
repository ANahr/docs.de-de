## <a name="example"></a><span data-ttu-id="6b37f-105">Beispiel</span><span class="sxs-lookup"><span data-stu-id="6b37f-105">Example</span></span>  
 <span data-ttu-id="6b37f-106">Im folgenden Beispiel wird CS0469 generiert:</span><span class="sxs-lookup"><span data-stu-id="6b37f-106">The following sample generates CS0469.</span></span>  
  
```csharp  
// CS0469.cs  
// compile with: /W:2  
class Test  
{  
   static void Main()  
   {  
      char c = (char)180;  
      switch (c)  
      {  
         case (char)127:  
            break;  
  
         case (char)180:   
            goto case 127;   // CS0469  
            // try the following line instead  
            // goto case (char) 127;  
      }  
   }  
}  
```
