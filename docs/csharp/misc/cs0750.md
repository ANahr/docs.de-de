# <a name="compiler-error-cs0750"></a><span data-ttu-id="90f20-102">Compilerfehler CS0750</span><span class="sxs-lookup"><span data-stu-id="90f20-102">Compiler Error CS0750</span></span>
<span data-ttu-id="90f20-103">Eine partielle Methode darf keine Zugriffsmodifizierer oder die Modifizierer "virtual", "abstract", "override", "new", "sealed" oder "extern" enthalten.</span><span class="sxs-lookup"><span data-stu-id="90f20-103">A partial method cannot have access modifiers or the virtual, abstract, override, new, sealed, or extern modifiers.</span></span>  
  
 <span data-ttu-id="90f20-104">Aufgrund ihres besonderen Verhaltens unterliegen partielle Methoden Beschränkungen hinsichtlich der Modifizierer, die sie akzeptieren können .</span><span class="sxs-lookup"><span data-stu-id="90f20-104">Because of their special behavior, partial methods are subject to restrictions as to the modifiers they can accept.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="90f20-105">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="90f20-105">To correct this error</span></span>  
  
1.  <span data-ttu-id="90f20-106">Entfernen Sie den unzulässigen Modifizierer aus der Methodendeklaration.</span><span class="sxs-lookup"><span data-stu-id="90f20-106">Remove the unauthorized modifier from the method declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="90f20-107">Beispiel</span><span class="sxs-lookup"><span data-stu-id="90f20-107">Example</span></span>  
 <span data-ttu-id="90f20-108">Im folgenden Beispiel wird der Fehler CS0750 generiert:</span><span class="sxs-lookup"><span data-stu-id="90f20-108">The following example generates CS0750:</span></span>  
  
```csharp  
// cs0750.cs  
using System;  
  
public class Base  
{  
    protected virtual void PartG()  
    {  
    }  
  
    protected void PartH()  
    {  
    }  
    protected virtual void PartI()  
    {  
    }  
}  
  
public partial class C:Base  
{  
    // All these partial method declarations  
    // will generate CS0750.  
    public partial void PartA();  
    private partial void PartB();  
    protected partial void PartC();  
    internal partial void PartD();  
    virtual partial void PartE();  
    abstract partial void PartF();  
    override partial void PartG();  
    new partial void PartH();  
    sealed override partial void PartI();  
    extern partial void PartJ();  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
