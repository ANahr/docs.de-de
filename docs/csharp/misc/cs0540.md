## <a name="example"></a><span data-ttu-id="65302-106">Beispiel</span><span class="sxs-lookup"><span data-stu-id="65302-106">Example</span></span>  
 <span data-ttu-id="65302-107">Im folgenden Beispiel wird CS0540 generiert:</span><span class="sxs-lookup"><span data-stu-id="65302-107">The following sample generates CS0540.</span></span>  
  
```  
// CS0540.cs  
interface I  
{  
   void m();  
}  
  
public class Clx  
{  
   void I.m() {}   // CS0540  
}  
  
// OK  
public class Cly : I  
{  
   void I.m() {}  
   public static void Main() {}  
}  
```
