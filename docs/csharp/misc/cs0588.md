# <a name="compiler-error-cs0588"></a>Compilerfehler CS0588
Interner Compilerfehler: Stufe 'LEX'  
  
 Versuchen Sie festzustellen, ob der Compiler einen Fehler verursacht, da unerwartete Syntax nicht analysiert werden kann. Wenn dies nicht der Fall ist, versuchen Sie es [sprechen Sie mit uns](/visualstudio/ide/talk-to-us).
