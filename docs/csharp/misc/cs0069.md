# <a name="compiler-error-cs0069"></a>Compilerfehler CS0069
Ein Ereignis in einer Schnittstelle kann keine add- oder remove-Accessoren haben.  
  
 Sie k√∂nnen die Accessorfunktionen eines Ereignisses nicht in einer [Schnittstelle](../../csharp/language-reference/keywords/interface.md)definieren. Weitere Informationen finden Sie unter [Ereignisse](../../csharp/programming-guide/events/index.md) und [Schnittstellen](../../csharp/programming-guide/interfaces/index.md).  
  
 Im folgenden Beispiel wird CS0069 generiert:  
  
```csharp  
// CS0069.cs  
// compile with: /target:library  
  
public delegate void EventHandler();  
  
public interface a  
{  
   event EventHandler Click { remove {} }   // CS0069  
   event EventHandler Click2;   // OK  
}  
```
