# Compilerfehler CS0014
Die erforderliche Datei "file" konnte nicht gefunden werden  
  
 Eine Datei wird vom Compiler benötigt, aber sie ist nicht auf dem System. Stellen Sie sicher, dass der Pfad korrekt ist. Wenn die Datei eine [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)]\-Systemdatei ist, dann müssen Sie möglicherweise die Installation reparieren oder aber [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)] vollständig entfernen und neu installieren.
