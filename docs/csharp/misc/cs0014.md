# <a name="compiler-error-cs0014"></a>Compilerfehler CS0014
Die erforderliche Datei "file" konnte nicht gefunden werden  
  
 Eine Datei wird vom Compiler benötigt, aber sie ist nicht auf dem System. Stellen Sie sicher, dass der Pfad korrekt ist. Wenn die Datei eine Systemdatei Visual Studio ist, müssen Sie möglicherweise die Installation reparieren, da sonst entfernen und installieren Sie Visual Studio vollständig.
