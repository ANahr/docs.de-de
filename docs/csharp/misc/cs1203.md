# Compilerwarnung (Stufe 1) CS1203
Die Funktion 'funktion' ist veraltet. Verwenden Sie stattdessen 'funktion'.  
  
 Die Funktion 'ungültige funktion' ist veraltet. Verwenden Sie stattdessen 'gültige funktion'.  
  
 Die Funktion, die Sie verwenden möchten, ist veraltet. Aktualisieren Sie den Code, um stattdessen die gültige Funktion zu verwenden.
