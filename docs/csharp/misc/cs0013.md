# Compilerfehler CS0013
Unerwarteter Fehler beim Schreiben der Metadaten in die Datei „ModelStore.dll“ \-\- Kein freier logischer Speicherplatz zum Erstellen weiterer Benutzerzeichenfolgen.  
  
 Die Common Language Runtime von .NET Framework hat keine Metadaten ausgegeben. Stellen Sie sicher, dass der Pfad richtig ist und dass der Datenträger nicht voll ist. Wenn das Problem weiterhin besteht, müssen Sie [!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)] und\/oder .NET Framework möglicherweise reparieren oder neu installieren.
