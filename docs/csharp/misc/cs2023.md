# <a name="compiler-warning-level-1-cs2023"></a>Compilerwarnung (Stufe 1) CS2023
Die /noconfig-Option wird ignoriert, da sie in einer Antwortdatei angegeben wurde.  
  
 Die Compileroption [/noconfig](../../csharp/language-reference/compiler-options/noconfig-compiler-option.md) wurde in einer Antwortdatei angegeben, was nicht zulässig ist.  
  
 Diese Warnung kann nicht durch die [/nowarn](../../csharp/language-reference/compiler-options/nowarn-compiler-option.md) -Option unterdrückt werden.
