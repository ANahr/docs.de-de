# Compilerfehler CS0008
Unerwarteter Fehler beim Lesen der Metadaten aus der Datei "Datei" \-\- "Beschreibung"  
  
 Eine DLL, aus der Metadaten abgerufen werden sollen, wurde erfolgreich geöffnet, ist jedoch so beschädigt, dass keine Daten aus ihr gelesen werden konnten. Weitere Informationen finden Sie unter [\/reference \(Import Metadata\)](../../csharp/language-reference/compiler-options/reference-compiler-option.md).
