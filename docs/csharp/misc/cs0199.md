 <span data-ttu-id="ea628-104">Eine [readonly](../../csharp/language-reference/keywords/readonly.md) -Variable muss dieselbe [static](../../csharp/language-reference/keywords/static.md) -Verwendung wie der Konstruktor aufweisen, in dem sie als [ref](../../csharp/language-reference/keywords/ref.md) - oder [out](../../csharp/language-reference/keywords/out.md) -Parameter übergeben werden soll.</span><span class="sxs-lookup"><span data-stu-id="ea628-104">A [readonly](../../csharp/language-reference/keywords/readonly.md) variable must have the same [static](../../csharp/language-reference/keywords/static.md) usage as the constructor in which you want to pass it as a [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out.md) parameter.</span></span> <span data-ttu-id="ea628-105">Weitere Informationen finden Sie unter [Übergeben von Parametern](../../csharp/programming-guide/classes-and-structs/passing-parameters.md).</span><span class="sxs-lookup"><span data-stu-id="ea628-105">For more information, see [Passing Parameters](../../csharp/programming-guide/classes-and-structs/passing-parameters.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="ea628-106">Beispiel</span><span class="sxs-lookup"><span data-stu-id="ea628-106">Example</span></span>  
 <span data-ttu-id="ea628-107">Im folgenden Beispiel wird CS0199 generiert:</span><span class="sxs-lookup"><span data-stu-id="ea628-107">The following sample generates CS0199:</span></span>  
  
```  
// CS0199.cs  
class MyClass  
{  
    public static readonly int TestInt = 6;  
  
    static void TestMethod(ref int testInt)  
    {  
        testInt = 0;  
    }  
  
    MyClass()  
    {  
        TestMethod(ref TestInt);   // CS0199, TestInt is static  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
