 <span data-ttu-id="77fd5-104">Das [Conditional](http://msdn.microsoft.com/en-us/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) -Attribut für für Methoden nicht zulässig, die mit dem [override](../../csharp/language-reference/keywords/override.md) -Schlüsselwort gekennzeichnet sind.</span><span class="sxs-lookup"><span data-stu-id="77fd5-104">The [Conditional](http://msdn.microsoft.com/en-us/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) attribute is not allowed on a method that is marked with the [override](../../csharp/language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="77fd5-105">Weitere Informationen finden Sie unter [zu wissen, bei der Verwendung außer Kraft zu setzen und neue Schlüsselwörter](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span><span class="sxs-lookup"><span data-stu-id="77fd5-105">For more information, see [Knowing When to Use Override and New Keywords](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
 <span data-ttu-id="77fd5-106">Der Compiler bindet niemals Überschreibungsmethoden. Er bindet nur die Basismethode und die Common Language Runtime ruft dann die Überschreibung entsprechend auf.</span><span class="sxs-lookup"><span data-stu-id="77fd5-106">The compiler never binds to override methods; it only binds to the base method and the common language runtime calls the override, as appropriate.</span></span>  
  
 <span data-ttu-id="77fd5-107">Im folgenden Beispiel wird CS0243 generiert:</span><span class="sxs-lookup"><span data-stu-id="77fd5-107">The following sample generates CS0243:</span></span>  
  
```  
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
