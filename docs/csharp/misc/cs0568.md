# <a name="compiler-error-cs0568"></a><span data-ttu-id="48ae3-102">Compilerfehler CS0568</span><span class="sxs-lookup"><span data-stu-id="48ae3-102">Compiler Error CS0568</span></span>
<span data-ttu-id="48ae3-103">Strukturen können keine expliziten parameterlosen Konstruktoren enthalten.</span><span class="sxs-lookup"><span data-stu-id="48ae3-103">Structs cannot contain explicit parameterless constructors</span></span>  
  
 <span data-ttu-id="48ae3-104">Jede [Struktur](../../csharp/language-reference/keywords/struct.md) verfügt bereits über einen Standardkonstruktor, der das Objekt zu null initialisiert.</span><span class="sxs-lookup"><span data-stu-id="48ae3-104">Each [struct](../../csharp/language-reference/keywords/struct.md) already has a default constructor that initializes the object to zero.</span></span> <span data-ttu-id="48ae3-105">Aus diesem Grund müssen die Konstruktoren, die Sie für eine Struktur erstellen, einen oder mehrere Parameter annehmen können.</span><span class="sxs-lookup"><span data-stu-id="48ae3-105">Therefore, the constructors that you can create for a struct must take one or more parameters.</span></span>  
  
 <span data-ttu-id="48ae3-106">Im folgenden Beispiel wird CS0568 generiert:</span><span class="sxs-lookup"><span data-stu-id="48ae3-106">The following sample generates CS0568:</span></span>  
  
```csharp  
// CS0568.cs  
public struct ClassY  
{  
   public int field1;  
   public ClassY(){}   // CS0568, cannot have no param constructor  
   // Try following instead:  
   // public ClassY(int i)  
   // {  
   //    field1 = i;  
   // }  
}  
  
public class ClassX  
{  
   public static void Main()  
   {  
   }  
}  
```
