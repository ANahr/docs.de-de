# <a name="compiler-error-cs0584"></a>Compilerfehler CS0584
Interner Compilerfehler: Stufe 'Stufe' Symbol 'Symbol'  
  
 Versuchen Sie festzustellen, ob der Compiler einen Fehler verursacht, da unerwartete Syntax nicht analysiert werden kann. Wenn dies nicht der Fall ist, versuchen Sie [sprechen Sie mit uns](https://docs.microsoft.com/visualstudio/ide/talk-to-us).
