 Versuchen Sie festzustellen, ob der Compiler einen Fehler verursacht, da unerwartete Syntax nicht analysiert werden kann. Wenn das nicht der Fall ist, versuchen Sie es [sprechen Sie mit uns](/visualstudio/ide/talk-to-us).
