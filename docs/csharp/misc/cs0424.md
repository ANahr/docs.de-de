<span data-ttu-id="b009c-103">"Klasse": Eine Klasse mit dem ComImport-Attribut kann keine Basisklasse angeben.</span><span class="sxs-lookup"><span data-stu-id="b009c-103">'class': a class with the ComImport attribute cannot specify a base class</span></span>  
  
 <span data-ttu-id="b009c-104">Die Angabe des <xref:System.Runtime.InteropServices.ComImportAttribute> -Attributs impliziert, dass die Implementierung für die Klasse von einem COM-Modul importiert werden muss.</span><span class="sxs-lookup"><span data-stu-id="b009c-104">Specifying the <xref:System.Runtime.InteropServices.ComImportAttribute> attribute implies that the implementation for the class is to be imported from a COM module.</span></span> <span data-ttu-id="b009c-105">Weitere von der Basisklasse geerbte Methoden oder Felder dürfen der im COM-Modul definierten Implementierung nicht hinzugefügt werden.</span><span class="sxs-lookup"><span data-stu-id="b009c-105">Additional methods or fields inherited from the base class are not allowed to be added to the implementation defined in the COM module.</span></span>  
  
 <span data-ttu-id="b009c-106">Im folgenden Beispiel wird CS0424 generiert:</span><span class="sxs-lookup"><span data-stu-id="b009c-106">The following sample generates CS0424:</span></span>

```  
// CS0424.cs

public class A {}

```
