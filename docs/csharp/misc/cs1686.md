 <span data-ttu-id="1d6cc-104">Dieser Fehler wird generiert, wenn Sie eine Variable verwenden und versuchen, deren Adresse zu übernehmen und eine dieser Aktionen in einer anonymen Methode erfolgt.</span><span class="sxs-lookup"><span data-stu-id="1d6cc-104">This error is generated when you use a variable, and attempt to take its address, and one of these actions is done inside an anonymous method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="1d6cc-105">Beispiel</span><span class="sxs-lookup"><span data-stu-id="1d6cc-105">Example</span></span>  
 <span data-ttu-id="1d6cc-106">Im folgenden Beispiel wird CS1686 generiert:</span><span class="sxs-lookup"><span data-stu-id="1d6cc-106">The following sample generates CS1686.</span></span>  
  
```csharp  
// CS1686.cs  
// compile with: /unsafe /target:library  
class MyClass  
{  
   public unsafe delegate int * MyDelegate();  
  
   public unsafe int * Test()  
   {  
      int j = 0;  
      MyDelegate d = delegate { return &j; };   // CS1686  
      return &j;   // OK  
   }  
}  
```
