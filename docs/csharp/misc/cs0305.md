## <a name="example"></a><span data-ttu-id="a1314-106">Beispiel</span><span class="sxs-lookup"><span data-stu-id="a1314-106">Example</span></span>

```csharp  
// CS0305.cs  
public class MyList<T> {}

class MyClass  
{  
   public static void Main()

      MyList<MyClass, MyClass> list1 = new MyList<MyClass>();   // CS0305  
      MyList<MyClass> list2 = new MyList<MyClass>();   // OK  
   }  
}  
```
