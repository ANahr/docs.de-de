## <a name="example"></a><span data-ttu-id="6d1a0-106">Beispiel</span><span class="sxs-lookup"><span data-stu-id="6d1a0-106">Example</span></span>

```csharp

public class MyClass<T> {}  
  
class MyClass  
{  
   public static void Main()

      MyList<MyClass, MyClass> list1 = new MyList<MyClass>();   // CS0305  
      MyList<MyClass> list2 = new MyList<MyClass>();   // OK  
   }  
}  
```
