# <a name="compiler-error-cs1583"></a>Compilerfehler CS1583
'Datei' ist keine gültige Win32-Ressourcendatei.  
  
 Dieser Fehler tritt bei der Angabe von Dateinamen mit der [/win32res](../../csharp/language-reference/compiler-options/win32res-compiler-option.md) -Compileroption, die nicht gültig oder ordnungsgemäß formatierte Win32-Ressourcendatei. Der Dateiname wird in Visual Studio im Anwendungsbereich des Projekt-Designers angegeben.
