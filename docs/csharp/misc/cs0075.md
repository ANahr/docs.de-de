 <span data-ttu-id="8d02f-107">*Aus der Regel folgt, dass, wenn x- und y Bezeichner sind, (x) y, ((x)(y) und ((x)(-y) Typumwandlungsausdrücke, aber (X) sind-y ist nicht der Fall, auch wenn x einen Typ bezeichnet. Wenn x ein Schlüsselwort, die einen vordefinierten Typ (z. B. However, if x is a keyword that identifies a predefined type (such as int), then all four forms are cast-expressions (because such a keyword could not possibly be an expression by itself).*</span></span>  
  
 <span data-ttu-id="8d02f-108">Mit dem folgenden Code wird CS0075 generiert:</span><span class="sxs-lookup"><span data-stu-id="8d02f-108">The following code generates CS0075:</span></span>  
  
```  
// CS0075  
namespace MyNamespace  
{  
    enum MyEnum { }  
    public class MyClass  
    {  
        public static void Main()  
        {  
            // To fix the error, place the negative  
            // values below in parentheses  
            int i = (System.Int32) - 4; //CS0075  
            MyEnum e = (MyEnum) - 1;    //CS0075  
            System.Console.WriteLine(i); //to avoid warning  
            System.Console.WriteLine(e); //to avoid warning  
        }  
    }  
}  
```
