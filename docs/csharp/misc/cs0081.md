# <a name="compiler-error-cs0081"></a><span data-ttu-id="3f3a7-102">Compilerfehler CS0081</span><span class="sxs-lookup"><span data-stu-id="3f3a7-102">Compiler Error CS0081</span></span>
<span data-ttu-id="3f3a7-103">Eine Typparameterdeklaration muss ein Bezeichner sein, kein Typ.</span><span class="sxs-lookup"><span data-stu-id="3f3a7-103">Type parameter declaration must be an identifier not a type</span></span>  
  
 <span data-ttu-id="3f3a7-104">Wenn Sie eine generische Methode oder einen generischen Typ deklarieren, geben Sie den Typparameter als Bezeichner an, z. B. „T“ oder „inputType“.</span><span class="sxs-lookup"><span data-stu-id="3f3a7-104">When you declare a generic method or type, specify the type parameter as an identifier, for example "T" or "inputType".</span></span> <span data-ttu-id="3f3a7-105">Wenn die Methode von Clientcode aufgerufen wird, liefert sie den Typ, der jedes Vorkommen des Bezeichners im Text der Methode oder Klasse ersetzt.</span><span class="sxs-lookup"><span data-stu-id="3f3a7-105">When client code calls the method, it supplies the type, which replaces each occurrence of the identifier in the method or class body.</span></span> <span data-ttu-id="3f3a7-106">Weitere Informationen finden Sie unter [generischen Typparameter](../../csharp/programming-guide/generics/generic-type-parameters.md).</span><span class="sxs-lookup"><span data-stu-id="3f3a7-106">For more information, see [Generic Type Parameters](../../csharp/programming-guide/generics/generic-type-parameters.md).</span></span>  
  
```csharp  
// CS0081.cs  
class MyClass  
{  
   public void F<int>() {}   // CS0081  
   public void F<T>(T input) {}   // OK  
  
   public static void Main()  
   {  
      MyClass a = new MyClass();  
      a.F<int>(2);  
      a.F<double>(.05);  
   }  
}  
```
