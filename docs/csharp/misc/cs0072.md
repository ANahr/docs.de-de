# <a name="compiler-error-cs0072"></a><span data-ttu-id="df5ba-102">Compilerfehler CS0072</span><span class="sxs-lookup"><span data-stu-id="df5ba-102">Compiler Error CS0072</span></span>
<span data-ttu-id="df5ba-103">'ereignis': Überschreiben nicht möglich; 'methode' ist kein Ereignis</span><span class="sxs-lookup"><span data-stu-id="df5ba-103">'event' : cannot override; 'method' is not an event</span></span>  
  
 <span data-ttu-id="df5ba-104">Ein [Ereignis](../../csharp/language-reference/keywords/event.md) kann nur ein anderes Ereignis überschreiben.</span><span class="sxs-lookup"><span data-stu-id="df5ba-104">An [event](../../csharp/language-reference/keywords/event.md) can only override another event.</span></span> <span data-ttu-id="df5ba-105">Weitere Informationen finden Sie unter [Ereignisse](../../csharp/programming-guide/events/index.md).</span><span class="sxs-lookup"><span data-stu-id="df5ba-105">For more information, see [Events](../../csharp/programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="df5ba-106">Im folgenden Beispiel wird CS0072 generiert:</span><span class="sxs-lookup"><span data-stu-id="df5ba-106">The following sample generates CS0072:</span></span>  
  
```  
// CS0072.cs  
delegate void MyDelegate();  
  
class Test1  
{  
   public virtual event MyDelegate MyEvent;  
   public virtual void VMeth()  
   {  
   }  
  
   public void FireAway()  
   {  
      if (MyEvent != null)  
         MyEvent();  
   }  
}  
  
class Test2 : Test1  
{  
   public override event MyDelegate VMeth   // CS0072  
   // uncomment the following lines to resolve  
   // public override event MyDelegate MyEvent  
   {  
      add  
      {  
         VMeth += value;  
         // MyEvent += value;  
      }  
      remove  
      {  
         VMeth -= value;  
         // MyEvent -= value;  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
