# <a name="serializing-object-graphs-that-contain-xelement-objects-c"></a>Serialisieren von Objektdiagrammen, die XElement-Objekte enthalten (C#)
Dieses Thema bietet eine Einführung in das Serialisieren von Objektdiagrammen, die Verweise auf Objekte des Typs <xref:System.Xml.Linq.XElement> enthalten. <xref:System.Xml.Linq.XElement> implementiert die <xref:System.Xml.Serialization.IXmlSerializable>-Schnittstelle, um diese Art der Serialisierung zu ermöglichen.  
  
 Beachten Sie, dass nur die Klasse <xref:System.Xml.Linq.XElement> die Serialisierung implementiert.
