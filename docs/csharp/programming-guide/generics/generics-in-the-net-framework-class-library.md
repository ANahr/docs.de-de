# <a name="generics-in-the-net-framework-class-library-c-programming-guide"></a>Generika in der .NET Framework-Klassenbibliothek (C#-Programmierhandbuch)
Version 2.0 der .NET Framework-Klassenbibliothek bietet einen neuen Namespace, <xref:System.Collections.Generic>, der mehrere gebrauchsfertige generische Auflistungsklassen und zugehörige Schnittstellen enthält. Andere Namespaces wie <xref:System> bieten auch neue generische Schnittstellen wie <xref:System.IComparable%601>. Diese Klassen und Schnittstellen sind effizienter und typsicherer als die nicht generischen Auflistungsklassen in früheren Versionen von .NET Framework. Berücksichtigen Sie vor dem Entwerfen und Implementieren eigener benutzerdefinierter Auflistungsklassen, ob Sie eine Klasse aus der .NET Framework-Klassenbibliothek verwenden oder davon eine Klasse ableiten können.
