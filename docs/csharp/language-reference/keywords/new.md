# <a name="new-c-reference"></a>new (C#-Referenz)
In C# kann das `new`-Schlüsselwort als Operator, Modifizierer oder als Einschränkung verwendet werden.  
  
 [new-Operator](../../../csharp/language-reference/keywords/new-operator.md)  
 Wird zum Erstellen von Objekten und zum Aufrufen von Konstruktoren verwendet.  
  
 [new-Modifizierer](../../../csharp/language-reference/keywords/new-modifier.md)  
 Wird zum Ausblenden eines von einem Basisklassenmember geerbten Members verwendet  
  
 [new-Einschränkung](../../../csharp/language-reference/keywords/new-constraint.md)  
 Wird zum Einschränken von Typen verwendet, die als Argumente für einen Typparameter in einer generischen Deklaration verwendet werden können
