# ascending (C#-Referenz)
Das `ascending`\-Kontextschlüsselwort wird in der [orderby\-Klausel](../../../csharp/language-reference/keywords/orderby-clause.md) in Abfrageausdrücken verwendet, um die Sortierreihenfolge vom kleinsten Element zum größten Element anzugeben.  Da `ascending` die Standardsortierreihenfolge ist, müssen Sie es nicht angeben.
