## <a name="remarks"></a><span data-ttu-id="629cc-132">Hinweise</span><span class="sxs-lookup"><span data-stu-id="629cc-132">Remarks</span></span>  
  
-   <span data-ttu-id="629cc-133">Die explizite numerische Konvertierung kann zu einem Genauigkeitsverlust führen oder eine Ausnahme auslösen.</span><span class="sxs-lookup"><span data-stu-id="629cc-133">The explicit numeric conversion may cause loss of precision or result in throwing exceptions.</span></span>  
  
-   <span data-ttu-id="629cc-134">Wenn Sie einen `decimal`-Wert in einen integralen Typ konvertieren, wird dieser Wert Richtung 0 (null) auf den nächsten Integralwert gerundet.</span><span class="sxs-lookup"><span data-stu-id="629cc-134">When you convert a `decimal` value to an integral type, this value is rounded towards zero to the nearest integral value.</span></span> <span data-ttu-id="629cc-135">Wenn der erzeugte Integralwert sich außerhalb des Bereichs des Zieltyps befindet, wird eine <xref:System.OverflowException> ausgelöst.</span><span class="sxs-lookup"><span data-stu-id="629cc-135">If the resulting integral value is outside the range of the destination type, an <xref:System.OverflowException> is thrown.</span></span>  
  
-   <span data-ttu-id="629cc-136">Wenn Sie einen `double`- oder `float`-Wert konvertieren, wird er Wert abgeschnitten.</span><span class="sxs-lookup"><span data-stu-id="629cc-136">When you convert from a `double` or `float` value to an integral type, the value is truncated.</span></span> <span data-ttu-id="629cc-137">Wenn der erzeugte Integralwert sich außerhalb des Bereichs des Zielwerts befindet, hängt das Ergebnis vom Kontext der Überlaufprüfung ab.</span><span class="sxs-lookup"><span data-stu-id="629cc-137">If the resulting integral value is outside the range of the destination value, the result depends on the overflow checking context.</span></span> <span data-ttu-id="629cc-138">In einem geprüften Kontext wird eine `OverflowException` ausgelöst, während das Ergebnis in einem ungeprüften Kontext ein nicht angegebener Wert des Zieltyps ist.</span><span class="sxs-lookup"><span data-stu-id="629cc-138">In a checked context, an `OverflowException` is thrown, while in an unchecked context, the result is an unspecified value of the destination type.</span></span>  
  
-   <span data-ttu-id="629cc-139">Wenn Sie `double` in `float` konvertieren, wird der `double`-Wert auf den nächsten `float`-Wert gerundet.</span><span class="sxs-lookup"><span data-stu-id="629cc-139">When you convert `double` to `float`, the `double` value is rounded to the nearest `float` value.</span></span> <span data-ttu-id="629cc-140">Wenn der `double`-Wert zu klein oder zu groß ist, um in den Zieltyp zu passen, ist das Ergebnis 0 (null) oder unendlich.</span><span class="sxs-lookup"><span data-stu-id="629cc-140">If the `double` value is too small or too large to fit into the destination type, the result will be zero or infinity.</span></span>  
  
-   <span data-ttu-id="629cc-141">Wenn Sie `float` oder `double` in `decimal` konvertieren, wird der Quellwert in eine `decimal`-Darstellung konvertiert und bei Bedarf auf die nächste Zahl nach der achtundzwanzigsten Dezimalstelle gerundet.</span><span class="sxs-lookup"><span data-stu-id="629cc-141">When you convert `float` or `double` to `decimal`, the source value is converted to `decimal` representation and rounded to the nearest number after the 28th decimal place if required.</span></span> <span data-ttu-id="629cc-142">Je nach Wert des Quellwerts kann eines der folgenden Ergebnisse auftreten:</span><span class="sxs-lookup"><span data-stu-id="629cc-142">Depending on the value of the source value, one of the following results may occur:</span></span>  
  
    -   <span data-ttu-id="629cc-143">Wenn der Quellwert zu klein ist, als dass er als `decimal` dargestellt werden könnte, ist das Ergebnis 0 (null).</span><span class="sxs-lookup"><span data-stu-id="629cc-143">If the source value is too small to be represented as a `decimal`, the result becomes zero.</span></span>  
  
    -   <span data-ttu-id="629cc-144">Wenn der Quellwert NaN (nicht numerisch), unendlich oder zu groß ist, um als `decimal` dargestellt zu werden, wird eine `OverflowException` ausgelöst.</span><span class="sxs-lookup"><span data-stu-id="629cc-144">If the source value is NaN (not a number), infinity, or too large to be represented as a `decimal`, an `OverflowException` is thrown.</span></span>  
  
-   <span data-ttu-id="629cc-145">Wenn Sie `decimal` in `float` oder `double` konvertieren, wird der `decimal`-Wert auf den nächsten `double`- oder `float`-Wert konvertiert.</span><span class="sxs-lookup"><span data-stu-id="629cc-145">When you convert `decimal` to `float` or `double`, the `decimal` value is rounded to the nearest `double` or `float` value.</span></span>  
  
 <span data-ttu-id="629cc-146">Weitere Informationen zu expliziten Konvertierungen finden Sie unter „Explicit“ in der C#-Sprachspezifikation.</span><span class="sxs-lookup"><span data-stu-id="629cc-146">For more information on explicit conversion, see Explicit in the C# Language Specification.</span></span> <span data-ttu-id="629cc-147">Weitere Informationen zum Zugriff auf die Spezifikation finden Sie in der [C#-Sprachspezifikation](../../../csharp/language-reference/language-specification/index.md).</span><span class="sxs-lookup"><span data-stu-id="629cc-147">For more information on how to access the spec, see [C# Language Specification](../../../csharp/language-reference/language-specification/index.md).</span></span>
