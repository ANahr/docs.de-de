# <a name="default-c-reference"></a>default (C#-Referenz)
Das `default`-Schlüsselwort.  
  
 Das `default`-Schlüsselwort kann in der `switch`-Anweisung oder in generischem Code verwendet werden:  
  
-   [Die switch-Anweisung](../../../csharp/language-reference/keywords/switch.md): Gibt die Standardbezeichnung an.  
  
-   [Generischer Code](../../../csharp/programming-guide/generics/default-keyword-in-generic-code.md): Gibt den Standardwert des Typparameters an. Dieser ist NULL für Verweistypen und 0 (null) für Werttypen.
