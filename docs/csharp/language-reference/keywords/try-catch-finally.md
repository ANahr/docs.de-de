# <a name="try-catch-finally-c-reference"></a>try-catch-finally (C#-Referenz)
Häufige Verwendungen von `catch` und `finally` sind das Abrufen und Verwenden von Ressourcen in einem `try`-Block, das Vorgehen bei außergewöhnlichen Umständen in einem `catch`-Block und das Freisetzen von Ressourcen im `finally`-Block.  
  
 Weitere Informationen und Beispiele zum erneuten Auslösen von Ausnahmen finden Sie unter [try-catch](../../../csharp/language-reference/keywords/try-catch.md) und [Auslösen von Ausnahmen](../../../standard/exceptions/index.md). Weitere Information über den `finally`-Block finden unter [try-finally](../../../csharp/language-reference/keywords/try-finally.md).
