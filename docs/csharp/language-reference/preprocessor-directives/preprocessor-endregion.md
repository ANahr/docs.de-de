# #endregion (C#-Referenz)
Mit `#endregion` wird das Ende eines [\#region](../../../csharp/language-reference/preprocessor-directives/preprocessor-region.md)\-Blocks gekennzeichnet.  Beispiele:  
  
```  
  
      #region MyClass definition  
class MyClass   
{  
    static void Main()   
    {  
    }  
}  
#endregion  
```
