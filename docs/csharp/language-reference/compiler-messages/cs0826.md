 <span data-ttu-id="0c566-104">Gemäß den vom Compiler verwendeten Rückschlussregeln müssen Arrayelemente alle denselben Typ aufweisen bzw. implizit in diesen Typ konvertierbar sein.</span><span class="sxs-lookup"><span data-stu-id="0c566-104">Array elements must all be the same type or implicitly convertible to the same type according to the type inference rules used by the compiler.</span></span> <span data-ttu-id="0c566-105">Der optimale Typ muss einer der im Arrayausdruck vorhandenen Typen sein.</span><span class="sxs-lookup"><span data-stu-id="0c566-105">The best type must be one of the types present in the array expression.</span></span> <span data-ttu-id="0c566-106">Elemente werden nicht in einen neuen Typ wie `object` konvertiert.</span><span class="sxs-lookup"><span data-stu-id="0c566-106">Elements will not be converted to a new type such as `object`.</span></span> <span data-ttu-id="0c566-107">Bei implizit typisierten Arrays muss der Compiler den Arraytyp anhand des Typs der ihm zugewiesenen Elemente ableiten.</span><span class="sxs-lookup"><span data-stu-id="0c566-107">For an implicitly typed array, the compiler must infer the array type based on the type of elements assigned to it.</span></span>

## <a name="example"></a><span data-ttu-id="0c566-112">Beispiel</span><span class="sxs-lookup"><span data-stu-id="0c566-112">Example</span></span>  
 <span data-ttu-id="0c566-113">Der folgende Code führt zu Fehler CS0826, da die Arrayelemente unterschiedliche Typen aufweisen und die Rückschlusslogik des Compilers keinen einzelnen optimalen Typ finden kann:</span><span class="sxs-lookup"><span data-stu-id="0c566-113">The following code generates CS0826 because the array elements are not all the same type, and the compiler's type inference logic does not find a single best type:</span></span>  
  
```csharp  
// cs0826.cs  
public class C  
{  
    public static int Main()  
    {  
        var x = new[] { 1, "str" }; // CS0826  
  
        char c = 'c';  
        short s1 = 0;  
        short s2 = -0;  
        short s3 = 1;  
        short s4 = -1;  
  
        var array1 = new[] { s1, s2, s3, s4, c, '1' }; // CS0826  
        return 1;  
    }  
}  
```
