# Compiler Warning (level 2) CS0618
"Member" ist veraltet: "Text"  
  
 Ein Klassenmember wurde mit dem `Obsolete`\-Attribut markiert, d.Â h. bei einem Verweis auf den Klassenmember wird eine Warnung ausgegeben.  Weitere Informationen finden Sie unter [Allgemeine Attribute](../Topic/Common%20Attributes%20\(C%23%20and%20Visual%20Basic\).md).  
  
 Im folgenden Beispiel wird CS0618 generiert:  
  
```  
// CS0618.cs  
// compile with: /W:2  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", false)]   // warn if referenced  
   public static void m2()  
   {  
   }  
  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m2();  // CS0618  
   }  
}  
```
