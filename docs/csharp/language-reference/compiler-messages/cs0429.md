# Compiler Warning (level 4) CS0429
Unerreichbarer Ausdruckscode wurde entdeckt.  
  
 Dieser Fehler tritt immer dann auf, wenn ein Teil eines Ausdrucks im Code nicht erreichbar ist.  Im folgenden Beispiel erf√ºllt die Bedingung `false && myTest()` dieses Kriterium, weil die `myTest()`\-Methode aufgrund der Tatsache niemals ausgewertet wird, dass die linke Seite der `&&`\-Operation immer den Wert false ergibt.  Sobald der Operator `&&` die Anweisung `false` als false auswertet, bricht er die Auswertung ab, ohne jemals die rechte Seite auszuwerten.  
  
## Beispiel  
 Der Fehler CS0429 wird durch folgenden Code generiert.  
  
```  
// CS0429.cs  
public class cs0429   
{  
    public static void Main()   
    {  
        if (false && myTest())  // CS0429  
        // Try the following line instead:  
        // if (true && myTest())  
        {  
        }  
        else  
        {  
            int i = 0;  
            i++;  
        }  
    }  
  
    static bool myTest() { return true; }  
}  
```
