## <a name="example"></a><span data-ttu-id="2bbd1-108">Beispiel</span><span class="sxs-lookup"><span data-stu-id="2bbd1-108">Example</span></span>  
 <span data-ttu-id="2bbd1-109">Im folgenden Beispiel wird der Fehler CS0592 generiert:</span><span class="sxs-lookup"><span data-stu-id="2bbd1-109">The following sample generates CS0592:</span></span>  
  
```csharp  
// CS0592.cs  
using System;  
  
[AttributeUsage(AttributeTargets.Interface)]  
public class MyAttribute : Attribute   
{  
}  
  
[MyAttribute]  
// Generates CS0592 because MyAttribute is not valid for a class.

{  
    public static void Main()  
    {  
    }  
}  
```
