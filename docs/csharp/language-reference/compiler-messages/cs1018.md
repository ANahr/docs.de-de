## <a name="example"></a><span data-ttu-id="e372b-105">Beispiel</span><span class="sxs-lookup"><span data-stu-id="e372b-105">Example</span></span>  
 <span data-ttu-id="e372b-106">Im folgenden Beispiel wird CS1018 generiert. Es werden verschiedene Möglichkeiten zur Behebung des Fehlers vorgeschlagen:</span><span class="sxs-lookup"><span data-stu-id="e372b-106">The following example generates CS1018, and suggests several ways to resolve the error:</span></span>  
  
```csharp  
// CS1018.cs  
public class C  
{  
}  
  
public class a : C  
{  
    public a(int i)  
    {  
    }  
  
    public a () :   // CS1018  
    // possible resolutions:  
    // public a () resolves by removing the colon  
    // public a () : base() calls C's default constructor  
    // public a () : this(1) calls the assignment constructor of class a  
    {  
    }  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
