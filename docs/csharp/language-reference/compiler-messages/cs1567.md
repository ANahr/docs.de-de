<span data-ttu-id="e5c2c-103">Fehler beim Generieren der Win32-Ressource: „Datei“.</span><span class="sxs-lookup"><span data-stu-id="e5c2c-103">Error generating Win32 resource: 'file'</span></span>  
  
 <span data-ttu-id="e5c2c-104">Die Kompilierung hat entweder die Compileroption [/win32icon](../../../csharp/language-reference/compiler-options/win32icon-compiler-option.md) verwendet, oder sie hat [/win32res](../../../csharp/language-reference/compiler-options/win32res-compiler-option.md) nicht verwendet.  <span data-ttu-id="e5c2c-105">Wenn Sie das Dateierstellungsproblem nicht beheben können, haben Sie die Möglichkeit, [/win32res](../../../csharp/language-reference/compiler-options/win32res-compiler-option.md) zu verwenden, wobei keine Datei mit Ressourceninformationen generiert wird.</span><span class="sxs-lookup"><span data-stu-id="e5c2c-105">If you are unable to resolve the file-generation problem, you could use [/win32res](../../../csharp/language-reference/compiler-options/win32res-compiler-option.md), which does not generate a file that contains resource information.</span></span>
