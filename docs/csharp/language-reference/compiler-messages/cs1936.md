## <a name="example"></a><span data-ttu-id="6b6ce-110">Beispiel</span><span class="sxs-lookup"><span data-stu-id="6b6ce-110">Example</span></span>  
 <span data-ttu-id="6b6ce-111">Im folgenden Beispiel wird CS1936 erzeugt:</span><span class="sxs-lookup"><span data-stu-id="6b6ce-111">The following example produces CS1936:</span></span>  
  
```csharp

class Test  
{  
    static int Main()  
    {  
        object obj;  
        IEnumerable e = from x in obj // CS1936  
                        select x;  
        return 0;  
    }  
}
```  
