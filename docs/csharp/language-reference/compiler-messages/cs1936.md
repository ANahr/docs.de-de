## <a name="example"></a><span data-ttu-id="b8a99-110">Beispiel</span><span class="sxs-lookup"><span data-stu-id="b8a99-110">Example</span></span>  
 <span data-ttu-id="b8a99-111">Im folgenden Beispiel wird CS1936 erzeugt:</span><span class="sxs-lookup"><span data-stu-id="b8a99-111">The following example produces CS1936:</span></span>  
  
```csharp  
// cs1936.cs  
using System.Collections;  
using System.Linq;  
class Test  
{  
    static int Main()  
    {  
        object obj;  
        IEnumerable e = from x in obj // CS1936  
                        select x;  
        return 0;  
    }  
}  
```  
  
 <span data-ttu-id="b8a99-112">Dieser Fehler tritt normalerweise auf, wenn Sie versehentlich versuchen, ein Objekt eines Typs abzufragen, statt einer Sammlung dieser Objekte.</span><span class="sxs-lookup"><span data-stu-id="b8a99-112">This error typically occurs when you accidentally try to query an object of some type instead of a collection of those objects.</span></span>
