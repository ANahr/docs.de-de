# <a name="compiler-error-cs0507"></a><span data-ttu-id="dbe5c-102">Compilerfehler CS0507</span><span class="sxs-lookup"><span data-stu-id="dbe5c-102">Compiler Error CS0507</span></span>
<span data-ttu-id="dbe5c-103">„Funktion1“: Die Zugriffsmodifizierer können beim Überschreiben des geerbten „Zugriff“-Members „Funktion2“ nicht geändert werden.</span><span class="sxs-lookup"><span data-stu-id="dbe5c-103">'function1' : cannot change access modifiers when overriding 'access' inherited member 'function2'</span></span>  
  
 <span data-ttu-id="dbe5c-104">Es wurde der Versuch unternommen, die Zugriffsspezifikation in einer Methodenüberschreibung zu ändern.</span><span class="sxs-lookup"><span data-stu-id="dbe5c-104">An attempt was made to change the access specification in a method override.</span></span>  
  
## <a name="example"></a><span data-ttu-id="dbe5c-105">Beispiel</span><span class="sxs-lookup"><span data-stu-id="dbe5c-105">Example</span></span>  
 <span data-ttu-id="dbe5c-106">Im folgenden Beispiel wird CS0507 generiert.</span><span class="sxs-lookup"><span data-stu-id="dbe5c-106">The following sample generates CS0507.</span></span>  
  
```csharp  
// CS0507.cs  
abstract public class clx  
{  
   virtual protected void f() {}  
}  
  
public class cly : clx  
{  
   public override void f() {}   // CS0507  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="dbe5c-107">Beispiel</span><span class="sxs-lookup"><span data-stu-id="dbe5c-107">Example</span></span>  
 <span data-ttu-id="dbe5c-108">CS0507 kann auch auftreten, wenn eine Klasse eine Methode zu überschreiben versucht, die als `protected internal` gekennzeichnet ist und dies in den Metadaten definiert ist, auf die verwiesen wird.</span><span class="sxs-lookup"><span data-stu-id="dbe5c-108">CS0507 can also occur if a class attempts to override a method marked as `protected internal` defined in referenced metadata.</span></span> <span data-ttu-id="dbe5c-109">In diesem Fall sollte die überschreibende Methode als `protected` gekennzeichnet werden.</span><span class="sxs-lookup"><span data-stu-id="dbe5c-109">In this situation, the overriding method should be marked as `protected`.</span></span>  
  
```csharp  
// CS0507_b.cs  
// compile with: /target:library  
abstract public class clx  
{  
   virtual protected internal void f() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="dbe5c-110">Beispiel</span><span class="sxs-lookup"><span data-stu-id="dbe5c-110">Example</span></span>  
 <span data-ttu-id="dbe5c-111">Im folgenden Beispiel wird CS0507 generiert.</span><span class="sxs-lookup"><span data-stu-id="dbe5c-111">The following sample generates CS0507.</span></span>
