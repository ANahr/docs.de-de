 <span data-ttu-id="156a6-105">Dieser Fehler zeigt auf, dass zwei Referenzen über dieselbe Assemblyidentität verfügen, da die entsprechenden Assemblys keine starken Namen haben, nicht signiert wurden und der Compiler deshalb in den Metadaten nicht zwischen ihnen unterscheiden kann.</span><span class="sxs-lookup"><span data-stu-id="156a6-105">This error points out that two references have the same assembly identity because the assemblies in question lack strong names, they were not signed, and thus the compiler has no way of distinguishing between them in metadata.</span></span> <span data-ttu-id="156a6-106">Deshalb werden die Version und die Eigenschaften des Kulturassemblynamens zur Laufzeit ignoriert.</span><span class="sxs-lookup"><span data-stu-id="156a6-106">Thus, the run time ignores the version and culture assembly name properties.</span></span> <span data-ttu-id="156a6-107">Der Benutzer muss die redundante Schnittstelle entfernen, eine der Referenzen umbenennen oder einen starken Namen für sie vergeben.</span><span class="sxs-lookup"><span data-stu-id="156a6-107">The user should remove the redundant reference, rename one of the references, or provide a strong name for them.</span></span>
