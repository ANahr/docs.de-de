<span data-ttu-id="4cd2f-103">Der Wert des erwarteten integralen Typs</span><span class="sxs-lookup"><span data-stu-id="4cd2f-103">A value of an integral type expected</span></span>  
  
 <span data-ttu-id="4cd2f-104">Eine Variable wurde dort benutzt, wo ein integraler Datentyp erforderlich war.</span><span class="sxs-lookup"><span data-stu-id="4cd2f-104">A variable was used in a situation where an integral data type was required.</span></span> <span data-ttu-id="4cd2f-105">Weitere Informationen finden Sie unter [Typen](../../../csharp/programming-guide/types/index.md).</span><span class="sxs-lookup"><span data-stu-id="4cd2f-105">For more information, see [Types](../../../csharp/programming-guide/types/index.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="4cd2f-106">Beispiel</span><span class="sxs-lookup"><span data-stu-id="4cd2f-106">Example</span></span>  
 <span data-ttu-id="4cd2f-107">Dieser Fehler kann auftreten, wenn keine Konvertierung vorgenommen wurde, oder wenn die verfügbaren impliziten Konvertierungen zu einer mehrdeutigen Situation führen.</span><span class="sxs-lookup"><span data-stu-id="4cd2f-107">This error can occur when there is no conversion or if the available implicit conversions result in an ambiguous situation.</span></span> <span data-ttu-id="4cd2f-108">Im folgenden Beispiel wird CS0151 generiert:</span><span class="sxs-lookup"><span data-stu-id="4cd2f-108">The following sample generates CS0151.</span></span>  
  
```csharp  
// CS0151.cs  
public class MyClass  
{  
   public static implicit operator int (MyClass aa)  
   {  
      return 0;  
   }  
  
   public static implicit operator long (MyClass aa)  
   {  
      return 0;  
   }  
  
   public static void Main()  
   {  
      MyClass a = new MyClass();  
  
      // Compiler cannot choose between int and long  
      switch (a)   // CS0151  
      // try the following line instead  
      // switch ((int)a)  
      {  
         case 1:  
            break;  
      }  
   }  
}  
```
