# <a name="compiler-error-cs1943"></a><span data-ttu-id="283a5-102">Compilerfehler CS1943</span><span class="sxs-lookup"><span data-stu-id="283a5-102">Compiler Error CS1943</span></span>
<span data-ttu-id="283a5-103">Ein Ausdruck vom Typ „Typ“ ist in einer nachfolgenden from-Klausel in einem Abfrageausdruck mit dem Quelltyp „Typ“ unzulässig.</span><span class="sxs-lookup"><span data-stu-id="283a5-103">An expression of type 'type' is not allowed in a subsequent from clause in a query expression with source type 'type'.</span></span> <span data-ttu-id="283a5-104">Fehler beim Typrückschluss im Aufruf von „Methode“.</span><span class="sxs-lookup"><span data-stu-id="283a5-104">Type inference failed in the call to 'method'.</span></span>  
  
 <span data-ttu-id="283a5-105">Alle Bereichsvariablen müssen abfragbare Typen darstellen.</span><span class="sxs-lookup"><span data-stu-id="283a5-105">All range variables must represent queryable types.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="283a5-106">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="283a5-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="283a5-107">Stellen Sie sicher, dass der Typ ein abfragbarer Typ ist, der `IEnumerable`, `IEnumerable<T>`, eine abgeleitete Schnittstelle oder einen anderen Typ mit definiertem Abfragemuster implementiert.</span><span class="sxs-lookup"><span data-stu-id="283a5-107">Make sure that the type is a queryable type that implements `IEnumerable`, `IEnumerable<T>` or a derived interface, or any other type which has a query pattern defined for it.</span></span>  
  
2.  <span data-ttu-id="283a5-108">Wenn der Typ ein nicht generischer `IEnumerable` ist, stellen Sie für die Bereichsvariable einen expliziten Typ bereit.</span><span class="sxs-lookup"><span data-stu-id="283a5-108">If the type is a non-generic `IEnumerable`, provide an explicit type on the range variable.</span></span>  
  
## <a name="example"></a><span data-ttu-id="283a5-109">Beispiel</span><span class="sxs-lookup"><span data-stu-id="283a5-109">Example</span></span>  
 <span data-ttu-id="283a5-110">Mit dem folgenden Code wird der Fehler CS1943 generiert:</span><span class="sxs-lookup"><span data-stu-id="283a5-110">The following code generates CS1943:</span></span>  
  
```csharp  
// cs1943.cs  
using System.Linq;  
class Test  
{  
    class TestClass  
    { }  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        TestClass tc = new TestClass();  
  
        var x = from n in nums  
                from s in tc // CS1943  
                select n + s;  
    }  
}  
```
