<span data-ttu-id="c7369-103">Auf einen nicht statischen Member des äußeren Typs „typ1“ kann nicht über den geschachtelten Typ „typ2“ zugegriffen werden.</span><span class="sxs-lookup"><span data-stu-id="c7369-103">Cannot access a nonstatic member of outer type 'type1' via nested type 'type2'</span></span>  
  
 <span data-ttu-id="c7369-104">Ein Feld in einer Klasse steht nicht automatisch für eine geschachtelte Klasse zur Verfügung.</span><span class="sxs-lookup"><span data-stu-id="c7369-104">A field in a class is not automatically available to a nested class.</span></span> <span data-ttu-id="c7369-105">Um es für eine geschachtelte Klasse verfügbar zu machen, muss das Feld [statisch](../../../csharp/language-reference/keywords/static.md) sein.</span><span class="sxs-lookup"><span data-stu-id="c7369-105">To be available to a nested class, the field must be [static](../../../csharp/language-reference/keywords/static.md).</span></span> <span data-ttu-id="c7369-106">Ansonsten müssen Sie eine Instanz der äußeren Klasse erstellen.</span><span class="sxs-lookup"><span data-stu-id="c7369-106">Otherwise, you must create an instance of the outer class.</span></span> <span data-ttu-id="c7369-107">Weitere Informationen finden Sie unter [Geschachtelte Typen](../../../csharp/programming-guide/classes-and-structs/nested-types.md).</span><span class="sxs-lookup"><span data-stu-id="c7369-107">For more information, see [Nested Types](../../../csharp/programming-guide/classes-and-structs/nested-types.md).</span></span>  
  
 <span data-ttu-id="c7369-108">Im folgenden Beispiel wird CS0038 generiert:</span><span class="sxs-lookup"><span data-stu-id="c7369-108">The following sample generates CS0038:</span></span>  
  
```csharp  
// CS0038.cs  
class OuterClass  
{  
   public int count;  
   // try the following line instead  
   // public static int count;  
  
   class InnerClass  
   {  
      void func()  
      {  
         // or, create an instance  
         // OuterClass class_inst = new OuterClass();  
         // int count2 = class_inst.count;  
         int count2 = count;   // CS0038  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
