<span data-ttu-id="02d6e-103">The type parameter 'type parameter' cannot be used with the 'as' operator because it does not have a class type constraint nor a 'class' constraint (Der Typparameter „typparameter“ kann nicht mit dem „as“-Operator verwendet werden, da er weder über eine Klassentypeinschränkung noch über eine Klasseneinschränkung verfügt)</span><span class="sxs-lookup"><span data-stu-id="02d6e-103">The type parameter 'type parameter' cannot be used with the 'as' operator because it does not have a class type constraint nor a 'class' constraint</span></span>  
  
 <span data-ttu-id="02d6e-104">Dieser Fehler tritt auf, wenn ein generischer Typ den [as](../../../csharp/language-reference/keywords/as.md)-Operator verwendet, aber keine Klassentypeinschränkung aufweist.</span><span class="sxs-lookup"><span data-stu-id="02d6e-104">This error occurs if a generic type uses the [as](../../../csharp/language-reference/keywords/as.md) operator, but that generic type does not have a class type constraint.</span></span> <span data-ttu-id="02d6e-105">Der `as`-Operator ist nur für Verweistypen zulässig, weshalb der Typparameter eingeschränkt werden muss, um zu garantieren, dass er kein Werttyp ist.</span><span class="sxs-lookup"><span data-stu-id="02d6e-105">The `as` operator is only allowed with reference types, so the type parameter must be constrained to guarantee that it is not a value type.</span></span> <span data-ttu-id="02d6e-106">Um diesen Fehler zu vermeiden, verwenden Sie eine Klassentypeinschränkung oder eine Verweistypeinschränkung.</span><span class="sxs-lookup"><span data-stu-id="02d6e-106">To avoid this error, use a class type constraint or a reference type constraint.</span></span>  
  
 <span data-ttu-id="02d6e-107">Dies liegt daran, dass der `as`-Operator `null` zurückgeben kann, was kein zulässiger Wert für einen Werttyp ist. Der Typparameter muss als Werttyp behandelt werden, es sei denn, er verfügt über eine Klassentypeinschränkung oder eine Verweistypeinschränkung.</span><span class="sxs-lookup"><span data-stu-id="02d6e-107">This is because the `as` operator could return `null`, which is not a possible value for a value type, and the type parameter must be treated as a value type unless it is a class type constraint or a reference type constraint.</span></span>  
  
## <a name="example"></a><span data-ttu-id="02d6e-108">Beispiel</span><span class="sxs-lookup"><span data-stu-id="02d6e-108">Example</span></span>  
 <span data-ttu-id="02d6e-109">Im folgenden Beispiel wird CS0413 generiert:</span><span class="sxs-lookup"><span data-stu-id="02d6e-109">The following sample generates CS0413.</span></span>  
  
```csharp  
// CS0413.cs  
// compile with: /target:library  
class A {}  
class B : A {}  
  
class CMain  
{  
   A a = null;  
   public void G<T>()  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  // CS0413  
   }  
  
   // OK  
   public void H<T>() where T : A  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  
   }  
}  
```
