# <a name="compiler-error-cs0433"></a><span data-ttu-id="5f846-102">Compilerfehler CS0433</span><span class="sxs-lookup"><span data-stu-id="5f846-102">Compiler Error CS0433</span></span>
<span data-ttu-id="5f846-103">Der Typ „Typname1“ ist sowohl in „Typname2“ als auch in „Typname3“ vorhanden.</span><span class="sxs-lookup"><span data-stu-id="5f846-103">The type TypeName1 exists in both TypeName2 and TypeName3</span></span>  
  
 <span data-ttu-id="5f846-104">Zwei Assemblys, auf die in der Anwendung verwiesen wird, enthalten den gleichen Namespace und den gleichen Typ, was zu Mehrdeutigkeit führt.</span><span class="sxs-lookup"><span data-stu-id="5f846-104">Two different assemblies referenced in your application contain the same namespace and type, which produces ambiguity.</span></span>  
  
 <span data-ttu-id="5f846-105">Verwenden Sie zum Lösen dieses Problems das Aliasfeature der Compileroption [/reference (C# Compiler Options) (/reference (C#-Compileroptionen))](../../../csharp/language-reference/compiler-options/reference-compiler-option.md), oder verweisen Sie auf keine der Assemblys.</span><span class="sxs-lookup"><span data-stu-id="5f846-105">To resolve this error, use the alias feature of the [/reference (C# Compiler Options)](../../../csharp/language-reference/compiler-options/reference-compiler-option.md) compiler option or do not reference one of your assemblies.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5f846-106">Beispiel</span><span class="sxs-lookup"><span data-stu-id="5f846-106">Example</span></span>

```csharp  
   public class AggPubImpAggPubImp {}  
}  
```

## <a name="example"></a><span data-ttu-id="5f846-110">Beispiel</span><span class="sxs-lookup"><span data-stu-id="5f846-110">Example</span></span>  
 <span data-ttu-id="5f846-111">Im folgenden Beispiel wird der Fehler CS0433 generiert.</span><span class="sxs-lookup"><span data-stu-id="5f846-111">The following example generates CS0433.</span></span>  
  
```csharp  
// CS0433_3.cs  
// compile with: /reference:cs0433_1.dll /reference:cs0433_2.dll  
using TypeBindConflicts;  
public class Test   
{  
   public static void Main()   
   {  
      AggPubImpAggPubImp n6 = new AggPubImpAggPubImp();   // CS0433  
   }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="5f846-112">Beispiel</span><span class="sxs-lookup"><span data-stu-id="5f846-112">Example</span></span>

```csharp  
// compile with: /reference:cs0433_1.dll /reference:TypeBindConflicts=cs0433_2.dll

}
```
