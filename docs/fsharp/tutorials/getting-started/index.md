# <a name="getting-started-with-f"></a>Erste Schritte mit F## #

Es gibt drei Hauptmethoden zum Einstieg in F#:

Wenn Sie in Windows sind, sehen Sie sich den Artikel [Getting Started with F# in Visual Studio (Erste Schritte mit F# in Visual Studio)](getting-started-visual-studio.md) an.

Wenn Sie am Schreiben von Code in eine einfache IDE unter Windows, Linux oder MacOS interessiert sind, sehen Sie sich den Artikel [Getting Started with F# in Visual Studio Code with Ionide (Erste Schritte mit F# in Visual Studio-Code mit Ionide)](getting-started-vscode.md) an.

Wenn Sie es vorziehen, die Befehlszeile unter jedem Betriebssystem zu verwenden, sehen Sie sich den Artikel [Getting Started with F# with Command-line Tools (Erste Schritte mit F# mit Befehlszeilentools)](getting-started-command-line.md) an.
