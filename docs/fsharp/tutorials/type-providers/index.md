# <a name="type-providers"></a><span data-ttu-id="e9ef4-103">Typanbieter</span><span class="sxs-lookup"><span data-stu-id="e9ef4-103">Type Providers</span></span>

<span data-ttu-id="e9ef4-104">Ein F#-Typanbieter ist eine Komponente, die Typen, Eigenschaften und Methoden zur Verwendung im Programm bereitstellt.</span><span class="sxs-lookup"><span data-stu-id="e9ef4-104">An F# type provider is a component that provides types, properties, and methods for use in your program.</span></span> <span data-ttu-id="e9ef4-105">Typanbieter erstellen, so genannten **bereitgestellten Typen**, dem werden vom f#-Compiler generiert und basieren auf einer externen Datenquelle.</span><span class="sxs-lookup"><span data-stu-id="e9ef4-105">Type Providers generate what are known as **Provided Types**, which are generated by the F# compiler and are based on an external data source.</span></span>

<span data-ttu-id="e9ef4-106">Beispielsweise können ein f#-Typanbieter für SQL Typen, die Tabellen und Spalten in einer relationalen Datenbank generieren.</span><span class="sxs-lookup"><span data-stu-id="e9ef4-106">For example, an F# Type Provider for SQL can generate types representing tables and columns in a relational database.</span></span> <span data-ttu-id="e9ef4-107">In der Tat dies Was ist die [SQLProvider](https://fsprojects.github.io/SQLProvider/) Typanbieter unterstützt.</span><span class="sxs-lookup"><span data-stu-id="e9ef4-107">In fact, this is what the [SQLProvider](https://fsprojects.github.io/SQLProvider/) Type Provider does.</span></span>

<span data-ttu-id="e9ef4-108">Vorausgesetzt, dass die Typen von Eingabeparametern für ein Typanbieter abhängig sind.</span><span class="sxs-lookup"><span data-stu-id="e9ef4-108">Provided Types depend on input parameters to a Type Provider.</span></span> <span data-ttu-id="e9ef4-109">Derartige Eingabe kann einer Beispieldatenquelle (z. B. eine JSON-Schemadatei), eine URL, die direkt an einen externen Dienst oder eine Verbindungszeichenfolge für eine Datenquelle verweist.</span><span class="sxs-lookup"><span data-stu-id="e9ef4-109">Such input can be a sample data source (such as a JSON schema file), a URL pointing directly to an external service, or a connection string to a data source.</span></span> <span data-ttu-id="e9ef4-110">Ein Typanbieter kann außerdem sicherstellen, dass Gruppen von Typen nur bei Bedarf erweitert werden; Das heißt, werden sie erweitert, wenn die Typen tatsächlich von Ihrer Anwendung verwiesen werden.</span><span class="sxs-lookup"><span data-stu-id="e9ef4-110">A Type Provider can also ensure that groups of types are only expanded on demand; that is, they are expanded if the types are actually referenced by your program.</span></span> <span data-ttu-id="e9ef4-111">Dies ermöglicht die direkte, bedarfsabhängige und stark typisierte Integration von umfangreichen Informationsquellen, z. B. Onlinedatenanbietern.</span><span class="sxs-lookup"><span data-stu-id="e9ef4-111">This allows for the direct, on-demand integration of large-scale information spaces such as online data markets in a strongly typed way.</span></span>

## <a name="generative-and-erased-type-providers"></a><span data-ttu-id="e9ef4-112">Generierende und gelöschte Typanbietern</span><span class="sxs-lookup"><span data-stu-id="e9ef4-112">Generative and Erased Type Providers</span></span>

<span data-ttu-id="e9ef4-113">Gibt zwei Arten von Typanbietern: generierende und gelöscht.</span><span class="sxs-lookup"><span data-stu-id="e9ef4-113">Type Providers come in two forms: Generative and Erased.</span></span>

<span data-ttu-id="e9ef4-114">Generierende Typanbieter erzeugen, Typen, die als .NET-oder Schematypen in der Assembly geschrieben werden können in denen sie erstellt wurden.</span><span class="sxs-lookup"><span data-stu-id="e9ef4-114">Generative Type Providers produce types that can be written as .NET types into the assembly in which they are produced.</span></span> <span data-ttu-id="e9ef4-115">Dadurch können sie aus Code in anderen Assemblys genutzt werden.</span><span class="sxs-lookup"><span data-stu-id="e9ef4-115">This allows them to be consumed from code in other assemblies.</span></span> <span data-ttu-id="e9ef4-116">Dies bedeutet, dass die typisierte Darstellung der Datenquelle in der Regel eine sein muss, die mit .NET-Typen darstellen möglich ist.</span><span class="sxs-lookup"><span data-stu-id="e9ef4-116">This means that the typed representation of the data source must generally be one that is feasible to represent with .NET types.</span></span>

<span data-ttu-id="e9ef4-117">Löschen Typanbieter erzeugen, Typen, die nur genutzt werden können, in der Assembly oder das Projekt, das sie aus generiert werden.</span><span class="sxs-lookup"><span data-stu-id="e9ef4-117">Erasing Type Providers produce types that can only be consumed in the assembly or project they are generated from.</span></span> <span data-ttu-id="e9ef4-118">Die Typen sind kurzlebige; d. h. werden nicht in eine Assembly geschrieben und können nicht von Code in anderen Assemblys genutzt werden.</span><span class="sxs-lookup"><span data-stu-id="e9ef4-118">The types are ephemeral; that is, they are not written into an assembly and cannot be consumed by code in other assemblies.</span></span> <span data-ttu-id="e9ef4-119">Sie können enthalten *verzögert* Member, da Sie die bereitgestellten Typen aus einem potenziell unbegrenzte Informationen.</span><span class="sxs-lookup"><span data-stu-id="e9ef4-119">They can contain *delayed* members, allowing you to use provided types from a potentially infinite information space.</span></span> <span data-ttu-id="e9ef4-120">Sie eignen sich für eine kleine Teilmenge der eine große und miteinander verbundene Datenquelle verwenden.</span><span class="sxs-lookup"><span data-stu-id="e9ef4-120">They are useful for using a small subset of a large and interconnected data source.</span></span>
