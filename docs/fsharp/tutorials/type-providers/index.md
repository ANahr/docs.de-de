wesentliche Schranke, die das Einschließen einer Informationsquelle in ein Programm verhindert, ist die Anforderung, die Informationen als Typen, Eigenschaften und Methoden darzustellen, damit sie in Onlinedatenanbietern.</span><span class="sxs-lookup"><span data-stu-id="4c572-120">This allows for the direct, on-demand integration of large-scale information spaces such as online data markets in a strongly typed way.</span></span>

<span data-ttu-id="4c572-121">F# enthält mehrere integrierte Typanbieter für Datendienste, die im Internet und in Unternehmen häufig zur Anwendung kommen.</span><span class="sxs-lookup"><span data-stu-id="4c572-121">F# contains several built-in type providers for commonly used Internet and enterprise data services.</span></span> <span data-ttu-id="4c572-122">Diese Typanbieter bieten einfachen und regulären Zugriff auf relationale SQL-Datenbanken und netzwerkbasierte OData- und WSDL-Dienste, die die Verwendung von F# LINQ-Abfragen für diese Datenquellen unterstützen.</span><span class="sxs-lookup"><span data-stu-id="4c572-122">These type providers give simple and regular access to SQL relational databases and network-based OData and WSDL services and support the use of F# LINQ queries against these data sources.</span></span>

<span data-ttu-id="4c572-123">Bei Bedarf können Sie eigene benutzerdefinierte Typanbieter erstellen oder auf Typanbieter verweisen, die von anderen Entwicklern erstellt wurden.</span><span class="sxs-lookup"><span data-stu-id="4c572-123">Where necessary, you can create your own custom type providers, or reference type providers that have been created by others.</span></span> <span data-ttu-id="4c572-124">Angenommen, in einer Organisation wird ein Datendienst verwendet, der eine große und wachsende Anzahl von benannten Datasets bereitstellt, die alle ein eigenes, stabiles Datenschema verwenden.</span><span class="sxs-lookup"><span data-stu-id="4c572-124">For example, assume your organization has a data service providing a large and growing number of named data sets, each with its own stable data schema.</span></span> <span data-ttu-id="4c572-125">Sie möchten einen Typanbieter erstellen, der die Schemas liest und die neuesten verfügbaren Datasets auf eine stark typisierte Weise für den Programmierer darstellt.</span><span class="sxs-lookup"><span data-stu-id="4c572-125">You may choose to create a type provider that reads the schemas and presents the latest available data sets to the programmer in a strongly typed way.</span></span>
