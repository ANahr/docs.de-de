## <a name="increase-the-maximum-open-file-limit"></a><span data-ttu-id="073ac-128">Erhöhen des Limits für die maximale Anzahl von geöffneten Dateien</span><span class="sxs-lookup"><span data-stu-id="073ac-128">Increase the maximum open file limit</span></span>

<span data-ttu-id="073ac-129">Das Standardlimit für geöffnete Dateien unter macOS ist für einige .NET Core-Workloads möglicherweise nicht ausreichend, z.B. für das Wiederherstellen von Projekten oder das Ausführen von Komponententests.</span><span class="sxs-lookup"><span data-stu-id="073ac-129">The default open file limit on macOS may not be sufficient for some .NET Core workloads, such as restoring projects or running unit tests.</span></span>

<span data-ttu-id="073ac-130">Sie können dieses Limit erhöhen, indem Sie diese Schritte befolgen:</span><span class="sxs-lookup"><span data-stu-id="073ac-130">You can increase this limit by following these steps:</span></span>

1. <span data-ttu-id="073ac-131">Erstellen Sie mithilfe eines Text-Editors die neue Datei _/Library/LaunchDaemons/limit.maxfiles.plist_, und speichern Sie die Datei mit diesem Inhalt:</span><span class="sxs-lookup"><span data-stu-id="073ac-131">Using a text editor, create a new file _/Library/LaunchDaemons/limit.maxfiles.plist_, and save the file with this content:</span></span>

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN"
        "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
  <dict>
    <key>Label</key>
    <string>limit.maxfiles</string>
    <key>ProgramArguments</key>
    <array>
      <string>launchctl</string>
      <string>limit</string>
      <string>maxfiles</string>
      <string>2048</string>
      <string>4096</string>
    </array>
    <key>RunAtLoad</key>
    <true/>
    <key>ServiceIPC</key>
    <false/>
  </dict>
</plist>
```

2. <span data-ttu-id="073ac-132">Führen Sie in einem Terminalfenster folgenden Befehl aus:</span><span class="sxs-lookup"><span data-stu-id="073ac-132">In a terminal window, run the following command:</span></span>

```console

```

3. <span data-ttu-id="073ac-133">Starten Sie Ihren Mac neu, um diese Einstellungen zu übernehmen.</span><span class="sxs-lookup"><span data-stu-id="073ac-133">Reboot your Mac to apply these settings.</span></span>
