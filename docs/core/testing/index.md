<span data-ttu-id="9f5a0-112">Eine Suite von automatisierten Tests ist eine der besten Möglichkeiten, um sicherzustellen, dass eine Softwareanwendung das tut, was von den Autoren verlangt wurde.</span><span class="sxs-lookup"><span data-stu-id="9f5a0-112">Having a suite of automated tests is one of the best ways to ensure a software application does what its authors intended it to do.</span></span> <span data-ttu-id="9f5a0-113">Es gibt verschiedene Arten von Tests für Softwareanwendungen, einschließlich Integrationstests, Webtests, Auslastungstests und Sonstige.</span><span class="sxs-lookup"><span data-stu-id="9f5a0-113">There are different kinds of tests for software applications, including integration tests, web tests, load tests, and others.</span></span> <span data-ttu-id="9f5a0-114">Komponententests, die einzelne Softwarekomponenten oder Methoden testen, sind Tests auf der untersten Ebene.</span><span class="sxs-lookup"><span data-stu-id="9f5a0-114">Unit tests that test individual software components or methods are the lowest level tests.</span></span> <span data-ttu-id="9f5a0-115">Unittests sollen nur Code im Steuerelement des Entwicklers testen und nicht Infrastrukturprobleme wie z.B.
