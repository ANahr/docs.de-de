<span data-ttu-id="f5c7d-110">Eine Suite von automatisierten Tests ist eine der besten Möglichkeiten, um sicherzustellen, dass eine Softwareanwendung das tut, was von den Autoren verlangt wurde.</span><span class="sxs-lookup"><span data-stu-id="f5c7d-110">Having a suite of automated tests is one of the best ways to ensure a software application does what its authors intended it to do.</span></span> <span data-ttu-id="f5c7d-111">Es gibt viele verschiedene Arten von Tests für Softwareanwendungen, einschließlich Integrationstests, Webtests, Auslastungstests und viele mehr.</span><span class="sxs-lookup"><span data-stu-id="f5c7d-111">There are many different kinds of tests for software applications, including integration tests, web tests, load tests, and many others.</span></span> <span data-ttu-id="f5c7d-112">Auf der untersten Ebene befinden sich die Unittests, die einzelne Softwarekomponenten oder Methoden testen.</span><span class="sxs-lookup"><span data-stu-id="f5c7d-112">At the lowest level are unit tests, which test individual software components or methods.</span></span> <span data-ttu-id="f5c7d-113">Unittests sollen nur Code im Steuerelement des Entwicklers testen und nicht Infrastrukturprobleme wie z.B.
